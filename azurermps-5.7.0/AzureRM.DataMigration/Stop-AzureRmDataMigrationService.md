# Stop-AzureRmDataMigrationService

## Sinossi
Interrompe un'istanza del servizio di migrazione del database di Azure in uno stato in corso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### ComponentNameParameterSet (impostazione predefinita)
```
Stop-AzureRmDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
```

### ComponentObjectParameterSet
```
Stop-AzureRmDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
```

### ResourceIdParameterSet
```
Stop-AzureRmDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm]
```

## Descrizione
Il cmdlet Stop-AzureRmDataMigrationService interrompe un'istanza del servizio di migrazione del database di Azure in uno stato in corso.

## ESEMPI

### Esempio 1
```
PS C:\> Stop-AzureRmDataMigrationService -ResourceGroupName MyResourceGroup â€“ServiceName TestService

```
Nell'esempio precedente viene arrestata un'istanza del servizio di migrazione del database di Azure denominata TestService in base al nome del servizio passato come parametro di input

### Esempio 2
```
PS C:\> Stop-AzureRmDataMigrationService -InputObject $TestService

```
L'esempio precedente interrompe un'istanza del servizio di migrazione del database di Azure basato sull'oggetto PSDataMigrationService passato come parametro di input.
