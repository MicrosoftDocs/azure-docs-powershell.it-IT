# Remove-AzureRmNetworkWatcherPacketCapture

## Sinossi
Rimuove una risorsa di acquisizione di pacchetti.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### SetByResource (impostazione predefinita)
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcher <PSNetworkWatcher> -PacketCaptureName <String>
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByName
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcherName <String> -ResourceGroupName <String>
 -PacketCaptureName <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
La Remove-AzureRmNetworkWatcherPacketCapture rimuove una risorsa di acquisizione di pacchetti. Ãˆ consigliabile chiamare Stop-AzureRmNetworkWatcherPacketCapture prima di chiamare Remove-AzureRmNetworkWatcherPacketCapture. Se la sessione di acquisizione di pacchetti viene eseguita quando si chiama Remove-AzureRmNetworkWatcherPacketCapture l'acquisizione di pacchetti potrebbe non essere salvata. Se la sessione viene interrotta prima della rimozione, il file CAP contenente i dati di acquisizione non viene rimosso. 

## ESEMPI

### ---Esempio 1: rimuovere una sessione di acquisizione di pacchetti
```
Remove-AzureRmNetworkWatcherPacketCapture -NetworkWatcher $networkWatcher -PacketCaptureName "PacketCaptureTest"
```

In questo esempio rimuoviamo una sessione di acquisizione di pacchetti esistente denominata "PacketCaptureTest".
