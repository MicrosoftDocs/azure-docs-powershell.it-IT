# Modulo AzureRM. Tags
## Descrizione
Questo argomento Mostra gli argomenti della Guida per i cmdlet di Azure tag.

## Cmdlet AzureRM. Tags
### [Get-AzureRmTag](Get-AzureRmTag.md)
Ottiene i contrassegni di Azure predefiniti.

### [New-AzureRmTag](New-AzureRmTag.md)
Crea un tag Azure predefinito o aggiunge valori a un tag esistente.

### [Remove-AzureRmTag](Remove-AzureRmTag.md)
Elimina i valori o i tag di Azure predefiniti.
