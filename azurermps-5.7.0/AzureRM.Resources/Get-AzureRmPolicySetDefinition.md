# Get-AzureRmPolicySetDefinition

## Sinossi
Ottiene le definizioni dei set di criteri.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### GetBySubscription (impostazione predefinita)
```
Get-AzureRmPolicySetDefinition [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByNameAndResourceGroup
```
Get-AzureRmPolicySetDefinition -Name <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzureRmPolicySetDefinition -Id <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmPolicySetDefinition** ottiene tutte le definizioni dei set di criteri o una definizione specifica del set di criteri identificata per nome o ID.

## ESEMPI

### Esempio 1: ottenere tutte le definizioni di set di criteri
```
PS C:\>Get-AzureRmPolicySetDefinition
```

Questo comando ottiene tutte le definizioni del set di criteri.

### Esempio 2: ottenere la definizione del set di criteri per nome
```
PS C:\>Get-AzureRmPolicySetDefinition -Name "VMPolicyDefinition"
```

Questo comando ottiene la definizione del set di criteri denominata VMPolicyDefinition.
