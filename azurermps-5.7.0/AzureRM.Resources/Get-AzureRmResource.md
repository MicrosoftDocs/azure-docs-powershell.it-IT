## Descrizione
Il cmdlet **Get-AzureRmResource** ottiene le risorse Azure.

## ESEMPI

### Esempio 1: ottenere tutte le risorse di un tipo specifico
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType microsoft.web/sites
```

Questo comando consente di ottenere una risorsa del tipo Microsoft. Web/sites in ResourceGroup11.

### Esempio 2: ottenere una risorsa per nome
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceName ContosoWebsite
```

Questo comando ottiene una risorsa denominata ContosoWebsite in ResourceGroup11.

### Esempio 3: visualizzare tutto lo stato degli account di archiviazione in un gruppo di risorse
```
PS C:\>Get-AzureRmResource -ResourceGroupName ResourceGroup11 -ResourceType Microsoft.ClassicStorage/storageAccounts -ExpandProperties |
   Select * -Expand Properties |
   Sort Name |
   Format-Table Name,Status*
```
