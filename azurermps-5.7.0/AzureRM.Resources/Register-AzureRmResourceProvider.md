# Register-AzureRmResourceProvider

## Sinossi
Registra un provider di risorse.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Register-AzureRmResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Register-AzureRmResourceProvider** registra un provider di risorse Azure.

## ESEMPI

### Esempio 1: registrare un provider
```
PS C:\>Register-AzureRmResourceProvider -ProviderNamespace Microsoft.Network
```

Questo registra il provider Microsoft. Network per l'account.
