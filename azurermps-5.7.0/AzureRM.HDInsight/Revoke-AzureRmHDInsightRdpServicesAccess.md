# Revoke-AzureRmHDInsightRdpServicesAccess

## Sinossi
Disabilita l'accesso RDP a un cluster Windows.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Revoke-AzureRmHDInsightRdpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Revoke-AzureRmHDInsightRdpServicesAccess** Disabilita l'accesso RDP (Remote Desktop Protocol) a un cluster HDInsight di Azure basato su Windows.

## ESEMPI

### Esempio 1: disabilitare l'accesso RDP a un cluster specificato
```
PS C:\>Revoke-AzureRmHDInsightRdpServicesAccess -ClusterName "your-hadoop-001"
```

Questo comando revoca l'accesso RDP al cluster denominato your-Hadoop-001.
