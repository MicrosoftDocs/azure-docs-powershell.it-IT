# Grant-AzureRmHDInsightRdpServicesAccess

## Sinossi
Concede l'accesso RDP al cluster Windows.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Grant-AzureRmHDInsightRdpServicesAccess [-ClusterName] <String> [-RdpCredential] <PSCredential>
 [-RdpAccessExpiry] <DateTime> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Grant-AzureRmHDInsightRdpServicesAccess** consente il protocollo RDP (Remote Desktop Protocol) per accedere a un cluster di HDInsight Azure basato su Windows.

## ESEMPI

### Esempio 1: concedere l'accesso RDP a un cluster di Azure HDInsight
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

PS C:\> Grant-AzureRmHDInsightRdpServicesAccess `
            -ClusterName $clusterName `
            -RdpCredential $newRdpCreds `
            -RdpAccessExpiry $newRdpExpiry
```

Questo comando concede l'accesso RDP al cluster denominato your-Hadoop-001.
