# New-AzureRmRecoveryServicesVault

## Sinossi
Crea un nuovo Vault di servizi di ripristino.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmRecoveryServicesVault -Name <String> -ResourceGroupName <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRmRecoveryServicesVault** crea un nuovo Vault di servizi di ripristino.

## ESEMPI

### Esempio 1
```
PS C:\> New-AzureRmRecoveryServicesVault -Name "vaultName" -ResourceGroupName "rg" -Location "eastasia"
```

Creare un Vault del servizio di recupero nel gruppo di risorse e nella posizione specificata.
