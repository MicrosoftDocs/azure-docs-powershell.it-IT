# Remove-AzureRmStreamAnalyticsJob

## Sinossi
Rimuove un processo di analisi del flusso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmStreamAnalyticsJob** Elimina in modo asincrono un processo specifico di analisi del flusso in Azure.

## ESEMPI

### ESEMPIO 1: rimuovere un processo
```
PS C:\>Remove-AzureRmStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Questo comando rimuove il job StreamingJob.
