# Remove-AzureRmStreamAnalyticsFunction

## Sinossi
Elimina una funzione da un processo di analisi del flusso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmStreamAnalyticsFunction** Elimina una funzione in modo asincrono da un processo di analisi di Azure Stream.

## ESEMPI

### Esempio 1: rimuovere una funzione di analisi del flusso
```
PS C:\>Remove-AzureRmStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Questo comando rimuove la funzione denominata ScoreTweet dal processo denominato StreamJob22.
