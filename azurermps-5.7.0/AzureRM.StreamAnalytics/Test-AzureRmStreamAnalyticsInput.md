# Test-AzureRmStreamAnalyticsInput

## Sinossi
Verifica lo stato di connessione di un input.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Test-AzureRmStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **test-AzureRmStreamAnalyticsInput** verifica la possibilitÃ  di una connessione di analisi dello stream a un input.

## ESEMPI

### ESEMPIO 1: verificare lo stato di connessione di un flusso di input
```
PS C:\>Test-AzureRmStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Questo verifica lo stato di connessione dell'input EntryStream in StreamingJob.
