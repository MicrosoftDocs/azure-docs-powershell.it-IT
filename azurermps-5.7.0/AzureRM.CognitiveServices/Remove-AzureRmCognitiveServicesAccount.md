# Remove-AzureRmCognitiveServicesAccount

## Sinossi
Elimina un account di servizi cognitivi.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmCognitiveServicesAccount** Elimina l'account di servizi cognitivi specificato.
