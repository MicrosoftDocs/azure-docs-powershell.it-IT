# Remove-AzureRmAutomationModule

## Sinossi
Rimuove un modulo dall'automazione.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmAutomationModule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmAutomationModule** rimuove un modulo da un account di automazione in Azure Automation.

## ESEMPI

### Esempio 1: rimuovere un modulo
```
PS C:\>Remove-AzureRmAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Questo comando rimuove un modulo denominato ContosoModule dall'account di automazione denominato Contoso17.
