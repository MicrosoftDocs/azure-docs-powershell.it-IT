# Remove-AzureRmDataLakeStoreTrustedIdProvider

## Sinossi
Rimuove il provider di identità attendibile specificato nell'archivio dati specificato del lago.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmDataLakeStoreTrustedIdProvider [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmDataLakeStoreTrustedIdProvider** rimuove il provider di identità attendibile specificato nell'archivio dati specificato del lago.

## ESEMPI

### Esempio 1: rimuovere un provider di identità attendibile.
```
PS C:\> Remove-AzureRmDataLakeStoreTrustedIdProvider -AccountName "ContosoADL" -Name MyProvider
```

Rimuove il provider "fornitore" dall'account "ContosoADL"
