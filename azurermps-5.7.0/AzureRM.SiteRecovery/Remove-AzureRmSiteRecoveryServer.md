# Remove-AzureRmSiteRecoveryServer

## Sinossi
Rimuove un server di ripristino del sito dall'archivio corrente.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmSiteRecoveryServer -Server <ASRServer> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmSiteRecoveryServer** Annulla la registrazione di un server di ripristino del sito di Azure che lo rimuove dalla volta corrente.
