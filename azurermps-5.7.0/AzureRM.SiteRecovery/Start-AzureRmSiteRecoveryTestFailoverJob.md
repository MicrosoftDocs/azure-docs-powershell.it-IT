# Start-AzureRmSiteRecoveryTestFailoverJob

## Sinossi
Avvia un failover di test per un'entità di protezione del ripristino del sito.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### ByPEObject (impostazione predefinita)
```
Start-AzureRmSiteRecoveryTestFailoverJob -ProtectionEntity <ASRProtectionEntity> -Direction <String>
 [-DataEncryptionPrimaryCertFile <String>] [-DataEncryptionSecondaryCertFile <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Start-AzureRmSiteRecoveryTestFailoverJob** avvia il failover dei test di un'entità o un piano di ripristino di un sito di Azure Protection.
Puoi verificare se il processo è riuscito usando il cmdlet Get-AzureRmSiteRecoveryJob.
