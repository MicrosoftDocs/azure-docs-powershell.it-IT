# Remove-AzureRmDataLakeAnalyticsAccount

## Sinossi
Elimina un account di analisi dei dati sul lago.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmDataLakeAnalyticsAccount** Elimina definitivamente un account di analisi di Azure Data Lake.

## ESEMPI

### Esempio 1: rimuovere un account
```
PS C:\>Remove-AzureRmDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Questo comando rimuove l'account di analisi dei dati del lago specificato.
