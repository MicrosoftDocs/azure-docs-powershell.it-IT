# Stop-AzureRmDataLakeAnalyticsJob

## Sinossi
Annulla un processo.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureRmDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureRmDataLakeAnalyticsJob** Annulla un processo di analisi del Lago di dati di Azure.

## ESEMPI

### Esempio 1: annullamento di un processo
```
PS C:\>Stop-AzureRmDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Questo comando Annulla il processo con l'ID specificato.
