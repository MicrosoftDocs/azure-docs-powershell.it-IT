# Remove-AzureRmDataLakeAnalyticsCatalogSecret

## Sinossi
Elimina un segreto di analisi dei dati di Lake.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmDataLakeAnalyticsCatalogSecret [-Account] <String> [-DatabaseName] <String> [[-Name] <String>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmDataLakeAnalyticsCatalogSecret** Elimina un segreto del catalogo di analisi del Lago di dati di Azure.

## ESEMPI

### Esempio 1: rimuovere un segreto
```
PS C:\>Remove-AzureRmDataLakeAnalyticsCatalogSecret -Account "ContosoAdla" -DatabaseName "DatabaseName" -Name "SecretName"
```

Questo comando rimuove il segreto del catalogo dei dati di analisi del lago specificato.
