# Modulo AzureRM. Subscription. Preview
## Descrizione
Gli argomenti di questa sezione documentano i cmdlet di PowerShell di Azure per l'abbonamento a Azure in Azure Resource Manager (ARM) Framework. I cmdlet sono presenti nello spazio dei nomi Microsoft. Azure. Commands. Subscription.

## Cmdlet di AzureRM. Subscription
### [Get-AzureRmSubscriptionDefinition](Get-AzureRmSubscriptionDefinition.md)
Ottiene una definizione di abbonamento.

### [New-AzureRmSubscriptionDefinition](New-AzureRmSubscriptionDefinition.md)
Crea una definizione di abbonamento.
