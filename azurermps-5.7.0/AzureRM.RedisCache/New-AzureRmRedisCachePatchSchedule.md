# New-AzureRmRedisCachePatchSchedule

## Sinossi
Aggiunge una pianificazione delle patch.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRmRedisCachePatchSchedule** aggiunge una pianificazione delle patch a una cache in Azure Redis cache.

## ESEMPI

### Esempio 1: creare e aggiungere una pianificazione delle patch in una cache
```
PS C:\>New-AzureRmRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzureRmRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Questo comando aggiunge una pianificazione della patch alla cache denominata RedisCache06.
Il parametro Entries accetta come valore un comando che usa **New-AzureRmRedisCacheScheduleEntry** per creare una pianificazione.
