# Remove-AzureRmVMAEMExtension

## Sinossi
Rimuove l'estensione AEM da una macchina virtuale.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [[-OSType] <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmVMAEMExtension** rimuove l'estensione AEM (Azure Enhanced Monitoring) da una macchina virtuale.

## ESEMPI

### Esempio 1: rimuovere l'estensione AEM
```
PS C:\> Remove-AzureRmVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Questo comando rimuove l'estensione AEM per la macchina virtuale denominata Contoso-server.
