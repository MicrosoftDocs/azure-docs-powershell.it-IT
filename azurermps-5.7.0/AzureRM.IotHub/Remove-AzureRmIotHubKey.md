# Remove-AzureRmIotHubKey

## Sinossi
Rimuove una chiave IotHub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Rimuove una chiave IotHub.
Se sono presenti piÃ¹ tasti con lo stesso nome, la prima nell'elenco viene rimossa.

## ESEMPI

### Esempio 1 eliminare un IotHub
```
PS C:\> Remove-AzureRmIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner1"
```

Rimuove la chiave denominata iothubowner1 dalla IotHub denominata "myiothub"
