# Modulo AzureRM. AKS
## Descrizione
Comandi per interagire con i cluster di Kubernetes gestiti di Azure.

## Cmdlet AzureRM. AKS
### [Get-AzureRmAks](Get-AzureRmAks.md)
Elencare i cluster gestiti di Kubernetes.

### [Import-AzureRmAksCredential](Import-AzureRmAksCredential.md)
Importare e unire la configurazione di Kubectl per un cluster Kubernetes gestito.

### [New-AzureRmAks](New-AzureRmAks.md)
Creare un nuovo cluster Kubernetes gestito.

### [Remove-AzureRmAks](Remove-AzureRmAks.md)
Eliminare un cluster Kubernetes gestito.

### [Set-AzureRmAks](Set-AzureRmAks.md)
Aggiornare o creare un cluster Kubernetes gestito.

### [Start-AzureRmAksDashboard](Start-AzureRmAksDashboard.md)
Creare un tunnel SSH di Kubectl nel dashboard del cluster gestito.

### [Stop-AzureRmAksDashboard](Stop-AzureRmAksDashboard.md)
Interrompi il tunnel di Kubectl SSH creato in Start-AzureRmKubernetesDashboard.
