# New-AzureRmServiceBusRule

## Sinossi
Crea una nuova regola per una determinata sottoscrizione di argomenti. 

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmServiceBusRule [-ResourceGroupName] <String> [-Namespace] <String> [-Topic] <String>
 [-Subscription] <String> [-Name] <String> [-SqlExpression] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRmServiceBusRule** crea una nuova regola per la sottoscrizione specificata.

## ESEMPI

### Esempio 1
```
PS C:\> New-AzureRmServiceBusRule -ResourceGroup Default-ServiceBus-WestUS -Namespace SBExample1 -Topic SBTopic -Subscription SBSubscription -Name SBRule -SqlExpression "mysqlexpression='test'"
```

Il cmdlet New-AzureRmServiceBusRule crea una nuova regola per l'abbonamento specificato.
