# Get-AzureRmSqlDatabase

## Sinossi
Ottiene uno o più database.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSqlDatabase [[-DatabaseName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlDatabase** ottiene uno o più database SQL di Azure da un server di database SQL di Azure.

Questo cmdlet è supportato anche dal servizio database stretch di SQL Server in Azure.
