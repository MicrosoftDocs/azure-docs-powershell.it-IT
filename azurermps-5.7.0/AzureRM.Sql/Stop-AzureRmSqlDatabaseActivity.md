# Stop-AzureRmSqlDatabaseActivity

## Sinossi
Annulla l'operazione di aggiornamento asincrono nel database.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureRmSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureRmSqlDatabaseActivity** Annulla l'operazione di aggiornamento asincrono nel database.
