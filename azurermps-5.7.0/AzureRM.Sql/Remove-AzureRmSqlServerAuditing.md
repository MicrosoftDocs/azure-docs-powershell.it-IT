# Remove-AzureRmSqlServerAuditing

## Sinossi
Rimuove il controllo di un server SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmSqlServerAuditing [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmSqlServerAuditing** rimuove il controllo di un server SQL di Azure.
Per usare questo cmdlet, specificare i parametri *ResourceGroupName* e *ServerName* per identificare il server.
Dopo aver eseguito questo cmdlet, non viene eseguito il controllo dei database in Azure SQL Server.
Se il comando riesce e si specifica il parametro *PassThru* , il cmdlet restituisce un oggetto che descrive il criterio di controllo corrente e gli identificatori di Azure SQL Server.
Gli identificatori del server includono **ResourceGroupName** e **ServerName**.

## ESEMPI

### Esempio 1: rimuovere il controllo di un server SQL di Azure
```
PS C:\>Remove-AzureRmSqlDatabaseAuditing -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Questo comando rimuove il controllo di tutti i database presenti in Server01 nel gruppo risorse.
