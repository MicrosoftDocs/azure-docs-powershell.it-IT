# Get-AzureRmSqlServerUpgrade

## Sinossi
Ottiene lo stato di un aggiornamento del server di database SQL di Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSqlServerUpgrade -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlServerUpgrade** ottiene lo stato di un aggiornamento del server di database SQL di Azure.

## ESEMPI

### Esempio 1: ottenere lo stato di un aggiornamento
```
PS C:\>Get-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Format-List
ResourceGroupName               : resourcegroup01
ServerName                      : server01
Status                          : Queued
```

Questo comando ottiene lo stato di un aggiornamento dal server denominato Server01 nel gruppo di risorse denominato ResourceGroup01.
