# Unregister-AzResourceProvider

## Sinossi
Annulla la registrazione di un provider di risorse.

## SINTASSI

```
Unregister-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Unregister-AzResourceProvider** Annulla la registrazione di un provider di risorse Azure.

## ESEMPI

### Esempio 1: annullamento della registrazione del provider di risorse con ProviderNamespace

```powershell
PS C:\>Unregister-AzResourceProvider -ProviderNamespace "Microsoft.support"
```

Questo comando Annulla la registrazione del provider di risorse "Microsoft. support".
