# Remove-AzResourceGroup

## Sinossi
Rimuove un gruppo di risorse.

## SINTASSI

### RemoveByResourceGroupName (impostazione predefinita)
```
Remove-AzResourceGroup [-Name] <String> [-Force] [-AsJob] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupId
```
Remove-AzResourceGroup -Id <String> [-Force] [-AsJob] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzResourceGroup** rimuove un gruppo di risorse Azure e le relative risorse dall'abbonamento corrente.
Per eliminare una risorsa, ma uscire dal gruppo risorse, usare il cmdlet Remove-AzResource.

## ESEMPI

### Esempio 1: rimuovere un gruppo di risorse
```
PS C:\>Remove-AzResourceGroup -Name "ContosoRG01"
```

Questo comando rimuove il gruppo di risorse ContosoRG01 dall'abbonamento.
Il cmdlet richiede la conferma e non restituisce alcun output.

### Esempio 2: rimuovere un gruppo di risorse senza conferma
```
PS C:\>Get-AzResourceGroup -Name "ContosoRG01" | Remove-AzResourceGroup -Force
```

Questo comando usa il cmdlet Get-AzResourceGroup per ottenere il gruppo di risorse ContosoRG01 e lo passa a **Remove-AzResourceGroup** usando l'operatore pipeline. Il parametro *Force* Elimina il messaggio di conferma.

### Esempio 3: rimuovere tutti i gruppi di risorse
```
PS C:\>Get-AzResourceGroup | Remove-AzResourceGroup
```

Questo comando usa il cmdlet **Get-AzResourceGroup** per ottenere tutti i gruppi di risorse, quindi li passa a **Remove-AzResourceGroup** usando l'operatore pipeline.
