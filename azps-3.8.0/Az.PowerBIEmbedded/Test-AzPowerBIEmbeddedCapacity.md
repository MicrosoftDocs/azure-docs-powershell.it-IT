# Test-AzPowerBIEmbeddedCapacity

## Sinossi
Verifica l'esistenza di un'istanza della capacità incorporata di PowerBI.

## SINTASSI

```
Test-AzPowerBIEmbeddedCapacity [-Name] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Test-AzPowerBIEmbeddedCapacity verifica l'esistenza di un'istanza della capacità incorporata di PowerBI

## ESEMPI

### Esempio 1
```
PS C:\> Test-AzPowerBIEmbeddedCapacity -Name "testcapacity"
True
```

Questo comando testerà se esiste una capacità denominata testcapacity
