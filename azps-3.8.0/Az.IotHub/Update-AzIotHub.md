# Update-AzIotHub

## Sinossi
Aggiornare un hub di Azure molto.

## SINTASSI

```
Update-AzIotHub -ResourceGroupName <String> -Name <String> -Tag <Hashtable> [-Reset]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Puoi aggiornare le proprietÃ  dei tag di un IotHub.

## ESEMPI

### Esempio 1
```
PS C:\> Update-AzIotHub -ResourceGroupName "myresourcegroup" -Name "myiotdps" -Tag @tags

                 vices/IotHubs/myiotdps
```

Aggiungi " @tags " al contrassegno di un hub di Azure molto "myiotdps".
