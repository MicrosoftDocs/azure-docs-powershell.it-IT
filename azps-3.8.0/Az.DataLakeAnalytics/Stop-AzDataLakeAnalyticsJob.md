# Stop-AzDataLakeAnalyticsJob

## Sinossi
Annulla un processo.

## SINTASSI

```
Stop-AzDataLakeAnalyticsJob [-Account] <String> [-JobId] <Guid> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzDataLakeAnalyticsJob** Annulla un processo di analisi del Lago di dati di Azure.

## ESEMPI

### Esempio 1: annullamento di un processo
```
PS C:\>Stop-AzDataLakeAnalyticsJob -Account "ContosoAdlAccout" -JobId "a0a78d72-3fa8-4564-9b18-6becb3fda48a"
```

Questo comando Annulla il processo con l'ID specificato.
