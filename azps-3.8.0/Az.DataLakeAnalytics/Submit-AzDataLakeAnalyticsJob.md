## Descrizione
Il cmdlet **Submit-AzDataLakeAnalyticsJob** Invia un processo di analisi del Lago di dati di Azure.

## ESEMPI

### Esempio 1: inviare un processo
```
PS C:\>Submit-AzDataLakeAnalyticsJob -Account "ContosoAdlAccount" -Name "New Job" -ScriptPath $LocalScriptPath -AnalyticsUnits 32
```

Questo comando Invia un processo di analisi dei dati sul lago.

### Esempio 2: inviare un processo con i parametri di script
```
PS C:\>$parameters = [ordered]@{}
$parameters["Department"] = "Sales"
$parameters["NumRecords"] = 1000
$parameters["StartDateTime"] = (Get-Date).AddDays(-14)
Submit-AzDataLakeAnalyticsJob -Account "ContosoAdlAccount" -Name "New Job" -ScriptPath $LocalScriptPath -AnalyticsUnits 32 -ScriptParameter $parameters
```

I parametri di script U-SQL sono preceduto sopra il contenuto dello script principale, ad esempio: DECLARE @Department String = "Sales"; DECLARE @NumRecords int = 1000; DECLARE @StartDateTime DateTime = New DateTime (2017, 12, 6, 0, 0, 0, 0);
