# Update-AzServiceFabricReliability

## Sinossi
Aggiornare il livello di affidabilità del tipo di nodo principale in un cluster.

## SINTASSI

```
Update-AzServiceFabricReliability [-ResourceGroupName] <String> [-Name] <String>
 -ReliabilityLevel <ReliabilityLevel> [-AutoAddNode] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Usare **Update-AzServiceFabricReliability** per aggiornare l'affidabilità del tipo di nodo primario in un cluster.

## ESEMPI

### Esempio 1
```
PS c:> Update-AzServiceFabricReliability -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -ReliabilityLevel Silver
```

Questo comando modifica il livello di affidabilità del tipo di nodo primario in Silver.
