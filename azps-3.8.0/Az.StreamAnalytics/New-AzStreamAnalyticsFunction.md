# New-AzStreamAnalyticsFunction

## Sinossi
Crea o sostituisce una funzione in un processo di analisi del flusso.

## SINTASSI

```
New-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzStreamAnalyticsFunction** crea una funzione in un processo di analisi di Azure Stream o sostituisce una funzione esistente.
Definire la funzione in un file JSON (JavaScript Object Notation).
Puoi specificare il nome della funzione usando il parametro *Name* o nel file JSON.
Se specifichi il nome in entrambi i modi, i nomi devono corrispondere.
Per sostituire una funzione esistente, specificare il nome della funzione esistente.

## ESEMPI

### Esempio 1: creare una funzione di analisi del flusso
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob07" -File "C:\Function07.json"
```

Questo comando crea una funzione dal file Function07.jsattivata.
Il nome della funzione viene archiviato nel file con estensione JSON.

### Esempio 2: creare una funzione di analisi dello stream denominata ScoreTweet
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet"
```

Questo comando crea una funzione nel processo denominato ScoreTweet.

### Esempio 3: sostituire una funzione di analisi del flusso
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet" -Force
```

Questo comando sostituisce la definizione della funzione esistente denominata ScoreTweet con la definizione in Function22.jsattivata.
