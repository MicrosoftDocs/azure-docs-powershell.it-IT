# Set-AzNetAppFilesAccount

## Sinossi
Aggiorna un account di Azure NetApp file (ANF) con il nuovo set di dati. Utile per l'eliminazione di Active Directory associate.

## SINTASSI

```
Set-AzNetAppFilesAccount -ResourceGroupName <String> -Location <String> -Name <String>
 [-ActiveDirectories <PSNetAppFilesActiveDirectory[]>] [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzNetAppFilesAccount** modifica un account ANF.

## ESEMPI

### Esempio 1: modificare un account di ANF
```
PS C:\>Set-AzNetAppFilesAccount -ResourceGroupName "MyRG" -l "westus2" -Name "MyAnfAccount"

Output:

Location          : westus2
Id                : /subscriptions/subsId/resourceGroups/MyRG/providers/Microsoft.NetApp/netAppAccounts/MyAnfAccount
Name              : MyAnfAccount
Type              : Microsoft.NetApp/netAppAccounts
Tags              :
AccountId         : 9fa2ca6d-1e48-4439-30e3-7de056e44e5a
ActiveDirectories : {}
ProvisioningState : Succeeded
```

Questo comando esegue un aggiornamento per l'account specifico. L'assenza di Active Directory significa che verr√† rimossa dall'account.
