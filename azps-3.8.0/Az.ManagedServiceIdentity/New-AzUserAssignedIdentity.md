# New-AzUserAssignedIdentity

## Sinossi
Crea una nuova identità assegnata dall'utente o aggiorna un'identità assegnata a un utente esistente.

## SINTASSI

```
New-AzUserAssignedIdentity [-ResourceGroupName] <String> [-Name] <String> [-Location <String>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-Tag <Hashtable>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzUserAssignedIdentity** crea una nuova identità assegnata dall'utente. Se usato con un'identità già esistente, ha aggiornato l'identità.
Per aggiungere tag di Azure Resource Manager all'identità, usare il cmdlet Set-AzResource.
