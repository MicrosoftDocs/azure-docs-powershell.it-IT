# Export-AzAnalysisServicesInstanceLog

## Sinossi
Esporta un log da un'istanza del server di Analysis Services nell'ambiente attualmente connesso, come specificato nel comando Add-AzAnalysisServicesAccount

## SINTASSI

```
Export-AzAnalysisServicesInstanceLog -OutputPath <String> [-Force] [-Instance] <String> [-PassThru] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Export-AzAnalysisServicesInstance Esporta il log da un'istanza di Azure Analysis Services server a file

## ESEMPI

### Esempio 1
```
PS C:\>Export-AzAnalysisServicesInstanceLog -Instance testserver -OutputPath C:\path\to\log\testserver.log
```

Questo comando esporterà il log dal server "TestServer" nell'ambiente specificato nel comando Add-AzAnalysisServicesAccount e lo salverà in file specificato in OutputPath "C:\path\to\log\testserver.log"
