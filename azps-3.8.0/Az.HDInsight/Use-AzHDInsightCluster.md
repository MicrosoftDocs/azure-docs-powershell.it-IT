# Use-AzHDInsightCluster

## Sinossi
Seleziona un cluster da usare con il cmdlet Invoke-RmAzureHDInsightHiveJob.

## SINTASSI

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **use-AzHDInsightCluster** seleziona il cluster Azure HDInsight per il cmdlet Invoke-AzHDInsightHiveJob da usare per inviare processi hive.

## ESEMPI

### Esempio 1: selezionare un cluster per l'invio di query hive
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Questo comando consente di selezionare un cluster per un invio di query hive.
