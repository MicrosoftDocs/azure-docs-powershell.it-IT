# Set-AzAutomationVariable

## Sinossi
Modifica una variabile di automazione.

## SINTASSI

### UpdateVariableValue
```
Set-AzAutomationVariable [-Name] <String> -Encrypted <Boolean> -Value <Object> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### UpdateVariableDescription
```
Set-AzAutomationVariable [-Name] <String> -Description <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzAutomationVariable** modifica il valore o la descrizione di una variabile in Azure Automation.
Per crittografare la variabile, specificare il parametro *crittografato* .
Non è possibile modificare lo stato crittografato di una variabile dopo la creazione.
Se si specifica *crittografato* per una variabile esistente non crittografata, non riesce.

## ESEMPI

### Esempio 1: impostare il valore di una variabile
```
PS C:\>Set-AzAutomationVariable -AutomationAccountName "Contoso17" -Name "StringVariable22" -ResourceGroupName "ResourceGroup01" -Value "New Value" -Encrypted $False
```

Questo comando imposta un nuovo valore per la variabile denominata StringVariable22 nell'account di automazione di Azure denominato Contoso17.
