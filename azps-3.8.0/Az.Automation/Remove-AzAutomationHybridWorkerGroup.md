# Remove-AzAutomationHybridWorkerGroup

## Sinossi
Rimuove il gruppo di lavoratori ibridi dall'automazione.

## SINTASSI

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet Remove-AzAutomationHybridWorkerGroup rimuove un gruppo di lavoro ibrido dall'automazione.

## ESEMPI

### Esempio 1
Questo comando rimuove un lavoratore ibrido per nome.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
