# Get-AzSqlServerAdvisor

## Sinossi
Ottiene uno o più consulenti per un server SQL di Azure.

## SINTASSI

```
Get-AzSqlServerAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlServerAdvisor** ottiene uno o più consulenti di SQL Server di Azure per un server SQL di Azure.
