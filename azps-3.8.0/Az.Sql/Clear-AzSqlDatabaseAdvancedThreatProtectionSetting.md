# Clear-AzSqlDatabaseAdvancedThreatProtectionSetting

## Sinossi
Rimuove le impostazioni avanzate di protezione dalle minacce da un database.

## SINTASSI

```
Clear-AzSqlDatabaseAdvancedThreatProtectionSetting [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Clear-AzSqlDatabaseAdvancedThreatProtectionSetting** rimuove le impostazioni avanzate di protezione dalle minacce da un database SQL di AzureAzure.
Per usare questo cmdlet, specificare i parametri *ResourceGroupName* e *ServerName* per identificare il database da cui questo cmdlet rimuove le impostazioni.

## ESEMPI

### Esempio 1: rimuovere le impostazioni avanzate di protezione dalle minacce per un database
```
PS C:\>Clear-AzSqlDatabaseAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01" -DatabaseName "Database01"
```

Questo comando rimuove le impostazioni avanzate di protezione dalle minacce da un database denominato Database01 nel server denominato Server01.
