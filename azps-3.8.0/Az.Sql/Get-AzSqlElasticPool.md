# Get-AzSqlElasticPool

## Sinossi
Ottiene i pool elastici e i relativi valori di proprietà in un database SQL di Azure.

## SINTASSI

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlElasticPool** ottiene i pool elastici e i relativi valori di proprietà.
Specificare il nome di un pool elastico esistente per visualizzare i valori delle proprietà solo per il pool.
