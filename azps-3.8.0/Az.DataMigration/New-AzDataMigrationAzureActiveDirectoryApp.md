# New-AzDataMigrationAzureActiveDirectoryApp

## Sinossi
Creare una nuova istanza di DataMigration Azure ActiveDirectory dettagli dell'applicazione.

## SINTASSI

```
New-AzDataMigrationAzureActiveDirectoryApp -ApplicationId <String> -AppKey <SecureString>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Creare una nuova istanza di DataMigration Azure ActiveDirectory dettagli dell'applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $secpasswd = ConvertTo-SecureString "Your Secret Key Here" -AsPlainText -Force
C:\> New-AzDmsAadApp -ApplicationId "Your AppId/Service Principal ID here" -AppKey $secpasswd
```
ApplicationId: "il tuo ID di AppId/Service Principal qui" AppKey: System. Security. SecureString ID tenant: "ID tenant"
