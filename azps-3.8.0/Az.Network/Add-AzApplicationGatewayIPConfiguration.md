# Add-AzApplicationGatewayIPConfiguration

## Sinossi
Aggiunge una configurazione IP a un gateway dell'applicazione.

## SINTASSI

### SetByResourceId
```
Add-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-SubnetId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByResource
```
Add-AzApplicationGatewayIPConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-Subnet <PSSubnet>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzApplicationGatewayIPConfiguration** aggiunge una configurazione IP a un gateway dell'applicazione.
Le configurazioni IP contengono la subnet in cui è distribuito il gateway dell'applicazione.

## ESEMPI

### Esempio 1: aggiungere una configurazione di rete virtuale a un gateway dell'applicazione
```
PS C:\>$Vnet = Get-AzVirtualNetwork -Name "Vnet01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Subnet = Get-AzVirtualNetworkSubnetConfig -Name "Subnet01" -VirtualNetwork $Vnet 
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Add-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Appgwsubnet01" -Subnet $Subnet
```

Il primo comando consente di ottenere una rete virtuale.
Il secondo comando ottiene una subnet usando la rete virtuale creata in precedenza.
Il terzo comando ottiene il gateway dell'applicazione e lo archivia nella variabile $AppGw.
Il quarto comando aggiunge la configurazione IP al gateway dell'applicazione archiviato in $AppGw.
