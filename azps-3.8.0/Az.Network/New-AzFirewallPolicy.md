# New-AzFirewallPolicy

## Sinossi
Crea un nuovo criterio di Azure firewall

## SINTASSI

```
New-AzFirewallPolicy -Name <String> -ResourceGroupName <String> -Location <String> [-ThreatIntelMode <String>]
 [-BasePolicy <String>] [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzFirewallPolicy** crea un criterio di Azure firewall.

## ESEMPI

### 1. creare un criterio vuoto
```powershell
PS C:\> New-AzFirewallPolicy -Name fp1 -ResourceGroupName TestRg
```

Questo esempio crea un criterio di Azure firewall

### 2. creare un criterio vuoto con la modalità ThreatIntel
```powershell
PS C:\> New-AzFirewallPolicy -Name fp1 -ResourceGroupName TestRg -ThreatIntelMode "Deny"
```

Questo esempio crea un criterio di Azure Firewall con una modalità Intel per le minacce
