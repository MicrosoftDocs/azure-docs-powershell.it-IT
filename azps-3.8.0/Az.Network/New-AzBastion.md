## Descrizione
Crea una risorsa Bastion. Sarà necessario un indirizzo IP pubblico e un VirtualNetwork. In questo VirtualNetwork deve essere presente una subnet con nome AzureBastionSubnet. l'indirizzo IP pubico deve essere creato con SKU standard.
