# Set-AzExpressRoutePort

## Sinossi
Modifica un ExpressRoutePort.

## SINTASSI

```
Set-AzExpressRoutePort -ExpressRoutePort <PSExpressRoutePort> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzExpressRoutePort** Salva il ExpressRoutePort modificato in Azure.

## ESEMPI

### Esempio 1
```powershell
$erport = Get-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
$erport.Links[0].AdminState = 'Enabled'
Set-AzExpressRoutePort -ExpressRoutePort $erport
```

### Esempio 2
```powershell
$erport = Get-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
$erport.Links[0].AdminState = 'Enabled'
Set-AzExpressRoutePort -InputObject $erport
```

Modifica lo stato di amministratore di un collegamento di un ExpressRoutePort
