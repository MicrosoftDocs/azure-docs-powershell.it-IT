# New-AzP2sVpnGateway

## Sinossi

## Descrizione
Il cmdlet **New-AzP2sVpnGateway** consente di creare un nuovo P2SVpnGateway in VirtualHub per la connettività del sito da Point a client del sito a Azure VirtualWan.
