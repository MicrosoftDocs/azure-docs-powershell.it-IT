# Get-AzPrivateLinkResource

## Sinossi
Ottiene una risorsa di collegamento privato.

## SINTASSI

### ByPrivateLinkResourceId (impostazione predefinita)
```
Get-AzPrivateLinkResource -PrivateLinkResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzPrivateLinkResource** recupera tutte le risorse di collegamento appartiene PrivateLinkResource.

## ESEMPI

### Esempio 1
```
Get-AzPrivateLinkResource -PrivateLinkResourceId '/subscriptions/00000000-0000-0000-0000-000000000000/resourceGroups/TestResourceGroup/providers/Microsoft.Sql/servers/mySql'
```

Questo esempio elenca tutte le risorse del collegamento privato nbelong a SQL Server denominato mySql.
