# Remove-AzRouteFilter

## Sinossi
Rimuove un filtro di route.

## SINTASSI

```
Remove-AzRouteFilter -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzRouteFilter** rimuove un filtro di route.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzRouteFilter -Name "RouteFilter01" -ResourceGroupName "ResourceGroup01"
```

Il comando rimuove il filtro della route denominato RouteFilter01 nel gruppo di risorse denominato ResourceGroup01.
