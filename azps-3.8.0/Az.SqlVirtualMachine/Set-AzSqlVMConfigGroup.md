# Set-AzSqlVMConfigGroup

## Sinossi
Impostare le informazioni relative a un gruppo di macchine virtuali SQL in una configurazione di una macchina virtuale SQL.

## SINTASSI

```
Set-AzSqlVMConfigGroup [-SqlVM] <AzureSqlVMModel> [-SqlVMGroup] <AzureSqlVMGroupModel>
 -ClusterOperatorAccountPassword <SecureString> -SqlServiceAccountPassword <SecureString>
 [-ClusterBootstrapAccountPassword <SecureString>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet Set-AzSqlVMConfigGroup imposta le informazioni necessarie per partecipare a un gruppo di macchine virtuali SQL per una configurazione di una macchina virtuale SQL.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $config = Set-AzSqlVMConfigGroup -SqlVM $config -SqlVMGroup $group -ClusterOperatorAccountPassword 'password' -SqlServiceAccountPassword 'password'
```

Aggiornare le informazioni di gruppo di una configurazione di una macchina virtuale SQL.
