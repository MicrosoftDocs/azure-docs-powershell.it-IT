# Update-AzWebAppAccessRestrictionConfig

## Sinossi
Aggiorna l'ereditariet√† della configurazione di Access restiction del sito principale in un sito SCM per una Web App Azure.

## SINTASSI

### InputValuesParameterSet (impostazione predefinita)
```
Update-AzWebAppAccessRestrictionConfig [-ResourceGroupName] <String> [-Name] <String>
 [-ScmSiteUseMainSiteRestrictionConfig] [-SlotName <String>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Update-AzWebAppAccessRestrictionConfig [-PassThru] -InputObject <PSAccessRestrictionConfig>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Update-AzWebAppAccessRestrictionConfig** aggiorna la configurazione delle restrizioni di Access per un'app Azure Web.

## ESEMPI

### Esempio 1: aggiornare un sito SCM di Web App per usare le restrizioni di accesso dal sito principale
```
PS C:\>Set-AzWebAppAccessRestriction -ResourceGroupName "Default-Web-WestUS" -Name "ContosoSite" -ScmSiteUseMainSiteRestrictionConfig
```

Questo comando aggiorna un'app Web denominata ContosoSite che appartiene al gruppo di risorse predefinito-Web-Westus per usare la configurazione delle restrizioni di Access del sito principale nel sito SCM.
