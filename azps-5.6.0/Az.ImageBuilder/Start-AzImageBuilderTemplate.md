# Start-AzImageBuilderTemplate

## SYNOPSIS
Creare elementi da un modello di immagine esistente

## SINTASSI

### Esegui (impostazione predefinita)
```
Start-AzImageBuilderTemplate -ImageTemplateName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### RunViaIdentity
```
Start-AzImageBuilderTemplate -InputObject <IImageBuilderIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIZIONE
Creare elementi da un modello di immagine esistente

## ESEMPI

### Esempio 1: Creare un modello di immagine
```powershell
PS C:\> Start-AzImageBuilderTemplate -ResourceGroupName wyunchi-imagebuilder -Name template-name-sn78hg

```

Questo comando avvia un modello di immagine.

### Esempio 2: Creare un modello di immagine
```powershell
PS C:\> $template = Get-AzImageBuilderTemplate -ResourceGroupName wyunchi-imagebuilder -Name template-name-sn78hg
PS C:\> Start-AzImageBuilderTemplate -InputObject $template

```

Questo comando avvia un modello di immagine.
