# Modulo Az.IotCentral
## Descrizione
Fornisce la gestione DELLE APPLICAZIONI PER IoT Central Applications.

## Cmdlet Az.IotCentral
### [Get-AzIotCentralApp](Get-AzIotCentralApp.md)
Ottiene le proprietà per una o più applicazioni ioT Central.

### [New-AzIotCentralApp](New-AzIotCentralApp.md)
Crea una nuova applicazione IoT Central.

### [Remove-AzIotCentralApp](Remove-AzIotCentralApp.md)
Elimina un'applicazione ioT central.

### [Set-AzIotCentralApp](Set-AzIotCentralApp.md)
Aggiorna i metadati per un'applicazione centrale IoT.
