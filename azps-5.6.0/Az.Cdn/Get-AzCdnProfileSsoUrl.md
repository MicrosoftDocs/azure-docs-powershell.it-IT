# Get-AzCdnProfileSsoUrl

## SYNOPSIS
Ottiene l'URL Single #A0 di un profilo CDN.

## SINTASSI

### ByFieldsParameterSet (Impostazione predefinita)
```
Get-AzCdnProfileSsoUrl -ProfileName <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectParameterSet
```
Get-AzCdnProfileSsoUrl -CdnProfile <PSProfile> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzCdnProfileSsoUrl** ottiene l'URL Single #A0 del profilo della rete per la distribuzione di contenuti (CDN) di Azure.
Questo URL consente agli utenti di connettersi a un portale supplementare e usare caratteristiche aggiuntive della rete CDN.
