# New-AzMetricAlertRuleV2DimensionSelection

## SYNOPSIS
Crea un oggetto di selezione della dimensione locale che può essere usato per creare criteri di avviso metrica.

## SINTASSI

```
New-AzMetricAlertRuleV2DimensionSelection -DimensionName <String> -ValuesToInclude <String[]>
 [-ValuesToExclude <String[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzMetricAlertRuleV2DimensionSelection** crea un oggetto di selezione della dimensione locale per facilitare la costruzione dei criteri di avviso della metrica usando il cmdlet [New-AzMetricAlertRuleV2Criteria.](https://docs.microsoft.com/powershell/module/az.monitor/new-azmetricalertrulev2criteria)

## ESEMPI

### Esempio 1: Creare un oggetto di selezione della dimensione

```powershell
PS C:\> New-AzMetricAlertRuleV2DimensionSelection -DimensionName LUN -ValuesToInclude 1,3,4

Dimension IncludeValues ExcludeValues
--------- ------------- -------------
LUN       {1, 3, 4}
```

Questo comando crea un oggetto di selezione della dimensione che specifica che i valori {1,3,4} devono essere selezionati per la dimensione "LUN"
