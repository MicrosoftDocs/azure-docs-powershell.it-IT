# Remove-AzLogProfile

## SYNOPSIS
Rimuove un profilo di log.

## SINTASSI

```
Remove-AzLogProfile -Name <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzLogProfile** rimuove un profilo di log.
Questo cmdlet implementa il criterio ShouldProcess, ad esempio potrebbe richiedere conferma all'utente prima di creare, modificare o rimuovere effettivamente la risorsa.

## ESEMPI

### Esempio 1

Rimuove un profilo di log. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzLogProfile -Name 'LogAlertRule1'
```
