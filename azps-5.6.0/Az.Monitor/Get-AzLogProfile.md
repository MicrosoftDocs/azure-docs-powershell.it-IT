# Get-AzLogProfile

## SYNOPSIS
Ottiene un profilo di log.

## SINTASSI

```
Get-AzLogProfile [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzLogProfile** ottiene un profilo di log.
