# New-AzMetricFilter

## SYNOPSIS
Crea un filtro delle dimensioni metriche che può essere usato per eseguire query sulle metriche.

## SINTASSI

```
New-AzMetricFilter [-Dimension] <String> [-Operator] <String> [-Value] <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzMetricFilter crea** un filtro delle dimensioni metriche che può essere usato per eseguire query sulle metriche.

## ESEMPI

### Esempio 1: Creare un filtro delle dimensioni metriche
```
PS C:\>New-AzMetricFilter -Dimension City -Operator eq -Value "Seattle","New York"
```

Questo comando crea un filtro metrico per le dimensioni nel formato "Città eq 'Seattle' o Città eq 'Roma'".
