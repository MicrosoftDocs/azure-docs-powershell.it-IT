# Get-AzAppServiceEnvironment

## SYNOPSIS
Ottiene l'ambiente del servizio app. Se è specificato solo gruppo di risorse, verrà restituito un elenco di tipi di risorse del gruppo di risorse.

## SINTASSI

```
Get-AzAppServiceEnvironment [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzAppServiceEnvironment** restituisce i file ASE che corrispondono alla query.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzAppServiceEnvironment -ResourceGroupName MyResourceGroup -Name MyAseName
```

restituisce uno specifico ambiente del servizio app <MyAseName> denominato in <MyResourceGroup>
