# Update-AzAutomationSourceControl

## SYNOPSIS
Aggiorna un controllo origine di Automazione di Azure.

## SINTASSI

```
Update-AzAutomationSourceControl -Name <String> [-AccessToken <SecureString>] [-FolderPath <String>]
 [-Branch <String>] [-Description <String>] [-AutoSync <Boolean>] [-PublishRunbook <Boolean>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il Update-AzAutomationSourceControl cmdlet modifica il valore di un campo in un controllo origine in Automazione di Azure.

## ESEMPI

### Esempio 1
Questo comando imposta il campo PublishRunbook su false per il controllo origine di Automazione di Azure denominato VSTSNative nell'account denominato devAccount.


```powershell
Update-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                      -AutomationAccountName "devAccount" `
                                      -Name "VSTSNative" `
                                      -PublishRunbook $false 

Name            SourceType Branch FolderPath  AutoSync PublishRunbook RepoUrl
----            ---------- ------ ----------  -------- -------------- -------
VSTSNative      VsoTfvc           /MyRunbooks False    False          https://contoso.visualstudio.com/_git/Fin...
```

### Esempio 2

Aggiorna un controllo origine di Automazione di Azure. (autogenerati)

<!-- Aladdin Generated Example -->
```powershell
Update-AzAutomationSourceControl -AccessToken <SecureString> -AutomationAccountName 'devAccount' -Name 'VSTSNative' -ResourceGroupName 'rg1'
```
