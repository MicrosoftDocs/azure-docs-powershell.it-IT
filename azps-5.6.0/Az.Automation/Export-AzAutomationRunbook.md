# Export-AzAutomationRunbook

## SYNOPSIS
Esporta un runbook di automazione.

## SINTASSI

```
Export-AzAutomationRunbook [-Name] <String> [-Slot <String>] [-OutputFolder <String>] [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Export-AzAutomationRunbook** esporta un runbook di automazione di Azure in un file di script di wps_2 (ps1), per runbook di wps_2 o flusso di lavoro di wps_2 oppure in un file di runbook grafico (con estensione graphrunbook) per runbook grafici.
Il nome del runbook diventa il nome del file esportato.

## ESEMPI

### Esempio 1: Esportare un runbook
```
PS C:\>Export-AzAutomationRunbook -ResourceGroupName "ResourceGroup01" -AutomationAccountName "ContosoAutomationAccount" -Name "Runbook03" -Slot "Published" -OutputFolder "C:\Users\PattiFuller\Desktop"
```

Questo comando esporta la versione pubblicata di un runbook di automazione in un desktop utente.
