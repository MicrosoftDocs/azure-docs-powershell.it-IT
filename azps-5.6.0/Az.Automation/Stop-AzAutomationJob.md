# Stop-AzAutomationJob

## SYNOPSIS
Interrompe un processo di automazione.

## SINTASSI

```
Stop-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzAutomationJob** interrompe un processo di automazione di Azure.
Specificare un processo di automazione in esecuzione.

## ESEMPI

### Esempio 1: Interrompere un processo
```
PS C:\>Stop-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Questo comando interrompe il processo con l'ID specificato.
