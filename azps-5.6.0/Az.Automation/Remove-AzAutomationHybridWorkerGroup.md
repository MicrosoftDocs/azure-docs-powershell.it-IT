# Remove-AzAutomationHybridWorkerGroup

## SYNOPSIS
Rimuove il gruppo di utenti ibridi dall'automazione.

## SINTASSI

```
Remove-AzAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Remove-AzAutomationHybridWorkerGroup rimuove un gruppo di utenti ibridi dall'automazione.

## ESEMPI

### Esempio 1
Questo comando rimuove un utente ibrido in base al nome.

```powershell
PS C:\> Remove-AzAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
