# Set-AzAutomationDscNode

## SYNOPSIS
Modifica la configurazione del nodo a cui Ã¨ mappato un nodo DSC.

## SINTASSI

```
Set-AzAutomationDscNode -Id <Guid> -NodeConfigurationName <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzAutomationDscNode** modifica una configurazione del nodo APS Desired State Configuration (DSC).
Azure Automation archivia la configurazione del nodo DSC come documento di configurazione MOF (Managed Object Format).

## ESEMPI

### Esempio 1: Modificare il mapping di configurazione del nodo
```
PS C:\>Set-AzAutomationDscNode -NodeConfigurationName "Contoso.NodeConfiguration01" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111c8a6067j8
```

Questo comando assegna la configurazione del nodo denominata Contoso.NodeConfiguration01 al nodo con il GUID specificato.
