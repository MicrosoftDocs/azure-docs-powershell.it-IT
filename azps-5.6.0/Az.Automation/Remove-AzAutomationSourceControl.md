# Remove-AzAutomationSourceControl

## SYNOPSIS
Rimuove un controllo origine di Automazione di Azure.

## SINTASSI

```
Remove-AzAutomationSourceControl [-Name] <String> [-PassThru] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il Remove-AzAutomationSourceControl cmdlet rimuove un controllo del codice sorgente dall'automazione di Azure.

## ESEMPI

### Esempio 1
Questo comando rimuove il controllo origine automazione denominato VSTSNative nell'account denominato devAccount.
Questo comando specifica il *parametro Force.* Di conseguenza, non richiede conferma.

```powershell
PS C:\> Remove-AzAutomationSourceControl -ResourceGroupName "rg1" `
                                              -AutomationAccountName "devAccount" `
                                              -Name "VSTSNative" `
                                              -Force
```
