# Set-AzAutomationSchedule

## SYNOPSIS
Modifica una pianificazione di automazione.

## SINTASSI

```
Set-AzAutomationSchedule [-Name] <String> [-IsEnabled <Boolean>] [-Description <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzAutomationSchedule** modifica una pianificazione nell'automazione di Azure.

## ESEMPI

### Esempio 1: Modificare una pianificazione
```
PS C:\>Set-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -Description "Automation Schedule" -ResourceGroupName "ResourceGroup01"
```

Questo comando modifica la descrizione della pianificazione denominata Pianificazione01.
