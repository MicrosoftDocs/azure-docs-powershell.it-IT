# Suspend-AzAutomationJob

## SYNOPSIS
Sospende un processo di automazione.

## SINTASSI

```
Suspend-AzAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Suspend-AzAutomationJob** sospende un processo di automazione di Azure.
Specificare un processo di automazione in esecuzione.
Per riprendere un processo sospeso, usare il cmdlet Resume-AzAutomationJob lavoro sospeso.

## ESEMPI

### Esempio 1: Sospendere un processo
```
PS C:\>Suspend-AzAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Questo comando sospende il processo con l'ID specificato.
