# Add-AzIotHubEventHubConsumerGroup

## SYNOPSIS
Crea un gruppo consumer eventhub.

## SINTASSI

```
Add-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubConsumerGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Crea un gruppo di utenti consumer in Eventhub associato all'oggetto IotHub specificato.

## ESEMPI

### Esempio 1: Aggiungere un gruppo consumer all'eventhub di telemetria
```
PS C:\> Add-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubConsumerGroupName "myconsumergroup"
```

Aggiunge un nuovo gruppo consumer denominato "myconsumergroup" all'eventhub per gli eventi di telemetria nel iothub denominato "myiothub"
