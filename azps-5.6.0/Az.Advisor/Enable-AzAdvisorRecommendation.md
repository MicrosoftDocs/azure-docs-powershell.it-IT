# Enable-AzAdvisorRecommendation

## SYNOPSIS
Abilita i suggerimenti di Azure Advisor.

## SINTASSI

### NameParameterSet (Default)
```
Enable-AzAdvisorRecommendation [-RecommendationName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSet
```
Enable-AzAdvisorRecommendation [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Enable-AzAdvisorRecommendation [-InputObject] <PsAzureAdvisorResourceRecommendationBase>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Questo cmdlet abilita un suggerimento non eliminato in precedenza. Ãˆ possibile rimuovere anche tutte le eliminazioni associate a un consiglio.

Rimuove tutte le eliminazioni per il consiglio specificato con il nome "recommendation_id".
