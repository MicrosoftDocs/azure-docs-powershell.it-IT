# Get-AzAdvisorConfiguration

## SYNOPSIS
Ottenere le configurazioni di Azure Advisor per la sottoscrizione o il gruppo di risorse specificato.

## SINTASSI

```
Get-AzAdvisorConfiguration [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Le configurazioni associate a un abbonamento sono di due tipi:

Configurazione a livello di abbonamento: per una sottoscrizione può essere presente una sola configurazione di questo tipo. LowCpuThreshold ed Exclude sono l'unica proprietà di questo tipo di configurazione.

Configurazione a livello di Gruppo Risorse: può essere presente una sola configurazione per ogni ResourceGroup di una sottoscrizione. Exclude è l'unica proprietà di questo tipo di configurazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\>$data = Get-AzAdvisorConfiguration
Id         : /subscriptions/{user_subscription}/providers/Microsoft.Advisor/configurations/{user_subscription}
Name       : {user_subscription}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

Id         : /subscriptions/{user_subscription}/providers/Microsoft.Advisor/configurations/{user_subscription}-{resourceGroupName}
Name       : {user_subscription}-{resourceGroupName}
Properties : Microsoft.Azure.Commands.Advisor.Cmdlets.Models.PsAzureAdvisorConfigurationProperties
Type       : Microsoft.Advisor/Configurations

PS C:\>$data[0].Properties
AdditionalProperties :
Exclude              : False
LowCpuThreshold      : 20

PS C:\>$data[1].Properties
AdditionalProperties :
Exclude              : True
LowCpuThreshold      : null

```
Recupera un elenco di configurazioni di Azure Advisor.
