# Remove-AzContainerRegistryTag

## SYNOPSIS
Rimuovere il contrassegno tag ACR.

## SINTASSI

```
Remove-AzContainerRegistryTag -RepositoryName <String> -Name <String> -RegistryName <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Rimuovere il contrassegno tag ACR.
Per usare questo cmdlet, esegui `Update-AzContainerRegistryRepository -RegistryName XXX -Repository XXX -DeleteEnable $true`
per prima cosa.

## ESEMPI

### Esempio 1
```powershell
Remove-AzContainerRegistryTag -RegistryName registry -RepositoryName alpine -Name latest
True
```

Rimuovere il contrassegno alpine:tag nel Registro di sistema.
