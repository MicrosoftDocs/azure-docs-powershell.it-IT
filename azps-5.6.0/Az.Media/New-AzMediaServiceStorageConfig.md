# New-AzMediaServiceStorageConfig

## SYNOPSIS
Creare una configurazione dell'account di archiviazione per i cmdlet del servizio multimediale.

## SINTASSI

```
New-AzMediaServiceStorageConfig [-DefaultProfile <IAzureContextContainer>] [-StorageAccountId] <String>
 [-IsPrimary] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzMediaServiceStorageConfig crea** una configurazione dell'account di archiviazione per i cmdlet del servizio multimediale.

## ESEMPI

### Esempio 1: Creare una configurazione di account di archiviazione per i cmdlet del servizio multimediale
```
PS C:\>
$StorageAccount = New-AzStorageAccount -ResourceGroupName $ResourceGroupName -Name "Storage1" -Location "East US" -Type "Standard_GRS"

PS C:\> New-AzMediaServiceStorageConfig -StorageAccountId $StorageAccount.Id -IsPrimary
```

Il primo comando crea un oggetto account di archiviazione usando il cmdlet **New-AzStorageAccount.**
Il comando chiama questo account di archiviazione Storage1 e il tipo Ã¨ denominato Standard_GRS e archivia il risultato nella variabile $StorageAccount.
Il secondo comando crea un oggetto configurazione di archiviazione come account di archiviazione principale associato al servizio multimediale usando le informazioni sugli ID dell'account di archiviazione archiviate nella variabile $StorageAccount archiviazione.
