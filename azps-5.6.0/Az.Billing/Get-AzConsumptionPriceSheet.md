# Get-AzConsumptionPriceSheet

## SYNOPSIS
Ottieni i listini prezzi dell'abbonamento.

## SINTASSI

```
Get-AzConsumptionPriceSheet [-BillingPeriodName <String>] [-ExpandMeterDetail] [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzConsumptionPriceSheet** ottiene i listini prezzi dell'abbonamento.
