# Get-AzAksVersion

## SYNOPSIS
Elencare la versione disponibile per la creazione del cluster Kubernetes gestito.

## SINTASSI

```
Get-AzAksVersion -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Elencare la versione disponibile per la creazione del cluster Kubernetes gestito.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzAksVersion -Location westus
```
