# Stop-AzAksDashboard

## SYNOPSIS
Interrompere il tunnel Kubectl SSH creato in Start-AzKubernetesDashboard.

## SINTASSI

```
Stop-AzAksDashboard [-PassThru] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Interrompere il tunnel Kubectl SSH creato in Start-AzKubernetesDashboard.

## ESEMPI

### Esempio 1
```
PS C:\> Stop-AzKubernetesDashboard
```

Interrompe la configurazione del tunnel SSH esistente eseguendo Start-AzKubernetesDashboard.
