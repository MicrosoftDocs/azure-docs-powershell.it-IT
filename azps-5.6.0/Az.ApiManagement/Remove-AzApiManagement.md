# Remove-AzApiManagement

## SYNOPSIS
Rimuove un servizio di gestione DELLE API.

## SINTASSI

```
Remove-AzApiManagement -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApiManagement rimuove** un servizio di gestione delle API di Azure.

## ESEMPI

### Esempio 1: Rimuovere un servizio di gestione DELLE API
```
PS C:\>Remove-AzApiManagement -ResourceGroupName "ContosoGroup02" -Name "ContosoApi"
```

Questo comando rimuove il servizio di gestione DELLE API denominato ContosoApi.
