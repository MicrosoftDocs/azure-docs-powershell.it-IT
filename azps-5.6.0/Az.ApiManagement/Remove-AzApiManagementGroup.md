# Remove-AzApiManagementGroup

## SYNOPSIS
Rimuove un gruppo di gestione delle API esistente.

## SINTASSI

```
Remove-AzApiManagementGroup -Context <PsApiManagementContext> -GroupId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApiManagementGroup** rimuove un gruppo di gestione delle API esistente.

## ESEMPI

### Esempio 1: Rimuovere un gruppo di gestione esistente
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementGroup -Context $apimContext -GroupId "Group0001" -Force
```

Questo comando rimuove un gruppo di gestione esistente denominato Group0001 e non richiede conferma all'utente.
