# Remove-AzApiManagementApiRevision

## SYNOPSIS
È stata rimossa una determinata revisione API

## SINTASSI

### ByApiId (impostazione predefinita)
```
Remove-AzApiManagementApiRevision -Context <PsApiManagementContext> -ApiId <String> -ApiRevision <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzApiManagementApiRevision -InputObject <PsApiManagementApi> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApiManagementApiRevision** rimuove una particolare revisione dell'API.

## ESEMPI

### Esempio 1: Rimuovere una revisione dell'API
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzAzureRmApiManagementApiRevision -Context $apimContext -ApiId "echo-api" -ApiRevision "2"
```

Questo comando rimuove la `2` revisione dell'API `echo-api` dal servizio di gestione delle API.

### Esempio 2

Questo comando rimuove la revisione 2 dell'API echo-api dal servizio di gestione DELLE API. (autogenerati)

```powershell
<!-- Aladdin Generated Example --> 
Remove-AzApiManagementApiRevision -ApiId '0001' -ApiRevision 6 -Context <PsApiManagementContext>
```
