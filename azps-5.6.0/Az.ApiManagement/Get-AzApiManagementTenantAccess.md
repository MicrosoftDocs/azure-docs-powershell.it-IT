# Get-AzApiManagementTenantAccess

## SYNOPSIS
Ottiene la configurazione di accesso per un tenant.

## SINTASSI

```
Get-AzApiManagementTenantAccess -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzApiManagementTenantAccess** ottiene la configurazione dell'accesso tenant per un tenant.
Le chiavi non verranno incluse nei dettagli dei risultati. Per ottenere il segreto client, usare **Get-AzApiManagementTenantAccessSecret.**

## ESEMPI

### Esempio 1: Ottenere la configurazione dell'accesso tenant
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementTenantAccess -Context $apimContext
```

Questo comando recupera la configurazione dell'accesso tenant per il contesto specificato.
