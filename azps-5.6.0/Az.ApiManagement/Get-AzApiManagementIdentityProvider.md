# Get-AzApiManagementIdentityProvider

## SYNOPSIS
Ottenere i dettagli di configurazione del provider di identità.

## SINTASSI

### AllIdentityProviders (Impostazione predefinita)
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdentityProviderByType
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Ottenere i dettagli di configurazione del provider di identità.
ClientSecret non verrà incluso nei dettagli dei risultati. Per ottenere il segreto client, usare **Get-AzApiManagementIdentityProviderClientSecret.**

## ESEMPI

### Esempio 1: Ottenere tutti i provider di identità

```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementIdentityProvider -Context $apimContext
```

Ottenere tutta la configurazione del provider di identità nel servizio.

### Esempio 2: Ottenere il provider di identità del tipo AAD
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\> Get-AzApiManagementIdentityProvider -Context $apimContext -Type Aad
```

Ottiene la configurazione del provider di identità di Azure Active Directory.
