# Remove-AzServiceBusNamespace

## SYNOPSIS
Rimuove lo spazio dei nomi dal gruppo di risorse specificato. 

## SINTASSI

### NamespacePropertiesSet (impostazione predefinita)
```
Remove-AzServiceBusNamespace [-ResourceGroupName] <String> [-Name] <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### NamespaceInputObjectSet
```
Remove-AzServiceBusNamespace [-InputObject] <PSNamespaceAttributes> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### NamespaceResourceIdParameterSet
```
Remove-AzServiceBusNamespace [-ResourceId] <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzServiceBus Consente di** rimuovere lo spazio dei nomi dal gruppo di risorse specificato.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzServiceBusNamespace -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Rimuove lo spazio dei nomi del bus di `SB-Example1` servizio dal gruppo di risorse `Default-ServiceBus-WestUS` specificato.

### Esempio 2.1 - InputObject - Uso della variabile:
```
PS C:\> $inputobject = Get-AzServiceBusNamespace <params>
PS C:\> Remove-AzServiceBusNamespace -InputObject $inputobject
```

Rimuove lo spazio dei nomi dei bus di servizio fornito tramite il $inputobject.

### Esempio 2.2 - InputObject - Uso di Piping:
```
PS C:\> Get-AzServiceBusNamespace <params> | Remove-AzServiceBusNamespace
```

Rimuove lo spazio dei nomi dei bus di servizio tramite Piping.

### Esempio 3 - ResourceId
```
PS c:\> $ResourceId = (Get-AzResource -ResourceType Microsoft.ServiceBus/namespaces).ResourceId
PS C:\> Remove-AzServiceBusNamespace -ResourceId $resourceid
```

Rimuove lo spazio dei nomi del bus di servizio fornito tramite ARM ID servizio in $resourceid parametro per -ResourceId o tramite piping.
