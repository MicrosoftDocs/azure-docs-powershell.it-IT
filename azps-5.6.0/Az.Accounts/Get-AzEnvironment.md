# Get-AzEnvironment

## SYNOPSIS
Ottenere endpoint e metadati per un'istanza dei servizi di Azure.

## SINTASSI

```
Get-AzEnvironment [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Get-AzEnvironment riceve gli endpoint e i metadati per un'istanza dei servizi di Azure.

## ESEMPI

### Esempio 1: Ottenere l'ambiente AzureCloud
```
PS C:\> Get-AzEnvironment AzureCloud

Name       Resource Manager Url          ActiveDirectory Authority
----       --------------------          -------------------------
AzureCloud https://management.azure.com/ https://login.microsoftonline.com/
```

Questo esempio mostra come ottenere gli endpoint e i metadati per l'ambiente AzureCloud (impostazione predefinita).
