# Save-AzContext

## SYNOPSIS
Salva le informazioni di autenticazione correnti per l'uso in altre sessioni di PowerShell.

## SINTASSI

```
Save-AzContext [[-Profile] <AzureRmProfile>] [-Path] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il Save-AzContext di autenticazione corrente salva le informazioni di autenticazione correnti per l'uso in altre sessioni di PowerShell.

## ESEMPI

### Esempio 1: Salvataggio del contesto della sessione corrente
```
PS C:\> Connect-AzAccount
PS C:\> Save-AzContext -Path C:\test.json
```

Questo esempio salva il contesto azure della sessione corrente nel file JSON fornito.

### Esempio 2: Salvataggio di un determinato contesto
```
PS C:\> Save-AzContext -Profile (Connect-AzAccount) -Path C:\test.json
```

Questo esempio salva il contesto di Azure passato al cmdlet nel file JSON fornito.
