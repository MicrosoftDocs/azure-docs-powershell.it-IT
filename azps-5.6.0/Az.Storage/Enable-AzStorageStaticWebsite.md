# Enable-AzStorageStaticWebsite

## SYNOPSIS
Abilitare il sito Web statico per l'account di archiviazione di Azure.

## SINTASSI

```
Enable-AzStorageStaticWebsite [[-IndexDocument] <String>] [[-ErrorDocument404Path] <String>] [-PassThru]
 [-Context <IStorageContext>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Enable-AzStorageStaticWebsite** abilita il sito Web statico per l'account di archiviazione di Azure.

## ESEMPI

### Esempio 1: Abilitare il sito Web statico per l'account di archiviazione di Azure
```
C:\PS>Enable-AzStorageStaticWebsite -IndexDocument $indexdoc -ErrorDocument404Path $errordoc
```

Questo comando abilita il sito Web statico per l'account di archiviazione di Azure.
