# Revoke-AzStorageAccountUserDelegationKeys

## SYNOPSIS
Revocare tutte le chiavi di delega degli utenti di un account di archiviazione.

## SINTASSI

### AccountName (impostazione predefinita)
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceGroupName] <String> [-StorageAccountName] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountObject
```
Revoke-AzStorageAccountUserDelegationKeys -InputObject <PSStorageAccount> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AccountResourceId
```
Revoke-AzStorageAccountUserDelegationKeys [-ResourceId] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Revoke-AzStorageAccountUserDelegationKeys** revoca tutte le chiavi di delega utente di un account di archiviazione, quindi verrà revocato anche tutto il token di firma di accesso condiviso di identità dell'account di archiviazione.

## ESEMPI

### Esempio 1: Revocare tutte le chiavi di delega utente di un account di archiviazione
```powershell
PS C:\>Revoke-AzStorageAccountUserDelegationKeys -ResourceGroupName "myresourcegroup" -AccountName "mystorageaccount"
```

Questo esempio revoca tutte le chiavi di delega utente di un account di archiviazione, quindi verrà revocato anche tutto il token di firma di accesso condiviso di identità generato dalle chiavi di delega utente.
