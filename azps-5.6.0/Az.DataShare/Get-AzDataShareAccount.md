# Get-AzDataShareAccount

## SYNOPSIS
Ottiene informazioni sugli account di Condivisione dati

## SINTASSI

### ByFieldsParameterSet (Impostazione predefinita)
```
Get-AzDataShareAccount [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceGroupParameterSet
```
Get-AzDataShareAccount -ResourceGroupName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareAccount -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzDataShareAccount** ottiene informazioni sugli account di condivisione dati in una sottoscrizione o un gruppo di risorse di Azure.
Se si specifica il nome di un account, questo cmdlet ottiene informazioni sull'account datshare.
Se non si specifica un nome, questo cmdlet ottiene informazioni su tutti gli account di condivisione dati in una sottoscrizione o un gruppo di risorse di Azure.
