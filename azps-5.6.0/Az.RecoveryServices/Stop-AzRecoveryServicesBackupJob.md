# Stop-AzRecoveryServicesBackupJob

## SYNOPSIS
Annulla un processo in esecuzione.

## SINTASSI

### JobFilterSet (impostazione predefinita)
```
Stop-AzRecoveryServicesBackupJob [-Job] <JobBase> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdFilterSet
```
Stop-AzRecoveryServicesBackupJob [-JobId] <String> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzRecoveryServicesBackupJob** annulla un processo di backup di Azure esistente.
Usare questo cmdlet per interrompere un processo che richiede troppo tempo e blocca altre attività.
È possibile annullare solo i tipi di processo di backup e ripristino.
Impostare il contesto del vault usando il cmdlet Set-AzRecoveryServicesVaultContext prima di usare il cmdlet corrente.

## ESEMPI

### Esempio 1: Interrompere un processo di backup
```
PS C:\>$Job = Get-AzRecoveryServicesBackupJob -Operation Backup
PS C:\> Stop-AzRecoveryServicesBackupJob -JobID $Job.InstanceId
```

Il primo comando ottiene un processo di backup e quindi lo archivia nella $Job variabile.
L'ultimo comando interrompe il processo specificando l'ID istanza del processo di backup in $Job.
