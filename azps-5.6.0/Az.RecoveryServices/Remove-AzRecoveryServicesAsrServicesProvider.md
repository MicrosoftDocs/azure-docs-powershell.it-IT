# Remove-AzRecoveryServicesAsrServicesProvider

## SYNOPSIS
Elimina/annulla la registrazione del provider di servizi di ripristino di Azure Site Recovery specificato dal vault dei servizi di ripristino.

## SINTASSI

```
Remove-AzRecoveryServicesAsrServicesProvider -InputObject <ASRRecoveryServicesProvider> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzRecoveryServicesAsrServicesProvider** rimuove il provider di servizi di ripristino dei siti di Azure specificato dal Vault.

## ESEMPI

### Esempio 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrServicesProvider -ServicesProvider $ServicesProvider
```

Avvia l'eliminazione/annullamento della registrazione del provider di servizi di Azure Site Recovery specificato e restituisce il processo ASR usato per tenere traccia dell'operazione.
