# Unregister-AzRecoveryServicesBackupManagementServer

## SYNOPSIS
Annulla la registrazione di un server SCDPM o di backup dal vault.

## SINTASSI

```
Unregister-AzRecoveryServicesBackupManagementServer [-AzureRmBackupManagementServer] <BackupEngineBase>
 [-PassThru] [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Unregister-AzRecoveryServicesBackupManagementServer** annulla la registrazione di un server System Center Data Protection Manager (SCDPM) o di un server di backup di Azure dal vault.
Questo cmdlet rimuove i riferimenti ai server di cui non è stata registrata la registrazione dal Vault.
Prima di annullare la registrazione di un contenitore, è necessario eliminare tutti i dati protetti associati a tale contenitore.
Impostare il contesto del vault usando il cmdlet Set-AzRecoveryServicesVaultContext prima di usare il cmdlet corrente.

## ESEMPI

### Esempio 1: Annullare la registrazione di un server SCDPM dal Vault
```powershell
PS C:\>$BMS = Get-AzRecoveryServicesBackupManagementServer -Name "dpmserver01.contoso.com"
PS C:\> Unregister-AzRecoveryServicesBackupManagementServer -AzBackupManagementServer $BMS
```

Il primo comando ottiene il server di gestione del backup dpmserver01.contoso.com e quindi lo archivia nella $BMS variabile.
Il secondo comando annulla la registrazione del server SCDPM dal Vault.

### Esempio 2

Annulla la registrazione di un server SCDPM o di backup dal vault. (autogenerati)

```powershell <!-- Aladdin Generated Example --> 
Unregister-AzRecoveryServicesBackupManagementServer -AzureRmBackupManagementServer <BackupEngineBase> -VaultId $vault.ID
```
