# Get-AzRecoveryServicesAsrAlertSetting

## SYNOPSIS
Recupera le impostazioni di notifica di Azure Site Recovery configurate per il vault.

## SINTASSI

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzRecoveryServicesAsrAlertSetting** ottiene le impostazioni di notifica di Azure Site Recovery configurate per il vault.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Impostazione di avviso/notifica per Azure Site Recovery.
