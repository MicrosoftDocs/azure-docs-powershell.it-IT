# Get-AzRecoveryServicesBackupContainer

## SYNOPSIS

Recupera i contenitori di backup.

## SINTASSI

```
Get-AzRecoveryServicesBackupContainer [-ContainerType] <ContainerType> [[-BackupManagementType] <String>]
 [[-FriendlyName] <String>] [[-ResourceGroupName] <String>] [[-Status] <ContainerRegistrationStatus>]
 [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE

Il cmdlet **Get-AzRecoveryServicesBackupContainer** ottiene un contenitore di backup. Un contenitore di backup incapsula le origini dati modellate come elementi di backup.
Per il tipo di contenitore "Azure VM", l'output elenca tutti i contenitori il cui nome corrisponde esattamente a quello passato come valore per il parametro Nome descrittivo. Per altri tipi di contenitori, l'output fornisce un elenco di contenitori con un nome simile al valore passato per il parametro nome descrittivo.
Impostare il contesto del vault usando il parametro -VaultId.

## ESEMPI

### Esempio 1: Ottenere un contenitore specifico

```powershell
PS C:\> $vault = Get-AzRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
PS C:\> Get-AzRecoveryServicesBackupContainer -ContainerType "AzureVM" -Status "Registered" -FriendlyName "V2VM" -VaultId $vault.ID
```

Questo comando recupera il contenitore denominato V2VM di tipo AzureVM.

### Esempio 2: Ottenere tutti i contenitori di un tipo specifico

```powershell
PS C:\> $vault = Get-AzRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
PS C:\> Get-AzRecoveryServicesBackupContainer -ContainerType Windows -BackupManagementType MARS -VaultId $vault.ID
```

Questo comando recupera tutti i contenitori di Windows protetti dall'agente di backup di Azure.
Il **parametro BackupManagementType** è necessario solo per i contenitori di Windows.
