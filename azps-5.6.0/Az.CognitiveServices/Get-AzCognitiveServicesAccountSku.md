# Get-AzCognitiveServicesAccountSku

## SYNOPSIS
Recupera gli SKU disponibili per un account.

## SINTASSI

```
Get-AzCognitiveServicesAccountSku [-Type <String>] [-Location <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzCognitiveServicesAccountSku** ottiene gli SKU disponibili per un account di Servizi cognitivi.
Lo SKU è il piano di livello per un account.
Definisce il prezzo, il limite di chiamata e la tariffa per l'account.
Lo SKU F0 è un livello gratuito.
I livelli a pagamento includono S0, S1, S2 e così via.

## ESEMPI

### Esempio 1
```powershell
PS C:\> (Get-AzCognitiveServicesAccountSku -Type 'TextAnalytics' -Location "westus").Value | Select-Object -E
xpandProperty Sku;

Name     Tier
----     ----
F0       Free
S0   Standard
```
