# Test-AzSynapseWorkspace

## SYNOPSIS
Verifica la presenza di un'area di lavoro Synapse Analytics.

## SINTASSI

```
Test-AzSynapseWorkspace [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Test-AzSynapseWorkspace** verifica l'esistenza di un'area di lavoro Synapse Analytics.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Test-AzSynapseWorkspace -Name ContosoWorkspace
```

Questo comando verifica l'esistenza di un'area di lavoro Synapse Analytics.
