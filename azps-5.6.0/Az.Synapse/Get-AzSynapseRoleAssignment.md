# Get-AzSynapseRoleAssignment

## SYNOPSIS
Ottiene un'assegnazione di ruolo di Synapse Analytics.

## DESCRIZIONE
Il cmdlet **Get-AzSynapseRoleAssignment** ottiene un'assegnazione di ruolo di Azure Synapse Analytics.
Se non si specifica una definizione di ruolo o un nome dell'entità utente, questo cmdlet ottiene tutte le assegnazioni di ruolo.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSynapseRoleAssignment -WorkspaceName ContosoWorkspace
```

Questo comando recupera tutte le assegnazioni di ruolo in un'area di lavoro.

### Esempio 2
```powershells
PS C:\> Get-AzSynapseRoleAssignment -WorkspaceName ContosoWorkspace -RoleDefinitionName ContosoRole
```

Questo comando recupera tutte le assegnazioni di ruolo nell'area di lavoro ContosoWorkspace con il nome di ruolo ContosoRole.

### Esempio 3
```powershells
PS C:\> Get-AzSynapseRoleAssignment -WorkspaceName ContosoWorkspace -RoleDefinitionName ContosoRole -SignInName ContosoName
```

Questo comando ottiene un'assegnazione di ruolo nell'area di lavoro ContosoWorkspace con il nome di ruolo ContosoRole e il nome dell'entità utente ContosoName.

### Esempio 4
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseRoleAssignment
```

Questo comando recupera tutte le assegnazioni di ruolo in un'area di lavoro tramite una pipeline.
