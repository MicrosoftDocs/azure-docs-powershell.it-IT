# Get-AzSynapseFirewallRule

## SYNOPSIS
Ottiene una regola del firewall di Synapse Analytics.

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzSynapseFirewallRule [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseFirewallRule -WorkSpaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSynapseFirewallRule** ottiene una regola firewall di Azure Synapse Analytics.
Se non si specifica un nome di regola, questo cmdlet ottiene tutte le regole.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSynapseFirewallRule -ResourceGroupName ContosoResourceGroup -WorkspaceName ContosoWorkspace
```

Questo comando recupera tutte le regole del firewall in un'area di lavoro.

### Esempio 2
```powershell
PS C:\> Get-AzSynapseFirewallRule -ResourceGroupName ContosoResourceGroup -WorkspaceName ContosoWorkspace -Name ContosoFirewallRule
```

Questo comando recupera la regola del firewall nell'area di lavoro ContosoWorkspace con il nome ContosoFirewallRule.

### Esempio 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseFirewallRule
```

Questo comando recupera tutte le regole del firewall in un'area di lavoro tramite pipeline.
