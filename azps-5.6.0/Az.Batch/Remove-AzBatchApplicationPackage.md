# Remove-AzBatchApplicationPackage

## SYNOPSIS
Elimina un record del pacchetto dell'applicazione e il file binario.

## SINTASSI

```
Remove-AzBatchApplicationPackage [-AccountName] <String> [-ResourceGroupName] <String>
 [-ApplicationName] <String> [-ApplicationVersion] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzBatchApplicationPackage** elimina un record del pacchetto dell'applicazione e il file binario da un account batch di Azure.

## ESEMPI

### Esempio 1: Eliminare un pacchetto dell'applicazione da un account Batch
```
PS C:\>Remove-AzBatchApplicationPackage -AccountName "ContosoBatch" -ResourceGroupName "ContosoBatchGroup" -ApplicationName "litware" -ApplicationVersion "1.0"
```

Questo comando elimina la versione 1.0 dell'applicazione Litware dall'account ContosoBatchGroup.
Il comando elimina sia il record del pacchetto che il BLOB che contiene il file binario del pacchetto.
