# Modulo Az.DedicatedHsm
## Descrizione
Microsoft Azure PowerShell: cmdlet DedicatedHsm

## Cmdlet Az.DedicatedHsm
### [Get-AzDedicatedHsm](Get-AzDedicatedHsm.md)
Ottiene l'HSM dedicato di Azure specificato.

### [New-AzDedicatedHsm](New-AzDedicatedHsm.md)
Creare o aggiornare un servizio HSM dedicato nell'abbonamento specificato.

### [Remove-AzDedicatedHsm](Remove-AzDedicatedHsm.md)
Elimina il servizio HSM dedicato di Azure specificato.

### [Update-AzDedicatedHsm](Update-AzDedicatedHsm.md)
Aggiornare un HSM dedicato nella sottoscrizione specificata.
