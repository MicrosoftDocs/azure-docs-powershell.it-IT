# Remove-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Rimuove una configurazione PrivateLink da un gateway applicazioni.

## SINTASSI

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGatewayPrivateLinkConfiguration** rimuove una configurazione privateLink da un gateway applicazione di Azure.

## ESEMPI

### Esempio 1: Rimuovere la configurazione PrivateLink di un gateway applicazione
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

Il primo comando ottiene un gateway applicazione e lo archivia nella $AppGw variabile.
Il secondo comando rimuove la configurazione privateLink denominata privateLinkConfig01 dal gateway di applicazione archiviato in $AppGw.
