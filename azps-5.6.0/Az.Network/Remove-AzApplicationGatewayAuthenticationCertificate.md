# Remove-AzApplicationGatewayAuthenticationCertificate

## SYNOPSIS
Rimuove un certificato di autenticazione da un gateway applicazioni.

## SINTASSI

```
Remove-AzApplicationGatewayAuthenticationCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGatewayAuthenticationCertificate rimuove** un certificato di autenticazione da un gateway applicazione di Azure.

## ESEMPI

### Esempio 1: Rimuovere un certificato di autenticazione da un gateway applicazioni
```
PS C:\> $appgw = Get-AzApplicationGateway -ResourceGroupName "rg" -Name "appGwName"
PS C:\> $appgw = Remove-AzApplicationGatewayAuthenticationCertificate -ApplicationGateway $appgw -Name "cert01"
PS C:\> $appgw = Set-AzApplicationGateway -ApplicationGateway $appgw
```

Il primo comando recupera il gateway applicazione denominato appGwName e archivia il risultato nella $appgw variabile.
Il secondo comando rimuove il certificato di autenticazione denominato cert01 dal gateway applicazione.
Il terzo comando aggiorna il gateway applicazione.
