# Get-AzApplicationGatewaySku

## SYNOPSIS
Ottiene lo SKU di un gateway applicazioni.

## SINTASSI

```
Get-AzApplicationGatewaySku -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzApplicationGatewaySku** ottiene l'unitÃ  di conservazione dei titoli azionari di un gateway applicazione.

## ESEMPI

### Esempio 1: Ottenere lo SKU di un gateway applicazione
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SKU = Get-AzApplicationGatewaySku -ApplicationGateway $AppGW
```

Il primo comando recupera il Gateway applicazione denominato ApplicationGateway01 e archivia il risultato nella variabile $AppGW.
Il secondo comando ottiene lo SKU di un gateway applicazione denominato ApplicationGateway01 e archivia il risultato nella variabile $SKU.
