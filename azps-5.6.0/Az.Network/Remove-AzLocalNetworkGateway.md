# Remove-AzLocalNetworkGateway

## SYNOPSIS
Elimina un Gateway di rete locale

## SINTASSI

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Gateway di rete locale è l'oggetto che rappresenta il dispositivo VPN locale.
Il cmdlet **Remove-AzLocalNetworkGateway** elimina l'oggetto che rappresenta il gateway locale in base al nome e al nome del gruppo di risorse.

## ESEMPI

### Esempio 1: Eliminare un Gateway di rete locale
```powershell
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

Elimina l'oggetto del Gateway di rete locale con il nome "myLocalGW" all'interno del gruppo di risorse "myRG" Nota: è necessario prima eliminare tutte le connessioni al Gateway di rete locale usando il cmdlet **Remove-AzVirtualNetworkGatewayConnection.**
