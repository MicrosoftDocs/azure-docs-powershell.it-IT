# Get-AzVirtualNetworkTap

## SYNOPSIS
Tocca una rete virtuale

## SINTASSI

### ListParameterSet (impostazione predefinita)
```
Get-AzVirtualNetworkTap [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzVirtualNetworkTap -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzVirtualNetworkTap** ottiene un tocco di rete virtuale di Azure o un elenco di tocchi di rete virtuale di Azure in un gruppo di risorse.

## ESEMPI

### Esempio 1: Toccare una rete virtuale
```
PS C:\> Get-AzVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
```

Questo comando ottiene un riferimento tocco VirtualNetwork per dato "VirtualTap1" in "ResourceGroup1".

### Esempio 2: Usare i filtri per ottenere tutti i tocchi della rete virtuale
```
PS C:\> Get-AzVirtualNetworkTap -Name "VirtualTap*"
```

Questo comando recupera tutti i riferimenti per il tocco di VirtualNetwork che iniziano con "VirtualTap".
