# Remove-AzLoadBalancerInboundNatRuleConfig

## SYNOPSIS
Rimuove una configurazione della regola NAT in ingresso da una bilanciamento del carico.

## SINTASSI

```
Remove-AzLoadBalancerInboundNatRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzLoadBalancerInboundNatRuleConfig** rimuove una configurazione della regola NAT (Network Address Translation) in ingresso da una bilanciamento del carico di Azure.

## ESEMPI

### 1: Eliminare una regola NAT in ingresso da una bilanciamento del carico di Azure
```
$loadbalancer = Get-AzLoadBalancer -Name mylb -ResourceGroupName myrg

 Remove-AzLoadBalancerInboundNatRuleConfig -Name "myinboundnatrule" -LoadBalancer $loadbalancer
```

Il primo comando carica una bilanciamento del carico gi√† esistente denominata "mylb" e la archivia nella variabile e $load bilanciamento. Il secondo comando rimuove la regola NAT in ingresso associata a questa bilanciamento del carico.
