# Get-AzApplicationGatewaySslPredefinedPolicy

## SYNOPSIS
Recupera i criteri SSL predefiniti forniti da Gateway applicazione.

## SINTASSI

```
Get-AzApplicationGatewaySslPredefinedPolicy [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzApplicationGatewaySslPredefinedPolicy** ottiene i criteri SSL predefiniti forniti da Gateway applicazione.
