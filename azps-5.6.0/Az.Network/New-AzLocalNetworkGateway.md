# New-AzLocalNetworkGateway

## SYNOPSIS
Crea un Gateway di rete locale

## SINTASSI

### ByLocalNetworkGatewayIpAddress
```
New-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> -Location <String>
 [-GatewayIpAddress <String>] [-AddressPrefix <String[]>] [-Asn <UInt32>] [-BgpPeeringAddress <String>]
 [-PeerWeight <Int32>] [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByLocalNetworkGatewayFqdn
```
New-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> -Location <String> [-Fqdn <String>]
 [-AddressPrefix <String[]>] [-Asn <UInt32>] [-BgpPeeringAddress <String>] [-PeerWeight <Int32>]
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Gateway di rete locale è l'oggetto che rappresenta il dispositivo VPN locale.
Il cmdlet **New-AzLocalNetworkGateway** crea l'oggetto che rappresenta il gateway locale in base al nome, al nome del gruppo di risorse, alla posizione e all'indirizzo IP del gateway, nonché al prefisso dell'indirizzo della rete locale che si connetterà ad Azure.

## ESEMPI

### Esempio 1: Creare un Gateway di rete locale
```powershell
New-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG -Location "West US" -GatewayIpAddress 23.99.221.164 -AddressPrefix "10.5.51.0/24"
```

Crea l'oggetto del Gateway di rete locale con il nome "myLocalGW" all'interno del gruppo di risorse "myRG" nella posizione "Stati Uniti occidentali" con l'indirizzo IP "23.99.221.164" e il prefisso dell'indirizzo "10.5.51.0/24" locale.
