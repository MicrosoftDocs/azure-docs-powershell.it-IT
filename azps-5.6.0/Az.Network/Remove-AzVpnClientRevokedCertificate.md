# Remove-AzVpnClientRevokedCertificate

## SYNOPSIS
Rimuove un certificato di revoca client VPN.

## SINTASSI

```
Remove-AzVpnClientRevokedCertificate -VpnClientRevokedCertificateName <String>
 -VirtualNetworkGatewayName <String> -ResourceGroupName <String> -Thumbprint <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzVpnClientRevokedCertificate** rimuove un certificato di revoca client da un gateway di rete virtuale.
I certificati di revoca client impediscono ai computer client di usare il certificato specificato per l'autenticazione.
Se si rimuove un certificato di revoca client, i computer client possono usare il certificato bloccato in precedenza per stabilire una connessione di rete privata virtuale (VPN).

## ESEMPI

### Esempio 1: Rimuovere un certificato di revoca client da un gateway di rete virtuale
```powershell
PS C:\>Remove-AzVpnClientRevokedCertificate -VirtualNetworkGatewayName "ContosoVirtualNetwork" -ResourceGroupName"ContosoResourceGroup" -VpnClientRevokedCertificateName "ContosoRevokedClientCertificate" -Thumbprint "E3A38EBA60CAA1C162785A2E1C44A15AD450199C3"
```

Questo comando rimuove un certificato di revoca client da un gateway di rete virtuale denominato ContosoVirtualNetwork.
Per rimuovere un certificato di revoca client, è necessario specificare sia il nome del certificato che l'impronta digitale del certificato.
