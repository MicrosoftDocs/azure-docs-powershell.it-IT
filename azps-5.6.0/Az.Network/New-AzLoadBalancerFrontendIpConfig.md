# New-AzLoadBalancerFrontendIpConfig

## SYNOPSIS
Crea una configurazione IP front-end per una bilanciamento del carico.

## SINTASSI

### SetByResourceSubnet (Impostazione predefinita)
```
New-AzLoadBalancerFrontendIpConfig -Name <String> [-PrivateIpAddress <String>]
 [-PrivateIpAddressVersion <String>] [-Zone <String[]>] -Subnet <PSSubnet>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```


## DESCRIZIONE
Il cmdlet **New-AzLoadBalancerFrontendIpConfig crea** una configurazione IP front-end per una bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1: Creare una configurazione IP front-end per una bilanciamento del carico
```
PS C:\> $publicip = New-AzPublicIpAddress -ResourceGroupName "MyResourceGroup" -Name "MyPublicIP" -Location "West US" -AllocationMethod "Dynamic"
PS C:\> New-AzLoadBalancerFrontendIpConfig -Name "FrontendIpConfig01" -PublicIpAddress $publicip
```

Il primo comando crea un indirizzo IP pubblico dinamico denominato MyPublicIP nel gruppo di risorse denominato MyResourceGroup e quindi lo archivia nella $publicip variabile.
Il secondo comando crea una configurazione IP front-end denominata FrontendIpConfig01 usando l'indirizzo IP pubblico in $publicip.

### Esempio 2: Creare una configurazione IP front-end per una bilanciamento del carico usando il prefisso IP
```
PS C:\> $publicipprefix = New-AzPublicIpPrefix -ResourceGroupName "MyResourceGroup" -name "MyPublicIPPrefix" -location "West US" -Sku Standard -PrefixLength 28
PS C:\> $frontend = New-AzLoadBalancerFrontendIpConfig -Name "FrontendIpConfig01" -PublicIpAddressPrefix $publicipprefix
```

Il primo comando crea un prefisso IP pubblico denominato MyPublicIP di lunghezza 28 nel gruppo di risorse denominato MyResourceGroup, quindi lo archivia nella variabile $publicipprefix variabile.
Il secondo comando crea una configurazione IP front-end denominata FrontendIpConfig01 usando il prefisso IP pubblico in $publicipprefix.
