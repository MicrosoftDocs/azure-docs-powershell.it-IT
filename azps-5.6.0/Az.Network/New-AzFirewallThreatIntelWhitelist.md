# New-AzFirewallThreatIntelWhitelist

## SYNOPSIS
Creare una nuova lista bianca di Threat Intelligence per il Firewall di Azure

## SINTASSI

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzFirewallThreatIntelWhitelist** crea un oggetto threat intel whitelist, che puÃ² essere usato quando si crea o si imposta un firewall di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

In questo esempio viene creata una lista bianca di informazioni sulle minacce che contiene un elenco di due voci FQDN e un elenco indirizzi Ip di due voci.
