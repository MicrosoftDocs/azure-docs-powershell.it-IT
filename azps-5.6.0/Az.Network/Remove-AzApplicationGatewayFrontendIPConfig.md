# Remove-AzApplicationGatewayFrontendIPConfig

## SYNOPSIS
Rimuove una configurazione IP front-end da un gateway applicazioni.

## SINTASSI

```
Remove-AzApplicationGatewayFrontendIPConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGatewayFrontendIPConfig** rimuove l'indirizzo IP di frontend da un gateway applicazione di Azure.

## ESEMPI

### Esempio 1: Rimuovere una configurazione IP front-end
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayFrontendIPConfig -ApplicationGateway $AppGw -Name "FrontEndIP02"
```

Il primo comando ottiene un gateway applicazione denominato ApplicationGateway01 e lo archivia nella $AppGw variabile.
Il secondo comando rimuove la configurazione IP front-end denominata FrontEndIP02 dal gateway di applicazione archiviato in $AppGw.
