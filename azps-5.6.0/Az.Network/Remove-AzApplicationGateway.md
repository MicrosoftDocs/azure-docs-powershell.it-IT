# Remove-AzApplicationGateway

## SYNOPSIS
Rimuove un gateway applicazione.

## SINTASSI

```
Remove-AzApplicationGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGateway** rimuove un gateway applicazione.

## ESEMPI

### Esempio 1: Rimuovere un gateway applicazione specificato
```
PS C:\>Remove-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
```

Questo comando rimuove il gateway applicazione denominato ApplicationGateway01 nel gruppo di risorse denominato ResourceGroup01.
