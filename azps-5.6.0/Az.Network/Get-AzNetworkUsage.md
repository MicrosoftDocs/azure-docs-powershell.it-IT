# Get-AzNetworkUsage

## SYNOPSIS
Elenca gli utilizzi della rete per una sottoscrizione

## SINTASSI

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Get-AzNetworkUsage ottiene limiti e l'utilizzo corrente per le risorse di rete.

## ESEMPI

### Esempio 1
