# Get-AzApplicationGatewayBackendHealth

## SYNOPSIS
Recupera l'integrità back-end del gateway applicazione.

## SINTASSI

```
Get-AzApplicationGatewayBackendHealth -Name <String> -ResourceGroupName <String> [-ExpandResource <String>]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Get-AzApplicationGatewayBackendHealth recupera l'integrità back-end del gateway applicazione.

## ESEMPI

### Esempio 1: recupera l'integrità del back-end senza risorse espanse.
```
PS C:\>$BackendHealth = Get-AzApplicationGatewayBackendHealth -Name ApplicationGateway01 -ResourceGroupName ResourceGroup01
```

Questo comando recupera l'integrità back-end del gateway applicazione denominato ApplicationGateway01 che appartiene al gruppo di risorse denominato ResourceGroup01 e lo archivia nella variabile $BackendHealth variabile.

### Esempio 2: Recupera l'integrità back-end con risorse espanse.
```
PS C:\>$BackendHealth = Get-AzApplicationGatewayBackendHealth -Name ApplicationGateway01 -ResourceGroupName ResourceGroup01 -ExpandResource "backendhealth/applicationgatewayresource"
```

Questo comando recupera l'integrità back-end (con risorse espanse) del gateway applicazione denominato ApplicationGateway01 che appartiene al gruppo di risorse denominato ResourceGroup01 e lo archivia nella variabile $BackendHealth variabile.
