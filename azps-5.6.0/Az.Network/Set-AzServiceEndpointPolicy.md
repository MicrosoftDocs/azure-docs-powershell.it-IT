# Set-AzServiceEndpointPolicy

## SYNOPSIS
Aggiorna i criteri degli endpoint del servizio.

## SINTASSI

```
Set-AzServiceEndpointPolicy -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzServiceEndpointPolicy** crea un criterio endpoint del servizio.

## ESEMPI

### Esempio 1: imposta un criterio endpoint del servizio
```
$serviceEndpointPolicy = Set-AzServiceEndpointPolicy -Name "Policy1" -ServiceEndpointPolicy $serviceEndpointPolicy -ResourceGroup "resourcegroup1"
```

Questo comando aggiorna un criterio endpoint di servizio denominato Criterio1 definito dall'oggetto $serviceEndpointPolicy appartiene al gruppo di risorse "resourcegroup1".
