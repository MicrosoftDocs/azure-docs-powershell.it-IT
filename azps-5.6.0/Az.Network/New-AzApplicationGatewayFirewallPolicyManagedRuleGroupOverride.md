# New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride

## SYNOPSIS
Crea la voce RuleGroupOverride in ManagedRuleSets per il criterio del firewall.

## SINTASSI

```
New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName <String>
 -Rule <PSApplicationGatewayFirewallPolicyManagedRuleOverride[]> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
**New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride** crea una voce ruleGroupOverride in un managedRuleSet per un criterio firewall.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $overrideEntry = New-AzApplicationGatewayFirewallPolicyManagedRuleGroupOverride -RuleGroupName $ruleName -Rules $rule1,$rule2
```

Crea una voce RuleGroupOverride con il nome del gruppo $ruleName e Rules come $rule 1, $rule 2. Assegna lo stesso valore a $overrideEntry
