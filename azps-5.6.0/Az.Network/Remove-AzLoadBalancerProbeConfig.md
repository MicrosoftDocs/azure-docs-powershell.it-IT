# Remove-AzLoadBalancerProbeConfig

## SYNOPSIS
Rimuove una configurazione da una bilanciamento del carico.

## SINTASSI

```
Remove-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzLoadBalancerProbeConfig rimuove** una configurazione di configurazione da una bilanciamento del carico.

## ESEMPI

### Esempio 1: Rimuovere una configurazione di configurazione da una bilanciamento del carico
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerProbeConfig -Name "MyProbe" -LoadBalancer $loadbalancer
```

Il primo comando recupera la bilanciamento del carico denominata MyLoadBalancer e quindi la archivia nella $loadbalancer variabile.
Il secondo comando elimina la configurazione denominata MyProbe dal servizio di bilanciamento del carico in $loadbalancer.
