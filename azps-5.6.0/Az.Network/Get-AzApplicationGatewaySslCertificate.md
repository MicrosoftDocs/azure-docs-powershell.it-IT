# Get-AzApplicationGatewaySslCertificate

## SYNOPSIS
Ottiene un certificato SSL per un gateway applicazione.

## SINTASSI

```
Get-AzApplicationGatewaySslCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzApplicationGatewaySslCertificate** ottiene un certificato SSL per un gateway applicazione.

## ESEMPI

### Esempio 1: Ottenere un certificato SSL specifico
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Cert = Get-AzApplicationGatewaySslCertificate -Name "Cert01" -ApplicationGateway $AppGW
```

Il primo comando recupera il gateway applicazione denominato ApplicationGateway01 e archivia il risultato nella variabile $AppGW.
Il secondo comando recupera il certificato SSL denominato Cert01 dal gateway applicazione archiviato nella variabile $AppGW.
Il comando archivia il certificato nella variabile denominata $Cert.

### Esempio 2: Ottenere un elenco di certificati SSL
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $Certs = Get-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW
```

Il primo comando recupera il gateway applicazione denominato ApplicationGateway01 e archivia il risultato nella variabile $AppGW.
Questo secondo comando ottiene un elenco di certificati SSL dal gateway applicazione archiviato nella variabile $AppGW.
Il comando archivia quindi i risultati nella variabile denominata $Certs.
