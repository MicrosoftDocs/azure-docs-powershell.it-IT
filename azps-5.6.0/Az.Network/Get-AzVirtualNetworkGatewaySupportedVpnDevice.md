# Get-AzVirtualNetworkGatewaySupportedVpnDevice

## SYNOPSIS
Questo commandlet restituisce un elenco di marche, modelli e versioni del firmware supportati per i dispositivi VPN.

## SINTASSI

```
Get-AzVirtualNetworkGatewaySupportedVpnDevice -Name <String> -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Questo commandlet restituisce un elenco di marche, modelli e versioni del firmware supportati per i dispositivi VPN.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzVirtualNetworkGatewaySupportedVpnDevice -ResourceGroupName TestRG -Name TestGateway 
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
```

Restituisce l'elenco dei marchi, dei modelli e delle versioni del firmware supportati per i dispositivi VPN:
<?xml version="1.0" encoding="utf-8"?>
<RpVpnDeviceList version="1.0">
  <Vendor name="Cisco-Test">
    <DeviceFamily name="IOS-Test">
       <FirmwareVersion name="20" />
    </DeviceFamily>
  </Vendor>
</RpVpnDeviceList>
