# Remove-AzApplicationGatewaySslCertificate

## SYNOPSIS
Rimuove un certificato SSL da un gateway applicazione di Azure.

## SINTASSI

```
Remove-AzApplicationGatewaySslCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGatewaySslCertificate** rimuove un certificato SSL (Secure Sockets Layer) da un gateway applicazione di Azure.

## ESEMPI

### Esempio 1: Rimuovere un certificato SSL da un gateway applicazioni
```
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslCertificate -ApplicationGateway $AppGW -Name "Cert02"
```

Il primo comando recupera il gateway applicazione denominato ApplicationGateway01 e archivia il risultato nella variabile $AppGW.
Il secondo comando rimuove il certificato SSL denominato Cert02 dal gateway di applicazione archiviato nella $AppGW variabile.
