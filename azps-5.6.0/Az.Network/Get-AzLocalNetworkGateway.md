# Get-AzLocalNetworkGateway

## SYNOPSIS
Ottiene un Gateway di rete locale

## SINTASSI

```
Get-AzLocalNetworkGateway [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Gateway di rete locale Ã¨ l'oggetto che rappresenta il dispositivo VPN locale.
Il cmdlet **Get-AzLocalNetworkGateway restituisce** l'oggetto che rappresenta il gateway locale in base al nome e al nome del gruppo di risorse.

## ESEMPI

### Esempio 1: Ottenere un Gateway di rete locale
```powershell
Get-AzLocalNetworkGateway -Name myLocalGW1 -ResourceGroupName myRG

                           }
```

Restituisce l'oggetto di Gateway di rete locale con il nome "myLocalGW1" all'interno del gruppo di risorse "myRG"
