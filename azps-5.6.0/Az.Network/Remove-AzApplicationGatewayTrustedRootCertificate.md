# Remove-AzApplicationGatewayTrustedRootCertificate

## SYNOPSIS
Rimuove un certificato radice attendibile da un gateway applicazioni.

## SINTASSI

```
Remove-AzApplicationGatewayTrustedRootCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGatewayTrustedRootCertificate rimuove** un certificato radice attendibile da un gateway applicazione esistente.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name "myRootCA"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Il primo comando ottiene un gateway applicazione e lo archivia nella $gw variabile.
Il secondo comando rimuove il certificato radice attendibile denominato myRootCA dal gateway applicazione archiviato in $gw.
Il terzo comando aggiorna il gateway applicazione in Azure.
