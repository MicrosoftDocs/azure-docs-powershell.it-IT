# Remove-AzNetworkSecurityGroup

## SYNOPSIS
Rimuove un gruppo di sicurezza di rete.

## SINTASSI

```
Remove-AzNetworkSecurityGroup -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzNetworkSecurityGroup** rimuove un gruppo di sicurezza di rete azure.

## ESEMPI

### Esempio 1: Rimuovere un gruppo di sicurezza di rete
```
PS C:\>Remove-AzNetworkSecurityGroup -Name "NSG-FrontEnd" -ResourceGroupName "TestRG"
```

Questo comando rimuove il gruppo di sicurezza NSG-FrontEnd gruppo di risorse denominato TestRG.
