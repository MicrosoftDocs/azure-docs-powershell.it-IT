# New-AzApplicationGatewayFrontendPort

## SYNOPSIS
Crea una porta front-end per un gateway applicazioni.

## SINTASSI

```
New-AzApplicationGatewayFrontendPort -Name <String> -Port <Int32> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzApplicationGatewayFrontendPort** crea una porta front-end per un gateway applicazione di Azure.

## ESEMPI

### Esempio 1: Esempio 1: Creare una porta front-end
```powershell
PS C:\>$FrontEndPort = New-AzApplicationGatewayFrontendPort -Name "FrontEndPort01" -Port 80
```

Questo comando crea una porta front-end denominata FrontEndPort01 sulla porta 80 e archivia il risultato nella variabile $FrontEndPort.
