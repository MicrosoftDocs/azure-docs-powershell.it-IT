# Set-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Modifica la catena di certificati CA del client attendibile di un gateway applicazioni.

## SINTASSI

```
Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzApplicationGatewayTrustedClientCertificate** modifica la catena di certificati CA del client attendibile di un gateway applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName --CertificateFile ".\clientCAUpdated.cer"
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Negli scenari di esempio precedenti viene illustrato come aggiornare un oggetto della catena di certificati CA del client attendibile esistente. Il primo comando ottiene un gateway applicazione e lo archivia nella $gw variabile. Il secondo comando modifica l'oggetto della catena di certificati CA del client attendibile esistente con un nuovo file della catena di certificati CA. Il terzo comando aggiorna il gateway applicazione in Azure.
