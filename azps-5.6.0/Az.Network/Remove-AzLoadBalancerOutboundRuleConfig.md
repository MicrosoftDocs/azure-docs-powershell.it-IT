# Remove-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
Rimuove una configurazione della regola in uscita da una bilanciamento del carico.

## SINTASSI

```
Remove-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzLoadBalancerOutboundRuleConfig rimuove** una configurazione della regola in uscita da una bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1: Eliminare una regola in uscita da una bilanciamento del carico di Azure
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Remove-AzLoadBalancerOutboundRuleConfig -Name "RuleName" -LoadBalancer $slb
PS C:\>Set-AzLoadBalancer -LoadBalancer $slb
```

Il primo comando ottiene il bilanciamento del carico associato alla configurazione della regola in uscita che si vuole rimuovere e quindi lo archivia nella variabile $slb in uscita.
Il secondo comando rimuove la configurazione della regola in uscita associata dal servizio di bilanciamento del carico.
Il terzo comando aggiorna il bilanciamento del carico.
