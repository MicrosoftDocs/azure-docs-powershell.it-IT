# Get-AzDataMigrationProject

## SYNOPSIS
Recupera le proprietà di un progetto di migrazione del database di Azure.

## SINTASSI

### ComponentNameParameterSet (Default)
```
Get-AzDataMigrationProject -ResourceGroupName <String> -ServiceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Get-AzDataMigrationProject [-InputObject] <PSDataMigrationService> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzDataMigrationProject [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il Get-AzDataMigrationProject cmdlet recupera le proprietà di un progetto di migrazione del database di Azure.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzDataMigrationProject -ServiceName testService -Name testProject -ResourceGroup testResourceGroup
```

L'esempio precedente recupera il progetto di migrazione del database di Azure denominato TestProject nel gruppo di risorse denominato testResourceGroup e nel servizio denominato testService

### Esempio 2
```
PS C:\> Get-AzDataMigrationProject -InputObject $myService
```

L'esempio precedente recupera il progetto di migrazione del database di Azure in base al parametro di input dell'oggetto PSProject passato.
