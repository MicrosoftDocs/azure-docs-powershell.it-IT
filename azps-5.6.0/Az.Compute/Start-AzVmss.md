# Start-AzVmss

## SYNOPSIS
Avvia VMSS o un set di macchine virtuali all'interno del sistema VMSS.

## SINTASSI

```
Start-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Start-AzVmss** avvia tutte le macchine virtuali nel set di scala delle macchine virtuali (VMSS) o in un set di macchine virtuali.
Ãˆ possibile usare il *parametro InstanceId* per selezionare un set di macchine virtuali.

## ESEMPI

### Esempio 1: Avviare un set specifico di macchine virtuali all'interno del sistema VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"-InstanceId "0", "1"
```

Questo comando avvia un set specifico di macchine virtuali specificato dalla matrice della stringa di ID di istanza che appartengono al sistema VMSS denominato ContosoVMSS.

### Esempio 2: Avviare tutte le macchine virtuali nel sistema VMSS
```
PS C:\> Start-AzVmss -ResourceGroupName "ContosOrg" -VMScaleSetName "ContosoVMSS"
```

Questo comando avvia tutte le macchine virtuali appartenenti a VMSS denominate ContosoVMSS.
