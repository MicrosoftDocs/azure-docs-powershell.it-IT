# Stop-AzVM

## SYNOPSIS
Arresta una macchina virtuale di Azure.

## SINTASSI

### ResourceGroupNameParameterSetName (impostazione predefinita)
```
Stop-AzVM [-Name] <String> [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-ResourceGroupName] <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Stop-AzVM [-Force] [-StayProvisioned] [-NoWait] [-SkipShutdown] [-Id] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzVM** interrompe una macchina virtuale di Azure.

## ESEMPI

### Esempio 1: Arrestare una macchina virtuale
```
PS C:\> Stop-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Questo comando interrompe la macchina virtuale denominata VirtualMachine07 in ResourceGroup11.
