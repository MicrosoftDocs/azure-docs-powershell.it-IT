# Revoke-AzSnapshotAccess

## SYNOPSIS
Revoca l'accesso a uno snapshot.

## SINTASSI

```
Revoke-AzSnapshotAccess [-ResourceGroupName] <String> [-SnapshotName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Revoke-AzSnapshotAccess** revoca l'accesso a uno snapshot.

## ESEMPI

### Esempio 1
```
PS C:\> Revoke-AzSnapshotAccess -ResourceGroupName 'ResourceGroup01' -SnapshotName 'Snapshot01'
```

Revocare l'accesso alla snapshot denominata 'Snapshot01' nel gruppo di risorse denominato 'ResourceGroup01'
