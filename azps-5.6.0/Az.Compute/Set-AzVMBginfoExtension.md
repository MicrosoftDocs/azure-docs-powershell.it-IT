# Set-AzVMBginfoExtension

## SYNOPSIS
Aggiunge l'estensione BGInfo a una macchina virtuale.

## SINTASSI

```
Set-AzVMBginfoExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name <String>]
 [-TypeHandlerVersion <String>] [-Location <String>] [-DisableAutoUpgradeMinorVersion] [-ForceRerun <String>]
 [-NoWait] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzVMBGInfoExtension aggiunge** l'estensione BGInfo a una macchina virtuale.

## ESEMPI

### Esempio 1: Aggiungere l'estensione BGInfo per una macchina virtuale
```
PS C:\> Set-AzVMBgInfoExtension -ResourceGroupName "ContosoRG" -VMName "ContosoVM" -Name "ExtensionName" -TypeHandlerVersion "2.1" -Location "West Europe"
```

Questo comando aggiunge l'estensione BGInfo alla macchina virtuale contosoVM.
Il comando specifica il gruppo di risorse e il percorso della macchina virtuale.
Il comando specifica il nome e la versione dell'estensione.
