# Remove-AzVmssDiagnosticsExtension

## SYNOPSIS
Rimuove un'estensione di diagnostica da VMSS.

## SINTASSI

```
Remove-AzVmssDiagnosticsExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzVmssDiagnosticsExtension** rimuove un'estensione di diagnostica dal set di scala di macchine virtuali (VMSS).

## ESEMPI

### Esempio 1: Rimuovere un'estensione di diagnostica da VMSS
```
PS C:\> Remove-AzVmssDiagnosticsExtension -VirtualMachineScaleSet $VMSS -Name $extName
```

Questo comando rimuove l'estensione di diagnostica da VMSS.
