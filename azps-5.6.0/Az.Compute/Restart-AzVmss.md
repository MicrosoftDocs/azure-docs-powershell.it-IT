# Restart-AzVmss

## SYNOPSIS
Riavvia il sistema VMSS o una macchina virtuale all'interno di VMSS.

## SINTASSI

```
Restart-AzVmss [-ResourceGroupName] <String> [-VMScaleSetName] <String> [[-InstanceId] <String[]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Restart-AzVmss** riavvia il set di scalabilità delle macchine virtuali (VMSS).
Questo cmdlet può essere usato anche per riavviare una macchina virtuale specifica all'interno di VMSS usando il *parametro InstanceId.*

## ESEMPI

### Esempio 1: Riavviare VMSS
```
PS C:\> Restart-AzVmss -ResourceGroupName "Group001" -VMScaleSetName "VMSS001";
```

Questo comando riavvia il sistema VMSS denominato VMSS001 che appartiene al gruppo di risorse denominato Group001.

### Esempio 2: Riavviare una macchina virtuale specifica all'interno di VMSS
```
PS C:\> Restart-AzVmss -ResourceGroupName "Group004" -VMScaleSetName "VMSS001" -InstanceId "1"
```

Questo comando riavvia una macchina virtuale il cui ID istanza è 1 nel sistema VMSS denominato VMSS001 che appartiene al gruppo di risorse denominato Group001.
