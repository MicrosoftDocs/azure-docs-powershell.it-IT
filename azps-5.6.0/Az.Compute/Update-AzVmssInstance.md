# Update-AzVmssInstance

## SYNOPSIS
Avvia un aggiornamento manuale dell'istanza di VMSS.

## SINTASSI

```
Update-AzVmssInstance [-ResourceGroupName] <String> [-VMScaleSetName] <String> [-InstanceId] <String[]>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il Update-AzVmssInstance cmdlet avvia un aggiornamento manuale dell'istanza di VMSS (Virtual Machine Scale Set) specificata.
Viene usato quando il criterio di aggiornamento del set di scale VMSS Ã¨ impostato su manuale.

## ESEMPI

### Esempio 1: Avviare un aggiornamento dell'istanza di VMSS
```
PS C:\> Update-AzVmssInstance -ResourceGroupName "Group011" -VMScaleSetName "VMScaleSet001" -InstanceId "0"
```

Questo comando avvia un aggiornamento del sistema VMSS denominato VMScaleSet001 con l'ID istanza 0.
