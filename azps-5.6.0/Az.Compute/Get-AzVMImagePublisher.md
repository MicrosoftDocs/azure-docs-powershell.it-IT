# Get-AzVMImagePublisher

## SYNOPSIS
Ottiene gli editori VMImage.

## SINTASSI

```
Get-AzVMImagePublisher -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzVMImagePublisher** ottiene gli editori VMImage.

## ESEMPI

### Esempio 1: Ottenere gli editori VMImage per un'area geografica
```
PS C:\> Get-AzVMImagePublisher -Location "Central US"
```

Questo comando recupera gli editori delle istanze di VMImage per l'area geografica Degli Stati Uniti centrali all'interno del profilo azure.
