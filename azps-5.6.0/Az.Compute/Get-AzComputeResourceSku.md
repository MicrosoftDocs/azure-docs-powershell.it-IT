# Get-AzComputeResourceSku

## SYNOPSIS
Elencare tutte le SKU delle risorse di calcolo

## SINTASSI

```
Get-AzComputeResourceSku [[-Location] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Elencare tutte le SKU delle risorse di calcolo

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzComputeResourceSku "westus";
```

Elencare tutte le SKU delle risorse di calcolo nell'area Degli Stati Uniti occidentali
