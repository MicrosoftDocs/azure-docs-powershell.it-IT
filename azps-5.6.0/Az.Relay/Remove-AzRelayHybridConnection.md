# Remove-AzRelayHybridConnection

## SYNOPSIS
Rimuove HybridConnection dallo spazio dei nomi HybridConnection specificato.

## SINTASSI

```
Remove-AzRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzRelayHybridConnection** rimuove HybridConnection dallo spazio dei nomi Relay specificato.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Rimuove HybridConnection `TestHybridConnection` dallo spazio dei `TestNameSpace-Relay1` nomi.
