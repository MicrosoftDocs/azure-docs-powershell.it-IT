# Modulo Az.ADDomainServices
## Descrizione
Cmdlet di Microsoft Azure PowerShell: AdDomainServices

## Cmdlet Az.ADDomainServices
### [Get-AzADDomainService](Get-AzADDomainService.md)
L'operazione Ottieni servizio di dominio recupera una rappresentazione json del servizio di dominio.

### [New-AzADDomainService](New-AzADDomainService.md)
L'operazione Crea servizio di dominio crea un nuovo servizio di dominio con i parametri specificati.
Se il servizio specifico esiste già, le eventuali proprietà modificabili verranno aggiornate e le eventuali proprietà non modificabili rimarranno invariate.

### [New-AzADDomainServiceForestTrust](New-AzADDomainServiceForestTrust.md)
Creare un oggetto in memoria per ForestTrust

### [New-AzADDomainServiceReplicaSet](New-AzADDomainServiceReplicaSet.md)
Creare un oggetto in memoria per ReplicaSet

### [Remove-AzADDomainService](Remove-AzADDomainService.md)
L'operazione Elimina servizio di dominio elimina un servizio di dominio esistente.

### [Update-AzADDomainService](Update-AzADDomainService.md)
L'operazione Update Domain Service può essere usata per aggiornare la distribuzione esistente.
La chiamata di aggiornamento supporta solo le proprietà elencate nel corpo PATCH.
