# Test-AzAnalysisServicesServer

## SYNOPSIS
Verifica la presenza di un'istanza del server Analysis Services

## SINTASSI

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Test-AzAnalysisServicesServer verifica l'esistenza di un'istanza del server Analysis Services

## ESEMPI

### Esempio 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Questo comando verifica se nel gruppo di test del gruppo di risorse Ã¨ presente un server denominato testserver
