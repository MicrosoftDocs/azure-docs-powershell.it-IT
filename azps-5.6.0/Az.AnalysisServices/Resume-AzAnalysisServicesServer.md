# Resume-AzAnalysisServicesServer

## SYNOPSIS
Riprende un'istanza del server Analysis Services

## SINTASSI

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Resume-AzAnalysisServicesServer riprendere un'istanza del server Analysis Services

## ESEMPI

### Esempio 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Questo comando riprenderà un server denominato testserver in pausa nel gruppo di test del gruppo di risorse
