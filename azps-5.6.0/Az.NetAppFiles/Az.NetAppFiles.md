# Modulo Az.NetAppFiles
## Descrizione
Gli argomenti di questa sezione documentano i cmdlet di Azure PowerShell per i file NetApp di Azure nel framework di Gestione risorse di Azure (ARM). I cmdlet sono presenti nello spazio dei nomi Microsoft.Azure.Commands.NetAppFiles.
