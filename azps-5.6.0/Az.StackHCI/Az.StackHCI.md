# Modulo Az.StackHCI
## Descrizione
Microsoft Azure PowerShell: cmdlet di registrazione HCI di Azure Stack

## Cmdlet Az.StackHCI
### [Register-AzStackHCI](Register-AzStackHCI.md)
Register-AzStackHCI crea una risorsa cloud Microsoft.AzureStackHCI che rappresenta il cluster locale e registra il cluster locale con Azure.

### [Test-AzStackHCIConnection](Test-AzStackHCIConnection.md)
Test-AzStackHCIConnection verifica la connettivit√† dai nodi cluster locali ai servizi di Azure richiesti da Azure Stack HCI.

### [Unregister-AzStackHCI](Unregister-AzStackHCI.md)
Unregister-AzStackHCI elimina la risorsa cloud Microsoft.AzureStackHCI che rappresenta il cluster locale e annulla la registrazione del cluster locale con Azure.
Le informazioni registrate disponibili nel cluster vengono usate per annullare la registrazione del cluster se non vengono passati parametri.
