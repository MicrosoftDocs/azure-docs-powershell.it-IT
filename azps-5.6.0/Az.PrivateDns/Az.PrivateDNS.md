# Modulo Az.PrivateDns
## Descrizione
Questo argomento visualizza gli argomenti della Guida per i cmdlet privati di DNS di Azure.

## Cmdlet Az.PrivateDns
### [Add-AzPrivateDnsRecordConfig](Add-AzPrivateDnsRecordConfig.md)
Aggiunge un record DNS privato a un oggetto set di record locale.

### [Get-AzPrivateDnsRecordSet](Get-AzPrivateDnsRecordSet.md)
Ottiene un set di record da un'area DNS privata.

### [Get-AzPrivateDnsVirtualNetworkLink](Get-AzPrivateDnsVirtualNetworkLink.md)
Ottiene un collegamento di rete virtuale associato all'area DNS privata specificata.

### [Get-AzPrivateDnsZone](Get-AzPrivateDnsZone.md)
Ottiene un'area DNS privata.

### [New-AzPrivateDnsRecordConfig](New-AzPrivateDnsRecordConfig.md)
Crea un nuovo oggetto locale del record DNS privato.

### [New-AzPrivateDnsRecordSet](New-AzPrivateDnsRecordSet.md)
Crea un set di record in un'area DNS privata.

### [New-AzPrivateDnsVirtualNetworkLink](New-AzPrivateDnsVirtualNetworkLink.md)
Crea un nuovo collegamento di rete virtuale DNS privato.

### [New-AzPrivateDnsZone](New-AzPrivateDnsZone.md)
Crea una nuova area DNS privata.

### [Remove-AzPrivateDnsRecordConfig](Remove-AzPrivateDnsRecordConfig.md)
Rimuove un record DNS privato da un oggetto set di record locale.

### [Remove-AzPrivateDnsRecordSet](Remove-AzPrivateDnsRecordSet.md)
Elimina un set di record da un'area DNS privata.

### [Remove-AzPrivateDnsVirtualNetworkLink](Remove-AzPrivateDnsVirtualNetworkLink.md)
Rimuove un collegamento di rete virtuale da un gruppo di risorse.

### [Remove-AzPrivateDnsZone](Remove-AzPrivateDnsZone.md)
Rimuove un'area DNS privata da un gruppo di risorse.

### [Set-AzPrivateDnsRecordSet](Set-AzPrivateDnsRecordSet.md)
Aggiorna/imposta un set di record in un'area DNS privata.

### [Set-AzPrivateDnsVirtualNetworkLink](Set-AzPrivateDnsVirtualNetworkLink.md)
Aggiorna/imposta un collegamento di rete virtuale associato a un'area privata e a un gruppo di risorse.

### [Set-AzPrivateDnsZone](Set-AzPrivateDnsZone.md)
Aggiorna un'area DNS privata da un gruppo di risorse.
