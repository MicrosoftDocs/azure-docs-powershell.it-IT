# Get-AzPrivateDnsVirtualNetworkLink

## SYNOPSIS
Ottiene un collegamento di rete virtuale associato all'area DNS privata specificata.

## SINTASSI

```
Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName <String> -ZoneName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzPrivateDnsVirtualNetworkLink** ottiene i collegamenti di rete virtuale associati a una particolare area DNS privata dal gruppo di risorse specificato.
Se si specifica il *parametro Name,* viene restituito un singolo **oggetto PSPrivateDnsVirtualNetworkLink.**
Se non si specifica il *parametro Name,* verrà restituita una matrice contenente tutti i collegamenti associati all'area nel gruppo di risorse specificato.
È possibile usare **l'oggetto PSPrivateDnsVirtualNetworkLink** per aggiornare il collegamento.

```
PS C:\> $Link = Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName "MyResourceGroup" -ZoneName "myzone.com" -Name "mylink"

ZoneName                : myzone.com

                          etwork/virtualNetworks/myvirtualnetwork
```

Questo esempio recupera il collegamento di rete virtuale associato all'area DNS privata denominata myzone.com dal gruppo di risorse specificato e quindi lo archivia nella variabile $Link privata.
