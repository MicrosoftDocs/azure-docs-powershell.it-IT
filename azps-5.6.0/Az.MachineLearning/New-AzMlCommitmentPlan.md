# New-AzMlCommitmentPlan

## SYNOPSIS
Crea un nuovo piano di impegno.

## SINTASSI

```
New-AzMlCommitmentPlan -ResourceGroupName <String> -Location <String> -Name <String> -SkuName <String>
 -SkuTier <String> [-SkuCapacity <Int32>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Crea un piano di impegno per Azure Machine Learning in un gruppo di risorse esistente.
Se nel gruppo di risorse è presente un piano di impegno con lo stesso nome, la chiamata funge da operazione di aggiornamento e il piano di impegno esistente viene sovrascritto.

## ESEMPI

### Esempio 1: Creare un nuovo piano di impegno
```
New-AzMlCommitmentPlan -ResourceGroupName "MyResourceGroup" -Name "MyCommitmentPlanName" -Location "South Central US" -SkuName DevTest -SkuTier Standard -SkuCapacity 1
```

Crea un nuovo piano di impegno di Azure Machine Learning denominato "MyCommitmentPlanName" nel gruppo "MyResourceGroup" e nell'area sud degli Stati Uniti. In questo esempio viene usato lo SKU DevTest/Standard, ovvero le risorse fornite dal piano di impegno saranno definite dai limiti di DevTest/Standard. SkuCapacity in questo esempio è 1, ovvero il costo del piano sarà 1x il costo di DevTest e le risorse contenute nel piano saranno 1x quanto fornito da DevTest.
