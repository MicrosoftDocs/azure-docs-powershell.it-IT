# Resume-AzDataFactoryPipeline

## SYNOPSIS
Riprende una pipeline sospesa in Data factory.

## SINTASSI

### ByFactoryName (Default)
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Resume-AzDataFactoryPipeline [-Name] <String> [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Resume-AzDataFactoryPipeline** riprende una pipeline sospesa in Data factory di Azure.

## ESEMPI

### Esempio 1: Riprendere una pipeline
```powershell
PS C:\>Resume-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF"
Confirm
Are you sure you want to resume pipeline 'DPWikisample' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Questo comando consente di riprendere la pipeline denominata DPWikisample nel data factory denominato WikiADF.
Usare il cmdlet **Suspend-AzDataFactoryPipeline** per sospendere una pipeline.
Il comando restituisce il valore $True.

### Esempio 2

Riprende una pipeline sospesa in Data factory. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Resume-AzDataFactoryPipeline -DataFactory $DataFactory -Name 'DPWikisample'
```
