# New-AzDataFactoryPipeline

## SYNOPSIS
Crea una pipeline in Data factory.

## SINTASSI

### ByFactoryName (Default)
```
New-AzDataFactoryPipeline [[-Name] <String>] [-DataFactoryName] <String> [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryPipeline [[-Name] <String>] [-DataFactory] <PSDataFactory> [-File] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzDataFactoryPipeline** crea una pipeline in Data factory di Azure.
Se si specifica un nome per una pipeline già esistente, il cmdlet richiede conferma prima che sostituisca la pipeline.
Se si specifica il *parametro Force,* il cmdlet sostituisce la pipeline esistente senza conferma.
Eseguire queste operazioni nell'ordine seguente: 
- Creare un data factory. 
- Creare servizi collegati. 
- Creare set di dati. 
- Creare una pipeline.
Se nel data factory è già presente una pipeline con lo stesso nome, questo cmdlet chiede di confermare se sovrascrivere la pipeline esistente con la nuova pipeline.
Se si conferma di sovrascrivere la pipeline esistente, viene sostituita anche la definizione della pipeline.

## ESEMPI

### Esempio 1: Creare una pipeline
```
PS C:\>New-AzDataFactoryPipeline -ResourceGroupName "ADF" -Name "DPWikisample" -DataFactoryName "WikiADF" -File "C:\DPWikisample.json" 
PipelineName      : DPWikisample
ResourceGroupName : ADF
DataFactoryName   : WikiADF11
Properties        : Microsoft.DataFactories.PipelineProperties
```

Questo comando crea una pipeline denominata DPWikisample nel data factory denominato ADF.
Il comando basa la pipeline sulle informazioni del DPWikisample.jssu file.
Questo file include informazioni su attività come l'attività di copia e l'attività HDInsight nella pipeline.
