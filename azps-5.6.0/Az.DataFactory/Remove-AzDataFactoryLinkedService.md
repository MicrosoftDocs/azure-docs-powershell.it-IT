# Remove-AzDataFactoryLinkedService

## SYNOPSIS
Rimuove un servizio collegato da Data factory di Azure.

## SINTASSI

### ByFactoryName (Default)
```
Remove-AzDataFactoryLinkedService [-Force] [-DataFactoryName] <String> [-Name] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryLinkedService [-Force] [-DataFactory] <PSDataFactory> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzDataFactoryLinkedService** rimuove un servizio collegato da Data factory di Azure.

## ESEMPI

### Esempio 1: Rimuovere un servizio collegato
```
PS C:\>Remove-AzDataFactoryLinkedService -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -Name "LinkedServiceTest"
Confirm
Are you sure you want to remove linked service 'LinkedServiceTest' in data factory 'WikiADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Questo comando rimuove il servizio collegato denominato LinkedServiceTest dal data factory denominato WikiADF.
Questo comando restituisce il valore $True.
