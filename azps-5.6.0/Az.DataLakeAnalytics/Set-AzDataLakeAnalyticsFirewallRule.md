# Set-AzDataLakeAnalyticsFirewallRule

## SYNOPSIS
Aggiorna una regola del firewall in un account di Data Lake Analytics.

## SINTASSI

```
Set-AzDataLakeAnalyticsFirewallRule [-Account] <String> [-Name] <String> [[-StartIpAddress] <String>]
 [[-EndIpAddress] <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzDataLakeAnalyticsFirewallRule** aggiorna una regola del firewall in un account di Azure Data Lake Analytics.

## ESEMPI

### Esempio 1: Aggiornare una regola del firewall
```
PS C:\>Set-AzDataLakeAnalyticsFirewallRule -Account "ContosoAdlAcct" -Name "My firewall rule" -StartIpAddress 127.0.0.1 -EndIpAddress 127.0.0.10
```

Questo comando aggiorna la regola del firewall denominata "regola del firewall" nell'account "ContosoAdlAcct" in modo che abbia il nuovo intervallo IP: 127.0.0.1 - 127.0.0.10
