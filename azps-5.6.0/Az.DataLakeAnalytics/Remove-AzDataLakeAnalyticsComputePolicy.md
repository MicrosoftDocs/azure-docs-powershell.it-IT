# Remove-AzDataLakeAnalyticsComputePolicy

## SYNOPSIS
Rimuove un criterio di elaborazione di Azure Data Lake Analytics specificato

## SINTASSI

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
**Remove-AzDataLakeAnalyticsComputePolicy** rimuove un criterio di calcolo di Azure Data Lake Analytics specificato.

## ESEMPI

### Esempio 1: Rimuovere un criterio di calcolo
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Questo comando rimuove il criterio di elaborazione specificato con il nome 'myPolicy' nell'account 'contosoadla'.
