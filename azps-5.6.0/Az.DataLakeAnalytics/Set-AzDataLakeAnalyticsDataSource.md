# Set-AzDataLakeAnalyticsDataSource

## SYNOPSIS
Modifica i dettagli di un'origine dati di un account Data Lake Analytics.

## SINTASSI

```
Set-AzDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-AccessKey] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzDataLakeAnalyticsDataSource** modifica i dettagli di un'origine dati di un account Azure Data Lake Analytics.

## ESEMPI

### Esempio 1: Cambiare la chiave di accesso per un'origine dati
```
PS C:\>Set-AzDataLakeAnalyticsDataSource -Account "ContosoAdlAccount" -Blob "contosowasb" -AccessKey "...newaccesskey..."
```

Questo comando modifica la chiave di accesso archiviata per un'origine dati di Archiviazione BLOB di Azure.
