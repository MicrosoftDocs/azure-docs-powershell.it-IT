# Get-AzHpcCacheUsageModel

## SYNOPSIS
Recupera tutti i modelli di utilizzo per destinazione di archiviazione NFS.

## SINTASSI

```
Get-AzHpcCacheUsageModel [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzHpcCacheUsageModel** restituisce un elenco di modelli di utilizzo per la destinazione di archiviazione NFS.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzHpcCacheUsageModel
```
