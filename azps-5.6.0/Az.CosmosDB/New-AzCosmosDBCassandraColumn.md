# New-AzCosmosDBCassandraColumn

## SYNOPSIS
Crea una nuova colonna CassandraDB.

## SINTASSI

```
New-AzCosmosDBCassandraColumn -Name <String> -Type <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
La **colonna New-AzCosmosDBCassandraColumn** crea una nuova colonnaDb Cassandra.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzCosmosDBCassandraColumn -Name "name" -Type int

Name Type
---- ----
name int
```
