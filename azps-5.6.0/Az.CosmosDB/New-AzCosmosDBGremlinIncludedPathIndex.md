# New-AzCosmosDBGremlinIncludedPathIndex

## SYNOPSIS
Crea un nuovo oggetto di tipo PSIndexes. Può essere passato come valore di parametro per Set-AzCosmosDBGremlinIncludedPath.

## SINTASSI

```
New-AzCosmosDBGremlinIncludedPathIndex -DataType <String> [-Precision <Int32>] -Kind <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Oggetto corrispondente agli indici di IncludedPath dell'API Di Gremlin.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzCosmosDBGremlinIncludedPathIndex -DataType String -Precision -1 -Kind Hash

DataType Precision Kind
-------- --------- ----
String          -1 Hash
```
