# Get-AzLegacyPeering

## SYNOPSIS
Consente di convertire le risorse peering legacy in risorse di Gestione ARM Azure. 

## SINTASSI

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il comando viene usato per visualizzare le risorse peering legacy, che verranno convertite in risorse di Gestione ARM Azure.
