# Get-AzStreamAnalyticsFunction

## SYNOPSIS
Ottiene funzioni in un processo di Analisi di flusso.

## SINTASSI

```
Get-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzStreamAnalyticsFunction** ottiene un elenco delle funzioni definite in un processo di Analisi di flusso di Azure o informazioni su una funzione specifica.

## ESEMPI

### Esempio 1: Ottenere tutte le funzioni di Analisi di flusso
```
PS C:\>Get-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22"
```

Questo comando ottiene le funzioni definite nel processo denominato StreamJob22.

### Esempio 2: Ottenere una funzione specifica di Analisi di flusso
```
PS C:\>Get-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Questo comando recupera informazioni sulla funzione denominata ScoreTweet definita nel processo denominato StreamJob22.
