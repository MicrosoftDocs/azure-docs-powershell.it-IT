# Remove-AzStreamAnalyticsJob

## SYNOPSIS
Rimuove un processo di Analisi di flusso.

## SINTASSI

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzStreamAnalyticsJob** elimina in modo asincrono uno specifico processo di Analisi di flusso in Azure.

## ESEMPI

### ESEMPIO 1: Rimuovere un processo
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Questo comando rimuove il processo StreamingJob.
