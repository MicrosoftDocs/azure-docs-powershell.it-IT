# New-AzStreamAnalyticsFunction

## SYNOPSIS
Crea o sostituisce una funzione in un processo di Analisi di flusso.

## SINTASSI

```
New-AzStreamAnalyticsFunction [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzStreamAnalyticsFunction** crea una funzione in un processo di Analisi di flusso di Azure o sostituisce una funzione esistente.
Definire la funzione in un file JSON (JavaScript Object Notation).
Ãˆ possibile specificare il nome della funzione usando il parametro *Name* o nel file JSON.
Se si specifica il nome in entrambi i modi, i nomi devono corrispondere.
Per sostituire una funzione esistente, specificare il nome della funzione esistente.

## ESEMPI

### Esempio 1: Creare una funzione di Analisi di flusso
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob07" -File "C:\Function07.json"
```

Questo comando crea una funzione dal Function07.jsfile.
Il nome della funzione viene archiviato nel file JSON.

### Esempio 2: Creare una funzione di Analisi di flusso denominata ScoreTweet
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet"
```

Questo comando crea una funzione nel processo denominato ScoreTweet.

### Esempio 3: Sostituire una funzione di Analisi di flusso
```
PS C:\>New-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -File "C:\Function22.json" -Name "ScoreTweet" -Force
```

Questo comando sostituisce la definizione della funzione esistente denominata ScoreTweet con la definizione in Function22.jsavanti.
