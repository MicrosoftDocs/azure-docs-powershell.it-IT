# Remove-AzStreamAnalyticsInput

## SYNOPSIS
Elimina un input da un processo di Analisi di flusso.

## SINTASSI

```
Remove-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzStreamAnalyticsInput** elimina in modo asincrono un input da un processo di Analisi di flusso in Azure.

## ESEMPI

### ESEMPIO 1: Rimuovere un flusso di input da un processo
```
PS C:\>Remove-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Questo comando rimuove l'input EventStream da StreamingJob.
