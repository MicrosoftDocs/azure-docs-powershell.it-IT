# Test-AzStreamAnalyticsInput

## SYNOPSIS
Verifica lo stato di connessione di un input.

## SINTASSI

```
Test-AzStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Test-AzStreamAnalyticsInput** verifica la capacitÃ  di Stream Analytics di connettersi a un input.

## ESEMPI

### Esempio 1: Verificare lo stato di connessione di uno stream di input
```powershell
PS C:\>Test-AzStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EntryStream"
```

Verifica lo stato di connessione dell'input EntryStream in StreamingJob.
