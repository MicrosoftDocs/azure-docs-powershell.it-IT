# Set-AzIntegrationAccountCertificate

## SYNOPSIS
Modifica un certificato dell'account di integrazione.

## SINTASSI

```
Set-AzIntegrationAccountCertificate -ResourceGroupName <String> -Name <String> -CertificateName <String>
 [-KeyName <String>] [-KeyVersion <String>] [-KeyVaultId <String>] [-PublicCertificateFilePath <String>]
 [-Metadata <Object>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Set-AzIntegrationAccountCertificate** modifica un certificato dell'account di integrazione.
Questo cmdlet restituisce un oggetto che rappresenta il certificato dell'account di integrazione.
Specificare il nome dell'account di integrazione, il nome del gruppo di risorse e il nome del certificato.
Questo modulo supporta i parametri dinamici.
Per usare un parametro dinamico, digitarlo nel comando.
Per individuare i nomi dei parametri dinamici, digitare un trattino (-) dopo il nome del cmdlet e quindi premere TAB piÃ¹ volte per scorrere i parametri disponibili.
Se si omette un parametro di modello obbligatorio, il cmdlet chiede di specificare il valore.
