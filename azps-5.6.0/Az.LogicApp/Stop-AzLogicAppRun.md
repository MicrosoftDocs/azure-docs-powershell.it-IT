# Stop-AzLogicAppRun

## SYNOPSIS
Annulla un'esecuzione di un'app per la logica.

## SINTASSI

```
Stop-AzLogicAppRun -ResourceGroupName <String> -Name <String> -RunName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzLogicAppRun** annulla un'esecuzione di un'app logica.
Specificare l'app per la logica, il gruppo di risorse ed eseguire.
Questo modulo supporta i parametri dinamici.
Per usare un parametro dinamico, digitarlo nel comando.
Per individuare i nomi dei parametri dinamici, digitare un trattino (-) dopo il nome del cmdlet e quindi premere TAB piÃ¹ volte per scorrere i parametri disponibili.
Se si omette un parametro di modello obbligatorio, il cmdlet chiede di specificare il valore.

## ESEMPI

### Esempio 1: Annullare un'esecuzione di un'app per la logica
```
PS C:\>Stop-AzLogicAppRun -ResourceGroupName "ResourceGroup11" -Name "LogicApp03" -RunName "08587489104702792076" -Force
```

Questo comando annulla un'esecuzione dell'app logica denominata LogicApp03.
