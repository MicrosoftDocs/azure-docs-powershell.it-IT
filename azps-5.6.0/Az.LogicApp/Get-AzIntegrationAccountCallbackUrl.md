# Get-AzIntegrationAccountCallbackUrl

## SYNOPSIS
Ottiene l'URL di un account di integrazione.

## SINTASSI

```
Get-AzIntegrationAccountCallbackUrl -ResourceGroupName <String> -Name <String> [-NotAfter <DateTime>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzIntegrationAccountUrlUrl** ottiene l'URL url di url url dell'account di integrazione da un gruppo di risorse.
Questo cmdlet restituisce un oggetto **UrlUrl** che rappresenta l'URL url url dell'account di integrazione.
Specificare il nome dell'account di integrazione e il nome del gruppo di risorse.
Questo modulo supporta i parametri dinamici.
Per usare un parametro dinamico, digitarlo nel comando.
Per individuare i nomi dei parametri dinamici, digitare un trattino (-) dopo il nome del cmdlet e quindi premere TAB piÃ¹ volte per scorrere i parametri disponibili.
Se si omette un parametro di modello obbligatorio, il cmdlet chiede di specificare il valore.
