# Modulo Az.Maps
## Descrizione
Questo argomento visualizza la Guida per i cmdlet di Azure Maps.

## Cmdlet di Az.Maps
### [Get-AzMapsAccount](Get-AzMapsAccount.md)
Ottiene l'account.

### [Get-AzMapsAccountKey](Get-AzMapsAccountKey.md)
Ottiene le chiavi API per un account.
Queste chiavi sono il meccanismo di autenticazione utilizzato nelle chiamate successive ad Azure Maps.

### [New-AzMapsAccount](New-AzMapsAccount.md)
Crea un account di Azure Maps.

### [New-AzMapsAccountKey](New-AzMapsAccountKey.md)
Rigenera una chiave account.

### [Remove-AzMapsAccount](Remove-AzMapsAccount.md)
Elimina un account di Azure Maps.
