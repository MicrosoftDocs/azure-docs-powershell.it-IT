# Remove-AzSecurityWorkspaceSetting

## SYNOPSIS
Elimina l'impostazione dell'area di lavoro di sicurezza per la sottoscrizione.

## SINTASSI

### SubscriptionLevelResource (Default)
```
Remove-AzSecurityWorkspaceSetting -Name <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzSecurityWorkspaceSetting -ResourceId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSecurityWorkspaceSetting -InputObject <PSSecurityWorkspaceSetting> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Elimina l'impostazione dell'area di lavoro di sicurezza per la sottoscrizione.
Questa azione consente agli agenti di sicurezza appena installati di segnalare l'area di lavoro predefinita di questa sottoscrizione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzSecurityWorkspaceSetting -Name "default"
```

Elimina l'impostazione dell'area di lavoro di sicurezza per la sottoscrizione.
