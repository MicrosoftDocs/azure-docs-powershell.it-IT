# Set-AzSecurityAutoProvisioningSetting

## SYNOPSIS
Aggiorna l'impostazione di provisioning automatico

## SINTASSI

### SubscriptionLevelResource (Default)
```
Set-AzSecurityAutoProvisioningSetting -Name <String> [-EnableAutoProvision]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Set-AzSecurityAutoProvisioningSetting [-EnableAutoProvision] -ResourceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Set-AzSecurityAutoProvisioningSetting -InputObject <PSSecurityAutoProvisioningSetting>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Attiva o disattiva le impostazioni di provisioning automatico per un abbonamento.
Le impostazioni di provisioning automatico consentono di decidere se si vuole che il Centro sicurezza di Azure eserviti il provisioning automatico di un agente di sicurezza che verrà installato nelle macchine virtuali.
L'agente di sicurezza monitorerà la macchina virtuale per creare avvisi di sicurezza e monitorare la conformità della macchina virtuale.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Set-AzSecurityAutoProvisioningSetting -Name "default" -EnableAutoProvision

--                                                                                                                ----    -------------
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/autoProvisioningSettings/default default On
```

Attiva l'impostazione di provisioning automatico per un abbonamento.

### Esempio 2

```powershell
PS C:\> Set-AzSecurityAutoProvisioningSetting -Name "default"
Id                                                                                                                Name

```

Disattiva l'impostazione di provisioning automatico per un abbonamento.
