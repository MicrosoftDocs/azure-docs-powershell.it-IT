# Get-AzRegulatoryComplainceAssessment

## SYNOPSIS
Ottiene le valutazioni di conformità alle normative

## SINTASSI

### SubscriptionLevelResource (Default)
```
Get-AzRegulatoryComplainceAssessment [-Name <String>] -StandardName <String> -ControlName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzRegulatoryComplainceAssessment -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Ottenere informazioni dettagliate sulla valutazione o elencare tutte le valutazioni sotto uno specifico standard di controllo e conformità alle normative.
