# Set-AzSecurityPricing

## SYNOPSIS

Abilita o disabilita i piani di Azure Defender per una sottoscrizione nel Centro sicurezza di Azure.

## SINTASSI

### SubscriptionLevelResource (Default)

```powershell
Set-AzSecurityPricing -Name <String> -PricingTier <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObject

```powershell
Set-AzSecurityPricing -InputObject <PSSecurityPricing> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE

Abilitare o disabilitare uno qualsiasi dei piani di Azure Defender per una sottoscrizione.

Per informazioni dettagliate su Azure Defender e sui piani disponibili, vedere [Introduzione ad Azure Defender.](https://docs.microsoft.com/azure/security-center/azure-defender)

## ESEMPI

### Esempio 1

```powershell
PS C:\> Set-AzSecurityPricing -Name "virtualmachines" -PricingTier "Standard"
```

Abilita **Azure Defender per i server** per la sottoscrizione.

"Standard" fa riferimento allo stato "Attivo" di un piano di Azure Defender, come mostrato nell'area prezzi e impostazioni del Centro sicurezza di Azure del portale di Azure.
