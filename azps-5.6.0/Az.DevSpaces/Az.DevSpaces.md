# Modulo Az.DevSpaces
## Descrizione
Comandi per interagire con il controller Azure DevSpaces.

## Cmdlet di Az.DevSpaces
### [Get-AzDevSpacesController](Get-AzDevSpacesController.md)
Ottieni o elenca il controller Azure DevSpaces.

### [New-AzDevSpacesController](New-AzDevSpacesController.md)
Crea un nuovo controller Azure DevSpaces.

### [Remove-AzDevSpacesController](Remove-AzDevSpacesController.md)
Eliminare un controller DevSpaces.

### [Update-AzDevSpacesController](Update-AzDevSpacesController.md)
Aggiorna il controller DevSpaces per aggiungere tag.
