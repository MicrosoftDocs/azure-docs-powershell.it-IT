# Get-AzFrontDoor

## SYNOPSIS
Ottenere una bilanciamento del carico anteriore

## SINTASSI

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzFrontDoorGet** ottiene tutte le porte anteriore esistenti nell'abbonamento corrente che corrispondono alle informazioni fornite.
