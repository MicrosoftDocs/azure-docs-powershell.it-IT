# Remove-AzRedisCacheFirewallRule

## SYNOPSIS
Rimuovere una regola del firewall da una cache di Ridisposizione.

## SINTASSI

### NormalParameterSet (impostazione predefinita)
```
Remove-AzRedisCacheFirewallRule [-ResourceGroupName <String>] -Name <String> -RuleName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PSRedisFirewallRuleObject
```
Remove-AzRedisCacheFirewallRule -InputObject <PSRedisFirewallRule> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Rimuovere una regola del firewall da una cache di Ridisposizione.

## ESEMPI

### Esempio 1: Rimuovere una singola regola del firewall
```
PS C:\>Remove-AzRedisCacheFirewallRule -Name "mycache" -RuleName "ruleone" -PassThru
True
```

Questo comando rimuove dalla cache di Redis una regola del firewall denominata ruleone denominata mycache.
