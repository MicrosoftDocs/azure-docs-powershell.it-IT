# Get-AzRedisCacheLink

## SYNOPSIS
Ottenere il collegamento di replica geografica per la cache di Redis.

## SINTASSI

### AllLinksForCache (impostazione predefinita)
```
Get-AzRedisCacheLink -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AllLinksForPrimaryCache
```
Get-AzRedisCacheLink -PrimaryServerName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### SingleLink
```
Get-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AllLinksForSecondaryCache
```
Get-AzRedisCacheLink -SecondaryServerName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Esistono quattro diversi modi per ottenere i dettagli del collegamento di replica geografica. Fornire il parametro Name o PrimaryServerName e/o SecondaryServerName. Viene assegnato il nome, quindi vengono restituiti tutti i collegamenti in cui è presente la cache. Se viene assegnato solo PrimaryServerName, verranno restituiti tutti i collegamenti in cui la cache è primaria. Se si riceve solo SecondaryServerName, verranno restituiti tutti i collegamenti in cui la cache è secondaria. Se vengono fornite entrambe le proprietà PrimaryServerName e SecondaryServerName, verrà restituito un collegamento specifico con il ruolo corretto. 

## ESEMPI

### Esempio 1: Ottenere il set di parametri Using AllLinksForCache
```
PS C:\>Get-AzRedisCacheLink -Name "mycache1"

        PrimaryServerName   : mycache1
        SecondaryServerName : mycache2
        ProvisioningState   : Succeeded
```

Questo comando recupera tutti i collegamenti di replica geografica per la cache di Redis denominata mycache1.

### Esempio 2: Ottenere l'uso del set di parametri AllLinksForPrimaryCache
```
PS C:\>Get-AzRedisCacheLink -PrimaryServerName "mycache1"

        PrimaryServerName   : mycache1
        SecondaryServerName : mycache2
        ProvisioningState   : Succeeded
```

Questo comando recupera i collegamenti di replica geografica in cui la cache di Redis denominata mycache1 è primaria.

### Esempio 3: Ottenere l'uso del set di parametri AllLinksForSecondaryCache
```
PS C:\>Get-AzRedisCacheLink -SecondaryServerName "mycache2"

        PrimaryServerName   : mycache1
        SecondaryServerName : mycache2
        ProvisioningState   : Succeeded
```

Questo comando recupera i collegamenti di replica geografica in cui la cache di Redis denominata mycache2 è secondaria.

### Esempio 4: Usare il set di parametri SingleLink
```
PS C:\>Get-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"

        PrimaryServerName   : mycache1
        SecondaryServerName : mycache2
        ProvisioningState   : Succeeded
```

Questo comando recupera un singolo collegamento di replica geografica in cui la cache di Redis denominata mycache1 è primaria e la cache di Redis denominata mycache2 è secondaria.
