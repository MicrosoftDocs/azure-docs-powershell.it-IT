# Remove-AzRedisCacheLink

## SYNOPSIS
Rimuovere un collegamento di replica geografica tra due cache di Redis.

## SINTASSI

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Rimuovere un collegamento di replica geografica tra due cache di Redis.

## ESEMPI

### Esempio 1: Rimuovere un collegamento di replica geografica
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Questo comando rimuove un collegamento di replica geografica in cui la cache di Redis denominata mycache1 è primaria e la cache di Redis denominata mycache2 è secondaria.
