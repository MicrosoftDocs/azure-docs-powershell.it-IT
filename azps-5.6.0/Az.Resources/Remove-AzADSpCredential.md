# Remove-AzADSpCredential

## SYNOPSIS
Rimuove le credenziali da un'entità servizio.

## SINTASSI

### ObjectIdWithKeyIdParameterSet (impostazione predefinita)
```
Remove-AzADSpCredential -ObjectId <String> [-KeyId <Guid>] [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SPNWithKeyIdParameterSet
```
Remove-AzADSpCredential -ServicePrincipalName <String> [-KeyId <Guid>] [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DisplayNameWithKeyIdParameterSet
```
Remove-AzADSpCredential -DisplayName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ServicePrincipalObjectParameterSet
```
Remove-AzADSpCredential -ServicePrincipalObject <PSADServicePrincipal> [-KeyId <Guid>] [-PassThru] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il Remove-AzADSpCredential cmdlet può essere usato per rimuovere una chiave delle credenziali da un'entità servizio in caso di compromissione o come parte della scadenza del rollover delle credenziali.
L'entità servizio viene identificata fornendo l'ID oggetto o il nome dell'entità servizio (SPN).
Le credenziali da rimuovere sono identificate dal relativo ID chiave se è necessario rimuovere singole credenziali o con un parametro "Tutte" per eliminare tutte le credenziali associate all'entità servizio.

## ESEMPI

### Esempio 1: Rimuovere credenziali specifiche da un'entità servizio

```powershell
PS C:\> Remove-AzADSpCredential -ObjectId 7663d3fb-6f86-4352-9e6d-cf9d50d5ee82 -KeyId 9044423a-60a3-45ac-9ab1-09534157ebb
```

Rimuove le credenziali con ID chiave '9044423a-60a3-45ac-9ab1-09534157ebb' dall'entità servizio con ID oggetto '7663d3fb-6f86-4352-9e6d-cf9d50d5ee82'.

### Esempio 2: Rimuovere tutte le credenziali da un'entità servizio

```powershell
PS C:\> Remove-AzADSpCredential -ServicePrincipalName http://test123
```

Rimuove tutte le credenziali dall'entità servizio con il nome SPN " http://test123 ".

### Esempio 3: Rimuovere tutte le credenziali da un'entità servizio tramite piping

```powershell
PS C:\> Get-AzADServicePrincipal -ObjectId 7663d3fb-6f86-4352-9e6d-cf9d50d5ee82 | Remove-AzADSpCredential
```

Ottiene l'entità servizio con ID oggetto '7663d3fb-6f86-4352-9e6d-cf9d50d5ee82' e pipe che al cmdlet di Remove-AzADSpCredential per rimuovere tutte le credenziali dall'entità servizio.
