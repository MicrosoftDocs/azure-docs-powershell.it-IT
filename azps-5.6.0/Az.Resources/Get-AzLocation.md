# Get-AzLocation

## SYNOPSIS
Recupera tutte le posizioni e i provider di risorse supportati per ogni posizione.

## SINTASSI

```
Get-AzLocation [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzLocation** ottiene tutte le posizioni e i provider di risorse supportati per ogni posizione.

## ESEMPI

### Esempio 1: Ottenere tutte le posizioni e i provider di risorse supportati
```
PS C:\>Get-AzLocation
```

Questo comando recupera tutte le posizioni e i provider di risorse supportati per ogni posizione.
