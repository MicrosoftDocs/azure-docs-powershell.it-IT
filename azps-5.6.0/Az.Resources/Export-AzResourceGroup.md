# Export-AzResourceGroup

## SYNOPSIS
Acquisisce un gruppo di risorse come modello e lo salva in un file.

## SINTASSI

```
Export-AzResourceGroup -ResourceGroupName <String> [-Path <String>] [-IncludeParameterDefaultValue]
 [-IncludeComments] [-SkipResourceNameParameterization] [-SkipAllParameterization] [-Resource <String[]>]
 [-Force] [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Export-AzResourceGroup** acquisisce il gruppo di risorse specificato come modello e lo salva in un file JSON. Ciò può risultare utile in scenari in cui sono già state create alcune risorse nel gruppo di risorse e quindi si vogliono sfruttare i vantaggi derivanti dall'uso di distribuzioni con modello supportato.
Questo cmdlet consente di creare facilmente il modello per le risorse esistenti nel gruppo di risorse.
In alcuni casi, questo cmdlet potrebbe non generare alcune parti del modello.
I messaggi di avviso informano dell'esito negativo delle risorse.
Il modello verrà comunque generato per le parti che hanno avuto esito positivo.

## ESEMPI

### Esempio 1: Esportare un gruppo di risorse
```
PS C:\>Export-AzResourceGroup -ResourceGroupName "TestGroup"
```

Questo comando acquisisce il gruppo di risorse denominato Gruppo di test come modello e lo salva in un file JSON nella directory corrente.

### Esempio 2: Esportare una singola risorsa da un gruppo di risorse
```
PS C:\>Export-AzResourceGroup -ResourceGroupName "TestGroup" -Resource "/subscriptions/5f43547b-1d2d-4a3e-ace4-88d4b600d568/resourceGroups/TestGroup/providers/Microsoft.Compute/virtualMachines/TestVirtualMachine"
```

Questo comando acquisisce la risorsa macchina virtuale denominata "TestVirtualMachine" dal gruppo di risorse "TestGroup" come modello e la salva in un file JSON nella directory corrente.

### Esempio 3: Esportare una selezione di risorse da un gruppo di risorse
```
PS C:\>Export-AzResourceGroup -ResourceGroupName "TestGroup" -SkipAllParameterization -Resource @(
  "/subscriptions/5f43547b-1d2d-4a3e-ace4-88d4b600d568/resourceGroups/TestGroup/providers/Microsoft.Compute/virtualMachines/TestVm",
  "/subscriptions/5f43547b-1d2d-4a3e-ace4-88d4b600d568/resourceGroups/TestGroup/providers/Microsoft.Network/networkInterfaces/TestNic"
)
```

Questo comando acquisisce due risorse dal gruppo di risorse "Gruppo di test" come modello e lo salva in un file JSON nella directory corrente. Il modello generato non conterrà parametri generati.
