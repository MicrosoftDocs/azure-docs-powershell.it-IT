# Register-AzResourceProvider

## SYNOPSIS
Registra un provider di risorse.

## SINTASSI

```
Register-AzResourceProvider -ProviderNamespace <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Register-AzResourceProvider** registra un provider di risorse di Azure.

## ESEMPI

### Esempio 1: Registrare un provider
```
PS C:\>Register-AzResourceProvider -ProviderNamespace Microsoft.Network
```

Il provider di rete Microsoft.Network viene registrato per l'account.
