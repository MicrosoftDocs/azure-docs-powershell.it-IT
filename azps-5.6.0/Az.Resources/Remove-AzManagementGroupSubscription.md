# Remove-AzManagementGroupSubscription

## SYNOPSIS
Rimuove una sottoscrizione da un gruppo di gestione.

## SINTASSI

```
Remove-AzManagementGroupSubscription [-GroupName] <String> [-SubscriptionId] <Guid> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzManagementGroupSubscription** rimuove una sottoscrizione da un gruppo di gestione.

## ESEMPI

### Esempio 1: Rimuovere la sottoscrizione da un gruppo di gestione
```powershell
PS C:\> Remove-AzManagementGroupSubscription -GroupName "TestGroup" -SubscriptionId 2120692d-35c3-44c8-81f5-631fa7351726
```
