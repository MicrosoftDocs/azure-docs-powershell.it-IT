# Get-AzTenantDeploymentOperation

## SYNOPSIS
Ottenere l'operazione di distribuzione per la distribuzione nell'ambito tenant

## SINTASSI

### GetByDeploymentName (impostazione predefinita)
```
Get-AzTenantDeploymentOperation -DeploymentName <String> [-OperationId <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByDeploymentObject
```
Get-AzTenantDeploymentOperation -DeploymentObject <PSDeployment> [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzTenantDeploymentOperation** elenca tutte le operazioni che erano parte della distribuzione nell'ambito tenant corrente per identificare e fornire altre informazioni sulle esatte operazioni non riuscite per una particolare distribuzione.

## ESEMPI

### Esempio 1: Ottenere le operazioni di distribuzione con un nome di distribuzione
```
PS C:\>Get-AzTenantDeploymentOperation -DeploymentName Deploy01
```

Ottiene le operazioni di distribuzione con il nome "Deploy01" nell'ambito tenant corrente.

### Esempio 2: Ottenere una distribuzione e ottenere le operazioni di distribuzione
```
PS C:\>Get-AzTenantDeployment -Name Deploy01 | Get-AzTenantDeploymentOperation
```

Questo comando recupera la distribuzione "Deploy01" nell'ambito tenant corrente e ottiene le operazioni di distribuzione.
