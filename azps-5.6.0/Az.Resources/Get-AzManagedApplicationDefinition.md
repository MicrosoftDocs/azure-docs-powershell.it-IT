# Get-AzManagedApplicationDefinition

## SYNOPSIS
Ottiene le definizioni delle applicazioni gestite

## SINTASSI

### GetByNameAndResourceGroup (impostazione predefinita)
```
Get-AzManagedApplicationDefinition [-Name <String>] -ResourceGroupName <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzManagedApplicationDefinition -Id <String> [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzManagedApplicationDefinition ottiene** le definizioni delle applicazioni gestite

## ESEMPI

### Esempio 1: Ottenere tutte le definizioni di applicazioni gestite in un gruppo di risorse
```
PS C:\>Get-AzManagedApplicationDefinition -ResourceGroupName "MyRG"
```

Questo comando recupera le definizioni delle applicazioni gestite nel gruppo di risorse "MyRG"

### Esempio 2: Ottenere la definizione di un'applicazione gestita
```
PS C:\>Get-AzManagedApplicationDefinition -ResourceGroupName "MyRG" -Name "myManagedAppDef"
```

Questo comando recupera la definizione di applicazione gestita "myManagedAppDef" nel gruppo di risorse "MyRG"
