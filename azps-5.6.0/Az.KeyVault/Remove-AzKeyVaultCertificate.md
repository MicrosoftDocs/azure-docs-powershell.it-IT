# Remove-AzKeyVaultCertificate

## SYNOPSIS
Rimuove un certificato da un vault delle chiavi.

## SINTASSI

### ByVaultNameAndName (impostazione predefinita)
```
Remove-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String> [-InRemovedState] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObject
```
Remove-AzKeyVaultCertificate [-InputObject] <PSKeyVaultCertificateIdentityItem> [-InRemovedState] [-Force]
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzKeyVaultCertificate** rimuove un certificato da un vault delle chiavi.

## ESEMPI

### Esempio 1: Rimuovere un certificato
```powershell
PS C:\> Remove-AzKeyVaultCertificate -VaultName "ContosoKV01" -Name "SelfSigned01" -PassThru -Force

SecretId           : https://contosokv01.vault.azure.net:443/secrets/selfsigned01/968c3920884a435abf8faea11f565456
Thumbprint         : XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
```

Questo comando rimuove il certificato denominato SelfSigned01 dalla chiave vault denominata ContosoKV01.
Questo comando specifica il *parametro Force.*
Di conseguenza, il cmdlet non richiede conferma.

### Esempio 2: Ripulire definitivamente il certificato eliminato dal vault delle chiavi
```powershell
PS C:\> Remove-AzKeyVaultCertificate -VaultName 'Contoso' -Name 'MyCert' -InRemovedState
```

Questo comando rimuove definitivamente il certificato denominato "MyCert" dalla chiave vault denominata "Contoso".
Per l'esecuzione di questo cmdlet Ã¨ necessaria l'autorizzazione di eliminazione, che deve essere stata concessa in precedenza ed esplicitamente all'utente in questo vault della chiave.
