# Remove-AzKeyVaultSecret

## SYNOPSIS
Elimina un segreto in un vault delle chiavi.

## SINTASSI

### ByVaultName (impostazione predefinita)
```
Remove-AzKeyVaultSecret [-VaultName] <String> [-Name] <String> [-Force] [-PassThru] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKeyVaultSecret [-InputObject] <PSKeyVaultSecretIdentityItem> [-Force] [-PassThru] [-InRemovedState]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il Remove-AzKeyVaultSecret cmdlet elimina un segreto in un vault delle chiavi.
Se il segreto è stato eliminato accidentalmente, il segreto può essere recuperato usando Undo-AzKeyVaultSecretRemoval da un utente con speciali autorizzazioni di "recupero".
Questo cmdlet ha un valore alto per la **proprietà ConfirmImpact.**

## ESEMPI

```powershell
Updated              : 4/26/2018 7:48:40 PM
Content Type         :

```

Questo comando rimuove il segreto denominato FinanceSecret dalla chiave vault denominata Contoso.'

### Esempio 2: Rimuovere un segreto da un vault delle chiavi senza conferma dell'utente
```powershell
PS C:\> Remove-AzKeyVaultSecret -VaultName 'Contoso' -Name 'FinanceSecret' -PassThru -Force

Vault Name           : Contoso
Name                 : FinanceSecret
Version              : f622abc7b1394092812f1eb0f85dc91c
Id                   : https://contoso.vault.azure.net:443/secrets/financesecret/f622abc7b1394092812f1eb0f85dc91c
Deleted Date         : 5/25/2018 4:45:34 PM
Scheduled Purge Date : 8/23/2018 4:45:34 PM
Enabled              : True
Expires              :
Not Before           :
Created              : 4/19/2018 5:56:02 PM
Updated              : 4/26/2018 7:48:40 PM
Content Type         :
Tags                 :
```

Questo comando rimuove il segreto denominato FinanceSecret dalla chiave vault denominata Contoso.
Il comando specifica i parametri *Force* e *Confirm* e, di conseguenza, il cmdlet non richiede conferma.

### Esempio 3: Ripulire il segreto eliminato dal vault delle chiavi in modo permanente
```powershell
PS C:\> Remove-AzKeyVaultSecret -VaultName 'Contoso' -Name 'FinanceSecret' -InRemovedState
```

Questo comando elimina definitivamente il segreto denominato FinanceSecret dalla chiave vault contoso.
Per l'esecuzione di questo cmdlet è necessaria l'autorizzazione di eliminazione, che deve essere stata concessa in precedenza ed esplicitamente all'utente per il vault della chiave.
