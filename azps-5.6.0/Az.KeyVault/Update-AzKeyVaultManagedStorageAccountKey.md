# Update-AzKeyVaultManagedStorageAccountKey

## SYNOPSIS
Rigenera la chiave specificata dell'account di archiviazione di Azure gestito da Vault delle chiavi.

## SINTASSI

### ByDefinitionName (Impostazione predefinita)
```
Update-AzKeyVaultManagedStorageAccountKey [-VaultName] <String> [-AccountName] <String> [-KeyName] <String>
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Update-AzKeyVaultManagedStorageAccountKey [-InputObject] <PSKeyVaultManagedStorageAccountIdentityItem>
 [-KeyName] <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Rigenera la chiave specificata dell'account di archiviazione di Azure gestito dal Vault delle chiavi e la imposta come chiave attiva. La chiave del Vault proxy la chiamata a Gestione risorse di Azure per rigenerare la chiave. Il chiamante deve avere le autorizzazioni per rigenerare le chiavi in un dato account di archiviazione di Azure.

## ESEMPI

### Esempio 1: Rigenerare una chiave
```powershell
PS C:\> Update-AzKeyVaultManagedStorageAccountKey -VaultName 'myvault' -AccountName 'mystorageaccount' -KeyName 'key1'
```

Rigenera 'key1' dell'account 'mystorageaccount' e imposta 'key1' come attivo dell'account di archiviazione di Azure gestito dal Vault delle chiavi.
