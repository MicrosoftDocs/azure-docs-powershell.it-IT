# Get-AzSqlDatabaseBackupLongTermRetentionPolicy

## SYNOPSIS
Ottiene i criteri di conservazione a lungo termine del database.

## SINTASSI

```
Get-AzSqlDatabaseBackupLongTermRetentionPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlDatabaseBackupLongTermRetentionPolicy** ottiene i criteri di conservazione a lungo termine registrati nel database.
Il criterio Ã¨ una risorsa di backup di Azure usata per definire i criteri di archiviazione di backup.

## ESEMPI

### Esempio 1: Ottenere la versione corrente dei criteri di conservazione a lungo termine
```powershell
PS C:\> Get-AzSqlDatabaseBackupLongTermRetentionPolicy -ResourceGroupName resourcegroup01 -ServerName server01 -DatabaseName database01


ResourceGroupName                      : resourcegroup01
ServerName                             : server01
DatabaseName                           : database01
WeeklyRetention                        : P2W
MonthlyRetention                       : PT0S
YearlyRetention                        : PT0S
WeekOfYear                             : 0
State                                  :
RecoveryServicesBackupPolicyResourceId :
Location                               :
```

Questo comando recupera la versione corrente dei criteri di conservazione a lungo termine per database01
