# Invoke-AzSqlInstanceFailover

## SYNOPSIS
Failover di un'istanza SQL azure.

## SINTASSI

```
Invoke-AzSqlInstanceFailover [-Name] <String> [-AsJob] [-PassThru] [-Force] [-ReadableSecondary]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Invoke-AzSqlInstanceFailover** failover un'istanza SQL Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01"
```

Questo comando failoverrà la replica principale dell'istanza denominata "ManagedInstance01".

### Esempio 2
```powershell
PS C:\> Invoke-AzSqlInstanceFailover -ResourceGroupName "ResourceGroup01" -Name "ManagedInstance01" -ReadableSecondary
```

Questo comando failoverrà la replica secondaria leggibile dell'istanza gestita "ManagedInstance01".
