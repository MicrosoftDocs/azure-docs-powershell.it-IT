# Invoke-AzSqlElasticPoolFailover

## SYNOPSIS
Failover di un pool flessibile.

## SINTASSI

```
Invoke-AzSqlElasticPoolFailover [-ElasticPoolName] <String> [-AsJob] [-PassThru] [-Force]
 [-ServerName] <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il Invoke-AzSqlElasticPoolFailover cmdlet failover di un pool flessibile. Il failover si verificherà in tutti i database nel pool elastico dopo l'esecuzione di questo cmdlet.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Invoke-AzSqlElasticPoolFailover -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Questo comando failoverrà il pool elastico denominato "ElasticPool01" nel server denominato "Server01".  Questo significa che il failover si verificherà in tutti i database nel pool elastico denominato "ElasticPool01".
