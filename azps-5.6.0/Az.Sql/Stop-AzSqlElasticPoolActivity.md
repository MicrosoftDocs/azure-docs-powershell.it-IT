# Stop-AzSqlElasticPoolActivity

## SYNOPSIS
Annulla l'operazione di aggiornamento asincrono in un pool elastico.

## SINTASSI

```
Stop-AzSqlElasticPoolActivity [-PassThru] [-ServerName] <String> [-ElasticPoolName] <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzSqlElasticPoolActivity** annulla l'operazione di aggiornamento asincrono in un pool flessibile.

## ESEMPI

### Esempio 1: Annullare l'operazione di aggiornamento asincrono in un pool elastico
```
PS C:\> Stop-AzSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -OperationId af97005d-9243-4f8a-844e-402d1cc855f5

ErrorCode       :

```

Questo comando annulla l'operazione di aggiornamento asincrono nel pool elastico.
