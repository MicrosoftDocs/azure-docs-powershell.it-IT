# Add-AzSqlServerKeyVaultKey

## SYNOPSIS
Aggiunge una chiave del Vault chiave a un SQL server.

## SINTASSI

```
Add-AzSqlServerKeyVaultKey [-KeyId] <String> [-AsJob] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Add-AzSqlServerKeyVaultKey aggiunge una chiave del Vault chiave al server SQL specificato.
Il server deve avere le autorizzazioni "get, wrapKey, unwrapKey" per il vault.

## ESEMPI

### Esempio 1: Aggiungere la chiave del Vault chiave
```
PS C:\> Add-AzSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Questo comando aggiunge la chiave del Vault chiave con ID ' ' al server SQL denominato 'ContosoServer' nel gruppo di risorse https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 'ContosoResourceGroup'.
