# Stop-AzSqlSyncGroupSync

## SYNOPSIS
Interrompe la sincronizzazione di un gruppo di sincronizzazione.

## SINTASSI

```
Stop-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzSqlSyncGroupSync** interrompe la sincronizzazione di un gruppo di sincronizzazione.

## ESEMPI

### Esempio 1
```
PS C:\> Stop-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Questo comando interrompe la sincronizzazione in corso per il gruppo di sincronizzazione mysg.
