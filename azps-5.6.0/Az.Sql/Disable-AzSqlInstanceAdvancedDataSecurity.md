# Disable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Disabilita Advanced Data Security in un'istanza gestita.

## SINTASSI

```
Disable-AzSqlInstanceAdvancedDataSecurity [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Disable-AzSqlInstanceAdvancedDataSecurity** disabilita Advanced Data Security per un'istanza gestita.

## ESEMPI

### Esempio 1 - Disabilitare Advanced Data Security per le istanze gestite
```powershell
PS C:\>  Disable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" `

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```

### Esempio 2 - Disabilitare La sicurezza avanzata dei dati del server dalla risorsa dell'istanza gestita
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Disable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : False
```
