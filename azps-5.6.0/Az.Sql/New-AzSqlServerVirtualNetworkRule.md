# New-AzSqlServerVirtualNetworkRule

## SYNOPSIS
Crea una regola di SQL Server virtuale di Azure. 

## SINTASSI

```
New-AzSqlServerVirtualNetworkRule -VirtualNetworkRuleName <String> -VirtualNetworkSubnetId <String>
 [-IgnoreMissingVnetServiceEndpoint] [-AsJob] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Crea una regola di SQL Server virtuale di Azure. Le regole di rete virtuale vengono usate per connettere il SQL Server di Azure a una rete virtuale specifica per limitare l'accesso al SQL Server di Azure in modo che sia disponibile solo all'interno della rete virtuale. 

## ESEMPI

### Esempio 1
```
PS C:\> $virtualNetworkRule = New-AzSqlServerVirtualNetworkRule -ResourceGroupName rg -ServerName serverName -VirtualNetworkRuleName virtualNetworkRuleName -VirtualNetworkSubnetId virtualNetworkSubnetId
```

Crea una regola di SQL Server virtuale di Azure
