# Remove-AzSqlServerDnsAlias

## SYNOPSIS
Rimuove l'alias DNS SQL Server Azure.

## SINTASSI

### Rimuovere un alias dns del server dai parametri di input del cmdlet
```
Remove-AzSqlServerDnsAlias -Name <String> -ServerName <String> [-ResourceGroupName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Rimuovere un alias dns del server dalla definizione di istanza di AzureSqlServerDnsAliasModel
```
Remove-AzSqlServerDnsAlias -InputObject <AzureSqlServerDnsAliasModel> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Rimuovere un alias dns del server da un ID di risorsa di Azure
```
Remove-AzSqlServerDnsAlias -ResourceId <String> [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Questo comando rimuove Azure SQL Server alias DNS dal server lasciando intatto il server.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzSqlServerDnsAlias -DnsAliasName aliasName -ServerName serverName -ResourceGroupName rg
```

Rimuove l'SQL Server DNS di Azure con il nome aliasName dal server con il nome serverName
