# Remove-AzSqlDatabaseFailoverGroup

## SYNOPSIS
Rimuove un gruppo di failover SQL database di Azure.

## SINTASSI

```
Remove-AzSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Questo comando rimuove il gruppo di failover con il nome specificato, lasciando inalterati tutti i database e le relazioni di replica. L'endpoint del listener verrà ne verrà annullata la registrazione dal DNS.
Il server primario del gruppo di failover deve essere usato per eseguire il comando.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Rimuovere un gruppo di failover.
