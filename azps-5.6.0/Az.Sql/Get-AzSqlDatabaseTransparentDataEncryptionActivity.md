# Get-AzSqlDatabaseTransparentDataEncryptionActivity

## SYNOPSIS
Recupera lo stato di un'analisi TDE di un database.

## SINTASSI

```
Get-AzSqlDatabaseTransparentDataEncryptionActivity [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlDatabaseTransparentDataEncryptionActivity** ottiene lo stato di avanzamento di un'analisi TDE (Transparent Data Encryption) di un database SQL Azure.
Se non è in esecuzione alcun intervallo di crittografia, questo cmdlet restituisce un elenco vuoto.
Per altre informazioni, vedere Transparent Data Encryption con Azure SQL Database https://msdn.microsoft.com/library/dn948096 ( nella Microsoft Developer Network https://msdn.microsoft.com/library/dn948096) Library.
Questo cmdlet è supportato anche dal servizio SQL Server di estensione del database in Azure.

## ESEMPI

### Esempio 1: Ottenere l'attività TDE per un database
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryptionActivity -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName : resourcegroup01
ServerName        : server01
DatabaseName      : database01
Status            : Encrypting
PercentComplete   : 3.662109
```

Questo comando recupera l'attività TDE per il database denominato database01 nel server denominato server01.
