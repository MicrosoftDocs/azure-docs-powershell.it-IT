# Remove-AzSqlSyncGroup

## SYNOPSIS
Rimuove un gruppo di sincronizzazione SQL database di Azure.

## SINTASSI

```
Remove-AzSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzSqlSyncGroup** rimuove un gruppo di SQL database di Azure.

## ESEMPI

### Esempio 1: Rimuovere un gruppo di sincronizzazione
```
PS C:\>Remove-AzSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

Questo comando rimuove il gruppo di SQL database di Azure denominato syncGroup01.
