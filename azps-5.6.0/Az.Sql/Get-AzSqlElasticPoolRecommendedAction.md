# Get-AzSqlElasticPoolRecommendedAction

## SYNOPSIS
Recupera una o più azioni consigliate per un consulente del pool SQL di Azure.

## SINTASSI

```
Get-AzSqlElasticPoolRecommendedAction [-RecommendedActionName <String>] -ServerName <String>
 -ElasticPoolName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlElasticPoolRecommendedAction** ottiene una o più azioni consigliate per un consulente del pool SQL elastico di Azure.
