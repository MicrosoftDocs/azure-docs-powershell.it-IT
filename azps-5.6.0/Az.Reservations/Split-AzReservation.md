# Split-AzReservation

## SYNOPSIS
Dividere un `Reservation` elemento .

## SINTASSI

### CommandLine (impostazione predefinita)
```
Split-AzReservation -ReservationOrderId <Guid> -ReservationId <Guid> -Quantity <Int32[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PipeObject
```
Split-AzReservation -Quantity <Int32[]> -Reservation <PSReservation> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Dividere una in `Reservation` due con la distribuzione di quantità `Reservation` specificata.

## ESEMPI

### Esempio 1
```
PS C:\> Split-AzReservation -ReservationOrderId "00000000-ffff-ffff-0000-00000fffff" -ReservationId "11111111-1111-1111-1111-1111111111" -Quantities 2,3
```

Dividere le specificate `Reservation` in due s con le quantità `Reservation` corrispondenti
