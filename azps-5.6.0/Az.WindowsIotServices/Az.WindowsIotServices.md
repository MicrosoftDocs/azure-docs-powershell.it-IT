# Modulo Az.WindowsIotServices
## Descrizione
Microsoft Azure PowerShell: cmdlet di WindowsIotServices

## Cmdlet Az.WindowsIotServices
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
Ottenere i metadati non correlati alla sicurezza di un servizio di dispositivi Windows IoT.

### [New-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Creare o aggiornare i metadati di un servizio dispositivi IoT di Windows.
Il modello più comune per modificare una proprietà è recuperare i metadati e i metadati di sicurezza di Windows IoT Device Service e quindi combinarli con i valori modificati in un nuovo corpo per aggiornare il servizio dispositivi Windows IoT.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Eliminare un servizio per dispositivi Windows IoT.

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Aggiorna i metadati di un servizio dispositivi Windows IoT.
Il modello più comune per modificare una proprietà è recuperare i metadati e i metadati di sicurezza di Windows IoT Device Service e quindi combinarli con i valori modificati in un nuovo corpo per aggiornare il servizio dispositivi Windows IoT.
