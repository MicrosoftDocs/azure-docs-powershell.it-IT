# Remove-AzDataLakeStoreAccount

## SYNOPSIS
Elimina definitivamente un account Data Lake Store.

## SINTASSI

```
Remove-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzDataLakeStoreAccount** elimina definitivamente un account Data Lake Store.

## ESEMPI

### Esempio 1: Rimuovere un account Data Lake Store
```
PS C:\>Remove-AzDataLakeStoreAccount -Name "ContosoADL"
```

Questo comando rimuove l'account denominato ContosoADL da Data Lake Store.
