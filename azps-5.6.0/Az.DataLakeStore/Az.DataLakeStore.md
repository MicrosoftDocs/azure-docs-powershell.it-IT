# Modulo Az.DataLakeStore
## Descrizione
Gli argomenti di questa sezione documentare i cmdlet di Azure PowerShell per Azure Data Lake Store nel framework di Gestione ARM Azure. I cmdlet sono presenti nello spazio dei nomi Microsoft.Azure.Commands.DataLakeStore. Questi cmdlet funzionano solo con gli account Azure Data Lake Storage Gen1.

## Cmdlet di Az.DataLakeStore
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Aggiunge una regola del firewall all'account Data Lake Store specificato.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Aggiunge contenuto a un elemento in un Data Lake Store.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Aggiunge un provider di identità attendibile all'account Data Lake Store specificato.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Aggiunge una regola di rete virtuale all'account Data Lake Store specificato.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Tenta di abilitare il Vault delle chiavi gestito dall'utente per la crittografia dell'account Data Lake Store specificato.

### [Export-AzDataLakeStoreItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Esporta le proprietà (Utilizzo disco e Acl) per l'intero albero dal percorso specificato in un percorso di output

### [Export-AzDataLakeStoreItem](Export-AzDataLakeStoreItem.md)
Scarica un file da Data Lake Store.

### [Get-AzDataLakeStoreAccount](Get-AzDataLakeStoreAccount.md)
Recupera i dettagli di un account Data Lake Store.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreChildItem.md)
Ottiene l'elenco degli elementi in una cartella di Data Lake Store.

### [Get-AzDataLakeStoreItemSummary](Get-AzDataLakeStoreChildItemSummary.md)
Recupera il riepilogo delle dimensioni totali, dei file e delle directory contenuti nel percorso specificato

### [Get-AzDataLakeStoreFirewallRule](Get-AzDataLakeStoreFirewallRule.md)
Ottiene le regole del firewall specificate nel Data Lake Store specificato.
Se non viene specificata alcuna regola del firewall, vengono elencate tutte le regole del firewall per l'account.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreItem.md)
Recupera i dettagli di un file o di una cartella in Data Lake Store.

### [Get-AzDataLakeStoreItemAclEntry](Get-AzDataLakeStoreItemAclEntry.md)
Ottiene una voce nell'elenco di controllo di accesso di un file o di una cartella in Data Lake Store.

### [Get-AzDataLakeStoreItemContent](Get-AzDataLakeStoreItemContent.md)
Recupera il contenuto di un file in Data Lake Store.

### [Get-AzDataLakeStoreItemOwner](Get-AzDataLakeStoreItemOwner.md)
Ottiene il proprietario di un file o di una cartella in Data Lake Store.

### [Get-AzDataLakeStoreItemPermission](Get-AzDataLakeStoreItemPermission.md)
Ottiene l'ottale di autorizzazione di un file o di una cartella in Data Lake Store.

### [Get-AzDataLakeStoreTrustedIdProvider](Get-AzDataLakeStoreTrustedIdProvider.md)
Ottiene il provider di identità attendibile specificato nel Data Lake Store specificato.
Se non viene specificato alcun provider, vengono elencati tutti i provider per l'account.

### [Get-AzDataLakeStoreVirtualNetworkRule](Get-AzDataLakeStoreVirtualNetworkRule.md)
Ottiene le regole di rete virtuale specificate nel Data Lake Store specificato.
Se non viene specificata alcuna regola di rete virtuale, vengono elencate tutte le regole di rete virtuale per l'account.

### [Import-AzDataLakeStoreItem](Import-AzDataLakeStoreItem.md)
Carica un file o una directory locale in Data Lake Store.

### [Join-AzDataLakeStoreItem](Join-AzDataLakeStoreItem.md)
Unisce uno o più file per crearne uno in Data Lake Store.

### [Move-AzDataLakeStoreItem](Move-AzDataLakeStoreItem.md)
Sposta o rinomina un file o una cartella in Data Lake Store.

### [New-AzDataLakeStoreAccount](New-AzDataLakeStoreAccount.md)
Crea un nuovo account Data Lake Store.

### [New-AzDataLakeStoreItem](New-AzDataLakeStoreItem.md)
Crea un nuovo file o una nuova cartella in Data Lake Store.

### [Remove-AzDataLakeStoreAccount](Remove-AzDataLakeStoreAccount.md)
Elimina definitivamente un account Data Lake Store.

### [Remove-AzDataLakeStoreFirewallRule](Remove-AzDataLakeStoreFirewallRule.md)
Rimuove la regola del firewall specificata nel Data Lake Store specificato.

### [Remove-AzDataLakeStoreItem](Remove-AzDataLakeStoreItem.md)
Elimina un file o una cartella in Data Lake Store.

### [Remove-AzDataLakeStoreItemAcl](Remove-AzDataLakeStoreItemAcl.md)
Cancella l'ACL di un file o di una cartella in Data Lake Store.

### [Remove-AzDataLakeStoreItemAclEntry](Remove-AzDataLakeStoreItemAclEntry.md)
Rimuove una voce dall'elenco di controllo di accesso di un file o di una cartella in Data Lake Store.

### [Remove-AzDataLakeStoreTrustedIdProvider](Remove-AzDataLakeStoreTrustedIdProvider.md)
Rimuove il provider di identità attendibile specificato nel Data Lake Store specificato.

### [Remove-AzDataLakeStoreVirtualNetworkRule](Remove-AzDataLakeStoreVirtualNetworkRule.md)
Rimuove la regola di rete virtuale specificata nel Data Lake Store specificato.

### [Set-AzDataLakeStoreAccount](Set-AzDataLakeStoreAccount.md)
Modifica un account Data Lake Store.

### [Set-AzDataLakeStoreFirewallRule](Set-AzDataLakeStoreFirewallRule.md)

### [Set-AzDataLakeStoreItemOwner](Set-AzDataLakeStoreItemOwner.md)
Modifica il proprietario di un file o di una cartella in Data Lake Store.

### [Set-AzDataLakeStoreItemPermission](Set-AzDataLakeStoreItemPermission.md)
Modifica l'ottale di un file o di una cartella in Data Lake Store.

### [Set-AzDataLakeStoreTrustedIdProvider](Set-AzDataLakeStoreTrustedIdProvider.md)
Modifica il provider di identità attendibile specificato nel Data Lake Store specificato.

### [Set-AzDataLakeStoreVirtualNetworkRule](Set-AzDataLakeStoreVirtualNetworkRule.md)
Modifica la regola di rete virtuale specificata nell'archivio dati specificato.

### [Test-AzDataLakeStoreAccount](Test-AzDataLakeStoreAccount.md)
Verifica la presenza di un account Data Lake Store.
