# Enable-AzDataLakeStoreKeyVault

## SYNOPSIS
Tenta di abilitare il Vault delle chiavi gestito dall'utente per la crittografia dell'account Data Lake Store specificato.

## SINTASSI

```
Enable-AzDataLakeStoreKeyVault [-Account] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Enable-AzDataLakeStoreKeyVault** tenta di abilitare un Vault delle chiavi gestito dall'utente per la crittografia dell'account Data Lake Store specificato.

## ESEMPI

### Esempio 1: Abilitare il Vault delle chiavi per l'account ContosoADLS
```
PS C:\>Enable-AzDataLakeStoreKeyVault -Name "ContosoADLS"
```

Questo comando tenta di abilitare il Vault delle chiavi gestito dall'utente per l'account Data Lake Store denominato ContosoADLS.
