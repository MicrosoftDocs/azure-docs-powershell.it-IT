# Set-AzDataLakeStoreFirewallRule

## SYNOPSIS
Modifica la regola del firewall specificata nel Data Lake Store specificato.

## SINTASSI

```
Set-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [[-StartIpAddress] <String>]
 [[-EndIpAddress] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzDataLakeStoreFirewallRule** modifica la regola del firewall specificata nel Data Lake Store specificato.

## ESEMPI

### Esempio 1: Aggiornare l'intervallo IP di inizio e di fine per una regola del firewall
```
PS C:\> Set-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Aggiorna la regola del firewall "MyFirewallRule" nell'account "ContosoADL" in modo che abbia un intervallo compreso tra 127.0.0.1 e 127.0.0.2
