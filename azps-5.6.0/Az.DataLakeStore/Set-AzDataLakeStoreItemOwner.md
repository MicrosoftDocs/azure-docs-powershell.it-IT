# Set-AzDataLakeStoreItemOwner

## SYNOPSIS
Modifica il proprietario di un file o di una cartella in Data Lake Store.

## SINTASSI

```
Set-AzDataLakeStoreItemOwner [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Type] <Owner>
 [-Id] <Guid> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzDataLakeStoreItemOwner** modifica il proprietario di un file o di una cartella in Data Lake Store.

## ESEMPI

### Esempio 1: Impostare il proprietario di un elemento
```
PS C:\>Set-AzDataLakeStoreItemOwner -AccountName "ContosoADL" -Path / -Type User -Id (Get-AzADUser -Mail "PattiFuller@contoso.com").ObjectId
```

Questo comando imposta il proprietario della directory radice su Patti Fuller.
