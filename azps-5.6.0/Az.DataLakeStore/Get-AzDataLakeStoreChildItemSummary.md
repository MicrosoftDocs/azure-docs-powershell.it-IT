# Get-AzDataLakeStoreChildItemSummary

## SYNOPSIS
Ottiene il riepilogo delle dimensioni totali, dei file e delle directory contenuti nel percorso specificato

## SINTASSI

```
Get-AzDataLakeStoreChildItemSummary [-Account] <String> [-Path] <DataLakeStorePathInstance>
 [-Concurrency <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
**Get-AzDataLakeStoreItemSummary** recupera il riepilogo del contenuto per un determinato percorso. Calcola in modo ricorsivo il numero totale di file, le directory e le dimensioni totali di tutti i file nel percorso specificato.

## ESEMPI

### Esempio 1: Ottenere il riepilogo del contenuto di una cartella
```
PS C:\> Get-AzDataLakeStoreChildItemSummary -Account ContosoADL -Path /a -Concurrency 128
```

Elenca il numero totale di directory, file e relative dimensioni contenuti in /a.
