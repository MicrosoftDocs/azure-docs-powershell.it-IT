# Test-AzDataLakeStoreAccount

## SYNOPSIS
Verifica la presenza di un account Data Lake Store.

## SINTASSI

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Test-AzDataLakeStoreAccount** verifica l'esistenza di un account Data Lake Store.

## ESEMPI

### Esempio 1: Testare un account
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Questo comando verifica l'esistenza dell'account denominato ContosoADL.
