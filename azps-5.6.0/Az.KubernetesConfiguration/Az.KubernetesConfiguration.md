# Modulo Az.KubernetesConfiguration
## Descrizione
Microsoft Azure PowerShell: Cmdlet KubernetesConfiguration

## Cmdlet Az.KubernetesConfiguration
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
Recupera i dettagli della configurazione del controllo del codice sorgente.

### [New-AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
Creare una nuova configurazione del controllo del controllo del codice sorgente di Kubernetes.

### [Remove-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
In questo modo verrà eliminato il file YAML usato per configurare la configurazione del controllo del codice sorgente, interrompendo così la futura sincronizzazione dal repo di origine.
