# Remove-AzSentinelBookmark

## SYNOPSIS
Eliminare un segnalibro.

## SINTASSI

### BookmarkId. (Impostazione predefinita)
```
Remove-AzSentinelBookmark -ResourceGroupName <String> -WorkspaceName <String> -BookmarkId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelBookmark -InputObject <PSSentinelBookmark> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzSentinelBookmark** elimina definitivamente un segnalibro da un'area di lavoro specificata.
È possibile passare un **oggetto Bookmark** usando l'operatore pipeline oppure specificare i parametri obbligatori.
È possibile usare il parametro Confirm $ConfirmPreference Windows PowerShell per controllare se il cmdlet richiede conferma.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzSentinelBookmark -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -BookmarkId "MyBookmarkId"
```

Questo comando rimuove il segnalibro dall'area di lavoro.
