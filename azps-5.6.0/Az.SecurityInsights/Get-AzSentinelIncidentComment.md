# Get-AzSentinelIncidentComment

## SYNOPSIS
Ottenere un commento per l'evento imprevisto.

## SINTASSI

### IncidentId (impostazione predefinita)
```
Get-AzSentinelIncidentComment -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IncidentCommentId
```
Get-AzSentinelIncidentComment -ResourceGroupName <String> -WorkspaceName <String> -IncidentId <String>
 -IncidentCommentId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSentinelIncidentComment -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzSentinelIncidentComment** ottiene un commento dell'evento dall'area di lavoro specificata.
Se si specificano *i parametri IncidentCommentId* e *IncidentId,* viene restituito **un singolo oggetto IncidentComment.**
Se non si specifica il parametro *IncidentCommentId,* verrÃ  restituita una matrice contenente tutti i commenti degli incidenti per l'evento imprevisto specificato nell'area di lavoro specificata.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $IncidentComments = Get-AzSentinelIncidentComment -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId"
```

Questo esempio recupera tutti i **incidentComments** per l'evento imprevisto specificato nell'area di lavoro specificata e quindi li archivia nella variabile $IncidentComments specificata.

### Esempio 2
```powershell
PS C:\> $IncidentComment = Get-AzSentinelIncidentComment -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -IncidentId "MyIncidentId" -IncidentCommentId "MyIncidentCommentId"
```

Questo esempio recupera un **commento per** l'evento imprevisto specificato nell'area di lavoro specificata e quindi lo archivia nella variabile $IncidentComment specificata.
