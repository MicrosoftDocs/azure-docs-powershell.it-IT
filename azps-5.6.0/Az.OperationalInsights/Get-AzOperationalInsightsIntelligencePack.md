# Get-AzOperationalInsightsIntelligencePack

## SYNOPSIS
Recupera i pacchetti di intelligence disponibili.

## SINTASSI

```
Get-AzOperationalInsightsIntelligencePack [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzOperationalInsightsIntelligencePack** ottiene i pacchetti di intelligence disponibili.

## ESEMPI

### Esempio 1: Ottenere i pacchetti di intelligence
```
PS C:\>Get-AzOperationalInsightsStorageInsight -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Questo comando recupera i pacchetti di intelligence disponibili.
