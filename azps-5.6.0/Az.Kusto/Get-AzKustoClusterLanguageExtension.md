# Get-AzKustoClusterLanguageExtension

## SYNOPSIS
Restituisce un elenco di estensioni delle lingue che possono essere eseguite all'interno di query KQL.

## SINTASSI

```
Get-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIZIONE
Restituisce un elenco di estensioni delle lingue che possono essere eseguite all'interno di query KQL.

## ESEMPI

### Esempio 1: Elencare tutte le estensioni delle lingue impostate per un cluster
```powershell
PS C:\> Get-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster

Name
----
R
PYTHON
```

Il comando precedente restituisce un elenco di estensioni delle lingue che possono essere eseguite all'interno di query KQL.
