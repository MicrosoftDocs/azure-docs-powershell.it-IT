# Remove-AzureStorageTable

## Sinossi
Rimuove una tabella di archiviazione.

## SINTASSI

```
Remove-AzureStorageTable [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureStorageTable** consente di rimuovere una o più tabelle di archiviazione da un account di archiviazione in Azure.

## ESEMPI

### Esempio 1: rimuovere una tabella
```
PS C:\>Remove-AzureStorageTable -Name "TableABC"
```

Questo comando rimuove una tabella.

### Esempio 2: rimuovere più tabelle
```
PS C:\>Get-AzureStorageTable table* | Remove-AzureStorageTable
```

Questo esempio usa un carattere jolly con il parametro *Name* per ottenere tutte le tabelle che corrispondono alla tabella pattern e quindi passa il risultato sulla pipeline per rimuovere le tabelle.
