# Stop-AzureStorageBlobCopy

## Sinossi
Interrompe un'operazione di copia.

## SINTASSI

### NamePipeline (impostazione predefinita)
```
Stop-AzureStorageBlobCopy [-Blob] <String> [-Container] <String> [-Force] [-CopyId <String>]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### BlobPipeline
```
Stop-AzureStorageBlobCopy -CloudBlob <CloudBlob> [-Force] [-CopyId <String>] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>] [-ConcurrentTaskCount <Int32>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ContainerPipeline
```
Stop-AzureStorageBlobCopy -CloudBlobContainer <CloudBlobContainer> [-Blob] <String> [-Force] [-CopyId <String>]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureStorageBlobCopy** interrompe un'operazione di copia nel BLOB di destinazione specificato.

## ESEMPI

### Esempio 1: interrompere l'operazione di copia per nome
```
PS C:\>Stop-AzureStorageBlobCopy -Container "ContainerName" -Blob "BlobName" -CopyId "CopyID"
```

Questo comando interrompe l'operazione di copia per nome.

### Esempio 2: interrompere l'operazione di copia usando la pipeline
```
PS C:\>Get-AzureStorageContainer container* | Stop-AzureStorageBlobCopy -Blob "BlobName"
```

Questo comando interrompe l'operazione di copia passando il contenitore sulla pipeline da **Get-AzureStorageContainer**.

### Esempio 3: interrompere l'operazione di copia usando la pipeline e Get-AzureStorageBlob
```
PS C:\>Get-AzureStorageBlob -Container "ContainerName" | Stop-AzureStorageBlobCopy -Force
```

Questo esempio interrompe l'operazione di copia passando il contenitore sulla pipeline dal cmdlet Get-AzureStorageBlob.
