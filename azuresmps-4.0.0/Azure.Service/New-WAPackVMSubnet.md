# New-WAPackVMSubnet

## Sinossi
Crea una subnet della macchina virtuale.

## SINTASSI

```
New-WAPackVMSubnet -VNet <VMNetwork> -Name <String> -Subnet <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-WAPackVMSubnet** crea una subnet macchina virtuale.

## ESEMPI

### Esempio 1: creare una subnet macchina virtuale
```
PS C:\> $VNet = Get-WAPackVNet -Name "ContosoVNet01"
PS C:\> New-WAPackVMSubnet -VNet $VNet -Name "ContosoVMSubnet01" -Subnet "192.168.1.0/24"
```

Il primo comando recupera prima di tutto la rete della macchina virtuale a cui si vuole aggiungere una nuova subnet della macchina virtuale.
Questa rete di macchine virtuali Ã¨ denominata ContosoVNet01.

Il secondo comando crea una subnet macchina virtuale usando la rete Virtual Machine di recupero in precedenza, un nome ContosoVMSubnet01 e una subnet 192.168.1.0/24.
