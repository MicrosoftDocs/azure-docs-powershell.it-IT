# Get-AzureWebsiteLocation

## Sinossi
Ottiene tutte le posizioni disponibili per il sito Web.

## SINTASSI

```
Get-AzureWebsiteLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Questo argomento descrive il cmdlet nella versione 0.8.10 del modulo PowerShell di Microsoft Azure.
Per ottenere la versione del modulo che si sta usando, nella console di PowerShell di Azure digitare `(Get-Module -Name Azure).Version` .

Ottiene tutti i percorsi del sito Web disponibili per l'abbonamento corrente

## ESEMPI

### Esempio 1: ottenere tutte le posizioni disponibili
```
PS C:\> Get-AzureWebsiteLocations
```

Questo esempio consente di ottenere tutti i percorsi del sito Web disponibili per l'abbonamento corrente.
