# Remove-AzureDeployment

## Sinossi
Elimina una distribuzione di un servizio cloud.

## SINTASSI

```
Remove-AzureDeployment [-ServiceName] <String> [-Slot] <String> [-DeleteVHD] [-Force]
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureDeployment** Elimina una distribuzione di un servizio cloud di Azure.
Per eliminare una distribuzione, sospenderla prima di tutto.

## ESEMPI

### Esempio 1: rimuovere una distribuzione
```
PS C:\> Remove-AzureDeployment -ServiceName "ContosoService"
```

Questo comando rimuove la distribuzione del servizio Azure denominato ContosoService.
Poiché questo comando non specifica uno slot, rimuove il servizio dall'ambiente di produzione.

### Esempio 2: rimuovere una distribuzione e dischi rigidi virtuali
```
PS C:\> Remove-AzureDeployment -ServiceName "ContosoService" -DeleteVHD
```

Questo comando rimuove la distribuzione del servizio denominato ContosoService dall'ambiente di produzione.
Il comando rimuove anche i dischi rigidi virtuali sottostanti.
