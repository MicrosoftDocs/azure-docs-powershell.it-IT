# Remove-AzureMediaServicesAccount

## Sinossi
Rimuove l'account di servizi di Azure Media specificato.

**Nota:**  Questa versione è deprecata, vedere la [versione più recente](https://docs.microsoft.com/powershell/module/azurerm.media/?view=azurermps-5.4.0#media_services).

## SINTASSI

```
Remove-AzureMediaServicesAccount -Name <String> [-Force] [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Questo argomento descrive il cmdlet nella versione 0.8.10 del modulo PowerShell di Microsoft Azure.
Per ottenere la versione del modulo che si sta usando, nella console di PowerShell di Azure digitare `(Get-Module -Name Azure).Version` .

Il cmdlet **Remove-AzureMediaServicesAccount** rimuove un account di servizi multimediali.

## ESEMPI

### Esempio 1: eliminare un account di servizi multimediali
```
PS C:\> Remove-AzureMediaServicesAccount -Name "mediaservicesaccount" -Force
```
