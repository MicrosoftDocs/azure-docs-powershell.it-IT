# Switch-AzureWebsiteSlot

## Sinossi
Scambia lo slot di produzione per un sito Web con un altro slot.

## SINTASSI

```
Switch-AzureWebsiteSlot [-Name <String>] [-Slot1 <String>] [-Slot2 <String>] [-Force]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Switch-AzureWebsiteSlot** scambia lo slot di produzione per un sito Web con un altro slot.
Questo funziona con i siti Web con due solo slot.

## ESEMPI

### Esempio 1: cambiare lo slot del sito Web
```
PS C:\> Switch-AzureWebsiteSlot -Name MyWebsite
```

Cambiare lo slot di backup del sito Web di Azure con lo slot di produzione.
