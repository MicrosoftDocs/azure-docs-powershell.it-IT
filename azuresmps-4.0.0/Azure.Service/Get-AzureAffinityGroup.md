# Get-AzureAffinityGroup

## Sinossi
Ottiene un oggetto Group Affinity di Azure.

## SINTASSI

```
Get-AzureAffinityGroup [[-Name] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureAffinityGroup** ottiene un gruppo di affinità Azure.
L'oggetto gruppo affinità include il nome del gruppo di affinità, la posizione, l'etichetta, la descrizione e i servizi di archiviazione e ospitati che fanno parte del gruppo affinità.

## ESEMPI

### Esempio 1: ottenere le proprietà di un gruppo di affinità
```
PS C:\> Get-AzureAffinityGroup -Name "South01"
```

Questo comando consente di ottenere le proprietà del gruppo affinità denominato South01.
