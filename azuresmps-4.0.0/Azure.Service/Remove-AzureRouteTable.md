# Remove-AzureRouteTable

## Sinossi
Rimuove una tabella di route.

## SINTASSI

```
Remove-AzureRouteTable -Name <String> [-PassThru] [-Force] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRouteTable** rimuove una tabella di route dall'abbonamento.
Se una tabella di route è associata a una subnet, non è possibile rimuovere la tabella.

## ESEMPI

### Esempio 1: rimuovere una tabella di route
```
PS C:\> Remove-AzureRouteTable -Name "PublicRouteTable"
```

Questo comando rimuove la tabella di route denominata PublicRouteTable.
