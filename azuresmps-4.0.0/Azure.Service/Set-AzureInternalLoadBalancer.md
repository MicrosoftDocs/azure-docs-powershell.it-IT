# Set-AzureInternalLoadBalancer

## Sinossi
Modifica una configurazione di bilanciamento del carico interna in un servizio Azure.

## SINTASSI

### ServiceAndSlot (impostazione predefinita)
```
Set-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### SubnetNameOnly
```
Set-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-SubnetName] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

### SubnetNameAndIP
```
Set-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-SubnetName] <String> [-StaticVNetIPAddress] <IPAddress> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureInternalLoadBalancer** modifica una configurazione di bilanciamento del carico interna in un servizio Azure.
Per una rete virtuale, Ã¨ possibile specificare una subnet o l'indirizzo IP del sistema di bilanciamento del carico interno.
