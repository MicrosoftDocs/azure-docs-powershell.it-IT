# Unpublish-AzureRemoteAppProgram

## Sinossi
Annulla la pubblicazione di un programma RemoteApp di Azure.

## SINTASSI

```
Unpublish-AzureRemoteAppProgram [-CollectionName] <String> [[-Alias] <String[]>] [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Unpublish-AzureRemoteAppProgram** Annulla la pubblicazione di un programma RemoteApp di Azure.
Dopo avere annullato la pubblicazione di un programma, non sarà più disponibile per gli utenti di una raccolta RemoteApp di Azure.
