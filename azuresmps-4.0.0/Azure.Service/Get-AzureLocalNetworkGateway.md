# Get-AzureLocalNetworkGateway

## Sinossi
Ottiene un gateway di rete locale.

## SINTASSI

```
Get-AzureLocalNetworkGateway [-GatewayId <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureLocalNetworkGateway** ottiene un gateway di rete locale.
