# Disable-AzureWebsiteApplicationDiagnostic

## Sinossi
Disabilita la diagnostica delle applicazioni per un sito Web di Azure.

## SINTASSI

```
Disable-AzureWebsiteApplicationDiagnostic [-PassThru] [-File] [-TableStorage] [-BlobStorage] [-Name <String>]
 [-Slot <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Questo argomento descrive il cmdlet nella versione 0.8.10 del modulo PowerShell di Microsoft Azure.
Per ottenere la versione del modulo che si sta usando, nella console di PowerShell di Azure digitare `(Get-Module -Name Azure).Version` .

Disabilita la diagnostica delle applicazioni per un sito Web di Azure.
Disabilita la registrazione configurata per l'archiviazione in un file System o in Azure.

## ESEMPI

### Esempio 1: disabilitare il file di registrazione dell'applicazione
```
PS C:\> Disable-AzureWebsiteApplicationDiagnostic -Name MyWebsite -File
```

L'esempio seguente disabilita la registrazione delle applicazioni nel file System.

### Esempio 2: disabilitare la registrazione con l'archiviazione di Azure
```
PS C:\> Disable-AzureWebsiteApplicationDiagnostic -Name MyWebsite -Storage
```

L'esempio seguente disabilita la registrazione delle applicazioni tramite lo spazio di archiviazione.
