# Remove-AzureLocalNetworkGateway

## Sinossi
Rimuove un gateway di rete locale di Azure.

## SINTASSI

```
Remove-AzureLocalNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureLocalNetworkGateway** rimuove un gateway di rete locale di Azure.
