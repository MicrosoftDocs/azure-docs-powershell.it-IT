# Get-AzureVNetSite

## Sinossi
Ottiene un oggetto elenco con informazioni sulle reti virtuali di Azure.

## SINTASSI

```
Get-AzureVNetSite [[-VNetName] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVNetSite** ottiene un oggetto elenco con informazioni sulle reti virtuali Azure per l'abbonamento corrente.
Se si specifica un nome di rete virtuale, vengono restituite solo le informazioni relative alla rete virtuale.

## ESEMPI

### Esempio 1: ottenere informazioni su tutte le reti virtuali nell'abbonamento corrente
```
PS C:\> Get-AzureVNetSite
```

Questo comando consente di ottenere informazioni su tutte le reti virtuali nell'abbonamento corrente.

### Esempio 2: ottenere informazioni su una rete virtuale specifica nell'abbonamento corrente
```
PS C:\> Get-AzureVNetSite -VNetName "MyProductionNetwork"
```

Questo comando recupera solo le informazioni sulla rete virtuale MyProductionNetwork.
