# Stop-AzureWebsiteJob

## Sinossi
Arresta un processo Web per un sito Web.

## SINTASSI

```
Stop-AzureWebsiteJob -JobName <String> [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureWebsiteJob** arresta un processo Web per un sito Web.

## ESEMPI

### Esempio 1: arrestare un processo Web per un sito Web
```
PS C:\> Stop-AzureWebsiteJob -Name MyWebsite -JobName MyWebJob
```

Interrompe un processo Web denominato MyWebJob per sito Web.
