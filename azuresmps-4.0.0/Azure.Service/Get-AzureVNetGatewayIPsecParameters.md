# Get-AzureVNetGatewayIPsecParameters

## Sinossi
Ottiene i parametri IPsec per la connessione tra un gateway di rete virtuale e un sito di rete locale.

## SINTASSI

```
Get-AzureVNetGatewayIPsecParameters -VNetName <String> -LocalNetworkSiteName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVNetGatewayIPsecParameters** restituisce i parametri IPSec (Internet Protocol Security) e Internet Key Exchange (IKE) correnti per la connessione tra un gateway di rete virtuale e un sito di rete locale.
