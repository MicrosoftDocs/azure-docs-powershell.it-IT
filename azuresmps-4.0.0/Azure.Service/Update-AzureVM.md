# Update-AzureVM

## Sinossi
Modifica la configurazione di una macchina virtuale di Azure.

## SINTASSI

```
Update-AzureVM [-Name] <String> -VM <PersistentVM> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Update-AzureVM** accetta le informazioni sull'aggiornamento per la macchina virtuale specificata e avvia l'aggiornamento.
È possibile aggiungere o rimuovere dischi di dati, modificare la modalità cache di dischi di dati o sistemi operativi, modificare gli endpoint di rete o modificare le dimensioni della macchina virtuale.

## ESEMPI

### Esempio 1: aggiornare le dimensioni di una macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "ContosoService03" -Name "VirtualMachine04" | Set-AzureVMSize -InstanceSize "Medium" | Update-AzureVM
```

Questo comando modifica le dimensioni della macchina virtuale denominata VirtualMachine04, in corso nel servizio denominato ContosoService03, su media.

### Esempio 2: aggiungere un disco di dati a una macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "ContosoService03" -Name "VirtualMachine05" | Add-AzureDataDisk -CreateNew -MediaLocation "https://ContosoStore1.blob.core.azure.com/vhds/Disk22.vhd" -DiskSizeInGB 128 -DiskLabel "Data-128" -LUN 0 | Update-AzureVM
```

Questo comando aggiunge un nuovo disco dati alla macchina virtuale denominata VirtualMachine05, in corso nel servizio denominato ContosoService03.
