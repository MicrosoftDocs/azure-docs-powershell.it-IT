# Get-AzureVMImageDiskConfigSet

## Sinossi
Ottiene un oggetto set di configurazione del disco dal contesto dell'immagine di input.

## SINTASSI

```
Get-AzureVMImageDiskConfigSet [[-ImageContext] <OSImageContext>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVMImageDiskConfigSet** ottiene un oggetto set di configurazione del disco dal contesto dell'immagine di input.

## ESEMPI

### Esempio 1: ottenere un oggetto set configurazione disco da una macchina virtuale
```
PS C:\> Get-AzureVMImage -ImageName $Img | Get-AzureVMImageDiskConfigSet
```

Questo comando ottiene un oggetto set di configurazione disco da una macchina virtuale.
