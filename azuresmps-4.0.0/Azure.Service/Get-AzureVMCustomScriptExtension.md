# Get-AzureVMCustomScriptExtension

## Sinossi
Ottiene le informazioni da un'estensione di script personalizzata di Azure Virtual Machine.

## SINTASSI

```
Get-AzureVMCustomScriptExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVMCustomScriptExtension** ottiene le informazioni da un'estensione di script personalizzata di Azure Virtual Machine.

## ESEMPI

### Esempio 1: ottenere un'estensione di script di una macchina virtuale Azure
```
PS C:\> Get-AzureVMCustomScriptExtension -VM $VM;
```

Questo comando consente di ottenere un'estensione di script di una macchina virtuale Azure archiviata nella variabile $VM.
