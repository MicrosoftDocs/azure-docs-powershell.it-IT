# Get-AzureInternalLoadBalancer

## Sinossi
Ottiene i dettagli della configurazione interna del bilanciamento del carico.

## SINTASSI

```
Get-AzureInternalLoadBalancer [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureInternalLoadBalancer** ottiene i dettagli della configurazione di bilanciamento del carico interna per un servizio Azure.

## ESEMPI

### Esempio 1: ottenere informazioni dettagliate per un bilanciamento del carico interno
```
PS C:\> Get-AzureService -ServiceName "ContosoService" | Get-AzureInternalLoadBalancer
```

Questo comando ottiene il servizio denominato ContosoService usando il cmdlet **Get-AzureService** .
Il comando passa il servizio al cmdlet corrente usando l'operatore pipeline.
Il cmdlet corrente ottiene i dettagli per il servizio di bilanciamento del carico interno per tale Service.
