# Reset-AzureVirtualNetworkGateway

## Sinossi
Reimposta un gateway di rete virtuale.

## SINTASSI

```
Reset-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Reset-AzureVirtualNetworkGateway** Reimposta un gateway di rete virtuale.
