# Remove-AzureApplicationGateway

## Sinossi
Rimuove un gateway dell'applicazione.

## SINTASSI

```
Remove-AzureApplicationGateway -Name <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureApplicationGateway** rimuove un gateway dell'applicazione.

## ESEMPI

### Esempio 1: rimuovere un gateway applicazione
```
PS C:\> Remove-AzureApplicationGateway -Name "ApplicationGateway06"
```

Questo comando rimuove il gateway dell'applicazione denominato ApplicationGateway06.
