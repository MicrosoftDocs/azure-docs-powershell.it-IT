# Get-AzureVNetGatewayKey

## Sinossi
Ottiene la chiave già condivisa per la connessione tra un gateway di rete virtuale e un sito di rete locale.

## SINTASSI

```
Get-AzureVNetGatewayKey -VNetName <String> -LocalNetworkSiteName <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVNetGatewayKey** ottiene la chiave già condivisa per la connessione tra un gateway di rete virtuale e un sito di rete locale.
