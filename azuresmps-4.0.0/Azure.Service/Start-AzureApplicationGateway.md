# Start-AzureApplicationGateway

## Sinossi
Avvia un gateway dell'applicazione.

## SINTASSI

```
Start-AzureApplicationGateway -Name <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Start-AzureApplicationGateway** avvia un gateway dell'applicazione.

## ESEMPI

### Esempio 1: avviare un gateway applicazione
```
PS C:\> Start-AzureApplicationGateway -Name "ApplicationGateway06"
```

Questo comando avvia il gateway dell'applicazione denominato ApplicationGateway06.
