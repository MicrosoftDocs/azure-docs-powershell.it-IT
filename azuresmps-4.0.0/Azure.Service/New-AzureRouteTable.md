# New-AzureRouteTable

## Sinossi
Crea una tabella di route.

## SINTASSI

```
New-AzureRouteTable -Name <String> -Location <String> [-Label <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRouteTable** crea una tabella di route in una posizione specificata.
È possibile aggiungere route definite dall'utente alla tabella di route.
È possibile associare la tabella di route a una subnet in una rete virtuale.
