# Get-AzureVMSqlServerExtension

## Sinossi
Ottiene le impostazioni dell'agente IaaS di SQL Server in una particolare macchina virtuale.

## SINTASSI

```
Get-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVMSqlServerExtension** ottiene le impostazioni dell'agente di SQL Server come servizio (IaaS) in una particolare macchina virtuale.

## ESEMPI

### Esempio 1: ottenere le impostazioni di un'estensione di SQL Server in una macchina virtuale
```
PS C:\> Get-AzureVMSqlServerExtension-VM $VM
          ExtensionName        : SqlIaaSAgent
          Publisher            : Microsoft.SqlServer.Management
          Version              : 1.0
          State                : Enable
          RoleName             : VMName
          AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
          AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Ottiene le impostazioni dell'estensione di SQL Server in una particolare macchina virtuale.

### Esempio 2: ottenere le impostazioni di un agente di IaaS di SQL Server in una macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "Service" -Name "VMName" | Get-AzureVMSqlServerExtension
          ExtensionName        : SqlIaaSAgent
          Publisher            : Microsoft.SqlServer.Management
          Version              : 1.0
          State                : Enable
          RoleName             : VMName
          AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
          AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Ottiene le impostazioni dell'agente IaaS di SQL Server in una particolare macchina virtuale usando l'input tramite pipe.

### Esempio 3: ottenere le impostazioni di un agente IaaS della versione specifica di SQL Server in una macchina virtuale
```
PS C:\> Get-AzureVMSqlServerExtension -VM $VM -Version "1.0"
          ExtensionName        : SqlIaaSAgent
          Publisher            : Microsoft.SqlServer.Management
          Version              : 1.0
          State                : Enable
          RoleName             : VMName
          AutoPatchingSettings : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoPatchingSettings
          AutoBackupSettings   : Microsoft.WindowsAzure.Commands.ServiceManagement.IaaS.Extensions.AutoBackupSettings
```

Questo comando consente di ottenere le impostazioni della versione specifica di agente IaaS di SQL Server in una macchina virtuale.
