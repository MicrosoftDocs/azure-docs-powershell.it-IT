# Remove-AzureVirtualNetworkGateway

## Sinossi
Rimuove un gateway di rete virtuale.

## SINTASSI

```
Remove-AzureVirtualNetworkGateway -GatewayId <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureVirtualNetworkGateway** rimuove un gateway di rete virtuale.
