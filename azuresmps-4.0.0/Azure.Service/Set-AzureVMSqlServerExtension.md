# Set-AzureVMSqlServerExtension

## Sinossi
Imposta l'estensione di Azure SQL Server in una macchina virtuale.

## SINTASSI

### EnableSqlServerExtension (impostazione predefinita)
```
Set-AzureVMSqlServerExtension [[-ReferenceName] <String>] [[-Version] <String>]
 [[-AutoPatchingSettings] <AutoPatchingSettings>] [[-AutoBackupSettings] <AutoBackupSettings>]
 [[-KeyVaultCredentialSettings] <KeyVaultCredentialSettings>] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

### DisableSqlServerExtension
```
Set-AzureVMSqlServerExtension [[-ReferenceName] <String>] [[-Version] <String>] [-Disable] -VM <IPersistentVM>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### UninstallSqlServerExtension
```
Set-AzureVMSqlServerExtension [[-ReferenceName] <String>] [[-Version] <String>] [-Uninstall]
 -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureVMSqlServerExtension** imposta l'estensione di Azure SQL Server in una macchina virtuale.

## ESEMPI

### Esempio 1: impostare le impostazioni di correzione automatica in una macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "ServiceName" -Name "VMName" | Set-AzureVMSqlServerExtension -AutoPatchingSettings $APS | Update-AzureVM
```

Questo comando imposta le impostazioni di correzione automatica in una macchina virtuale di Azure.

### Esempio 2: impostare le impostazioni di backup automatico in una macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "ServiceName" -Name "VMName" | Set-AzureVMSqlServerExtension -AutoBackupSettings $ABS | Update-AzureVM
```

Questo comando imposta le impostazioni di backup automatico in Azure Virtual Machine.

### Esempio 3: disabilitare un'estensione di SQL Server in una macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "Service" -Name "VMName" | Set-AzureVMSqlServerExtension -Disable
```

Questo comando Disabilita l'estensione della macchina virtuale di SQL Server in una determinata macchina virtuale.

### Esempio 4: disinstallare un'estensione di SQL Server in una specifica macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "Service" -Name "VMName" | Set-AzureVMSqlServerExtension -Uninstall
```

Questo comando disinstalla un'estensione della macchina virtuale di SQL Server nella macchina virtuale denominata VMName.
