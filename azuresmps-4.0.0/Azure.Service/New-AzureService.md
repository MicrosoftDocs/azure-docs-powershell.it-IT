# New-AzureService

## Sinossi
Crea un servizio Azure.

## SINTASSI

### ParameterSetAffinityGroup (impostazione predefinita)
```
New-AzureService [-ServiceName] <String> [-AffinityGroup] <String> [[-Label] <String>]
 [[-Description] <String>] [[-ReverseDnsFqdn] <String>] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

### ParameterSetLocation
```
New-AzureService [-ServiceName] <String> [-Location] <String> [[-Label] <String>] [[-Description] <String>]
 [[-ReverseDnsFqdn] <String>] [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureService** crea un nuovo servizio Azure nell'abbonamento corrente.

## ESEMPI

### Esempio 1: creare un servizio
```
PS C:\> New-AzureService -ServiceName "MySvc01" -Label "MyTestService" -Location "South Central US"
```

Questo comando crea un nuovo servizio denominato MySvc01 nella posizione centro sud degli Stati Uniti.

### Esempio 2: creare un servizio in un gruppo di affinità
```
PS C:\> New-AzureService -ServiceName "MySvc01" -AffinityGroup NorthRegion
```

Questo comando crea un nuovo servizio denominato MySvc01 usando il gruppo di affinità NorthRegion.
