# Get-AzureNetworkSecurityGroupConfig

## Sinossi
Ottiene i dettagli per un gruppo di sicurezza di rete.

## SINTASSI

```
Get-AzureNetworkSecurityGroupConfig [-Detailed] -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureNetworkSecurityGroupConfig** restituisce i dettagli per un gruppo di sicurezza di rete.
Specificare il parametro *dettagliato* per visualizzare le regole di sicurezza della rete.
