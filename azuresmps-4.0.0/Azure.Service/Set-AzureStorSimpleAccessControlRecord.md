# Set-AzureStorSimpleAccessControlRecord

## Sinossi
Aggiorna la IQN di un record di controllo di Access.

## SINTASSI

```
Set-AzureStorSimpleAccessControlRecord -ACRName <String> -IQNInitiatorName <String> [-WaitForComplete]
 [-NewName <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureStorSimpleAccessControlRecord** aggiorna il nome completo iSCSI (IQN) di un record di controllo di Access esistente.

## ESEMPI
