# Get-AzureVirtualNetworkGatewayKey

## Sinossi
Ottiene la chiave per un gateway di rete virtuale di Azure.

## SINTASSI

```
Get-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVirtualNetworkGatewayKey** ottiene la chiave per un gateway di rete virtuale di Azure.
