# Restart-AzureWebsite

## Sinossi
Si arresta e quindi riavvia il sito Web specificato.

## SINTASSI

```
Restart-AzureWebsite [-PassThru] [-Name <String>] [-Slot <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Questo argomento descrive il cmdlet nella versione 0.8.10 del modulo PowerShell di Microsoft Azure.
Per ottenere la versione del modulo che si sta usando, nella console di PowerShell di Azure digitare `(Get-Module -Name Azure).Version` .

Il cmdlet **Restart-AzureWebsite** si arresta e quindi riavvia il sito Web specificato.

## ESEMPI

### Esempio 1: riavviare un sito Web
```
PS C:\> Restart-AzureWebsite -Name MyWebsite
```

In questo esempio viene riavviato un sito Web denominato sito Web.
