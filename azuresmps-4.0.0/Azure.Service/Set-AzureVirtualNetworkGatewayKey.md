# Set-AzureVirtualNetworkGatewayKey

## Sinossi
Imposta la chiave per un gateway di rete virtuale di Azure.

## SINTASSI

```
Set-AzureVirtualNetworkGatewayKey -GatewayId <String> -ConnectedEntityId <String> -SharedKey <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureVirtualNetworkGatewayKey** imposta la chiave per un gateway di rete virtuale di Azure.
