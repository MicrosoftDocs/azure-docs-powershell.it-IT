# Get-AzureSubnet

## Sinossi
Ottiene un elenco di subnet associate alla macchina virtuale di Azure specificata.

## SINTASSI

```
Get-AzureSubnet -VM <IPersistentVM> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureSubnet** restituisce un elenco delle subnet associate alla macchina virtuale specificata.
USA **Get-AzureVM** per specificare la macchina virtuale.

## ESEMPI

### Esempio 1: ottenere subnet per una macchina virtuale
```
PS C:\> $VM = Get-AzureVM -ServiceName "ContosoService03" -Name "VirtualMachine01"
C:\PS> Get-AzureSubnet -VM $VM
```

Il primo comando consente di ottenere una macchina virtuale denominata VirtualMachine01 nel servizio denominato ContosoService03 e quindi di archiviarla nella variabile $VM.

Il secondo comando consente di ottenere le subnet di Azure per la macchina virtuale in $VM.
