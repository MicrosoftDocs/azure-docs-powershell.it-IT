# Add-AzureInternalLoadBalancer

## Sinossi
Aggiunge un bilanciamento del carico interno a un servizio Azure.

## SINTASSI

### ServiceAndSlot (impostazione predefinita)
```
Add-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

### SubnetNameOnly
```
Add-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-SubnetName] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

### SubnetNameAndIP
```
Add-AzureInternalLoadBalancer [-InternalLoadBalancerName] <String> [-ServiceName] <String>
 [-SubnetName] <String> [-StaticVNetIPAddress] <IPAddress> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzureInternalLoadBalancer** aggiunge una configurazione di bilanciamento del carico interna a un servizio Azure.
Per una rete virtuale, Ã¨ possibile specificare una subnet o l'indirizzo IP del sistema di bilanciamento del carico interno.

## ESEMPI

### Esempio 1: aggiungere un bilanciamento del carico interno
```
PS C:\> Add-AzureInternalLoadBalancer -ServiceName "ContosoWebsite01" -InternalLoadBalancerName "ContosoILB"
```

Questo comando aggiunge un bilanciamento del carico interno denominato ContosoILB al servizio denominato ContosoWebsite01.

### Esempio 2: aggiungere un bilanciamento del carico interno per una subnet specificata
```
PS C:\> Add-AzureInternalLoadBalancer -ServiceName "ContosoWebsite01" -InternalLoadBalancerName "ContosoILB" -SubnetName "FrontEndSubnet"
```

Questo comando aggiunge un bilanciamento del carico interno denominato ContosoILB al servizio denominato ContosoWebsite01.
Il comando specifica la subnet denominata FrontEndSubnet.

### Esempio 3: aggiungere un bilanciamento del carico interno per una subnet e un indirizzo specificati
```
PS C:\> Add-AzureInternalLoadBalancer -ServiceName "ContosoWebsite01" -InternalLoadBalancerName "ContosoILB" -SubnetName "FrontEndSubnet" -StaticVNetIPAddress 192.168.4.7
```

Questo comando aggiunge un bilanciamento del carico interno denominato ContosoILB al servizio denominato ContosoWebsite01.
Il comando specifica la subnet denominata FrontEndSubnet e l'indirizzo statico della rete virtuale.
