# Get-AzureVMDiagnosticsExtension

## Sinossi
Ottiene le impostazioni dell'estensione di diagnostica di Azure in una macchina virtuale.

## SINTASSI

```
Get-AzureVMDiagnosticsExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVMDiagnosticsExtension** ottiene le impostazioni di Microsoft Azure Diagnostics Extension in una macchina virtuale.

## ESEMPI

### Esempio 1: ottenere l'estensione di diagnostica applicata a una macchina virtuale
```
PS C:\> Get-AzureVMDiagnosticsExtension -VM $VM
```

Questo comando consente di ottenere l'estensione di diagnostica applicata alla macchina virtuale specificata come archiviato nella variabile $VM.
