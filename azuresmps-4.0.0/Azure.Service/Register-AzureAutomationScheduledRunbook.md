# Register-AzureAutomationScheduledRunbook

## Sinossi

Associa un Runbook a una pianificazione.

## SINTASSI

### ByRunbookName (impostazione predefinita)
```
Register-AzureAutomationScheduledRunbook -AutomationAccountName <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

### ByRunbookNameAndScheduleName
```
Register-AzureAutomationScheduledRunbook -RunbookName <String> -ScheduleName <String>
 [-Parameters <IDictionary>] -AutomationAccountName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Il cmdlet **Register-AzureAutomationScheduledRunbook** associa un Runbook a una pianificazione.
Runbook inizia in base alla pianificazione specificata usando il parametro *ScheduleName* .

## ESEMPI

### Esempio 1: associare un Runbook a una pianificazione
```
PS C:\> Register-AzureAutomationScheduledRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ScheduleName "Sched01"
```

Questo comando associa il Runbook denominato Runbk01 con la pianificazione denominata Sched01 nell'account di automazione di Azure denominato Contoso17.
