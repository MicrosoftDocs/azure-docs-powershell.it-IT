# New-WAPackCloudService

## Sinossi
Crea un servizio cloud.

## SINTASSI

```
New-WAPackCloudService -Name <String> -Label <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Questi argomenti sono deprecati e verranno rimossi in futuro.
Questo argomento descrive il cmdlet nella versione 0.8.1 del modulo PowerShell di Microsoft Azure.
Per scoprire la versione del modulo che si sta usando, nella console di PowerShell di Azure digitare `(Get-Module -Name Azure).Version` .

Il cmdlet **New-WAPackCloudService** crea un servizio cloud.

## ESEMPI

### Esempio 1: creare un servizio cloud
```
PS C:\> New-WAPackCloudService -Name "ContosoCloudService01" -Label "A label"
```

Il comando crea un servizio cloud denominato ContosoCloudService01 con un'etichetta.
