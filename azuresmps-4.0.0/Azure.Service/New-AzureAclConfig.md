# New-AzureAclConfig

## Sinossi
Crea un oggetto di configurazione ACL vuoto.

## SINTASSI

```
New-AzureAclConfig [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureAclConfig** crea un oggetto di configurazione dell'elenco di controllo di accesso (ACL) vuoto.

## ESEMPI

### Esempio 1: creare un oggetto di configurazione ACL
```
PS C:\> $Acl = New-AzureAclConfig
```

Questo comando crea un oggetto di configurazione ACL vuoto e lo archivia nella variabile $Acl.
