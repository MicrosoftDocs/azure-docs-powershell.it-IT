# Remove-AzureVMImageOSDiskConfig

## Sinossi
Rimuove la configurazione del disco del sistema operativo dall'oggetto DiskConfigSet.

## SINTASSI

```
Remove-AzureVMImageOSDiskConfig [-DiskConfig] <VirtualMachineImageDiskConfigSet>
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureVMImageOSDiskConfig** rimuove la configurazione del disco del sistema operativo dall'oggetto DiskConfigSet.

## ESEMPI

### Esempio 1: rimuovere la configurazione del disco del sistema operativo da un DiskConfigSet
```
PS C:\> $Disk = New-AzureDiskConfigSet
PS C:\> $Disk = Set-AzureOSDiskConfig -DiskConfig $Disk -HostCaching ReadWrite
PS C:\> Remove-AzureVMImageOSDiskConfig -DiskConfig $Disk
```

Questo comando rimuove la configurazione del disco del sistema operativo dall'oggetto DiskConfigSet
