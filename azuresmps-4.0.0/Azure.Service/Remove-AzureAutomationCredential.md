# Remove-AzureAutomationCredential

## Sinossi

Rimuove una credenziale dall'automazione.

## SINTASSI

```
Remove-AzureAutomationCredential -Name <String> [-Force] -AutomationAccountName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Il cmdlet **Remove-AzureAutomationCredential** consente di rimuovere una credenziale da Microsoft Azure Automation.

## ESEMPI

### Esempio 1: rimuovere una credenziale
```
PS C:\> Remove-AzureAutomationCredential -AutomationAccountName "Contoso17" -Name "MyCredential"
```

Questo comando consente di rimuovere una credenziale denominata credenziali nell'account di automazione denominato Contoso17.
