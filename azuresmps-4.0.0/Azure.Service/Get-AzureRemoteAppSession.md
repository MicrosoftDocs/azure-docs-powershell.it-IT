# Get-AzureRemoteAppSession

## Sinossi
Elenca tutte le sessioni RemoteApp di Azure attive e disconnesse per una raccolta.

## SINTASSI

```
Get-AzureRemoteAppSession [-CollectionName] <String> [[-UserUpn] <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRemoteAppSession** elenca tutte le sessioni RemoteApp di Azure attive e disconnesse per una raccolta RemoteApp di Azure.

## ESEMPI

### Esempio 1: elencare tutte le sessioni in una raccolta
```
PS C:\> Get-AzureRemoteAppSession -CollectionName "ContosoApps"
```

Questo comando elenca tutte le sessioni della raccolta RemoteApp di Azure denominata ContosoApps.
