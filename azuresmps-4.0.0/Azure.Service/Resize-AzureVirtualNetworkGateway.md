# Resize-AzureVirtualNetworkGateway

## Sinossi
Ridimensiona un gateway di rete virtuale.

## SINTASSI

```
Resize-AzureVirtualNetworkGateway -GatewayId <String> -GatewaySKU <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet Resize-AzureVirtualNetworkGateway ridimensiona un gateway di rete virtuale.
