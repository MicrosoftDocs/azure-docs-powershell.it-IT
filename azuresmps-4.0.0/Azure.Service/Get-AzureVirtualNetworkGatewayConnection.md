# Get-AzureVirtualNetworkGatewayConnection

## Sinossi
Ottiene una connessione di gateway di rete virtuale.

## SINTASSI

```
Get-AzureVirtualNetworkGatewayConnection [-GatewayId <String>] [-ConnectedEntityId <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureVirtualNetworkGatewayConnection** ottiene una connessione di gateway di rete virtuale Azure.
