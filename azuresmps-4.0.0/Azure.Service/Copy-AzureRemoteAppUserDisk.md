# Copy-AzureRemoteAppUserDisk

## Sinossi
Copia il disco utente di un utente da una raccolta RemoteApp di Azure a un'altra.

## SINTASSI

```
Copy-AzureRemoteAppUserDisk [-SourceCollectionName] <String> [-DestinationCollectionName] <String>
 [-UserUpn] <String> [-OverwriteExistingUserDisk] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Copy-AzureRemoteAppUserDisk** copia il disco utente di un utente da una raccolta RemoteApp di Azure a un'altra.

## ESEMPI

### Esempio 1: copiare un disco utente
```
PS C:\> Copy-AzureRemoteAppUserDisk -DestinationCollectionName "Contoso02" -SourceCollectionName "Contoso01" -UserUpn "PattiFuller@contoso.com" -OverwriteExistingUserDisk
```

Questo comando copia il disco utente di un utente di Azure Active Directory che ha l'UPN della PattiFuller@contoso.com raccolta Contoso01 alla raccolta Contoso02.
Se un disco utente PattiFuller@contoso.com esiste già in Contoso02, questo comando lo sovrascrive.
