# Remove-AzureVMSqlServerExtension

## Sinossi
Rimuove un'estensione di SQL Server di una macchina virtuale di Azure da un oggetto macchina virtuale.

## SINTASSI

```
Remove-AzureVMSqlServerExtension -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureVMSqlServerExtension** rimuove un'estensione di SQL Server di una macchina virtuale Azure da un oggetto macchina virtuale.
