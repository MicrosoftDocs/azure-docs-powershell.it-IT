# Remove-AzureNetworkSecurityGroup

## Sinossi
Elimina un gruppo di sicurezza di rete.

## SINTASSI

```
Remove-AzureNetworkSecurityGroup -Name <String> [-Force] [-PassThru] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureNetworkSecurityGroup** Elimina un gruppo di sicurezza di rete di Azure dall'abbonamento.
