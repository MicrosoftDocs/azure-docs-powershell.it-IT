# Publish-AzureRemoteAppProgram

## Sinossi
Pubblica un programma RemoteApp di Azure.

## SINTASSI

### ID app (impostazione predefinita)
```
Publish-AzureRemoteAppProgram [-CollectionName] <String> [-StartMenuAppId] <String> [-CommandLine <String>]
 [-DisplayName <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### Percorso dell'app
```
Publish-AzureRemoteAppProgram [-CollectionName] <String> [-FileVirtualPath] <String> [-CommandLine <String>]
 [-DisplayName <String>] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Publish-AzureRemoteAppProgram** pubblica un programma RemoteApp di Azure, che lo rende disponibile per gli utenti della raccolta RemoteApp di Azure.

## ESEMPI

### Esempio 1: pubblicare un programma in una raccolta
```
PS C:\> Publish-AzureRemoteAppProgram -CollectionName "ContosoApps" -StartMenuAppId "a3732322-89a5-4cbc-9844-9634c74d337f" -DisplayName "Finance App"
```

Questo comando pubblica il programma nella raccolta ContosoApps con il *StartMenuAppId* specificato per aggiungere il programma al menu Start.
