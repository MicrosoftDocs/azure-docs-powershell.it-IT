# Remove-AzureReservedIP

## Sinossi
Rimuove un indirizzo IP riservato in base al nome.

## SINTASSI

```
Remove-AzureReservedIP [-ReservedIPName] <String> [-Force] [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureReservedIP** rimuove un indirizzo IP riservato in base al nome.

## ESEMPI

### Esempio 1: rimuovere un indirizzo IP riservato in base al nome
```
PS C:\> Remove-AzureReservedIP -ReservedIPName $name
```

Questo comando rimuove un indirizzo IP riservato in base al nome.
