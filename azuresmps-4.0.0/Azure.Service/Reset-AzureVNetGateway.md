# Reset-AzureVNetGateway

## Sinossi
Reimposta un gateway VPN di rete virtuale.

## SINTASSI

```
Reset-AzureVNetGateway -VNetName <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Reset-AzureVNetGateway** Reimposta e riavvia un gateway di rete virtuale (VPN) Virtual Private Network.

## ESEMPI

### Esempio 1: reimpostare un gateway di rete virtuale
```
PS C:\> Reset-AzureVNetGateway -VNetName "ContosoVN07"
```

Questo comando Reimposta il gateway di rete virtuale dalla rete virtuale denominata ContosoVN07.
