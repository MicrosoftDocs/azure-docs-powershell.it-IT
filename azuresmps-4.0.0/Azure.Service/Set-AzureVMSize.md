# Set-AzureVMSize

## Sinossi
Imposta le dimensioni di una macchina virtuale di Azure.

## SINTASSI

```
Set-AzureVMSize [-InstanceSize] <String> -VM <IPersistentVM> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureVMSize** aggiorna le dimensioni di una macchina virtuale.
Ha due parametri: *InstanceSize* , che è la nuova dimensione della macchina virtuale e *VM* , un oggetto macchina virtuale recuperato usando il cmdlet **Get-AzureVM** .
Il risultato di **set-AzureVMSize** può essere inviato tramite pipe al cmdlet **Update-AzureVM** o archiviato in una variabile per un uso successivo.
Non viene eseguita alcuna modifica effettiva finché non viene eseguito **Update-AzureVM** .

Nota: questo cmdlet richiede che la macchina virtuale venga revisionata e potrebbe ottenere un nuovo indirizzo IP.

## ESEMPI

### Esempio 1: impostare le dimensioni di una macchina virtuale
```
PS C:\> Get-AzureVM -ServiceName "MySvc1" -Name "MyVM3" | Set-AzureVMSize "Small" | Update-AzureVM
```

Questo comando aggiorna una macchina virtuale in dimensioni "piccole".
