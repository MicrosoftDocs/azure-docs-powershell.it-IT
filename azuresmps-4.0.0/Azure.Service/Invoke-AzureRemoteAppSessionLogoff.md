# Invoke-AzureRemoteAppSessionLogoff

## Sinossi
Disconnette immediatamente una sessione RemoteApp di Azure.

## SINTASSI

```
Invoke-AzureRemoteAppSessionLogoff [-CollectionName] <String> [-UserUpn] <String> [-Profile <AzureSMProfile>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Invoke-AzureRemoteAppSessionLogoff** disconnette immediatamente un utente da una sessione remota in uso in Azure RemoteApp.

## ESEMPI

### Esempio 1: disconnettersi da un utente
```
PS C:\> Invoke-AzureRemoteAppSessionLogoff -CollectionName ContosoApps -UserUpn PattiFuller@contoso.com
```

Questo comando disconnette l'utente il cui UPN è PattiFuller@contoso.com .
