# Remove-AzureAutomationModule

## Sinossi

Rimuove un modulo dall'automazione.

## SINTASSI

```
Remove-AzureAutomationModule -Name <String> [-Force] -AutomationAccountName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione

[!INCLUDE [aa-deprecation](../include/aa-deprecation.md)]

Il cmdlet **Remove-AzureAutomationModule** rimuove un account di automazione da Microsoft Azure Automation.

## ESEMPI

### Esempio 1: rimuovere un modulo
```
PS C:\> Remove-AzureAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule"
```

Questo comando rimuove un modulo denominato ContosoModule dall'account di automazione denominato Contoso17.
