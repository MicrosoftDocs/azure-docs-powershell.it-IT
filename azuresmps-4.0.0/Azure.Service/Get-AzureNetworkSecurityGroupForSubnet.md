# Get-AzureNetworkSecurityGroupForSubnet

## Sinossi
Ottiene il gruppo di sicurezza di rete per una subnet.

## SINTASSI

```
Get-AzureNetworkSecurityGroupForSubnet -VirtualNetworkName <String> -SubnetName <String> [-Detailed]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureNetworkSecurityGroupForSubnet** ottiene il gruppo di sicurezza di rete associato a una subnet.
