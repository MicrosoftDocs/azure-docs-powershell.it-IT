# Get-AzureRemoteAppPlan

## SYNOPSIS
Elenca i piani di Azure RemoteApp.

## SINTASSI

```
Get-AzureRemoteAppPlan [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzureRemoteAppPlan** elenca i piani di Azure RemoteApp.

## PARAMETERS
