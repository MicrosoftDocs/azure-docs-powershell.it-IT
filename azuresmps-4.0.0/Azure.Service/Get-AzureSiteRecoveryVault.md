# Get-AzureSiteRecoveryVault

## Sinossi
Recupera i Vault di ripristino del sito dall'abbonamento corrente.

## SINTASSI

### Predefinito (impostazione predefinita)
```
Get-AzureSiteRecoveryVault [-Profile <AzureSMProfile>] [<CommonParameters>]
```

### ByName
```
Get-AzureSiteRecoveryVault -Name <String> [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureSiteRecoveryVault** ottiene un elenco dei Vault di ripristino di siti di Azure o di uno specifico Vault di ripristino del sito dall'abbonamento corrente.

## ESEMPI

### Esempio 1: ottenere il Vault attivo
```
PS C:\> Get-AzureSiteRecoveryVault
Name             : ContosoVault
ID               : 6467459117394545458
CloudServiceName : CS-West-US-RecoveryServices
SubscriptionId   : a5aa5997-33e5-46cc-8ab8-b8d89b76b7ba
StatusReason     : 
Status           : Active
Location         : West US
```

Questo comando consente di ottenere il Vault di ripristino del sito Azure attivo.
