# Remove-AzureWebsiteJob

## Sinossi
Rimuove un processo Web esistente per un sito Web.

## SINTASSI

```
Remove-AzureWebsiteJob -JobName <String> -JobType <WebJobType> [-Force] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## Descrizione
Rimuove un processo Web esistente per un sito Web.

## ESEMPI

### Esempio 1: rimuovere un processo Web esistente per un sito Web
```
PS C:\> Remove-AzureWebsiteJob -Name MyWebsite -JobName MyWebJob -JobType Continuous
```

Rimuove un processo Web denominato MyWebJob per sito Web.
