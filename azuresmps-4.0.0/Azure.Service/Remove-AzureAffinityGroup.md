# Remove-AzureAffinityGroup

## Sinossi
Elimina un gruppo di affinità in un abbonamento.

## SINTASSI

```
Remove-AzureAffinityGroup [-Name] <String> [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureAffinityGroup** Elimina un gruppo di affinità di Azure nell'abbonamento corrente.
Non è possibile eliminare un gruppo di affinità che include membri.
È prima di tutto necessario eliminare tutti i membri di un gruppo di affinità.

## ESEMPI

### Esempio 1: rimuovere un gruppo di affinità
```
PS C:\> Remove-AzureAffinityGroup -Name "South01"
```

Questo comando Elimina il gruppo di affinità South01 nell'abbonamento corrente.
