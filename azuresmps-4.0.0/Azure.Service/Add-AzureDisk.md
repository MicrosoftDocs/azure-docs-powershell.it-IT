# Add-AzureDisk

## Sinossi
Aggiunge un disco all'archivio di Azure disk.

## SINTASSI

```
Add-AzureDisk [-DiskName] <String> [-MediaLocation] <String> [-Label <String>] [-OS <String>]
 [-Profile <AzureSMProfile>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzureDisk** aggiunge un disco all'archivio di Azure disk nell'abbonamento corrente.
Questo cmdlet puÃ² aggiungere un disco di sistema o un disco di dati.
Per aggiungere un disco di sistema, specificare un tipo di sistema operativo usando il parametro *OS* .

## ESEMPI

### Esempio 1: aggiungere un disco di avvio che usa il sistema operativo Windows
```
PS C:\> Add-AzureDisk -DiskName "MyWinDisk" -MediaLocation "http://contosostorage.blob.core.azure.com/vhds/winserver-system.vhd" -Label "StartupDisk" -OS "Windows"
```

Questo comando aggiunge un disco di sistema al repository del disco.
Il disco di sistema usa il sistema operativo Windows.

### Esempio 2: aggiungere un disco dati
```
PS C:\> Add-AzureDisk -DiskName "MyDataDisk" -MediaLocation "http://yourstorageaccount.blob.core.azure.com/vhds/winserver-data.vhd" -Label "DataDisk"
```

Questo comando aggiunge un disco dati.

### Esempio 3: aggiungere un disco di sistema Linux
```
PS C:\> Add-AzureDisk -DiskName "MyLinuxDisk" -MediaLocation "http://yourstorageaccount.blob.core.azure.com/vhds/linuxsys.vhd" -OS "Linux"
```

Questo comando aggiunge un disco di sistema Linux.
