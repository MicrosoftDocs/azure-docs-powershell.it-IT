# Set-AzureDns

## Sinossi
Modifica l'indirizzo IP di un server DNS.

## SINTASSI

```
Set-AzureDns [-Name] <String> [-IPAddress] <String> [-ServiceName] <String> [-Profile <AzureSMProfile>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureDns** modifica l'indirizzo IP di un server DNS per un servizio Azure.

## ESEMPI

### Esempio 1: modificare l'indirizzo IP di un server DNS
```
PS C:\> Set-AzureDns -ServiceName "ContosoService" -IPAddress 10.1.2.5 -Name "Dns07"
```

Questo comando consente di modificare l'indirizzo IP del server DNS denominato Dns07 per il servizio denominato ContosoService.
