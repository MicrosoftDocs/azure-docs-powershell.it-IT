# Get-AzRecoveryServicesAsrAlertSetting

## Sinossi
Ottiene le impostazioni di notifica di ripristino del sito Azure configurate per il Vault.

## SINTASSI

```
Get-AzRecoveryServicesAsrAlertSetting [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzRecoveryServicesAsrAlertSetting** ottiene le impostazioni di notifica di ripristino di Azure site configurate per il Vault.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzRecoveryServicesAsrAlertSetting

CustomEmailAddress     EmailSubscriptionOwner Locale
------------------     ---------------------- ------
{abcxxxx@xxxx.com} On                     en-US
```

Ottenere l'impostazione avviso/notifica per il ripristino del sito Azure.
