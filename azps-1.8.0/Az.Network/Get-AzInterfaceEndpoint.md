# Get-AzInterfaceEndpoint

## Sinossi
Il cmdlet Get-AzInterfaceEndpoint ottiene un endpoint di interfaccia.

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzInterfaceEndpoint [-Name <String>] -ResourceGroupName <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzInterfaceEndpoint -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzInterfaceEndpoint** ottiene un endpoint di interfaccia.

## ESEMPI

### Esempio 1
```
$interfaceendpoint = Get-AzInterfaceEndpoint -Name "InterfaceEndpoint1" -ResourceGroupName "ResourceGroup01"
```

Questo comando ottiene l'endpoint di interfaccia denominato InterfaceEndpoint1 che appartiene al gruppo di risorse denominato ResourceGroup01 e lo archivia nella variabile $interfaceendpoint.

### Esempio 2
```
$interfaceendpoint = Get-AzInterfaceEndpoint -ResourceId "/subscriptions/sub1/resourceGroups/chsriniIEtest1/providers/Microsoft.Network/interfaceEndpoints/ie1.10"
```

Questo comando ottiene l'endpoint dell'interfaccia con resourceId/subscriptions/sub1/resourceGroups/chsriniIEtest1/providers/Microsoft.Network/interfaceEndpoints/ie1.10 e lo archivia nella variabile $interfaceendpoint.

### Esempio 3
```
$interfaceendpoint = Get-AzInterfaceEndpoint -Name InterfaceEndpoint*
```

Questo comando consente di ottenere gli endpoint di interfaccia che iniziano con "InterfaceEndpoint"
