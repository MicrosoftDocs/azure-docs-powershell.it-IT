# Get-AzApplicationGatewayIdentity

## Sinossi
Ottenere l'identità assegnata al gateway dell'applicazione.

## SINTASSI

```
Get-AzApplicationGatewayIdentity -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzApplicationGatewayIdentity** Ottiene l'identità assegnata al gateway dell'applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $identity = Get-AzApplicationGatewayIdentity -ApplicationGateway $gw
```

In questo esempio viene illustrato come ottenere l'identità del gateway applicazione dal gateway dell'applicazione.
