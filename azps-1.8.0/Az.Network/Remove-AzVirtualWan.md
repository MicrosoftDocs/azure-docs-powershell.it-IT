# Remove-AzVirtualWan

## Sinossi
Rimuove una WAN virtuale di Azure.

## SINTASSI

### ByVirtualWanName (impostazione predefinita)
```
Remove-AzVirtualWan -ResourceGroupName <String> -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanObject
```
Remove-AzVirtualWan -InputObject <PSVirtualWan> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByVirtualWanResourceId
```
Remove-AzVirtualWan -ResourceId <String> [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Rimuove una WAN virtuale di Azure.

## ESEMPI

### Esempio 1

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Passthru
```

Questo esempio crea una WAN virtuale in un gruppo di risorse e quindi la Elimina immediatamente. Per eliminare la richiesta quando si elimina la rete WAN virtuale, usare il contrassegno-forza.

### Esempio 2

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -InputObject $virtualWan -Passthru
```

Questo esempio crea una WAN virtuale in un gruppo di risorse e quindi la Elimina immediatamente. Questa operazione viene eseguita usando l'oggetto WAN virtuale restituito da New-AzVirtualWan.
Per eliminare la richiesta quando si elimina la rete WAN virtuale, usare il contrassegno-forza.

### Esempio 3

```powershell
PS C:\> New-AzResourceGroup -Name "TestResourceGroup" -Location "Central US"
PS C:\> $virtualWan = New-AzVirtualWan -Name "MyVirtualWan" -ResourceGroupName "TestResourceGroup" -Location "Central US"
PS C:\> Remove-AzVirtualWan -ResourceId $virtualWan.Id -Passthru
```

Questo esempio crea una WAN virtuale in un gruppo di risorse e quindi la Elimina immediatamente. Questa operazione viene eseguita usando l'ID risorsa WAN virtuale restituito da New-AzVirtualWan.
Per eliminare la richiesta quando si elimina la rete WAN virtuale, usare il contrassegno-forza.
