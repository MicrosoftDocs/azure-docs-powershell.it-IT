# Get-AzLoadBalancerInboundNatPoolConfig

## Sinossi
Ottiene una o più configurazioni di pool NAT in ingresso da un bilanciamento del carico.

## SINTASSI

```
Get-AzLoadBalancerInboundNatPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzLoadBalancerInboundNatPoolConfig** ottiene una o più configurazioni di pool NAT in ingresso da un bilanciamento del carico.

## ESEMPI

### 1: ottenere
```
PS C:\> $slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Get-AzLoadBalancerInboundNatPoolConfig -Name myInboundNatPool
```
