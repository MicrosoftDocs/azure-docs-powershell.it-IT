# Get-AzNetworkWatcherTopology

## SYNOPSIS
Ottiene una visualizzazione a livello di rete delle risorse e delle relative relazioni in un gruppo di risorse.

## SINTASSI

### SetByResource (Default)
```
Get-AzNetworkWatcherTopology -NetworkWatcher <PSNetworkWatcher> -TargetResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByName
```
Get-AzNetworkWatcherTopology -NetworkWatcherName <String> -ResourceGroupName <String>
 -TargetResourceGroupName <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SetByLocation
```
Get-AzNetworkWatcherTopology -Location <String> -TargetResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il Get-AzNetworkWatcherTopology cmdlet una visualizzazione a livello di rete delle risorse e le relative relazioni in un gruppo di risorse. Nota: se le risorse di pi√π aree geografiche si trovano nel gruppo di risorse, solo le risorse nella stessa area geografica di Network Watcher verranno incluse nell'output JSON.
