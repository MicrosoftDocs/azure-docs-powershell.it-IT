# Revoke-AzHDInsightHttpServicesAccess

## Sinossi
Disabilita l'accesso HTTP al cluster.

## SINTASSI

```
Revoke-AzHDInsightHttpServicesAccess [-ClusterName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Revoke-AzHDInsightHttpServicesAccess** Disabilita l'accesso HTTP a un cluster HDInsight di Azure per ODBC, Ambari, oozie e webHCatalog Web Services.

## ESEMPI

### Esempio 1: disabilitare l'accesso HTTP a un cluster
```
PS C:\>Revoke-AzHDInsightHttpServicesAccess `
       -ClusterName "your-hadoop_001"
```

Questo comando revoca l'accesso HTTP al cluster denominato your-hadoop_001.
