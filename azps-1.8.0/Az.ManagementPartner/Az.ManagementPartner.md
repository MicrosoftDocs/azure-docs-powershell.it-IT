# Modulo AZ. ManagementPartner
## Descrizione
Cmdlet per i partner di gestione per Azure Resource Manager.

## Cmdlet AZ. ManagementPartner
### [Get-AzManagementPartner](Get-AzManagementPartner.md)
Ottiene l'ID Microsoft Partner Network (MPN) dell'utente autenticato o dell'entità di servizio corrente. 

### [New-AzManagementPartner](New-AzManagementPartner.md)
Associa un ID Microsoft Partner Network (MPN) all'utente autenticato o all'entità di servizio corrente.

### [Remove-AzManagementPartner](Remove-AzManagementPartner.md)
Eliminare l'ID Microsoft Partner Network (MPN) dell'utente o dell'entità di servizio corrente autenticato.

### [Update-AzManagementPartner](Update-AzManagementPartner.md)
Aggiorna l'ID Microsoft Partner Network (MPN) dell'utente o dell'entità di servizio corrente autenticato.
