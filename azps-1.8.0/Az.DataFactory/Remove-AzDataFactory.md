# Remove-AzDataFactory

## Sinossi
Rimuove una data factory.

## SINTASSI

### ByFactoryName (impostazione predefinita)
```
Remove-AzDataFactory [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactory [-DataFactory] <PSDataFactory> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzDataFactory** rimuove una data factory.

## ESEMPI

### Esempio 1: rimuovere una data factory
```
PS C:\>Remove-AzDataFactory -Name "WikiADF" -ResourceGroupName "ADF"
Confirm
Are you sure you want to remove data factory 'WikiADF' in resource group 'ADF'? 
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): Y
True
```

Questo comando rimuove la factory di dati denominata WikiADF dal gruppo di risorse denominato ADF.
Questo comando restituisce un valore di $True.
