# New-AzSignalR

## Sinossi
Creare un servizio di segnalazione.

## SINTASSI

```
New-AzSignalR [-ResourceGroupName <String>] [-Name] <String> [-Location <String>] [-Sku <String>]
 [-UnitCount <Int32>] [-Tag <System.Collections.Generic.IDictionary`2[System.String,System.String]>] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Creare un servizio di segnalazione.
I valori seguenti verranno usati per i parametri se non sono specificati:
* `ResourceGroupName`: gruppo di risorse predefinito impostato da `Set-AzDefault -ResourceGroupName` .
* `Location`: la posizione del gruppo risorse
* `Sku`: `Standard_S1`

## ESEMPI

### Creare un serivce di segnalazione
```powershell
PS C:\> New-AzSignalR -ResourceGroupName myResourceGroup1 -Name mysignalr1 -Location eastus -Sku Standard_S1

HostName                                           Location       ServerPort PublicPort ProvisioningState Version
--------                                           --------       ---------- ---------- ----------------- -------
mysignalr1.service.signalr.net                     eastus         5002       5001       Succeeded         1.0-preview
```
