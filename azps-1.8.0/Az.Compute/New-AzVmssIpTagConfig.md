# New-AzVmssIpTagConfig

## Sinossi
Crea un oggetto Tag IP per un'interfaccia di rete di un VMSS.

## SINTASSI

```
New-AzVmssIpTagConfig [-IpTagType] <String> [-Tag <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzVmssIpTagConfig** crea un oggetto di configurazione di tag IP per un'interfaccia di rete di un set di scale della macchina virtuale (VMSS).
Specifica la configurazione di questo cmdlet come parametro *IPTag* del cmdlet New-AzVmssIpConfig.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $iptag = New-AzVmssIpTagConfig -IpTagType 'FirstPartyUsage' -Tag 'Sql'
PS C:\> $ipCfg = New-AzVmssIPConfig -Name 'test' -SubnetId $subnetId -IpTag $ipTag;
```

Questo comando crea un oggetto local Tag IP con il tipo "FirstPartyUsage" e il tag "SQL" e quindi crea una configurazione IP con questo tag IP.
