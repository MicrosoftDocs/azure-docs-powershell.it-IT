# New-AzMetricAlertRuleV2DimensionSelection

## Sinossi
Crea un oggetto di selezione della dimensione locale che pu√≤ essere usato per creare criteri di avviso metrici.

## SINTASSI

```
New-AzMetricAlertRuleV2DimensionSelection -DimensionName <String> -ValuesToInclude <String[]>
 [-ValuesToExclude <String[]>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzMetricAlertRuleV2DimensionSelection** crea un oggetto di selezione della dimensione locale per facilitare la costruzione di criteri di avviso metrico usando il cmdlet **New-AzMetricAlertRuleV2Criteria** .

## ESEMPI

### Esempio 1: creare un oggetto selezione dimensione

```powershell
PS C:\> New-AzMetricAlertRuleV2DimensionSelection -DimensionName LUN -ValuesToInclude 1,3,4

Dimension IncludeValues ExcludeValues
--------- ------------- -------------
LUN       {1, 3, 4}     
```

Questo comando crea un oggetto selezione dimensione che specifica che i valori {1,3,4} devono essere selezionati per la dimensione "lun"
