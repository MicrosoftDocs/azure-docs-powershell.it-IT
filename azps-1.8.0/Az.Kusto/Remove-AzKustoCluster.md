# Remove-AzKustoCluster

## Sinossi
Elimina un cluster di Kusto esistente.

## SINTASSI

### ByNameAndResourceGroup (impostazione predefinita)
```
Remove-AzKustoCluster [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzKustoCluster [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzKustoCluster [-InputObject] <PSKustoCluster> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Elimina un cluster di Kusto esistente.

## ESEMPI

### Esempio 1-eliminare un cluster kusto esistente per nome

```
PS C:\> Remove-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster
```

Il comando precedente elimina il cluster kusto denominato "mykustocluster" nel gruppo di risorse "testrg".

### Esempio 2-eliminare un cluster di Kusto esistente tramite piping

```
PS C:\> Get-AzKustoCluster -ResourceGroupName testrg -Name mykustocluster | Remove-AzKustoCluster
```

Il comando precedente restituisce il cluster kusto denominato "mykustocluster" nel gruppo di risorse "testrg" con il `Get-AzKustoCluster` cmdlet e quindi convoglia il risultato per `Remove-AzKustoCluster` eliminarlo.
