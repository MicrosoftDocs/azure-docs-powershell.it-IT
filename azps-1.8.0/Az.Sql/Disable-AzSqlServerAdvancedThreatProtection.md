# Disable-AzSqlServerAdvancedThreatProtection

## Sinossi
Disabilita la protezione avanzata delle minacce in un server.

## SINTASSI

```
Disable-AzSqlServerAdvancedThreatProtection [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Disable-AzSqlServerAdvancedThreatProtection Disabilita** la protezione avanzata delle minacce in un server.

## ESEMPI

### Esempio 1-disabilitare la protezione avanzata delle minacce del server
```powershell
PS C:\>  Disable-AzSqlServerAdvancedThreatProtection `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```

### Esempio 2-disabilitare la protezione avanzata delle minacce server dalla risorsa server
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Disable-AzSqlServerAdvancedThreatProtection

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```
