# Remove-AzSqlServerThreatDetectionPolicy

## Sinossi
Rimuove i criteri di rilevamento delle minacce da un server.

## SINTASSI

```
Remove-AzSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Remove-AzSqlServerThreatDetectionPolicy rimuove i criteri di rilevamento delle minacce da un server SQL di Azure.
Per usare questo cmdlet, specificare i parametri ResourceGroupName e ServerName per identificare il server da cui questo cmdlet rimuove il criterio.

## ESEMPI

### Esempio 1: rimuovere un criterio di rilevamento delle minacce per un database
```
PS C:\> Remove-AzSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

Questo comando rimuove i criteri di rilevamento delle minacce da un server denominato Server01.
