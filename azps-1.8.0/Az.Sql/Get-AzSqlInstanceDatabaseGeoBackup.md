# Get-AzSqlInstanceDatabaseGeoBackup

## Sinossi
Restituisce informazioni sul backup ridondante del database di istanze gestite di Azure SQL.

## SINTASSI

```
Get-AzSqlInstanceDatabaseGeoBackup [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlInstanceDatabaseGeoBackup** ottiene uno o piÃ¹ database SQL di Azure Backup ridondanti da un'istanza gestita di database SQL di Azure.
