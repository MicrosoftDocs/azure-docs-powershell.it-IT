# Get-AzSqlInstance

## Sinossi
Restituisce informazioni sull'istanza di database gestito di Azure SQL.

## SINTASSI

```
Get-AzSqlInstance [[-Name] <String>] [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlInstance** restituisce informazioni su una o piÃ¹ istanze gestite da SQL Azure.
Specificare il nome di un'istanza per visualizzare le informazioni solo per quell'istanza.
