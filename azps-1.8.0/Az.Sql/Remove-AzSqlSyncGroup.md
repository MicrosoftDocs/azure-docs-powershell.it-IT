# Remove-AzSqlSyncGroup

## Sinossi
Rimuove un gruppo di sincronizzazione di database SQL di Azure.

## SINTASSI

```
Remove-AzSqlSyncGroup [-Name] <String> [-Force] [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzSqlSyncGroup** rimuove un gruppo di sincronizzazione di database SQL di Azure.

## ESEMPI

### Esempio 1: rimuovere un gruppo di sincronizzazione
```
PS C:\>Remove-AzSqlSyncGroup -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "database01" -Name "syncGroup01"
```

Questo comando rimuove il gruppo di sincronizzazione di database SQL di Azure denominato syncGroup01.
