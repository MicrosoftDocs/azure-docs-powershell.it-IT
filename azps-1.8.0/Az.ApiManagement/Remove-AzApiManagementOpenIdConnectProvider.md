# Remove-AzApiManagementOpenIdConnectProvider

## Sinossi
Rimuove un provider OpenID Connect.

## SINTASSI

```
Remove-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> -OpenIdConnectProviderId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApiManagementOpenIdConnectProvider** rimuove un provider OpenID Connect per la gestione delle API di Azure.

## ESEMPI

### Esempio 1: rimuovere un provider
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvicer01" -PassThru
```

Questo comando rimuove un provider con ID OICProvicer01.
