# Remove-AzDataLakeAnalyticsComputePolicy

## Sinossi
Rimuove un criterio di calcolo di analisi dei dati di Azure Data Lake specificato

## SINTASSI

```
Remove-AzDataLakeAnalyticsComputePolicy [-ResourceGroupName <String>] [-Account] <String> [-Name] <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
**Remove-AzDataLakeAnalyticsComputePolicy** rimuove un criterio di calcolo di analisi dei dati di Azure Data Lake specificato.

## ESEMPI

### Esempio 1: rimuovere un criterio di calcolo
```
PS C:\>Remove-AzDataLakeAnalyticsComputePolicy -Account "contosoadla" -Name myPolicy
```

Questo comando rimuove il criterio di calcolo specificato con il nome "criterio" nell'account "contosoadla".
