# New-AzCognitiveServicesAccount

## Sinossi
Crea un account di servizi cognitivi.

## SINTASSI

```
New-AzCognitiveServicesAccount [-ResourceGroupName] <String> [-Name] <String> [-Type] <String>
 [-SkuName] <String> [-Location] <String> [-Tag <Hashtable[]>] [-CustomSubdomainName <String>] [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzCognitiveServicesAccount** crea un account di servizi cognitivi con il tipo e SKU specificati.
