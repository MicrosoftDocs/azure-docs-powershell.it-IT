# Get-AzDataFactoryV2

## SYNOPSIS
Ottiene informazioni su Data factory.

## SINTASSI

### BySubscriptionId (impostazione predefinita)
```
Get-AzDataFactoryV2 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryName
```
Get-AzDataFactoryV2 [-ResourceGroupName] <String> [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il Get-AzDataFactoryV2 cmdlet recupera informazioni sulle data factories in un gruppo di risorse di Azure.
Se si specifica il nome di un data factory, questo cmdlet ottiene informazioni su tale data factory.
Se non si specifica un nome, questo cmdlet ottiene informazioni su tutti i data factories in un gruppo di risorse di Azure.

## ESEMPI

### Esempio 1: Ottenere tutti i data factories
```
PS C:\> Get-AzDataFactoryV2 -ResourceGroupName "ADF"

    DataFactoryName   : WikiADF
    DataFactoryId     : /subscriptions/3e8e61b5-9a7d-4952-bfae-545ab997b9ea/resourceGroups/adf/providers/Microsoft.DataFactory/factories/wikiadf
    ResourceGroupName : ADF
    Location          : EastUS
    Tags              : {}
    Identity          : Microsoft.Azure.Management.DataFactory.Models.FactoryIdentity
    ProvisioningState : Succeeded

    DataFactoryName   : WikiADF2
    DataFactoryId     : /subscriptions/3e8e61b5-9a7d-4952-bfae-545ab997b9ea/resourceGroups/adf/providers/Microsoft.DataFactory/factories/wikiadf2
    ResourceGroupName : ADF
    Location          : EastUS
    Tags              : {}
    Identity          :
    ProvisioningState : Succeeded
```

Visualizza informazioni su tutti i data factories nella sottoscrizione di Azure.

### Esempio 2: Ottenere uno specifico data factory
```
PS C:\> $DataFactory = Get-AzDataFactoryV2 -ResourceGroupName "ADF" -Name "WikiADF"

    DataFactoryName   : WikiADF
    DataFactoryId     : /subscriptions/3e8e61b5-9a7d-4952-bfae-545ab997b9ea/resourceGroups/adf/providers/Microsoft.DataF
                        actory/factories/wikiadf
    ResourceGroupName : ADF
    Location          : EastUS
    Tags              : {}
    Identity          : Microsoft.Azure.Management.DataFactory.Models.FactoryIdentity
    ProvisioningState : Succeeded
```

Questo comando visualizza informazioni sul data factory denominato WikiADF nella sottoscrizione del gruppo di risorse denominato ADF e quindi lo archivia nella $DataFactory variabile.
Specificare il parametro DataFactory nei cmdlet successivi per usare il data factory archiviato in $DataFactory.
