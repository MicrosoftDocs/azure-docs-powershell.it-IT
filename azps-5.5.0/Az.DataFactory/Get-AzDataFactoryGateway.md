# Get-AzDataFactoryGateway

## SYNOPSIS
Ottiene informazioni sui gateway logici in Data factory di Azure.

## SINTASSI

### ByFactoryName (Default)
```
Get-AzDataFactoryGateway [-DataFactoryName] <String> [[-Name] <String>] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGateway [-DataFactory] <PSDataFactory> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzDataFactoryGateway** ottiene informazioni sui gateway logici in Data factory di Azure.
Se si specifica il nome di un gateway, questo cmdlet ottiene informazioni sul gateway.
Se non si specifica un nome, questo cmdlet ottiene informazioni su tutti i gateway per un data factory.
Se si vuole aggiungere un Microsoft SQL Server locale come servizio collegato a un data factory, Ã¨ necessario installare un gateway nel computer locale.

Questo comando recupera informazioni su tutti i gateway logici per il data factory denominato WikiADF nel gruppo di risorse denominato ADF.
