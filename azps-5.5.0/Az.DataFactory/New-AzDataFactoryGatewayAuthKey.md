# New-AzDataFactoryGatewayAuthKey

## SYNOPSIS
Crea una chiave di autenticazione per un Gateway di Data factory di Azure.

## SINTASSI

### ByFactoryName (Default)
```
New-AzDataFactoryGatewayAuthKey [-DataFactoryName] <String> [-GatewayName] <String> [-KeyName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByFactoryObject
```
New-AzDataFactoryGatewayAuthKey [-InputObject] <PSDataFactory> [-GatewayName] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzDataFactoryGatewayAuthKey crea** la chiave di autenticazione del gateway per un gateway di Data factory di Azure specificato.
Per registrare il gateway in un servizio cloud, usare questa chiave.

## ESEMPI

### Esempio 1: Crea una chiave di autenticazione del gateway per Key1
```
PS C:\> New-AzDataFactoryGatewayAuthKey -ResourceGroup ADFResource -GatewayName 'MyGateway' -DataFactoryName MyADF -KeyName key1
Key1 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@BH0EV9hu/o2IYGQzfYYD203XhdS6Tty
       fkYwYFbG6wBU=
Key2 :
```

Questo comando crea una chiave di autenticazione del gateway di Key1 per il gateway del data factory denominato MyGateway.
