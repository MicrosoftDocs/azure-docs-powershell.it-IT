# Get-AzDataFactoryV2Trigger

## SYNOPSIS
Ottiene informazioni sui trigger in un data factory.

## SINTASSI

### ByFactoryName (Default)
```
Get-AzDataFactoryV2Trigger [[-Name] <String>] [-ResourceGroupName] <String> [-DataFactoryName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryV2Trigger [[-Name] <String>] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceId
```
Get-AzDataFactoryV2Trigger [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzDataFactoryV2Trigger** ottiene informazioni sui trigger in un data factory. Se si specifica il nome di un trigger, il cmdlet riceverÃ  informazioni sul trigger. Se non si specifica un nome, il cmdlet riceve informazioni su tutti i trigger nel data factory.

## ESEMPI

### Esempio 1: Ottenere informazioni su tutti i trigger
```
PS C:\> Get-AzDataFactoryV2Trigger -ResourceGroupName "ADF" -DataFactoryName "WikiADF"

    TriggerName       : ScheduledTrigger
    ResourceGroupName : ADF
    DataFactoryName   : WikiADF
    Properties        : Microsoft.Azure.Management.DataFactory.Models.ScheduleTrigger
    RuntimeState      : Stopped

    TriggerName       : ScheduledTrigger2
    ResourceGroupName : ADF
    DataFactoryName   : WikiADF
    Properties        : Microsoft.Azure.Management.DataFactory.Models.ScheduleTrigger
    RuntimeState      : Stopped
```

Ottiene un elenco di tutti i trigger creati nel data factory "WikiADF".

### Esempio 2: Ottenere informazioni su un trigger specifico
```
Get-AzDataFactoryV2Trigger -ResourceGroupName "ADF" -DataFactoryName "WikiADF" -TriggerName "ScheduledTrigger"

    TriggerName       : ScheduledTrigger
    ResourceGroupName : ADF
    DataFactoryName   : WikiADF
    Properties        : Microsoft.Azure.Management.DataFactory.Models.ScheduleTrigger
    RuntimeState      : Stopped
```

Ottiene un singolo trigger denominato "ScheduledTrigger" nel data factory "WikiADF".
