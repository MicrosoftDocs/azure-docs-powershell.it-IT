# Remove-AzDataFactoryHub

## SYNOPSIS
Rimuove un hub da Data factory di Azure.

## SINTASSI

### ByFactoryName (Default)
```
Remove-AzDataFactoryHub [-Name] <String> [-Force] [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryHub [-Name] <String> [-Force] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzDataFactoryHub** rimuove un hub da Data factory di Azure nel gruppo di risorse azure specificato e nel data factory specificato.
Se si rimuove un hub, vengono rimossi anche tutti i servizi e le pipeline collegati nell'hub.

## ESEMPI

### Esempio 1: Rimuovere un hub
```
PS C:\>Remove-AzDataFactoryHub -ResourceGroupName "ADFResourceGroup" -DataFactoryName "ADFDataFactory" -Name "ContosoDataHub"
```

Questo comando rimuove l'hub denominato ContosoDataHub dal gruppo di risorse azure denominato ADFResourceGroup e dal data factory denominato ADFDataWith.
