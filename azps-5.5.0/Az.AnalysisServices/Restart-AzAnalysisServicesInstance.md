# Restart-AzAnalysisServicesInstance

## SYNOPSIS
Riavvia un'istanza del server di Analysis Services nell'ambiente attualmente connesso, come specificato nel comando Add-AzAnalysisServicesAccount>

## SINTASSI

```
Restart-AzAnalysisServicesInstance [-Instance] <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Restart-AzAnalysisServicesInstance riavvia un'istanza del server di Azure Analysis Services

## ESEMPI

### Esempio 1
```
PS C:\>Restart-AzAnalysisServicesInstance
Instance: testserver
```

Questo comando riavvier√† il server 'testserver' nell'ambiente specificato nel comando Add-AzAnalysisServicesAccount
