# Update-AzCosmosDBAccountRegion

## SYNOPSIS
Aggiornare le aree geografiche di un account diDbDb.

## SINTASSI

### ByNameParameterSet (Impostazione predefinita)
```
Update-AzCosmosDBAccountRegion -ResourceGroupName <String> -Name <String> [-Location <String[]>]
 [-LocationObject <PSLocation[]>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Update-AzCosmosDBAccountRegion [-Location <String[]>] [-LocationObject <PSLocation[]>] -ResourceId <String>
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByObjectParameterSet
```
Update-AzCosmosDBAccountRegion [-Location <String[]>] [-LocationObject <PSLocation[]>]
 -InputObject <PSDatabaseAccountGetResults> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Aggiornare le aree geografiche di un account diDbDb. La posizione può essere fornita come oggetto di tipo PSLocation o come stringa di Nome posizione ordinata in base alla priorità di failover.
Il parametro LocationObject prevede l'aggiunta dell'elenco delle posizioni correnti (priorità di failover incluse) aggiunto dai nuovi LocationObjects corrispondenti alle nuove posizioni.
Il parametro Location prevede l'elenco della posizione corrente (ordinata in base alla priorità di failover) e delle nuove posizioni. Tieni presente che supportiamo solo l'aggiunta di aree geografiche. Specifica Location o LocationObject.
