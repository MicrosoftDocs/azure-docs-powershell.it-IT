# New-AzCosmosDBCassandraClusterKey

## SYNOPSIS
Crea una nuova chiave cluster CassammasDB Cassandra.

## SINTASSI

```
New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
La **nuova chiave cluster New-AzCosmosDBCassandraClusterKey** crea una nuova chiave cluster Diedb Cassandra.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzCosmosDBCassandraClusterKey -Name <String> -OrderBy <String>

Name   OrderBy
----   -------
{name}  Asc
```
