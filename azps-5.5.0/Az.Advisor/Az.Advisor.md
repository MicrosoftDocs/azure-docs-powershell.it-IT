# Modulo Az.Advisor
## Descrizione
Comandi per interagire con Azure Advisor.

## Cmdlet di Az.Advisor
### [Get-AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
Ottiene un elenco di consigli di Azure Advisor.

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
Disabilitare un consiglio di Azure Advisor.

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
Abilita i suggerimenti di Azure Advisor.

### [Get-AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
Ottenere le configurazioni di Azure Advisor per la sottoscrizione o il gruppo di risorse specificato.

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
Aggiorna o crea la configurazione di Azure Advisor.
