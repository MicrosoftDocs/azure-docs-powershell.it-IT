# Get-AzStorageTableStoredAccessPolicy

## SYNOPSIS
Recupera i criteri di accesso archiviati per una tabella di archiviazione di Azure.

## SINTASSI

```
Get-AzStorageTableStoredAccessPolicy [-Table] <String> [[-Policy] <String>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzStorageTableStoredAccessPolicy** elenca i criteri di accesso archiviato per una tabella di archiviazione di Azure.

## ESEMPI

### Esempio 1: Ottenere criteri di accesso archiviato in una tabella di archiviazione
```
PS C:\>Get-AzStorageTableStoredAccessPolicy -Table "Table02" -Policy "Policy50"
```

Questo comando recupera i criteri di accesso denominati Policy50 nella tabella di archiviazione denominata Table02.

### Esempio 2: Ottenere tutti i criteri di accesso archiviato in una tabella di archiviazione
```
PS C:\>Get-AzStorageTableStoredAccessPolicy -Table "Table02"
```

Questo comando recupera tutti i criteri di accesso nella tabella denominata Table02.
