# Remove-AzStorageTable

## SYNOPSIS
Rimuove una tabella di archiviazione.

## SINTASSI

```
Remove-AzStorageTable [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzStorageTable** rimuove una o più tabelle di archiviazione da un account di archiviazione in Azure.

## ESEMPI

### Esempio 1: Rimuovere una tabella
```
PS C:\>Remove-AzStorageTable -Name "TableABC"
```

Questo comando rimuove una tabella.

### Esempio 2: Rimuovere più tabelle
```
PS C:\>Get-AzStorageTable table* | Remove-AzStorageTable
```

Questo esempio usa un carattere jolly con il parametro *Name* per ottenere tutte le tabelle che corrispondono alla tabella dei criteri e quindi passa il risultato alla pipeline per rimuovere le tabelle.
