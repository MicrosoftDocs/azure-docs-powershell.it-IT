# Get-AzStorageBlobCopyState

## SYNOPSIS
Ottiene lo stato della copia di un BLOB di Archiviazione di Azure.

## SINTASSI

### NamePipeline (impostazione predefinita)
```
Get-AzStorageBlobCopyState [-Blob] <String> [-Container] <String> [-WaitForComplete]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

### BLOBPipeline
```
Get-AzStorageBlobCopyState -CloudBlob <CloudBlob> [-WaitForComplete] [-Context <IStorageContext>]
 [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

### ContainerPipeline
```
Get-AzStorageBlobCopyState -CloudBlobContainer <CloudBlobContainer> [-Blob] <String> [-WaitForComplete]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzStorageBlobCopyState** ottiene lo stato di copia di un BLOB di Archiviazione di Azure.
Deve essere eseguito nel BLOB di destinazione della copia.

## ESEMPI

### Esempio 1: Ottenere lo stato di copia di un BLOB
```
C:\PS>Get-AzStorageBlobCopyState -Blob "ContosoPlanning2015" -Container "ContosoUploads"
```

Questo comando recupera lo stato della copia del BLOB denominato ContosoPlanning2015 nel contenitore ContosoUploads.

### Esempio 2: Ottenere lo stato di copia di un BLOB tramite la pipeline
```
C:\PS>Get-AzStorageBlob -Blob "ContosoPlanning2015" -Container "ContosoUploads" | Get-AzStorageBlobCopyState
```

Questo comando recupera il BLOB denominato ContosoPlanning2015 nel contenitore ContosoUploads usando il cmdlet **Get-AzStorageBlob** e quindi passa il risultato al cmdlet corrente usando l'operatore della pipeline.
Il **cmdlet Get-AzStorageBlobCopyState** ottiene lo stato di copia per il BLOB.

### Esempio 3: Ottenere lo stato di copia di un BLOB in un contenitore usando la pipeline
```
C:\PS>Get-AzStorageContainer -Name "ContosoUploads" | Get-AzStorageBlobCopyState -Blob "ContosoPlanning2015"
```

Questo comando ottiene il contenitore denominato con il cmdlet **Get-AzStorageBlob** e quindi passa il risultato al cmdlet corrente.
Il cmdlet **Get-AzStorageContainer** ottiene lo stato di copia per il BLOB denominato ContosoPlanning2015 in tale contenitore.

### Esempio 4: Avviare la copia e la pipeline per ottenere lo stato di copia
```
C:\PS> $destBlob = Start-AzStorageBlobCopy -SrcContainer "contosouploads" -SrcBlob "ContosoPlanning2015" -DestContainer "contosouploads2" -DestBlob "ContosoPlanning2015_copy"

C:\PS> $destBlob | Get-AzStorageBlobCopyState
```

Il primo comando avvia la copia del BLOB "ContosoPlanning2015" in "ContosoPlanning2015_copy" e restituisce l'output dell'oggetto BLOB destiantion. Il secondo comando pipeline l'oggetto BLOB destiantion a Get-AzStorageBlobCopyState, per ottenere lo stato di copia BLOB.
