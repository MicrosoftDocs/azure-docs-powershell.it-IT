# Remove-AzStorageQueue

## SYNOPSIS
Rimuove una coda di archiviazione.

## SINTASSI

```
Remove-AzStorageQueue [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzStorageQueue** rimuove una coda di archiviazione.

## ESEMPI

### Esempio 1: Rimuovere una coda di archiviazione in base al nome
```
PS C:\>Remove-AzStorageQueue "ContosoQueue01"
```

Questo comando rimuove una coda denominata ContosoQueue01.

### Esempio 2: Rimuovere piÃ¹ code di archiviazione
```
PS C:\>Get-AzStorageQueue "Contoso*" | Remove-AzStorageQueue
```

Questo comando rimuove tutte le code con nomi che iniziano con Contoso.
