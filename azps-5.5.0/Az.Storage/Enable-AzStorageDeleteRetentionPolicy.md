# Enable-AzStorageDeleteRetentionPolicy

## SYNOPSIS
Abilitare l'eliminazione dei criteri di conservazione per il servizio BLOB Archiviazione di Azure.

## SINTASSI

```
Enable-AzStorageDeleteRetentionPolicy [-RetentionDays] <Int32> [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Enable-AzStorageDeleteRetentionPolicy** abilita l'eliminazione dei criteri di conservazione per il servizio BLOB archiviazione di Azure.

## ESEMPI

### Esempio 1: Abilitare l'eliminazione dei criteri di conservazione per il servizio BLOB
```
C:\PS>Enable-AzStorageDeleteRetentionPolicy -RetentionDays 3
```

Questo comando abilita i criteri di conservazione per il servizio BLOB e imposta i giorni di conservazione BLOB eliminati su 3.
