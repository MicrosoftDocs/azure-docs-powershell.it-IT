# Get-AzIntegrationAccount

## SYNOPSIS
Recupera gli account di integrazione.

## SINTASSI

```
Get-AzIntegrationAccount [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzIntegrationAccount** ottiene gli account di integrazione da un gruppo di risorse. Specificare il nome di un account di integrazione e di un gruppo di risorse.
Questo modulo supporta i parametri dinamici.
Per usare un parametro dinamico, digitarlo nel comando.
Per individuare i nomi dei parametri dinamici, digitare un trattino (-) dopo il nome del cmdlet e quindi premere TAB piÃ¹ volte per scorrere i parametri disponibili.
Se si omette un parametro di modello obbligatorio, il cmdlet chiede di specificare il valore.

```

```

Questo comando recupera un account di integrazione denominato IntegrationAccount31 dal gruppo di risorse specificato.

### Esempio 2: Ottenere account di integrazione in un gruppo di risorse
```
PS C:\>Get-AzIntegrationAccount -ResourceGroupName "ResourceGroup11"
Id          : /subscriptions/<SubscriptionId>/resourceGroups/ResourceGroup1/providers/Microsoft.Logic/integrationAccounts/IntegrationAccount31

```

Questo comando recupera gli account di integrazione da un gruppo di risorse denominato ResourceGroup11.

### Esempio 3: Ottenere tutti gli account di integrazione
```
PS C:\>Get-AzIntegrationAccount
```

Questo comando recupera tutti gli account di integrazione nella sottoscrizione di Azure.
