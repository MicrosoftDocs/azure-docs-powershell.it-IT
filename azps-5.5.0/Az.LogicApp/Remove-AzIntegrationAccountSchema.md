# Remove-AzIntegrationAccountSchema

## SYNOPSIS
Rimuove uno schema dell'account di integrazione.

## SINTASSI

```
Remove-AzIntegrationAccountSchema -ResourceGroupName <String> -Name <String> -SchemaName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzIntegrationAccountSchema rimuove** uno schema dell'account di integrazione da un gruppo di risorse.
Specificare il nome dell'account di integrazione, il nome del gruppo di risorse e il nome dello schema.
Questo modulo supporta i parametri dinamici.
Per usare un parametro dinamico, digitarlo nel comando.
Per individuare i nomi dei parametri dinamici, digitare un trattino (-) dopo il nome del cmdlet e quindi premere TAB piÃ¹ volte per scorrere i parametri disponibili.
Se si omette un parametro di modello obbligatorio, il cmdlet chiede di specificare il valore.

## ESEMPI

### Esempio 1: Rimuovere uno schema dell'account di integrazione
```
PS C:\>Remove-AzIntegrationAccountSchema -ResourceGroupName "ResourceGroup11" -Name "IntegrationAccount31" -SchemaName "IntegrationAccountSchema43"
```

Questo comando rimuove uno schema dell'account di integrazione denominato IntegrationAccountSchema43.
