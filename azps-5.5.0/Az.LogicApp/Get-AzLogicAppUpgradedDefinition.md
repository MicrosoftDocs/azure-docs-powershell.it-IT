# Get-AzLogicAppUpgradedDefinition

## SYNOPSIS
Ottiene la definizione aggiornata per un'app per la logica.

## SINTASSI

```
Get-AzLogicAppUpgradedDefinition -ResourceGroupName <String> -Name <String> -TargetSchemaVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzLogicAppUpgradedDefinition** ottiene la definizione aggiornata per l'app per la versione e la logica dello schema da un gruppo di risorse.
Questo cmdlet restituisce un oggetto che rappresenta la definizione dell'app per la logica aggiornata.
Specificare il nome del gruppo di risorse, il nome dell'app per la logica e la versione dello schema di destinazione.
Questo modulo supporta i parametri dinamici.
Per usare un parametro dinamico, digitarlo nel comando.
Per individuare i nomi dei parametri dinamici, digitare un trattino (-) dopo il nome del cmdlet e quindi premere TAB piÃ¹ volte per scorrere i parametri disponibili.
Se si omette un parametro di modello obbligatorio, il cmdlet chiede di specificare il valore.
