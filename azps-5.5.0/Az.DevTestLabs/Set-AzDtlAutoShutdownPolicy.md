# Set-AzDtlAutoShutdownPolicy

## SYNOPSIS
Imposta i criteri di arresto automatico dei laboratori DevTest Lab.

## SINTASSI

### Abilita (impostazione predefinita)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Disabilita
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzDtlAutoShutdownPolicy** imposta il criterio di arresto automatico di un lab, che chiude automaticamente tutte le macchine virtuali nel lab in un'ora specificata del giorno.
Il cmdlet usa il gruppo di risorse specificato e il nome del lab per impostare il criterio.

## ESEMPI

### Esempio 1

Imposta i criteri di arresto automatico dei laboratori DevTest Lab. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Set-AzDtlAutoShutdownPolicy -Enable -LabName <String> -ResourceGroupName MyResourceGroup -Time <DateTime>
```
