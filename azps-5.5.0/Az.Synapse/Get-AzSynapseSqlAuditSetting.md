# Get-AzSynapseSqlAuditSetting

## SYNOPSIS
Recupera le impostazioni di controllo di un'area di lavoro di analisi di Azure Synapse.

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzSynapseSqlAuditSetting [-ResourceGroupName <String>] -WorkspaceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByInputObjectParameterSet
```
Get-AzSynapseSqlAuditSetting -InputObject <PSSynapseWorkspace> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSqlAuditSetting -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSynapseSqlAuditSetting** ottiene le impostazioni di controllo di un'area di lavoro di analisi di Azure Synapse.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSynapseSqlAuditSetting -WorkspaceName ContosoWorkspace
```

Ottiene le impostazioni di controllo di un'area di lavoro denominata ContosoWorkspace.

### Esempio 2
```powershell
PS C:\> Get-AzSynapseWorkspace -Name ContosoWorkspace | Get-AzSynapseSqlAuditSetting
```

Ottiene le impostazioni di controllo di un'area di lavoro denominata ContosoWorkspace tramite pipeline.
