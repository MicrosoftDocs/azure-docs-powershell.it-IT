# New-AzHDInsightClusterAutoscaleConfiguration

## SYNOPSIS
Crea un oggetto non permanente che descrive la configurazione di gradazione automatica di un cluster HDInsight di Azure.

## SINTASSI

### LoadAutoscaleParameterSet (impostazione predefinita)
```
New-AzHDInsightClusterAutoscaleConfiguration -MinWorkerNodeCount <Int32> -MaxWorkerNodeCount <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ScheduleAutoscaleParameterSet
```
New-AzHDInsightClusterAutoscaleConfiguration -TimeZone <String>
 -Condition <System.Collections.Generic.List`1[Microsoft.Azure.Commands.HDInsight.Models.AzureHDInsightAutoscaleCondition]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzHDInsightClusterAutoscaleConfiguration** crea un oggetto non permanente che descrive la configurazione della scala automatica di un cluster Azure HDInsight.

## ESEMPI

### Esempio 1: Creare un oggetto che descrive la configurazione della gradazione automatica basata sul carico
```powershell
PS C:\> New-AzHDInsightClusterAutoscaleConfiguration -MinWorkerNodeCount 3 -MaxWorkerNodeCount 5
```

Questo comando crea un oggetto che descrive la configurazione della gradazione automatica basata sul carico.

### Esempio 2: Creare un oggetto che descrive la configurazione della scala automatica basata su pianificazione
```powershell
# Create an autoscale condition firstly
PS C:\> $condition=New-AzHDInsightClusterAutoscaleScheduleCondition -Day Monday -Time 09:00 -WorkerNodeCount 5
PS C:\> New-AzHDInsightClusterAutoscaleConfiguration -TimeZone ([System.TimeZoneInfo]::Local).Id `
        -Condition $condition
```

Questo comando crea un oggetto che descrive la configurazione della gradazione automatica basata su pianificazione.
