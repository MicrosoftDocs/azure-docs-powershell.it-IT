# Set-AzHDInsightPersistedScriptAction

## SYNOPSIS
Imposta un'azione script eseguita in precedenza come azione di script persistente.

## SINTASSI

```
Set-AzHDInsightPersistedScriptAction [-ClusterName] <String> [-ScriptExecutionId] <Int64>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzHDInsightPersistedScriptAction** imposta un'azione script eseguita in precedenza come azione di script persistente.
L'azione di script specificata deve avere avuto esito positivo in precedenza.
L'azione di script verrà eseguita ogni volta che il cluster Azure HDInsight viene ridimensionato.

## ESEMPI

### Esempio 1: Impostare il persistere di un'azione di script riuscita in precedenza o eseguirlo sul cluster
```
PS C:\>Set-AzHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -ScriptExecutionId "<id>"
```

Questo comando imposta un'azione di script eseguita in precedenza come azione di script persistente.
