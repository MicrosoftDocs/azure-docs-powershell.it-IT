# Disable-AzHDInsightMonitoring

## SYNOPSIS
Disabilita il monitoraggio in un cluster HDInsight e il flusso dei log pertinenti nell'area di lavoro di monitoraggio specificata durante l'abilitazione non verrà più attivato.

## SINTASSI

```
Disable-AzHDInsightMonitoring [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Disable-AzHDInsightMonitoring** disabilita il monitoraggio in un cluster Azure HDInsight.

## ESEMPI

### Esempio 1
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster

True
```

Il monitoraggio verrà disabilitato nel cluster HDInsight e i log pertinenti non verranno più efluiranno nell'area di lavoro di monitoraggio.

### Esempio 2
```
PS C:\> Disable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg

True
```

Il monitoraggio verrà disabilitato nel cluster HDInsight e il flusso dei log pertinenti nell'area di lavoro di monitoraggio non verrà più disattivato.
