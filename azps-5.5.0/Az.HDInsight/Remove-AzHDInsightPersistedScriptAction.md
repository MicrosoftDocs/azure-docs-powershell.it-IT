# Remove-AzHDInsightPersistedScriptAction

## SYNOPSIS
Rimuove un'azione di script persistente da un cluster HDInsight.

## SINTASSI

```
Remove-AzHDInsightPersistedScriptAction [-ClusterName] <String> [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzHDInsightPersistedScriptAction** rimuove un'azione script persistente dall'elenco di azioni degli script persistenti del cluster Azure HDInsight specificato.
Lo script rimosso non verrà più eseguito quando il cluster viene ridimensionato.

## ESEMPI

### Esempio 1: Rimuovere un'azione script dall'elenco di azioni di script persistenti in un cluster
```
PS C:\>Remove-AzHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -Name "Scriptaction"
```

Questo comando rimuove l'azione script denominata Scriptaction dall'elenco di azioni di script persistenti nel cluster specificato.
