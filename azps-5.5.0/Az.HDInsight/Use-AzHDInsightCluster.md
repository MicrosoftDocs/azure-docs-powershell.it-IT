# Use-AzHDInsightCluster

## SYNOPSIS
Seleziona un cluster da usare con il cmdlet Invoke-RmAzureHDInsightHiveJob cluster.

## SINTASSI

```
Use-AzHDInsightCluster [-ClusterName] <String> [-HttpCredential] <PSCredential> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Use-AzHDInsightCluster** seleziona il cluster Azure HDInsight per il cmdlet Invoke-AzHDInsightHiveJob da usare per inviare i processi Hive.

## ESEMPI

### Esempio 1: Selezionare un cluster per l'invio di query Hive
```
PS C:\># Cluster info
PS C:\>$clusterName = "your-hadoop-001"
PS C:\>$clusterCreds = Get-Credential

PS C:\>Use-AzHDInsightCluster `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Questo comando seleziona un cluster per l'invio di una query Hive.
