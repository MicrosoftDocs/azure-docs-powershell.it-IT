# Enable-AzHDInsightMonitoring

## SYNOPSIS
Consente il monitoraggio in un cluster HDInsight e i log pertinenti verranno inviati all'area di lavoro di monitoraggio specificata durante l'abilitazione.

## SINTASSI

```
Enable-AzHDInsightMonitoring [-Name] <String> [-WorkspaceId] <String> [-PrimaryKey] <String>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Enable-AzHDInsightMonitoring** abilita il monitoraggio in un cluster Azure HDInsight.

## ESEMPI

### Esempio 1
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

Il monitoraggio verrà abilitato nel cluster HDInsight e i log pertinenti verranno inviati all'area di lavoro di monitoraggio con ID 1d364e89-bb71-4503-aa3d-a23535aea7bd

### Esempio 2
```
PS C:\> Enable-AzHDInsightMonitoring -Name testcluster -ResourceGroupName testrg -WorkspaceId 1d364e89-bb71-4503-aa3d-a23535aea7bd -PrimaryKey <key for workspace 1d364e89-bb71-4503-aa3d-a23535aea7bd>

True
```

Il monitoraggio verrà abilitato nel cluster HDInsight e i log pertinenti verranno inviati all'area di lavoro di monitoraggio con ID 1d364e89-bb71-4503-aa3d-a23535aea7bd
