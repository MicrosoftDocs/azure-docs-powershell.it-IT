# Remove-AzStreamAnalyticsFunction

## SYNOPSIS
Elimina una funzione da un processo di Analisi di flusso.

## SINTASSI

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzStreamAnalyticsFunction** elimina una funzione in modo asincrono da un processo di Analisi di flusso di Azure.

## ESEMPI

### Esempio 1: Rimuovere una funzione di Analisi di flusso
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Questo comando rimuove la funzione denominata ScoreTweet dal processo denominato StreamJob22.
