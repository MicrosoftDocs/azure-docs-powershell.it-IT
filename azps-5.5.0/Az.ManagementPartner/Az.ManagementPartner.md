# Modulo Az.ManagementPartner
## Descrizione
Cmdlet dei partner di gestione per Gestione risorse di Azure.

## Cmdlet Az.ManagementPartner
### [Get-AzManagementPartner](Get-AzManagementPartner.md)
Ottiene l'ID Microsoft Partner Network (MPN) dell'utente autenticato corrente o dell'entità servizio. 

### [New-AzManagementPartner](New-AzManagementPartner.md)
Associa un ID Microsoft Partner Network (MPN) all'utente autenticato corrente o all'entità servizio.

### [Remove-AzManagementPartner](Remove-AzManagementPartner.md)
Eliminare l'ID Microsoft Partner Network (MPN) dell'utente autenticato corrente o dell'entità servizio.

### [Update-AzManagementPartner](Update-AzManagementPartner.md)
Aggiorna l'ID Microsoft Partner Network (MPN) dell'utente autenticato corrente o dell'entità servizio.
