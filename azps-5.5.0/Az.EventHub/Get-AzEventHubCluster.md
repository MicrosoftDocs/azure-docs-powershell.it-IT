# Get-AzEventHubCluster

## SYNOPSIS
Recupera i dettagli di un singolo cluster di hub eventi o ottiene un elenco di cluster di hub eventi.

## SINTASSI

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il Get-AzEventHubCluster cmdlet restituisce i dettagli di un cluster di hub eventi o un elenco di tutti i cluster di hub eventi in un determinato gruppo di risorse.
Se viene specificato il nome del cluster, vengono restituiti i dettagli di un singolo cluster.
Se non viene fornito il nome di un cluster, viene restituito un elenco di tutti i cluster nel gruppo di risorse specificato.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzEventHubCluster -ResourceGroupName RSG-Cluster27651 -Name Eventhub-Cluster-5557

Id        : /subscriptions/326100e2-f69d-4268-8503-075374f62b6e/resourceGroups/RSG-Cluster27651/providers/Microsoft.Eve
            ntHub/clusters/Eventhub-Cluster-5557
Name      : Eventhub-Cluster-5557
Location  : southcentralus
CreatedAt : 09/10/2020 22:09:57
UpdatedAt : 09/10/2020 23:02:48
MetricId  :
Status    :
Sku       : Microsoft.Azure.Commands.EventHub.Models.PSEventHubsClusterSkuAttributes
Tags      : {[ClusterTag2, Tag4], [ClusterTag1, Tag3]}

```

Restituisce i detial del cluster 'Eventhub-Cluster-5557' dal gruppo di risorse 'RSG-Cluster27651'
