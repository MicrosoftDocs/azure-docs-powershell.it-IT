# Get-AzEventHubClustersAvailableRegion

## SYNOPSIS
Recupera i dettagli di un singolo cluster Eventhub o dell'elenco di cluster nel gruppo di risorse specificato

## SINTASSI

```
Get-AzEventHubClustersAvailableRegion [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Elenco Get-AzEventHubClustersAvailableRegion cmdlet di aree geografiche in cui sono disponibili elementi dedicati per la creazione.

## ESEMPI

### Esempio 1
