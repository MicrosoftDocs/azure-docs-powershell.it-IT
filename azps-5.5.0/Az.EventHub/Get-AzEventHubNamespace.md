# Get-AzEventHubNamespace

## SYNOPSIS
Recupera i dettagli di uno spazio dei nomi Hub eventi o ottiene un elenco di tutti gli spazi dei nomi Hub eventi nella sottoscrizione di Azure corrente.

## SINTASSI

```
Get-AzEventHubNamespace [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Get-AzEventHubNamespace deve recuperare i dettagli di uno spazio dei nomi Hub eventi specificato oppure un elenco di tutti gli spazi dei nomi Hub eventi nella sottoscrizione di Azure corrente.
Se si specifica il nome dello spazio dei nomi, vengono restituiti i dettagli di un singolo spazio dei nomi Hub eventi.
Se il nome dello spazio dei nomi non viene specificato, viene restituito un elenco di spazi dei nomi.

