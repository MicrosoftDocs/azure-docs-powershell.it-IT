# Add-AzDataLakeStoreFirewallRule

## SYNOPSIS
Aggiunge una regola del firewall all'account Data Lake Store specificato.

## SINTASSI

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Add-AzDataLakeStoreFirewallRule** aggiunge una regola del firewall all'account Data Lake Store specificato.

## ESEMPI

### Esempio 1: Aggiungere una nuova regola del firewall a un account Data Lake Store
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

Viene creata una nuova regola del firewall denominata "MyRule" nell'account "ContosoADL" con un intervallo IP compreso tra 127.0.0.1 e 127.0.0.2.
