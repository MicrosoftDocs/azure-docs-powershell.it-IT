# Get-AzDataLakeStoreItemOwner

## SYNOPSIS
Ottiene il proprietario di un file o di una cartella in Data Lake Store.

## SINTASSI

```
Get-AzDataLakeStoreItemOwner [-Account] <String> [-Path] <DataLakeStorePathInstance> [-Type] <Owner>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzDataLakeStoreItemOwner** ottiene il proprietario di un file o di una cartella in Data Lake Store.

## ESEMPI

### Esempio 1: Ottenere il proprietario di una directory
```
PS C:\>Get-AzDataLakeStoreItemOwner -AccountName "ContosoADL" -Path / -Type User
```

Questo comando ottiene il proprietario dell'utente per la directory radice dell'account ContosoADL.
