# Remove-AzDataLakeStoreVirtualNetworkRule

## SYNOPSIS
Rimuove la regola di rete virtuale specificata nel Data Lake Store specificato.

## SINTASSI

```
Remove-AzDataLakeStoreVirtualNetworkRule [-Account] <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzDataLakeStoreVirtualNetworkRule** rimuove la regola di rete virtuale specificata nel data lake store specificato.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzDataLakeStoreVirtualNetworkRule -Account "dls" -Name "myVNET"
```

Rimuove la regola di rete virtuale "myVNET" dall'account "dls"
