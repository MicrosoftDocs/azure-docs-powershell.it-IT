# New-AzCostManagementExport

## SYNOPSIS
Operazione di creazione o aggiornamento di un'esportazione.
Per l'operazione di aggiornamento è necessario impostare l'eTag più recente nella richiesta.
È possibile ottenere l'eTag più recente eseguendo un'operazione get.
L'operazione di creazione non richiede un eTag.

## SINTASSI

```
New-AzCostManagementExport -Name <String> -Scope <String> [-ConfigurationColumn <String[]>]
 [-DataSetGranularity <GranularityType>] [-DefinitionTimeframe <TimeframeType>] [-DefinitionType <ExportType>]
 [-DestinationContainer <String>] [-DestinationResourceId <String>] [-DestinationRootFolderPath <String>]
 [-Format <FormatType>] [-RecurrencePeriodFrom <DateTime>] [-RecurrencePeriodTo <DateTime>]
 [-ScheduleRecurrence <RecurrenceType>] [-ScheduleStatus <StatusType>] [-TimePeriodFrom <DateTime>]
 [-TimePeriodTo <DateTime>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIZIONE
Operazione di creazione o aggiornamento di un'esportazione.
Per l'operazione di aggiornamento è necessario impostare l'eTag più recente nella richiesta.
È possibile ottenere l'eTag più recente eseguendo un'operazione get.
L'operazione di creazione non richiede un eTag.

### Esempio 1: Creare un'istanza di AzCostManagementExport
