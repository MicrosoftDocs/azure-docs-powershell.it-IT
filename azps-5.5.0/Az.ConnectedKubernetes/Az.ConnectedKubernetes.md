# Modulo Az.ConnectedKubernetes
## Descrizione
Microsoft Azure PowerShell: cmdlet ConnectedKubernetes

## Cmdlet di Az.ConnectedKubernetes
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Restituisce le proprietà del cluster connesso specificato, inclusi il nome, l'identità, le proprietà e altri dettagli del cluster.

### [New-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
API per registrare un nuovo cluster K8s e quindi creare una risorsa tracciata in ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Eliminare un cluster connesso, rimuovendo la risorsa monitorata in Gestione risorse di Azure (ARM).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
API per aggiornare determinate proprietà della risorsa cluster connessa
