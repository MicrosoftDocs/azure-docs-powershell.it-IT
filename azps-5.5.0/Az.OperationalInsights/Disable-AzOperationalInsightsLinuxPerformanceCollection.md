# Disable-AzOperationalInsightsLinuxPerformanceCollection

## SYNOPSIS
Interrompe la raccolta di contatori delle prestazioni dai computer Linux.

## SINTASSI

### ByWorkspaceName (Impostazione predefinita)
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByWorkspaceObject
```
Disable-AzOperationalInsightsLinuxPerformanceCollection [-Workspace] <PSWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Disable-AzOperationalInsightsLinuxPerformanceCollection** interrompe la raccolta di contatori delle prestazioni dai computer Linux connessi in un'area di lavoro.
