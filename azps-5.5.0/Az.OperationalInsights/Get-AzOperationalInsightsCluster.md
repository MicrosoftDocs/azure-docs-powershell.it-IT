# Get-AzOperationalInsightsCluster

## SYNOPSIS
Ottenere o elencare cluster

## SINTASSI

```
Get-AzOperationalInsightsCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Ottenere o elencare cluster, cluster di elenchi in un gruppo di risorse quando non Ã¨ stato fornito "-ClusterName", cluster di elenchi in abbonamento quando non sono stati forniti "-ClusterName" e "ResourceGroupName".
