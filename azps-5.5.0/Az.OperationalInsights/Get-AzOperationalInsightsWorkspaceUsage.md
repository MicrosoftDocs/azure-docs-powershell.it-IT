# Get-AzOperationalInsightsWorkspaceUsage

## SYNOPSIS
Recupera i dati di utilizzo per un'area di lavoro.

## SINTASSI

```
Get-AzOperationalInsightsWorkspaceUsage [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzOperationalInsightsWorkspaceUsage** recupera i dati di utilizzo per un'area di lavoro.
In questo modo viene esponga la quantitÃ  di dati analizzati dall'area di lavoro in un determinato periodo.

## ESEMPI

### Esempio 1: Ottenere i dati di utilizzo in base al nome dell'area di lavoro
```
PS C:\>Get-AzOperationalInsightsWorkspaceUsage -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace"
```

Questo comando recupera i dettagli sull'utilizzo dell'area di lavoro denominata MyWorkspace nel gruppo di risorse specificato.

### Esempio 2: Ottenere i dati di utilizzo usando la pipeline
```
PS C:\>Get-AzOperationalInsightsWorkspace -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace" | Get-AzOperationalInsightsWorkspaceUsage
```

Questo comando recupera l'area di lavoro denominata MyWorkSpace usando il cmdlet Get-AzOperationalInsightsWorkspace e quindi passa l'area di lavoro al cmdlet corrente.
Il comando ottiene i dettagli di utilizzo per quell'area di lavoro.
