# Get-AzOperationalInsightsStorageInsight

## SYNOPSIS
Ottiene informazioni su informazioni sull'archiviazione.

## SINTASSI

### ByWorkspaceName (Impostazione predefinita)
```
Get-AzOperationalInsightsStorageInsight [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByWorkspaceObject
```
Get-AzOperationalInsightsStorageInsight [-Workspace] <PSWorkspace> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzOperationalInsightsStorageInsight** ottiene informazioni su un approfondimento di archiviazione esistente.
Se si specifica un nome di Informazioni dettagliate di archiviazione, questo cmdlet ottiene informazioni su tale informazione.
Se non si specifica un nome, questo cmdlet riceve informazioni su tutte le informazioni di archiviazione in un'area di lavoro.

## ESEMPI

### Esempio 1: Ottenere informazioni dettagliate sull'archiviazione in base al nome
```
PS C:\>Get-AzOperationalInsightsStorageInsight -Name "MyStorageInsight" -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace"
```

Questo comando recupera le informazioni di archiviazione denominate MyStorageInsight dall'area di lavoro ContosoWorkspace.

### Esempio 2: Ottenere informazioni dettagliate sull'archiviazione usando un oggetto area di lavoro
```
PS C:\>$Workspace = Get-AzOperationalInsightsWorkspace -ResourceGroupName "ContosoResourceGroup" -Name "MyWorkspace"
PS C:\>Get-AzOperationalInsightsStorageInsight -Workspace $Workspace -Name "MyStorageInsight"
```

Il primo comando usa il cmdlet **Get-AzOperationalInsightsWorkspace** per ottenere un'area di lavoro di Operational Insights e quindi la archivia nella variabile $Workspace lavoro.
Il secondo comando ottiene le informazioni di archiviazione denominate MyStorageInsight per l'area di lavoro di $Workspace.
