# Set-AzOperationalInsightsIntelligencePack

## SYNOPSIS
Abilita o disabilita il Pacchetto di intelligence specificato.

## SINTASSI

```
Set-AzOperationalInsightsIntelligencePack [-ResourceGroupName] <String> [-WorkspaceName] <String>
 [-IntelligencePackName] <String> [-Enabled] <Boolean> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzOperationalInsightsIntelligencePack** abilita il Pacchetto di intelligence specificato se *Enabled* è impostato su $True e lo disabilita se *Enabled* è impostato su $False.

## ESEMPI

### Esempio 1: Impostare i pacchetti di intelligence
```
PS C:\>Set-AzOperationalInsightsIntelligencePack -ResourceGroupName "ContosoResourceGroup" -WorkspaceName "ContosoWorkspace" -IntelligencePackName "ContosoWorkspace" -Enabled $True
```

Questo comando abilita il Pacchetto di intelligence specificato.
