# Get-AzSecurityPricing

## SYNOPSIS

Ottiene i piani di Azure Defender per una sottoscrizione in Azure Security Center.

## SINTASSI

### SubscriptionScope (impostazione predefinita)

```powershell
Get-AzSecurityPricing [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource

```powershell
Get-AzSecurityPricing -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId

```powershell
Get-AzSecurityPricing -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE

È possibile visualizzare ogni piano di Azure Defender, per ogni abbonamento, usando questo cmdlet.

Per informazioni dettagliate su Azure Defender e sui piani disponibili, vedere [Introduzione ad Azure Defender.](https://docs.microsoft.com/azure/security-center/azure-defender)

### Esempio 2

```powershell
PS C:\> Get-AzSecurityPricing -ResourceId
```

Recupera i dettagli sui prezzi dell'ID risorsa specifico. Dove ResourceId è uno degli ID restituiti da `Get-AzSecurityPricing` .

### Esempio 3

```powershell
PS C:\> Get-AzSecurityPricing -Name
```

Recupera i dettagli sui prezzi del piano di Azure Defender denominato. Dove `name` si trova uno dei nomi restituiti da `Get-AzSecurityPricing` .
