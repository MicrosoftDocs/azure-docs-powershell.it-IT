# Get-AzRegulatoryComplianceControl

## SYNOPSIS
Ottiene i controlli di conformità alle normative

## SINTASSI

### SubscriptionLevelResource (Default)
```
Get-AzRegulatoryComplianceControl [-Name <String>] -StandardName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzRegulatoryComplianceControl -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Ottenere informazioni dettagliate sui controlli oppure elencare tutti i controlli in base a specifici standard di conformità alle normative.
