# Get-AzSecurityCompliance

## SYNOPSIS
Ottenere la conformità della sicurezza di un abbonamento nel corso del tempo

## SINTASSI

### SubscriptionScope (impostazione predefinita)
```
Get-AzSecurityCompliance [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource
```
Get-AzSecurityCompliance -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSecurityCompliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Ottiene la conformità della sicurezza di un abbonamento in base al rapporto corrente delle risorse integre e non protette dell'abbonamento.
La conformità alla sicurezza viene calcolata ogni giorno e la cronologia viene salvata.
