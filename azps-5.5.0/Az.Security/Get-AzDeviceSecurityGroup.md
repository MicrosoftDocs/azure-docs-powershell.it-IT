# Get-AzDeviceSecurityGroup

## SYNOPSIS
Ottenere il gruppo di sicurezza del dispositivo (sicurezza dell'Hub IoT)

## SINTASSI

### ResourceIdScope (impostazione predefinita)
```
Get-AzDeviceSecurityGroup -HubResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdLevelResource
```
Get-AzDeviceSecurityGroup -HubResourceId <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il Get-AzDeviceSecurityGroup cmdlet di sicurezza del dispositivo restituisce un gruppo di sicurezza dei dispositivi definito nella soluzione di sicurezza iot
