# New-AzApiManagementCertificate

## SYNOPSIS
Crea un certificato di gestione API da usare durante l'autenticazione con back-end.

## SINTASSI

### LoadFromFile (impostazione predefinita)
```
New-AzApiManagementCertificate -Context <PsApiManagementContext> [-CertificateId <String>]
 -PfxFilePath <String> -PfxPassword <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Non elaborati
```
New-AzApiManagementCertificate -Context <PsApiManagementContext> [-CertificateId <String>] -PfxBytes <Byte[]>
 -PfxPassword <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzApiManagementCertificate** crea un certificato di Gestione API di Azure.

## ESEMPI

### Esempio 1: Creare e caricare un certificato
```powershell
PS C:\>$ApiMgmtContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>New-AzApiManagementCertificate -Context $ApiMgmtContext -PfxFilePath "C:\contoso\certificates\apimanagement.pfx" -PfxPassword "1111"
```

Questo comando carica un certificato in Gestione API. Questo certificato può essere usato per l'autenticazione a comune con back-end mediante criteri.

### Esempio 2

Crea un certificato di gestione API da usare durante l'autenticazione con back-end. (autogenerated)

```powershell
<!-- Aladdin Generated Example --> 
New-AzApiManagementCertificate -CertificateId '0123456789' -Context <PsApiManagementContext> -PfxFilePath 'C:\contoso\certificates\apimanagement.pfx' -PfxPassword '1111'
```
