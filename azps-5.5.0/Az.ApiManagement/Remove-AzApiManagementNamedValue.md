# Remove-AzApiManagementNamedValue

## SYNOPSIS
Rimuove un valore denominato gestione API.

## SINTASSI

```
Remove-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApiManagementNamedValue** rimuove un valore denominato Gestione API **di** Azure.

## ESEMPI

### Esempio 1: Rimuovere il valore denominato
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -PassThru
```

Questo comando rimuove il valore denominato con ID Property11.
