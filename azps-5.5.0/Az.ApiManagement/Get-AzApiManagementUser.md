# Get-AzApiManagementUser

## SYNOPSIS
Ottiene uno o più utenti.

## SINTASSI

### GeAllUsers (impostazione predefinita)
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByUserId
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-UserId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByUser
```
Get-AzApiManagementUser -Context <PsApiManagementContext> [-FirstName <String>] [-LastName <String>]
 [-State <PsApiManagementUserState>] [-Email <String>] [-GroupId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzApiManagementUser** ottiene un utente specificato o tutti gli utenti, se non è specificato alcun utente.

## ESEMPI

### Esempio 1: Ottenere tutti gli utenti
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext
```

Questo comando consente di ottenere tutti gli utenti.

### Esempio 2: Ottenere un utente in base all'ID
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -UserId "0123456789"
```

Questo comando ottiene un utente in base all'ID.

### Esempio 3: Ottenere utenti per cognome
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -LastName "Fuller"
```

Questo comando consente di ottenere gli utenti con un cognome specificato, più completo.

### Esempio 4: Ottenere un utente in base all'indirizzo di posta elettronica
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -Email "user@contoso.com"
```

Questo comando consente di ottenere l'utente che ha l'indirizzo di posta elettronica specificato.

### Esempio 5: Ottenere tutti gli utenti all'interno di un gruppo
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementUser -Context $apimContext -GroupId "0001"
```

Questo comando recupera tutti gli utenti all'interno del gruppo specificato.
