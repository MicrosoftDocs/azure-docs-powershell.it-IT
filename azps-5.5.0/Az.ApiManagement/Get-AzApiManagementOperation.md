# Get-AzApiManagementOperation

## SYNOPSIS
Ottiene un elenco o un'operazione API specificata.

## SINTASSI

### GetAllApiOperations (Impostazione predefinita)
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetById
```
Get-AzApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> [-ApiRevision <String>]
 -OperationId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
**Get-AzApiManagementOperation** ottiene un elenco o un'operazione API specificata.

## ESEMPI

### Esempio 1: Ottenere tutte le operazioni di gestione delle API
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId
```

Questo comando recupera tutte le operazioni di gestione delle API.

### Esempio 2: Ottenere un'operazione di gestione delle API in base all'ID operazione
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOperation -Context $apimContext -ApiId $APIId -OperationId "Operation003"
```

Questo comando recupera un'operazione di gestione api in base all'ID operazione denominato Operation0003.
