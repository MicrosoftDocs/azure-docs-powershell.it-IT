# Uninstall-AzureRm

## SYNOPSIS
Rimuove tutti i moduli di AzureRm da un computer.

## SINTASSI

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Rimuove tutti i moduli di AzureRm da un computer.

## ESEMPI

### Esempio 1
```
PS C:\> Uninstall-AzureRm
```

L'esecuzione di questo comando rimuove tutti i moduli di AzureRm dal computer per la versione di PowerShell in cui viene eseguito il cmdlet.
