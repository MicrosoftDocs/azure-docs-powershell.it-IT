# Enable-AzureRmAlias

## SYNOPSIS
Abilita gli alias dei prefissi di AzureRm per i moduli Az.

## SINTASSI

```
Enable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Abilita gli alias dei prefissi di AzureRm per i moduli Az. Se si specificato -Module, solo i moduli elencati avranno gli alias abilitati. In caso contrario, sono abilitati tutti gli alias di AzureRm.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Enable-AzureRmAlias
```

Abilita tutti i prefissi AzureRm per la sessione di PowerShell corrente.

### Esempio 2
```powershell
PS C:\> Enable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Abilita gli alias AzureRm per il modulo Az.Accounts sia per il processo corrente che per l'utente corrente.
