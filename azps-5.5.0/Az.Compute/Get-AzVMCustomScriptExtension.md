# Get-AzVMCustomScriptExtension

## SYNOPSIS
Ottiene informazioni su un'estensione di script personalizzata.

## SINTASSI

```
Get-AzVMCustomScriptExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzVMCustomScriptExtension** ottiene informazioni su uno script personalizzato Virtual Machine Extension in una macchina virtuale.

## ESEMPI

### Esempio 1: Ottenere un'estensione di script personalizzata
```
PS C:\> $VMCustomScriptExtension = Get-AzVMCustomScriptExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoCustomScript"
```

Questo comando ottiene l'estensione di script personalizzata denominata ContosoCustomScript per la macchina virtuale denominata VirtualMachine07.

### Esempio 2: Ottenere la visualizzazione dell'istanza di un'estensione di script personalizzata
```
PS C:\> $VMCustomScriptExtension = Get-AzVMCustomScriptExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine07" -Name "ContosoCustomScript" -Status
```

Questo comando ottiene la visualizzazione dell'istanza dell'estensione di script personalizzata denominata ContosoCustomScript per la macchina virtuale denominata VirtualMachine07.
