# Get-AzVmssSku

## SYNOPSIS
Ottiene gli SKU disponibili per VMSS.

## SINTASSI

```
Get-AzVmssSku [-ResourceGroupName] <String> [-VMScaleSetName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzVmssSku** ottiene gli SKU disponibili per il set di scala delle macchine virtuali (VMSS).

## ESEMPI

### Esempio 1: Ottenere tutti gli SKU disponibili da VMSS
```
PS C:\> Get-AzVmssSku -ResourceGroupName "ContosoGroup" -VMScaleSetName "ContosoVMSS"
```

Questo comando recupera tutti gli SKU disponibili dal sistema VMSS denominato ContosoVMSS che appartiene al gruppo di risorse denominato ContosoGroup.
