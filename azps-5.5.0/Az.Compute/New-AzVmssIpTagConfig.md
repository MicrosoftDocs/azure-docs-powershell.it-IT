# New-AzVmssIpTagConfig

## SYNOPSIS
Crea un oggetto tag IP per un'interfaccia di rete di un VMSS.

## SINTASSI

```
New-AzVmssIpTagConfig [-IpTagType] <String> [-Tag <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzVmssIpTagConfig** crea un oggetto di configurazione del tag IP per un'interfaccia di rete di un set di scala di macchine virtuali (VMSS).
Specificare la configurazione di questo cmdlet come *parametro IPTag* del cmdlet New-AzVmssIpConfig.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $iptag = New-AzVmssIpTagConfig -IpTagType 'FirstPartyUsage' -Tag 'Sql'
PS C:\> $ipCfg = New-AzVmssIPConfig -Name 'test' -SubnetId $subnetId -IpTag $ipTag;
```

Questo comando crea un oggetto locale tag IP con tipo 'FirstPartyUsage' e tag 'Sql', quindi crea una configurazione IP con questo tag IP.
