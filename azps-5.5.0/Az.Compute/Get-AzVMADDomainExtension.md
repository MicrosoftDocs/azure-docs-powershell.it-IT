# Get-AzVMADDomainExtension

## SYNOPSIS
Ottiene informazioni su un'estensione di dominio Active Directory.

## SINTASSI

```
Get-AzVMADDomainExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzVMADDomainExtension** ottiene informazioni sull'estensione di dominio di Azure Active Directory (AD) specificata.

## ESEMPI

### Esempio 1

Ottiene informazioni su un'estensione di dominio Active Directory. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Get-AzVMADDomainExtension -Name 'AgentPool01' -ResourceGroupName myresourcegroup -VMName 'VM01'
```
