# Get-AzVMDiagnosticsExtension

## SYNOPSIS
Recupera le impostazioni dell'estensione di diagnostica in una macchina virtuale.

## SINTASSI

```
Get-AzVMDiagnosticsExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzVMDiagnosticsExtension** ottiene le impostazioni dell'estensione di diagnostica di Azure in una macchina virtuale.

## ESEMPI

### Esempio 1: Ottenere l'estensione di diagnostica applicata a una macchina virtuale
```powershell
PS C:\> Get-AzVMDiagnosticsExtension -ResourceGroupName "ResourceGroup11" -VMName "ContosoVM22"
```

Questo comando applica l'estensione di diagnostica alla macchina virtuale ContosoVM22.

### Esempio 2

Recupera le impostazioni dell'estensione di diagnostica in una macchina virtuale. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Get-AzVMDiagnosticsExtension -Name 'AgentPool01' -ResourceGroupName 'ResourceGroup11' -Status -VMName 'ContosoVM22'
```
