# Get-AzVMExtension

## SYNOPSIS
Ottiene le proprietà di Virtual Machine Extensions installate in una macchina virtuale.

## SINTASSI

### GetExtensionParameterSet (impostazione predefinita)
```
Get-AzVMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### VMParameterSet
```
Get-AzVMExtension [-Status] [-VMObject <PSVirtualMachine>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzVMExtension [-Status] [-ResourceId <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzVMExtension** ottiene le proprietà delle estensioni delle macchine virtuali installate in una macchina virtuale.
Specificare il nome di un'estensione per cui ottenere le proprietà.
Per ottenere solo la visualizzazione dell'istanza di un'estensione, specificare il parametro Status.

## ESEMPI

### Esempio 1: Ottenere le proprietà di un'estensione
```
PS C:\> Get-AzVMExtension -ResourceGroupName "ResourceGroup11" -VMName "VirtualMachine22" -Name "CustomScriptExtension"
```
