# Add-AzVmssWinRMListener

## SYNOPSIS
Aggiunge un listener WinRM al sistema VMSS.

## SINTASSI

```
Add-AzVmssWinRMListener [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Protocol] <ProtocolTypes>]
 [[-CertificateUrl] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Add-AzVmssWinRM Consente** di aggiungere un listener Windows Remote Management (WinRM) nel set di scala delle macchine virtuali (VMSS).

## ESEMPI

### Esempio 1: Aggiungere un listener WinRM a VMSS
```
PS C:\> $VMSS = New-AzVmssConfig
PS C:\> Add-AzVmssWinRMListener -VirtualMachineScaleSet $VMSS -Protocol Https -CertificateUrl "http://keyVaultName.vault.contoso.net/secrets/secretName/secretVersion"
```

Questo esempio aggiunge un listener WinRM a VMSS.
Il primo comando usa il cmdlet **New-AzVmssConfig** per creare un oggetto di configurazione VMSS e archivia il risultato nella variabile $VMSS.
Il secondo comando aggiunge un listener WinRM del protocollo HTTP con il certificato all'URL specificato al sistema VMSS.
