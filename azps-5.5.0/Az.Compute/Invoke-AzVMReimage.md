# Invoke-AzVMReimage

## SYNOPSIS
Eseguire di nuovo l'immagine di una macchina virtuale di Azure.

## SINTASSI

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Invoke-AzVMReimage** ricrea un'immagine di una macchina virtuale di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Questo comando mostra nuovamente la macchina virtuale denominata VirtualMachine07 in ResourceGroup11.
