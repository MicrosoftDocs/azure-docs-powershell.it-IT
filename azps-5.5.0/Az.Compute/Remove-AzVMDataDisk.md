# Remove-AzVMDataDisk

## SYNOPSIS
Rimuove un disco dati da una macchina virtuale.

## SINTASSI

```
Remove-AzVMDataDisk [-VM] <PSVirtualMachine> [[-DataDiskNames] <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzVMDataDisk** rimuove un disco dati da una macchina virtuale.

## ESEMPI

### Esempio 1: Rimuovere un disco dati da una macchina virtuale
```
PS C:\> $VirtualMachine = Get-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07" 
PS C:\> Remove-AzVMDataDisk -VM $VirtualMachine -Name "Disk3"
PS C:\> Update-AzVM -ResourceGroupName "ResourceGroup11" -VM $VirtualMachine
```

Il primo comando recupera la macchina virtuale denominata VirtualMachine07 usando il cmdlet **Get-AzVM.**
Il comando archivia la macchina virtuale nella $VirtualMachine variabile.
Il secondo comando rimuove il disco dati denominato Disco3 dalla macchina virtuale archiviata in $VirtualMachine.
Il comando finale aggiorna lo stato della macchina virtuale archiviata in $VirtualMachine in ResourceGroup11.
