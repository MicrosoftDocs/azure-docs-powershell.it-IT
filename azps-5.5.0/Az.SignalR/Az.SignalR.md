# Modulo Az.SignalR
## Descrizione
Cmdlet per interagire con il servizio Azure SignalR.

## Cmdlet Az.SignalR
### [Get-AzSignalR](Get-AzSignalR.md)
Ottieni un servizio SignalR specifico o tutti i servizi SignalR in un gruppo di risorse o un abbonamento.

### [Get-AzSignalRKey](Get-AzSignalRKey.md)
Ottieni i tasti di scelta di un servizio SignalR.

### [Get-AzSignalRUsage](Get-AzSignalRUsage.md)
Ottenere la quota di utilizzo di un abbonamento.

Crea un servizio SignalR.
