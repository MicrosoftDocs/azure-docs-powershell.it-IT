# Remove-AzTrafficManagerEndpoint

## SYNOPSIS
Rimuove un endpoint da Gestione traffico.

## SINTASSI

### Campi
```
Remove-AzTrafficManagerEndpoint -Name <String> -Type <String> -ProfileName <String> -ResourceGroupName <String>
 [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Oggetto
```
Remove-AzTrafficManagerEndpoint -TrafficManagerEndpoint <TrafficManagerEndpoint> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzTrafficManagerEndpoint** rimuove un endpoint da Gestione traffico di Azure.

Questo cmdlet esegue il commit di ogni modifica nel servizio Di gestione del traffico.
Per rimuovere più endpoint da un oggetto profilo di Gestione traffico locale ed eseguire il commit delle modifiche in una singola operazione, usare il cmdlet Remove-AzTrafficManagerEndpointConfig locale.

È possibile usare l'operatore della pipeline per passare un oggetto **TrafficManagerEndpoint** a questo cmdlet oppure è possibile specificare un **oggetto TrafficManagerEndpoint** usando il *parametro TrafficManagerEndpoint.*

In alternativa, è possibile specificare il nome e il tipo dell'endpoint usando i parametri *Name* e *Type,* insieme ai parametri *ProfileName* e *ResourceGroupName.*

## ESEMPI

### Esempio 1: Rimuovere un endpoint da un profilo
```
PS C:\>Remove-AzTrafficManagerEndpoint -Name "contoso" -ProfileName "ContosoProfile" -ResourceGroupName "ResourceGroup11" -Type AzureEndpoints
```

Questo comando rimuove l'endpoint di Azure denominato contoso dal profilo denominato ContosoProfile nel gruppo di risorse denominato ResourceGroup11.
