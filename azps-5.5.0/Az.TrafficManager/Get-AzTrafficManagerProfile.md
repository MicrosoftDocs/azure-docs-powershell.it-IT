# Get-AzTrafficManagerProfile

## SYNOPSIS
Ottiene un profilo di Gestione traffico.

## SINTASSI

### ResourceGroupParameterSet
```
Get-AzTrafficManagerProfile [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### AccountNameParameterSet
```
Get-AzTrafficManagerProfile [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzTrafficManagerProfile** ottiene un profilo di Gestione traffico di Azure e restituisce un oggetto che rappresenta tale profilo.
Specificare un profilo in base al nome e al gruppo di risorse.

Ãˆ possibile modificare l'oggetto in locale e quindi applicarvi le modifiche usando il cmdlet Set-AzTrafficManagerProfile>

## ESEMPI

### Esempio 1: Ottenere un profilo
```
PS C:\>Get-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11"
```

Questo comando recupera il profilo denominato ContosoProfile in ResourceGroup11.
