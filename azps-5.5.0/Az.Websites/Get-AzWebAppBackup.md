# Get-AzWebAppBackup

## SYNOPSIS

## SINTASSI

### FromResourceName
```
Get-AzWebAppBackup [-BackupId] <String> [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### FromWebApp
```
Get-AzWebAppBackup [-BackupId] <String> [-WebApp] <PSSite> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzWebAppBackup** ottiene il backup specificato di un'app Web di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\>Get-AzWebAppBackup -ResourceGroupName "Default-Web-WestUS" -Name "WebAppStandard" -BackupId "12345"
```

Questo comando recupera il backup con ID "12345" dall'app Web denominata WebAppStandard che appartiene al gruppo di risorse Default-Web-WestUS.
