# Remove-AzAppServicePlan

## SYNOPSIS
Rimuove un piano di servizio app di Azure.

## SINTASSI

### S1
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### S2
```
Remove-AzAppServicePlan [-Force] [-AsJob] [-AppServicePlan] <PSAppServicePlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzAppServicePlan** rimuove un piano di servizio app di Azure.

## ESEMPI

### Esempio 1: Rimuovere un piano di servizio app
```
PS C:\>Remove-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

Questo comando rimuove il piano di servizio app di Azure denominato ContosoASP che appartiene al gruppo di risorse denominato Default-Web-WestUS.
