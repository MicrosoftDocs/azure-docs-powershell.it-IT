# Restore-AzWebAppSnapshot

## SYNOPSIS
Ripristina uno snapshot di un'app Web.

## SINTASSI

### FromResourceName
```
Restore-AzWebAppSnapshot [-RecoverConfiguration] [-UseDisasterRecovery] [-Force] [-AsJob]
 [-ResourceGroupName] <String> [-Name] <String> [[-Slot] <String>] [-DefaultProfile <IAzureContextContainer>]
 [-InputObject] <AzureWebAppSnapshot> [-WhatIf] [-Confirm] [<CommonParameters>]
```

### FromWebApp
```
Restore-AzWebAppSnapshot [-RecoverConfiguration] [-UseDisasterRecovery] [-Force] [-AsJob] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [-InputObject] <AzureWebAppSnapshot> [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Ripristina uno snapshot di un'app Web nell'app Web. Il ripristino di uno snapshot sovrascrive tutti i file di un'app Web con i file contenuti nello snapshot. Anche per ripristinare le impostazioni, usare il parametro RecoverConfiguration. È possibile ripristinare uno snapshot di un'app Web in qualsiasi altra app Web nello stesso abbonamento.

## ESEMPI

### Esempio 1
```
PS C:\> $snapshot = (Get-AzWebAppSnapshot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoApp" -Slot "Staging")[0]
PS C:\> Restore-AzWebAppSnapshot -ResourceGroupName "Default-Web-WestUS" -Name "ContosoApp" -Slot "Restore" -InputObject $snapshot -RecoverConfiguration
```

Ottiene lo snapshot più recente di un'app Web denominata "ContosoApp" con uno slot denominato "Gestione temporanea" nel gruppo di risorse "Default-Web-WestUS". Ripristina lo snapshot nello slot "Ripristina" dell'app Web.
