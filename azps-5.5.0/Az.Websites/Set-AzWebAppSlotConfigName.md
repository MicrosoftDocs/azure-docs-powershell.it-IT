# Set-AzWebAppSlotConfigName

## SYNOPSIS
Impostare i nomi di configurazione degli slot per le app Web

## SINTASSI

### S1
```
Set-AzWebAppSlotConfigName [[-AppSettingNames] <String[]>] [[-ConnectionStringNames] <String[]>]
 [-RemoveAllAppSettingNames] [-RemoveAllConnectionStringNames] [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Set-AzWebAppSlotConfigName [[-AppSettingNames] <String[]>] [[-ConnectionStringNames] <String[]>]
 [-RemoveAllAppSettingNames] [-RemoveAllConnectionStringNames] [-WebApp] <PSSite>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzWebAppSlotConfigName** contrassegna le impostazioni delle app e le stringhe di connessione come impostazioni dello slot

## ESEMPI

### Esempio 1
```powershell
PS C:\> Set-AzWebAppSlotConfigName -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -RemoveAllAppSettingNames -RemoveAllConnectionStringNames
```

Questo comando rimuove tutte le impostazioni dell'app e le stringhe di connessione per l'app Web ContosoWebApp associata al gruppo di risorse Default-Web-WestUS

### Esempio 2

Impostare i nomi di configurazione degli slot per le app Web. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Set-AzWebAppSlotConfigName -AppSettingNames <String[]> -Name 'ContosoWebApp' -ResourceGroupName 'Default-Web-WestUS'
```
