# Remove-AzRedisCacheDiagnostic

## SYNOPSIS
Disabilita la diagnostica in una cache di Ridis di Azure.

## SINTASSI

```
Remove-AzRedisCacheDiagnostic [-ResourceGroupName <String>] -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzRedisCacheDiagnostic** disabilita la diagnostica in una cache di Azure Redis.

## ESEMPI

### Esempio 1: Disabilitare la diagnostica
```
PS C:\>Remove-AzRedisCacheDiagnostic -ResourceGroupName "ContosoResourceGroup" -Name "PeakCache" -Force
```

Questo comando disabilita la diagnostica nella cache di Ridisposizione di Azure specificata.
In questo modo la diagnostica per tutte le cache di Azure Redis nella stessa area geografica viene disabilitata per la sottoscrizione.
