# Get-AzRedisCacheKey

## SYNOPSIS
Recupera i tasti di scelta per una cache di Ridis.

## SINTASSI

```
Get-AzRedisCacheKey [-ResourceGroupName <String>] -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzRedisCacheKey** ottiene le chiavi di accesso per una cache di Azure Redis.

## ESEMPI

### Esempio 1: Ottenere le chiavi di accesso per una cache di Redis
```
PS C:\>Get-AzRedisCacheKey -ResourceGroupName "MyResourceGroup" -Name "MyCacheKey"
PrimaryKey        : pJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
SecondaryKey      : sJ+jruGKPHDKsEC8kmoybobH3TZx2njBR3ipEsquZFo=
```

Questo comando recupera le chiavi di accesso denominate MyCacheKey.
