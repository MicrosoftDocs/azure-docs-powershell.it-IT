# New-AzRedisCachePatchSchedule

## SYNOPSIS
Aggiunge una pianificazione delle patch.

## SINTASSI

```
New-AzRedisCachePatchSchedule [-ResourceGroupName <String>] -Name <String> -Entries <PSScheduleEntry[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzRedisCachePatchSchedule aggiunge** una pianificazione delle patch a una cache in Azure Redis Cache.

## ESEMPI

### Esempio 1: Creare e aggiungere una pianificazione delle patch in una cache
```
PS C:\>New-AzRedisCachePatchSchedule -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -Entries @(New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00")
```

Questo comando aggiunge una pianificazione delle patch alla cache denominata RedisCache06.
Il parametro Entries accetta come valore un comando che usa **New-AzRedisCacheScheduleEntry** per creare una pianificazione.
