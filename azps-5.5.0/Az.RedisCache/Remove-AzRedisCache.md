# Remove-AzRedisCache

## SYNOPSIS
Rimuove una cache di Ridis.

## SINTASSI

```
Remove-AzRedisCache [-ResourceGroupName <String>] -Name <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzRedisCache** rimuove una cache di Azure Redis.

## ESEMPI

### Esempio 1: Rimuovere una cache di Ridis e restituire il risultato
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force -PassThru
True
```

Questo comando rimuove una cache di Ridis e indica se l'operazione riesce.

### Esempio 2: Rimuovere una cache di Ridis e non visualizzare il risultato
```
PS C:\>Remove-AzRedisCache -ResourceGroupName "ResourceGroup03" -Name "myCache" -Force
```

Questo comando rimuove una cache di Ridis.
Poiché il *parametro PassThru* non è specificato, il risultato dell'operazione non viene visualizzato.
