# Get-AzSqlDatabaseTransparentDataEncryption

## SYNOPSIS
Ottiene lo stato TDE per un database.

## SINTASSI

```
Get-AzSqlDatabaseTransparentDataEncryption [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlDatabaseTransparentDataEncryption** ottiene lo stato di Transparent Data Encryption (TDE) per un database SQL Azure.
Per altre informazioni, vedere Transparent Data Encryption con Azure SQL Database https://msdn.microsoft.com/library/dn948096 ( nella Microsoft Developer Network https://msdn.microsoft.com/library/dn948096) Library.
Questo cmdlet ottiene lo stato corrente di TDE, ma sia la crittografia che la decrittografia possono essere operazioni a esecuzione lunga.
Per vedere lo stato dell'analisi di crittografia, eseguire il cmdlet Get-AzSqlDatabaseTransparentDataEncryptionActivity crittografia.
Questo cmdlet è supportato anche dal servizio SQL Server di estensione del database in Azure.

## ESEMPI

### Esempio 1: Ottenere lo stato TDE per un database
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryption -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName             ServerName                    DatabaseName                                          State
-----------------             ----------                    ------------                                          -----
resourcegroup01               server01                      database01                                            Disabled
```

Questo comando ottiene lo stato di TDE per il database denominato Database01 nel server denominato server01.
