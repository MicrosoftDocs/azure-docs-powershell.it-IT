# Get-AzSqlDatabaseGeoBackupPolicy

## SYNOPSIS
Ottiene i criteri di backup geografici del database.

## SINTASSI

```
Get-AzSqlDatabaseGeoBackupPolicy [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzSqlDatabaseGeoBackupPolicy** ottiene il criterio di backup geografico registrato nel database.
Si tratta di una risorsa di backup di Azure usata per definire i criteri di archiviazione di backup.

## ESEMPI

### Esempio 1

Ottiene i criteri di backup geografici del database. (autogenerated)

```powershell
<!-- Aladdin Generated Example --> 
Get-AzSqlDatabaseGeoBackupPolicy -DatabaseName db1 -ResourceGroupName MyResourceGroup -ServerName s1
```
