# Enable-AzSqlInstanceAdvancedDataSecurity

## SYNOPSIS
Abilita Advanced Data Security per un'istanza gestita.

## SINTASSI

```
Enable-AzSqlInstanceAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Enable-AzSqlInstanceAdvancedDataSecurity** abilita Advanced Data Security in un'istanza gestita. Advanced Data Security è un pacchetto di sicurezza unificato che include la classificazione dei dati, la valutazione delle vulnerabilità e Advanced Threat Protection per l'istanza gestita. Verrà creato automaticamente un nuovo account di archiviazione per il salvataggio delle valutazioni di vulnerabilità. Se un account di archiviazione è stato creato in precedenza per questo scopo, verrà usato al suo posto)

## ESEMPI

### Esempio 1: Abilitare Advanced Data Security per le istanze gestite
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Esempio 2: Abilitare Advanced Data Security per le istanze gestite dalla risorsa server
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Esempio 3

Abilita Advanced Data Security per un'istanza gestita. (autogenerated)

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlInstanceAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -InstanceName 'ContosoManagedInstanceName' -ResourceGroupName MyResourceGroup
```
