# Disable-AzSqlServerAdvancedDataSecurity

## SYNOPSIS
Disabilita Advanced Data Security in un server.

## SINTASSI

```
Disable-AzSqlServerAdvancedDataSecurity [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Disable-AzSqlServerAdvancedDataSecurity** disabilita Advanced Data Security in un server.

## ESEMPI

### Esempio 1
### Esempio 2: Disabilitare Advanced Data Security del server
```powershell
PS C:\>  Disable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```

### Esempio 3: Disabilitare Advanced Data Security del server dalla risorsa server
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Disable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : False
```
