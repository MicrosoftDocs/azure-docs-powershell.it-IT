# Remove-AzSqlServerDisasterRecoveryConfiguration

## SYNOPSIS
Rimuove una configurazione SQL ripristino del sistema del server di database.

## SINTASSI

```
Remove-AzSqlServerDisasterRecoveryConfiguration [-VirtualEndpointName] <String> [-Force] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzSqlServerDisasterRecoveryConfiguration** rimuove una SQL di ripristino del sistema del server di database.
