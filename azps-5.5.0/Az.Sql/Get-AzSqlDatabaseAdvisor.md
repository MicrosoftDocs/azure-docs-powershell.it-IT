# Get-AzSqlDatabaseAdvisor

## SYNOPSIS
Ottiene uno o più consulenti per un database SQL Azure.

## SINTASSI

```
Get-AzSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlDatabaseAdvisor** ottiene uno o più consulenti database di Azure SQL per un database SQL Azure.
