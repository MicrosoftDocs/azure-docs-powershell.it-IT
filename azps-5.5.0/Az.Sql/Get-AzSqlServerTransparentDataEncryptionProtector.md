# Get-AzSqlServerTransparentDataEncryptionProtector

## SYNOPSIS
Ottiene la protezione TDE (Transparent Data Encryption)

## SINTASSI

```
Get-AzSqlServerTransparentDataEncryptionProtector [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il Get-AzSqlServerTransparentDataEncryptionProtector cmdlet riceve informazioni sulla protezione TDE.

## ESEMPI

### Esempio 1: Ottenere la protezione TDE (Transparent Data Encryption)
```
PS C:\> Get-AzSqlServerTransparentDataEncryptionProtector -ServerName 'ContosoServer' -ResourceGroup 'ContosoResourceGroup'
```

Questo comando ottiene la protezione TDE per il server denominato ContosoServer nel gruppo di risorse ContosoResourceGroup.
