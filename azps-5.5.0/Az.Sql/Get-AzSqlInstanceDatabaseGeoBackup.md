# Get-AzSqlInstanceDatabaseGeoBackup

## SYNOPSIS
Restituisce informazioni sul backup ridondante del database SQL istanza gestita di Azure.

## SINTASSI

```
Get-AzSqlInstanceDatabaseGeoBackup [[-Name] <String>] [-InstanceName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlInstanceDatabaseGeoBackup** ottiene uno o piÃ¹ backup ridondanti di Azure SQL da un'istanza gestita di database SQL Azure.
