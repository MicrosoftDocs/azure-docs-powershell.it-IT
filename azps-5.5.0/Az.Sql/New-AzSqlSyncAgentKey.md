# New-AzSqlSyncAgentKey

## SYNOPSIS
Crea una chiave dell'SQL di sincronizzazione di Azure.

## SINTASSI

```
New-AzSqlSyncAgentKey [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzSqlSyncAgentKey** crea una chiave dell'SQL di sincronizzazione di Azure.

## ESEMPI

### Esempio 1: Creare una chiave dell'agente di sincronizzazione per un SQL Azure.
```
PS C:\> New-AzSqlSyncAgentKey -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01"
SyncAgentKey                  : Key
```

Questo comando crea una chiave dell'agente di sincronizzazione per un agente di SQL Azure.
