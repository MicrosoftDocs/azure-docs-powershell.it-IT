# Stop-AzSqlDatabaseActivity

## SYNOPSIS
Annulla l'operazione di aggiornamento asincrono sul database.

## SINTASSI

```
Stop-AzSqlDatabaseActivity [-ServerName] <String> [-ElasticPoolName <String>] -DatabaseName <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Stop-AzSqlDatabaseActivity** annulla l'operazione di aggiornamento asincrono nel database.

## ESEMPI

### Esempio 1: Annullare l'operazione di aggiornamento asincrono sul database
