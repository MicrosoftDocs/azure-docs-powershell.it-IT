# Remove-AzSqlServerMSSupportAudit

## SYNOPSIS
Rimuove le impostazioni di controllo delle operazioni di supporto Microsoft di un server SQL Azure.

## SINTASSI

### ServerParameterSet (impostazione predefinita)
```
Remove-AzSqlServerMSSupportAudit [-ResourceGroupName] <String> [-ServerName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ServerObjectParameterSet
```
Remove-AzSqlServerMSSupportAudit -ServerObject <AzureSqlServerModel> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzSqlServerMSSupportAudit** rimuove le impostazioni di controllo delle operazioni di supporto Microsoft di un server SQL Azure.
Per usare il cmdlet, usare i *parametri ResourceGroupName* e *ServerName* per identificare il server.

## ESEMPI

### Esempio 1: Rimuovere le impostazioni di controllo delle operazioni di supporto Microsoft di un server SQL Azure
```powershell
PS C:\>Remove-AzSqlServerMSSupportAudit -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

### Esempio 2: Rimuovere tramite pipeline le impostazioni di controllo del supporto Microsoft di un server SQL Azure
```
PS C:\> Get-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01" | Remove-AzSqlServerMSSupportAudit
```
