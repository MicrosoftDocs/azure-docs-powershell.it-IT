# Get-AzSqlDatabaseReplicationLink

## SYNOPSIS
Recupera i collegamenti di replica geografica tra un database di SQL Azure e un gruppo di risorse o un SQL Server.

## SINTASSI

```
Get-AzSqlDatabaseReplicationLink [-DatabaseName] <String> -PartnerResourceGroupName <String>
 [-PartnerServerName <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlDatabaseReplicationLink** sostituisce il cmdlet **Get-AzSqlDatabaseCopy.**
Recupera tutti i collegamenti di replica geografica tra il database di SQL azure specificato e un gruppo di risorse o un server AzureSQL.

## ESEMPI

### Esempio 1

Recupera i collegamenti di replica geografica tra un database di SQL Azure e un gruppo di risorse o un SQL Server. (autogenerated)

```powershell
<!-- Aladdin Generated Example --> 
Get-AzSqlDatabaseReplicationLink -DatabaseName db1 -PartnerResourceGroupName rg2 -ResourceGroupName MyResourceGroup -ServerName s1
```
