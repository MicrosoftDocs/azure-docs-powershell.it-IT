# New-AzSqlServerCommunicationLink

## SYNOPSIS
Crea un collegamento di comunicazione per le transazioni di database elastiche tra due SQL server di database.

## SINTASSI

```
New-AzSqlServerCommunicationLink -LinkName <String> -PartnerServer <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzSqlServerCommunicationLink** crea un collegamento di comunicazione per le transazioni di database elastiche tra due server logici in Azure SQL Database.
Le transazioni di database di tipo elastico possono estendersi su database di uno dei server associati.
È possibile creare più collegamenti in un server.
Di conseguenza, le transazioni di database elastiche possono estendersi su un numero maggiore di server.

## ESEMPI

### Esempio 1: Creare un collegamento di comunicazione
```
PS C:\>New-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01" -PartnerServer "ContosoServer02"
```

Questo comando crea un collegamento denominato Link01 tra ContosoServer17 e ContosoServer02.
