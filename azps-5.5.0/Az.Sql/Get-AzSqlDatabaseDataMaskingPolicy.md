# Get-AzSqlDatabaseDataMaskingPolicy

## SYNOPSIS
Ottiene i criteri di maschera dati per un database.

## SINTASSI

```
Get-AzSqlDatabaseDataMaskingPolicy [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzSqlDatabaseDataMaskingPolicy** ottiene i criteri di maschera dei dati di un database SQL Azure.
Per usare questo cmdlet, usare i *parametri ResourceGroupName,* *ServerName* e *DatabaseName* per identificare il database.
Questo cmdlet è supportato anche dal servizio SQL Server di estensione del database in Azure.

## ESEMPI

### Esempio 1: Ottenere i criteri di maschera dati per un database SQL Azure
```
PS C:\>Get-AzSqlDatabaseDataMaskingPolicy -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
DatabaseName      : Database01
ResourceGroupName : ResourceGroup01
ServerName        : Server01
DataMaskingState  : Enabled
PrivilegedUsers  :
```

Questo comando recupera il criterio di maschera dati dal database Database01 nel gruppo di risorse ResourceGroup01 sul server Server01.
