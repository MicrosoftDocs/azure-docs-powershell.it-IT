# Get-AzSqlElasticPool

## SYNOPSIS
Recupera i pool elastici e i relativi valori di proprietà in un database SQL Azure.

## SINTASSI

```
Get-AzSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlElasticPool** ottiene pool di elastici e i relativi valori di proprietà.
Specificare il nome di un pool flessibile esistente per visualizzare i valori delle proprietà solo per tale pool.
