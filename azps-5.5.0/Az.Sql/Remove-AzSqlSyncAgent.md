# Remove-AzSqlSyncAgent

## SYNOPSIS
Rimuove un agente di sincronizzazione SQL Azure.

## SINTASSI

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzSqlSyncAgent** rimuove un agente di SQL Azure.

## ESEMPI

### Esempio 1: Rimuovere un agente di sincronizzazione
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Questo comando rimuove l'agente di SQL Azure denominato syncAgent01.
