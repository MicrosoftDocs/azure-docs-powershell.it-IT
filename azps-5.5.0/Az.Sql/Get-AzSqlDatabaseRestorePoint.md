# Get-AzSqlDatabaseRestorePoint

## SYNOPSIS
Recupera i diversi punti di ripristino da cui è possibile ripristinare SQL data warehouse.

## SINTASSI

```
Get-AzSqlDatabaseRestorePoint [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlDatabaseRestorePoint** recupera i distinti punti di ripristino da cui è possibile ripristinare un data warehouse di SQL Azure.
Per un database SQL Azure, la finestra di ripristino è continua.
Questo significa che qualsiasi momento nel periodo di conservazione del backup del database può essere usato come punto di ripristino.
Questo cmdlet è supportato anche dal servizio SQL Server di estensione del database in Azure.

## ESEMPI

### Esempio 1: Ottenere tutti i punti di ripristino
```
PS C:\>Get-AzSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : CONTINUOUS
RestorePointCreationDate : 
EarliestRestoreDate      : 8/12/2015 12:00:00 AM
RestorePointLabel        : RestorePoint01
```

Questo comando restituisce tutti i punti di ripristino disponibili per il database SQL Azure denominato Database01.
