# Get-AzSqlDatabaseIndexRecommendation

## SYNOPSIS
Recupera le operazioni di indice consigliate per un server o un database.

## SINTASSI

```
Get-AzSqlDatabaseIndexRecommendation -ServerName <String> [-DatabaseName <String>] [-TableName <String>]
 [-IndexRecommendationName <String>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlDatabaseIndexRecommendation** ottiene le operazioni di indice consigliate per un database o un server di SQL azure.

## ESEMPI

### Esempio 1: Ottenere consigli sugli indici per tutti i database nel server
```
PS C:\>Get-AzSqlDatabaseIndexRecommendation -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Questo comando restituisce gli indici consigliati per tutti i database sul server server01.

### Esempio 2: Ottenere consigli sull'indice per un database specifico
```
PS C:\>Get-AzSqlDatabaseIndexRecommendation -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
```

Questo comando restituisce gli indici consigliati per un database specifico.

### Esempio 3: Ottenere un singolo suggerimento per l'indice in base al nome
```
PS C:\>Get-AzSqlDatabaseIndexRecommendation -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01" -IndexRecommendationName "INDEX_NAME"
```

Questo comando restituisce un singolo suggerimento per l'indice in base al nome.
