# Get-AzSqlElasticPoolDatabase

## SYNOPSIS
Recupera i database elastici in un pool flessibile e i relativi valori di proprietà.

## SINTASSI

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlElasticPoolDatabase** ottiene database elastici in un pool flessibile e i relativi valori di proprietà.
È possibile specificare il nome di un database elastico in Azure SQL database per visualizzare i valori delle proprietà solo per tale database.

## ESEMPI

### Esempio 1: Recuperare tutti i database in un pool flessibile
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Questo comando recupera tutti i database in un pool elastico denominato ElasticPool01.

### Esempio 2: Recuperare tutti i database in un pool flessibile usando i filtri
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

Questo comando recupera tutti i database di un pool elastico denominato ElasticPool01 che iniziano con "Database".
