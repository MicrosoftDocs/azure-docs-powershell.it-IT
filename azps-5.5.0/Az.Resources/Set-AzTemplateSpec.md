## DESCRIZIONE
Modifica una specifica di Templace. Se la specifica del modello con il nome e/o la versione specificata non esiste già, verrà creata.

Quando si modifica il contenuto del modello ARM Di una versione specifica del modello, il contenuto può essere derivato da una stringa JSON non elaborata (usando il set di parametri **UpdateVersionByNameFromJsonParameterSet)** o da un file JSON specificato (usando il set di parametri **UpdateVersionByNameFromJsonFileParameterSet).**

## ESEMPI

### Esempio 1:
```powershell
PS C:\> $templateJson = @"
{
    "$schema": "http://schema.management.azure.com/schemas/2015-01-01/deploymentTemplate.json#",
    "contentVersion": "1.0.0.0",
    "parameters": {},
    "resources": []
}
"@
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v1.0' -Location 'West US' -TemplateJson $templateJson
```

Modifica la versione "v1.0" di una specifica di modello denominata "myTemplateSpec". La versione specificata avrà $templateJson contenuto del modello ARM versione. Se la versione radice delle specifiche del modello e/o non esiste già, verranno create.


**Note:** 

* Il ARM predefinito nell'esempio non ha alcuna attività perché non contiene risorse effettive.
* Il percorso è obbligatorio solo se la specifica del modello non esiste già

### Esempio 2:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec' -Version 'v2.0' -Location 'West US' -TemplateFile 'myTemplateContent.json'
```

Modifica la versione "v2.0" di una specifica di modello denominata "myTemplateSpec". La versione specificata avrà il contenuto del file locale "myTemplateContent.js" come contenuto del modello ARM versione. Se la versione radice delle specifiche del modello e/o non esiste già, verranno create.

**Nota:** Il percorso è obbligatorio solo se la specifica del modello non esiste già

### Esempio 3:
```powershell
PS C:\> Set-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'myTemplateSpec'  -Location 'West US' -Description 'My updated Template Spec'
```

Modifica la descrizione della specifica del modello denominata "myTemplateSpec" nel gruppo di risorse "myRG". Se la specifica del modello non esiste già, verrà creata.

**Nota:** Il percorso è obbligatorio solo se la specifica del modello non esiste già
