# Get-AzPolicyAlias

## SYNOPSIS
Get-AzPolicyAlias recupera e restituisce i tipi di risorse del provider di Azure che hanno alias definiti e corrispondenti ai valori dei parametri specificati. Se non vengono forniti parametri, verranno restituiti tutti i tipi di risorse provider che contengono un alias.
L'opzione -ListAvailable modifica questo comportamento elencando tutti i tipi di risorse corrispondenti, inclusi quelli senza alias.

## SINTASSI

```
Get-AzPolicyAlias [-NamespaceMatch <String>] [-ResourceTypeMatch <String>] [-AliasMatch <String>]
 [-PathMatch <String>] [-ApiVersionMatch <String>] [-LocationMatch <String>] [-ListAvailable]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzPolicyAlias** ottiene un elenco di alias dei criteri.
Gli alias dei criteri vengono usati dai criteri di Azure per fare riferimento alle proprietà del tipo di risorsa.
Vengono forniti parametri che limitano gli elementi nell'elenco abbinando varie proprietà del tipo di risorsa o i relativi alias.
Un determinato valore di corrispondenza corrisponde se la stringa di destinazione lo contiene usando un confronto senza distinzione tra maiuscole e minuscole.
