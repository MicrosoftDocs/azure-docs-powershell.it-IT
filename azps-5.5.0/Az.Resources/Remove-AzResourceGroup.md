# Remove-AzResourceGroup

## SYNOPSIS
Rimuove un gruppo di risorse.

## SINTASSI

### RemoveByResourceGroupName (impostazione predefinita)
```
Remove-AzResourceGroup [-Name] <String> [-Force] [-AsJob] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupId
```
Remove-AzResourceGroup -Id <String> [-Force] [-AsJob] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzResourceGroup** rimuove un gruppo di risorse di Azure e le relative risorse dalla sottoscrizione corrente.
Per eliminare una risorsa, ma uscire dal gruppo di risorse, usare il cmdlet Remove-AzResource risorse.

## ESEMPI

### Esempio 1: Rimuovere un gruppo di risorse
```
PS C:\>Remove-AzResourceGroup -Name "ContosoRG01"
```

Questo comando rimuove il gruppo di risorse ContosoRG01 dalla sottoscrizione.
Il cmdlet chiede conferma e non restituisce alcun output.

### Esempio 2: Rimuovere un gruppo di risorse senza conferma
```
PS C:\>Get-AzResourceGroup -Name "ContosoRG01" | Remove-AzResourceGroup -Force
```

Questo comando usa il cmdlet Get-AzResourceGroup per ottenere il gruppo di risorse ContosoRG01 e quindi lo passa a **Remove-AzResourceGroup** usando l'operatore della pipeline. Il *parametro Force* elimina la richiesta di conferma.

### Esempio 3: Rimuovere tutti i gruppi di risorse
```
PS C:\>Get-AzResourceGroup | Remove-AzResourceGroup
```

Questo comando usa il cmdlet **Get-AzResourceGroup** per ottenere tutti i gruppi di risorse e quindi li passa **a Remove-AzResourceGroup usando** l'operatore della pipeline.
