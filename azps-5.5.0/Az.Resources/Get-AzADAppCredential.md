# Get-AzADAppCredential

## SYNOPSIS
Recupera un elenco di credenziali associate a un'applicazione.

## SINTASSI

### ApplicationObjectIdParameterSet (impostazione predefinita)
```
Get-AzADAppCredential -ObjectId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ApplicationIdParameterSet
```
Get-AzADAppCredential -ApplicationId <Guid> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Get-AzADAppCredential -DisplayName <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ApplicationObjectParameterSet
```
Get-AzADAppCredential -ApplicationObject <PSADApplication> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il Get-AzADAppCredential cmdlet può essere usato per recuperare un elenco di credenziali associate a un'applicazione.
Questo comando recupera tutte le proprietà delle credenziali (ma non il valore delle credenziali) associate all'applicazione.

## ESEMPI

### Esempio 1: Ottenere le credenziali dell'applicazione in base all'ID oggetto

```powershell
PS C:\> Get-AzADAppCredential -ObjectId 1f99cf81-0146-4f4e-beae-2007d0668476
```

Restituisce un elenco di credenziali associate all'applicazione con ID oggetto '1f99cf81-0146-4f4e-beae-2007d0668476'.

### Esempio 2: Ottenere le credenziali dell'applicazione tramite piping

```powershell
PS C:\> Get-AzADApplication -ObjectId 1f99cf81-0146-4f4e-beae-2007d0668476 | Get-AzADAppCredential
```

Ottiene l'applicazione con ID oggetto '1f99cf81-0146-4f4e-beae-2007d0668476' e la pipe al cmdlet di Get-AzADAppCredential per elencare tutte le credenziali per l'applicazione.
