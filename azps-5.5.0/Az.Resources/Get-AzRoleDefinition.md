# Get-AzRoleDefinition

## SYNOPSIS
Elenco di tutti i ruoli di Azure RBAC disponibili per l'assegnazione.

## SINTASSI

### RoleDefinitionNameParameterSet (Impostazione predefinita)
```
Get-AzRoleDefinition [[-Name] <String>] [-Scope <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### RoleDefinitionIdParameterSet
```
Get-AzRoleDefinition -Id <Guid> [-Scope <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### RoleDefinitionCustomParameterSet
```
Get-AzRoleDefinition [-Scope <String>] [-Custom] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Usare il Get-AzRoleDefinition comando Visualizza con un particolare nome di ruolo per visualizzarne i dettagli.
Per esaminare le singole operazioni a cui un ruolo concede l'accesso, esaminare le proprietÃ  Actions e NotActions del ruolo.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzRoleDefinition -Name Reader
```

Ottenere la definizione del ruolo lettore

### Esempio 2
```
PS C:\> Get-AzRoleDefinition
```

Elenco di tutte le definizioni di ruoli RBAC
