# Remove-AzManagedApplicationDefinition

## SYNOPSIS
Rimuove una definizione di applicazione gestita

## SINTASSI

### RemoveByNameAndResourceGroup (impostazione predefinita)
```
Remove-AzManagedApplicationDefinition -Name <String> -ResourceGroupName <String> [-Force]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### RemoveById
```
Remove-AzManagedApplicationDefinition -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzManagedApplicationDefinition rimuove** una definizione di applicazione gestita

## ESEMPI

### Esempio 1: Rimuovere la definizione dell'applicazione gestita in base all'ID risorsa
```
PS C:\>$ApplicationDefinition = Get-AzManagedApplicationDefinition -Name "myAppDef" -ResourceGroupName "myRG"
PS C:\>Remove-AzManagedApplicationDefinition -Id $ApplicationDefinition.ResourceId -Force
```

Il primo comando ottiene una definizione di applicazione gestita denominata myAppDef usando il cmdlet Get-AzManagedApplicationDefinition.
Il comando lo archivia nella $ApplicationDefinition variabile.
Il secondo comando rimuove la definizione di applicazione gestita identificata dalla **proprietà ResourceId** dell'$ApplicationDefinition.
