# Remove-AzDeployment

## SYNOPSIS
Rimuove una distribuzione ed eventuali operazioni associate

## SINTASSI

### RemoveByDeploymentName (impostazione predefinita)
```
Remove-AzDeployment [-Name] <String> [-AsJob] [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByDeploymentId
```
Remove-AzDeployment -Id <String> [-AsJob] [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByInputObject
```
Remove-AzDeployment -InputObject <PSDeployment> [-AsJob] [-PassThru] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzDeployment rimuove** una distribuzione di Azure con ambito di sottoscrizione ed eventuali operazioni associate.

## ESEMPI

### Esempio 1: Rimuovere una distribuzione con un nome assegnato
```
PS C:\>Remove-AzDeployment -Name "RolesDeployment"
```

Questo comando rimuove la distribuzione "RolesDeployment" nell'ambito della sottoscrizione corrente.

### Esempio 2: Ottenere una distribuzione e rimuoverla
```
PS C:\>Get-AzDeployment -Name "RolesDeployment" | Remove-AzDeployment
```

Questo comando recupera la distribuzione "RolesDeployment" nell'ambito di sottoscrizione corrente e la rimuove.
