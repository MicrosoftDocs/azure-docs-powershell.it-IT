# Remove-AzADGroupMember

## SYNOPSIS
Rimuove un utente da un gruppo active directory.

## SINTASSI

### ExplicitParameterSet (impostazione predefinita)

Remove-AzADGroupMember [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

```

## DESCRIZIONE
Rimuove un utente da un gruppo active directory.

## ESEMPI

### Esempio 1: Rimuovere un utente da un gruppo per ID oggetto

```powershell
PS C:\> Remove-AzADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405 -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Rimuove l'utente con ID oggetto 'D9076BBC-D62C-4105-9C78-A7F5BC4A3405' dal gruppo con ID oggetto '85F89C90-780E-4AA6-9F4F-6F268D322EEE'.

### Esempio 2: Rimuovere un utente da un gruppo tramite piping

```powershell
PS C:\> Get-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Remove-AzADGroupMember -MemberObjectId D9076BBC-D62C-4105-9C78-A7F5BC4A3405
```

Ottiene il gruppo con ID oggetto '85F89C90-780E-4AA6-9F4F-6F268D322EEE' e lo sua pipe al cmdlet di Remove-AzADGroupMember per rimuovere l'utente al gruppo.
