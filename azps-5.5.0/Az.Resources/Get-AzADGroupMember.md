# Get-AzADGroupMember

## SYNOPSIS
Elenca i membri di un gruppo Active Directory nel tenant corrente.

## SINTASSI

### ObjectIdParameterSet (impostazione predefinita)
```
Get-AzADGroupMember [-GroupObjectId <String>] [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Get-AzADGroupMember -GroupDisplayName <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### GroupObjectParameterSet
```
Get-AzADGroupMember -GroupObject <PSADGroup> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

## DESCRIZIONE
Elenca i membri di un gruppo Active Directory nel tenant corrente.

## ESEMPI

### Esempio 1: Elencare i membri in base all'ID oggetto gruppo Active Directory

```powershell
PS C:\> Get-AzADGroupMember -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

Elenca i membri del gruppo Active Directory con ID oggetto '85F89C90-780E-4AA6-9F4F-6F268D322EEE'.

### Esempio 2: Elencare i membri in base all'ID oggetto gruppo di Active Directory usando il paging

```powershell
PS C:\> Get-AzADGroupMember -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE -First 100
```

Elenca i primi 100 membri del gruppo Active Directory con ID oggetto '85F89C90-780E-4AA6-9F4F-6F268D322EEE'.

### Esempio 3: Elencare i membri tramite piping

```powershell
PS C:\> Get-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Get-AzADGroupMember
```

Ottiene il gruppo Active Directory con ID oggetto '85F89C90-780E-4AA6-9F4F-6F268D322EEE' e lo pipe al cmdlet di Get-AzADGroupMember per elencare tutti i membri del gruppo.
