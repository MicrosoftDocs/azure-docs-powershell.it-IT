# Import-AzBlueprintWithArtifact

## SYNOPSIS
Importare un file di progetto in formato JSON in un oggetto progetto e salvarlo all'interno della sottoscrizione o del gruppo di gestione specificato.

## SINTASSI

```
Import-AzBlueprintWithArtifact -Name <String> [-SubscriptionId <String>] [-ManagementGroupId <String>]
 -InputPath <String> [-IncludeSubFolders] [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Importare una definizione di progetto con i relativi elementi. 

## ESEMPI

### Esempio 1
```powershell
PS C:\> Import-AzBlueprintWithArtifact -Name MySimpleBlueprint -SubscriptionId 00000000-1111-0000-1111-000000000000 -InputPath  C:\Blueprints\SimpleBlueprint
```

Importare una definizione di progetto con i relativi elementi e salvarla all'interno di una sottoscrizione.
