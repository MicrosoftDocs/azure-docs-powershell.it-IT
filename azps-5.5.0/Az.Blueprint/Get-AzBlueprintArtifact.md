# Get-AzBlueprintArtifact

## SYNOPSIS
Recuperare elementi da una definizione di progetto.

## SINTASSI

```
Get-AzBlueprintArtifact [-Name <String>] -Blueprint <PSBlueprintBase> [-BlueprintVersion <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Recuperare elementi da una definizione di progetto. Se non si specifica una versione della definizione del progetto, viene recuperata la bozza. Nel caso in cui non sia presente una bozza, viene restituita l'ultima definizione del progetto pubblicata.
