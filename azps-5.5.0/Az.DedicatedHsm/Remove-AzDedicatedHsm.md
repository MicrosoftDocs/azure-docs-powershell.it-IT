# Remove-AzDedicatedHsm

## SYNOPSIS
Elimina il servizio HSM dedicato di Azure specificato.

## SINTASSI

### Elimina (impostazione predefinita)
```
Remove-AzDedicatedHsm -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzDedicatedHsm -InputObject <IDedicatedHsmIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## DESCRIZIONE
Elimina il servizio HSM dedicato di Azure specificato.

## ESEMPI

### Esempio 1: Rimuovere un servizio HSM dedicato in base al nome
```powershell
PS C:\> Remove-AzDedicatedHsm -Name hsm-7t2xaf -ResourceGroupName lucas-manual-test

```

Questo commnad rimuove un modulo di sicurezza hardware (HSM) in base al nome.

### Esempio 2: Rimuovere un HSM dedicato in base all'oggetto
```powershell
PS C:\> $hsm = Get-AzDedicatedHsm -Name hsm-7t2xaf -ResourceGroupName dedicatedhsm-rg-n359cz
PS C:\> Remove-AzDedicatedHsm -InputObject  $hsm

```

Questo commnad rimuove un HSM dedicato in base all'oggetto.
