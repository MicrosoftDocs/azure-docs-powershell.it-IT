# Get-AzMariaDbReplica

## SYNOPSIS
Elencare tutte le replica per un determinato server.

## SINTASSI

```
Get-AzMariaDbReplica -ResourceGroupName <String> -ServerName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIZIONE
Elencare tutte le replica per un determinato server.

## ESEMPI

### Esempio 1: Elencare tutti i database di replica in un database di MariaDB
```powershell
PS C:\> Get-AzMariaDbReplica -ServerName mariadb-test-szp6dt -ResourceGroupName mariadb-test-qu5ov0

Name                       Location AdministratorLogin Version StorageProfileStorageMb SkuName   SkuTier        SslEnforcement
```

Questo comando elenca tutti i database di replica in un database di MariaDB.
