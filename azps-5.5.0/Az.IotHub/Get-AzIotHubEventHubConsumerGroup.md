# Get-AzIotHubEventHubConsumerGroup

## SYNOPSIS
Ottiene tutti i gruppi consumer di eventhub.

## SINTASSI

```
Get-AzIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Ottiene tutti i gruppi consumer di eventhub per i diversi EventHub usati da IotHub.

## ESEMPI

### Esempio 1 ottiene tutti i gruppi consumer di eventhub per il eventhub di telemetria
```
PS C:\> Get-AzIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub"
```

Ottiene tutti i gruppi consumer di eventhub per la telemetria eventhub per il iothub denominato myiothub
