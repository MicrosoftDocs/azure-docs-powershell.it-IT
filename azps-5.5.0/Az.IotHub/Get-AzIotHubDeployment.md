# Get-AzIotHubDeployment

## SYNOPSIS
Elenca tutte le distribuzioni Edge IoT o una particolare.

## SINTASSI

### ResourceSet (impostazione predefinita)
```
Get-AzIotHubDeployment [-ResourceGroupName] <String> [-IotHubName] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObjectSet
```
Get-AzIotHubDeployment [-InputObject] <PSIotHub> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIdSet
```
Get-AzIotHubDeployment [-ResourceId] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Ottenere i dettagli di una distribuzione Edge IoT o elencare le distribuzioni Edge IoT in un Hub IoT.
Per https://docs.microsoft.com/azure/iot-edge/module-deployment-monitoring altre informazioni, vedere.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Name "deploy1"
```

Ottenere i dettagli di una distribuzione Edge IoT.

### Esempio 2
```powershell
PS C:\> Get-AzIotHubDeployment -ResourceGroupName "myresourcegroup" -IotHubName "myiothub"
```

Elencare tutte le distribuzioni Edge IoT in un Hub IoT.
