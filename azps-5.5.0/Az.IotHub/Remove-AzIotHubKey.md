# Remove-AzIotHubKey

## SYNOPSIS
Rimuove una chiave IotHub.

## SINTASSI

### ResourceSet (impostazione predefinita)
```
Remove-AzIotHubKey [-ResourceGroupName] <String> [-Name] <String> [-KeyName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Remove-AzIotHubKey [-HubId] <String> [-KeyName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Rimuove una chiave IotHub.
Se sono presenti piÃ¹ chiavi con lo stesso nome, la prima nell'elenco viene rimossa.

## ESEMPI

### Esempio 1: Eliminare un iotHub
```
PS C:\> Remove-AzIotHubKey -ResourceGroupName "myresourcegroup" -Name "myiothub" -KeyName "iothubowner1"
```

Rimuove la chiave denominata iothubowner1 da IotHub denominata "myiothub"
