# Stop-AzDataMigrationService

## SYNOPSIS
Interrompe un'istanza del servizio di migrazione del database di Azure in esecuzione.

## SINTASSI

### ComponentNameParameterSet (Impostazione predefinita)
```
Stop-AzDataMigrationService -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ComponentObjectParameterSet
```
Stop-AzDataMigrationService [-InputObject] <PSDataMigrationService> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Stop-AzDataMigrationService [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet Stop-AzDataMigrationService interrompe un'istanza del servizio di migrazione del database di Azure in esecuzione.

## ESEMPI

### Esempio 1
```
PS C:\> Stop-AzDataMigrationService -ResourceGroupName MyResourceGroup -ServiceName TestService
```

L'esempio precedente interrompe un'istanza del servizio di migrazione del database di Azure denominata TestService in base al nome del servizio passato come parametro di input

### Esempio 2
```
PS C:\> Stop-AzDataMigrationService -InputObject $TestService
```

L'esempio precedente interrompe un'istanza del servizio di migrazione del database di Azure in base all'oggetto PSDataMigrationService passato come parametro di input.
