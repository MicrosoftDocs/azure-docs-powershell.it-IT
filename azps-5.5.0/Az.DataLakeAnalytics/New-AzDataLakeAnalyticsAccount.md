# New-AzDataLakeAnalyticsAccount

## SYNOPSIS
Crea un account Data Lake Analytics.

## SINTASSI

```
New-AzDataLakeAnalyticsAccount [-ResourceGroupName] <String> [-Name] <String> [-Location] <String>
 [-DefaultDataLakeStore] <String> [[-Tag] <Hashtable>] [-MaxAnalyticsUnits <Int32>] [-MaxJobCount <Int32>]
 [-QueryStoreRetention <Int32>] [-Tier <TierType>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzDataLakeAnalyticsAccount** crea un account Azure Data Lake Analytics.

## ESEMPI

### Esempio 1: Creare un account Data Lake Analytics
```
PS C:\>New-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount" -ResourceGroupName "ContosoOrg" -Location "East US 2" -DefaultDataLakeStore "ContosoAdlStore"
```

Questo comando crea un account Data Lake Analytics denominato ContosoAdlAccount che usa l'archivio dati ContosoAdlStore nel gruppo di risorse denominato ContosoOrg.
