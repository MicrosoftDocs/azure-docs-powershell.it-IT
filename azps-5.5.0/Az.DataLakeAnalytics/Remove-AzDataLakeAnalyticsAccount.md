# Remove-AzDataLakeAnalyticsAccount

## SYNOPSIS
Elimina un account di Data Lake Analytics.

## SINTASSI

```
Remove-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzDataLakeAnalyticsAccount** elimina definitivamente un account Azure Data Lake Analytics.

## ESEMPI

### Esempio 1: Rimuovere un account
```
PS C:\>Remove-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Questo comando rimuove l'account di Data Lake Analytics specificato.
