# Test-AzDataLakeAnalyticsAccount

## SYNOPSIS
Verifica l'esistenza di un account Data Lake Analytics.

## SINTASSI

```
Test-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Test-AzDataLakeAnalyticsAccount** verifica l'esistenza di un account Data Lake Analytics.

## ESEMPI

### Esempio 1: Verificare l'esistenza di un account
```
PS C:\>Test-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Questo comando verifica l'esistenza dell'account denominato ContosoAdlAccount.
