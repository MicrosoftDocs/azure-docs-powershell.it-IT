# New-AzKeyVaultCertificateOrganizationDetail

## SYNOPSIS
Crea un oggetto dettagli dell'organizzazione di certificati in memoria.

## SINTASSI

```
New-AzKeyVaultCertificateOrganizationDetail [-Id <String>]
 [-AdministratorDetails <System.Collections.Generic.List`1[Microsoft.Azure.Commands.KeyVault.Models.PSKeyVaultCertificateAdministratorDetails]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzKeyVaultCertificateOrganizationDetail** crea un oggetto dettagli organizzazione certificato in memoria.

## ESEMPI

### Esempio 1: Creare un oggetto dettagli organizzazione
```powershell
PS C:\> $AdminDetails = New-AzKeyVaultCertificateAdministratorDetail -FirstName "Patti" -LastName "Fuller" -EmailAddress "Patti.Fuller@contoso.com" -PhoneNumber "1234567890"
PS C:\> New-AzKeyVaultCertificateOrganizationDetail -AdministratorDetails $AdminDetails

Id AdministratorDetails
-- --------------------
   {Patti}
```

Il primo comando crea un oggetto dettagli dell'amministratore del certificato e quindi lo archivia nella $AdminDetails variabile.
Il secondo comando crea un oggetto dettagli organizzazione certificato e quindi lo archivia nella variabile $OrgDetails certificato.
