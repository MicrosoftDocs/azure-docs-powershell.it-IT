# Update-AzKeyVault

## SYNOPSIS
Aggiornare lo stato di un vault delle chiavi di Azure.

## SINTASSI

### UpdateByNameParameterSet (impostazione predefinita)
```
Update-AzKeyVault -ResourceGroupName <String> -VaultName <String> [-EnablePurgeProtection]
 [-EnableRbacAuthorization <Boolean>] [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### UpdateByInputObjectParameterSet
```
Update-AzKeyVault -InputObject <PSKeyVault> [-EnablePurgeProtection] [-EnableRbacAuthorization <Boolean>]
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UpdateByResourceIdParameterSet
```
Update-AzKeyVault -ResourceId <String> [-EnablePurgeProtection] [-EnableRbacAuthorization <Boolean>]
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Questo cmdlet aggiorna lo stato di un vault delle chiavi di Azure.

## ESEMPI

### Esempio 1: Abilitare la protezione contro l'eliminazione
```powershell
PS C:\> Get-AzKeyVault -VaultName $keyVaultName -ResourceGroupName $resourceGroupName | Update-AzKeyVault -EnablePurgeProtection
```

Abilita la protezione dalla ripulitura usando la sintassi delle piping.

### Esempio 2: Abilitare l'autorizzazione RBAC
```powershell
PS C:\> Get-AzKeyVault -VaultName $keyVaultName -ResourceGroupName $resourceGroupName | Update-AzKeyVault -EnableRbacAuthorization $true
```

Abilita l'autorizzazione RBAC usando la sintassi di piping.

### Esempio 3: Impostare i contrassegni
```powershell
PS C:\> Get-AzKeyVault -VaultName $keyVaultName | Update-AzKeyVault -Tags @{key = "value"}
```

Imposta i tag di un vault delle chiavi denominato $keyVaultName.

### Esempio 4: Pulire i contrassegni
```powershell
PS C:\> Get-AzKeyVault -VaultName $keyVaultName | Update-AzKeyVault -Tags @{}
```

Elimina tutti i tag di un vault delle chiavi denominato $keyVaultName.
