# Import-AzAutomationRunbook

## SYNOPSIS
Importa un runbook di automazione.

## SINTASSI

```
Import-AzAutomationRunbook [-Path] <String> [-Description <String>] [-Name <String>] [-Tags <IDictionary>]
 -Type <String> [-LogProgress <Boolean>] [-LogVerbose <Boolean>] [-Published] [-Force]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Import-AzAutomationRunbook** importa un runbook di automazione di Azure. Specificare il percorso di un file di script di wps_2 (ps1) da importare per i runbook del flusso di lavoro di wps_2 e wps_2, il file (graphrunbook) per i runbook grafici o (py) per i runbook python 2. Per wps_2 runbook del flusso di lavoro, lo script deve contenere un solo wps_2 flusso di lavoro che corrisponda al nome del file.

## ESEMPI

### Esempio 1: Importare un runbook da un file
```powershell
PS C:\> $Tags = @{"tag01"="value01"; "tag02"="value02"}
PS C:\> Import-AzAutomationRunbook -Path .\GraphicalRunbook06.graphrunbook -Tags $Tags -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -Type GraphicalPowershell
```

Il primo comando assegna due coppie chiave/valore alla $Tags variabile.
Il secondo comando importa un runbook grafico denominato GraphicalRunbook06 nell'account di automazione denominato AutomationAccount01.
Il comando assegna anche i tag archiviati in $Tags.

### Esempio 2

Importa un runbook di automazione. (autogenerated)
