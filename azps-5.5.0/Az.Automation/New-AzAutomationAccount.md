# New-AzAutomationAccount

## SYNOPSIS
Crea un account di automazione.

## SINTASSI

```
New-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-Plan <String>]
 [-Tags <IDictionary>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzAutomationAccount** crea un account di automazione di Azure in un gruppo di risorse.
Un account di automazione Ã¨ un contenitore per le risorse di automazione isolato dalle risorse di altri account di automazione. Le risorse di automazione includono runbook, configurazioni DSC (Desired State Configuration), processi e risorse.

## ESEMPI

### Esempio 1: Creare un account di automazione
```powershell
PS C:\> New-AzAutomationAccount -Name "ContosoAutomationAccount" -Location "East US" -ResourceGroupName "ResourceGroup01"
```

Questo comando crea un nuovo account di automazione denominato ContosoAutomationAccount nell'area degli Stati Uniti orientali.

### Esempio 2

Crea un account di automazione. (autogenerated)
