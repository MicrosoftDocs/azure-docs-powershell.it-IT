# Get-AzAutomationModule

## SYNOPSIS
Recupera i metadati per i moduli dall'automazione.

## SINTASSI

### Pertutti (impostazione predefinita)
```
Get-AzAutomationModule [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationModule [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzAutomationModule** recupera i metadati per i moduli dall'automazione di Azure.

## ESEMPI

### Esempio 1: Ottenere tutti i moduli
```
PS C:\>Get-AzAutomationModule -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01"
```

Questo comando recupera tutti i moduli nell'account di automazione denominato Contoso17.

### Esempio 2: Ottenere un modulo
```
PS C:\>Get-AzAutomationModule -AutomationAccountName "Contoso17" -Name "ContosoModule" -ResourceGroupName "ResourceGroup01"
```

Questo comando ottiene un modulo denominato ContosoModule nell'account di automazione denominato Contoso17.
