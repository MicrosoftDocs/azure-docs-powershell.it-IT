# Get-AzAutomationRegistrationInfo

## SYNOPSIS
Recupera le informazioni di registrazione per l'onboarding di un nodo DSC o di un utente ibrido all'automazione.

## SINTASSI

```
Get-AzAutomationRegistrationInfo [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzAutomationRegistrationInfo** recupera gli endpoint e le chiavi necessari per aggiungere un nodo DSC (Desired State Configuration) o un utente ibrido in un account di automazione di Azure.

## ESEMPI

### Esempio 1: Ottenere le informazioni di registrazione
```
PS C:\>Get-AzAutomationRegistrationInfo -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01"
```

Questo comando recupera le informazioni di registrazione per l'account di automazione denominato AutomationAccount01 nel gruppo di risorse denominato ResourceGroup01.
