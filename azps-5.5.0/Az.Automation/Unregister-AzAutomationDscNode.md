# Unregister-AzAutomationDscNode

## SYNOPSIS
Rimuove un nodo DSC dalla gestione tramite un account di automazione.

## SINTASSI

```
Unregister-AzAutomationDscNode -Id <Guid> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Unregister-AzAutomationDscNode** rimuove un nodo APS Desired State Configuration (DSC) dalla gestione da parte di un account di automazione di Azure.

## ESEMPI

### Esempio 1: Rimuovere un nodo DSC di Azure dalla gestione tramite un account di automazione
```
PS C:\>Unregister-AzAutomationDscNode -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01" -Id 064a8929-c98b-25e4-80hh-111ca86067j8
```

Questo comando rimuove il nodo DSC con il GUID specificato dalla gestione dall'account di automazione denominato Contoso17.
