# Get-AzAutomationDscConfiguration

## SYNOPSIS
Recupera le configurazioni DSC dall'automazione.

## SINTASSI

### Pertutti (impostazione predefinita)
```
Get-AzAutomationDscConfiguration [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByConfigurationName
```
Get-AzAutomationDscConfiguration [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzAutomationDscConfiguration** recupera le configurazioni DSC (Desired State Configuration) di APS dall'automazione di Azure.

## ESEMPI

### Esempio 1: Ottenere tutte le configurazioni DSC
```
PS C:\>Get-AzAutomationDscConfiguration -ResourceGroupName "ResourceGroup03" -AutomationAccountName "Contoso17"
```

Questo comando recupera i metadati per tutte le configurazioni DSC nell'account di automazione denominato Contoso17.

### Esempio 2: Ottenere una configurazione DSC in base al nome
```
PS C:\>Get-AzAutomationDscConfiguration -ResourceGroupName "ResourceGroup03" -AutomationAccountName "Contoso17" -Name "ContosoConfiguration"
```

Questo comando recupera i metadati per una configurazione DSC denominata MyConfiguration nell'account di automazione denominato Contoso17.
