# Get-AzAutomationAccount

## SYNOPSIS
Recupera gli account di automazione in un gruppo di risorse.

## SINTASSI

### Pertutti (impostazione predefinita)
```
Get-AzAutomationAccount [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByAutomationAccountName
```
Get-AzAutomationAccount [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzAutomationAccount** ottiene gli account di Automazione di Azure in un gruppo di risorse.
Per altre informazioni sugli account di automazione, vedere il cmdlet New-AzAutomationAccount.

## ESEMPI

### Esempio 1: Ottenere tutti gli account
```
PS C:\>Get-AzAutomationAccount -ResourceGroupName "ResourceGroup03"
```

Questo comando recupera tutti gli account di automazione nel gruppo di risorse denominato ResourceGroup03.

### Esempio 2: Ottenere un account
```
PS C:\>Get-AzAutomationAccount -ResourceGroupName "ResourceGroup03" -Name "ContosoAutomationAccount"
```

Questo comando recupera l'account di automazione denominato ContosoAutomationAccount nel gruppo di risorse denominato ContosoResourceGroup.
