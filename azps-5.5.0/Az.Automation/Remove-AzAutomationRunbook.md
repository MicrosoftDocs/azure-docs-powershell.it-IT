# Remove-AzAutomationRunbook

## SYNOPSIS
Rimuove un runbook.

## SINTASSI

```
Remove-AzAutomationRunbook [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzAutomationRunbook** rimuove un runbook dall'automazione di Azure.

## ESEMPI

### Esempio 1: Rimuovere un runbook
```
PS C:\>Remove-AzAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook03" -ResourceGroupName "ResourceGroup01"
```

Questo comando rimuove il runbook denominato Runbook03 nell'account di automazione di Azure denominato Contoso17.
