# Set-AzDeploymentManagerServiceTopology

## SYNOPSIS
Aggiorna la topologia del servizio.

## SINTASSI

```
Set-AzDeploymentManagerServiceTopology [-InputObject] <PSServiceTopologyResource>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzDeploymentManagerServiceTopology** aggiorna una topologia di servizio con l'oggetto topologia di servizio specificato.
Il cmdlet restituisce l'oggetto topologia del servizio aggiornato.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceTopologyObject
```

Questo comando aggiorna una topologia di servizio il cui nome e GruppoSocietà Risorse corrispondono rispettivamente alle proprietà Name e ResourceGroupName del $serviceTopologyObject.
Il comando restituisce l'oggetto topologia di servizio aggiornato.
