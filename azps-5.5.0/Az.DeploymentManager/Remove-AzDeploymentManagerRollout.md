# Remove-AzDeploymentManagerRollout

## SYNOPSIS
Elimina l'implementazione.

## SINTASSI

### Interattivo (impostazione predefinita)
```
Remove-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Remove-AzDeploymentManagerRollout [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzDeploymentManagerRollout** elimina un'implementazione in uno stato terminale.
Specificare l'implementazione in base al nome e al nome del gruppo di risorse. In alternativa, è possibile specificare l'oggetto Rollout o ResourceId.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout
```

Questo comando elimina un'implementazione denominata ContosoRollout in ContosoResourceGroup.

### Esempio 2: Eliminare un'implementazione con l'identificatore di risorsa
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Questo comando elimina un'implementazione denominata ContosoRollout in ContosoResourceGroup.

### Esempio 3: Eliminare un'implementazione usando l'oggetto di implementazione.
```powershell
PS C:\> Remove-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Questo comando elimina un'implementazione il cui nome e Gruppo Risorse corrispondono rispettivamente alle proprietà Name e ResourceGroupName dell'$rolloutObject.
