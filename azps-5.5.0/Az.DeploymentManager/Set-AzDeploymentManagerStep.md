# Set-AzDeploymentManagerStep

## SYNOPSIS
Aggiorna il passaggio.

## SINTASSI

```
Set-AzDeploymentManagerStep [-InputObject] <PSStepResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Set-AzDeploymentManagerStep** aggiorna un passaggio con l'oggetto passaggio specificato.
Il cmdlet restituisce l'oggetto passaggio aggiornato.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Set-AzDeploymentManagerStep -InputObject $stepObject
```

Questo comando aggiorna un passaggio il cui nome e GruppoSocietà Risorse corrispondono rispettivamente alle proprietà Name e ResourceGroupName dell'$stepObject risorsa.
Il passaggio verrà aggiornato alle proprietà impostate nel $stepObject.
