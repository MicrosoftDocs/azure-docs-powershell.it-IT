# Set-AzDeploymentManagerService

## SYNOPSIS
Aggiorna il servizio.

## SINTASSI

```
Set-AzDeploymentManagerService [-InputObject] <PSServiceResource> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzDeploymentManagerService** aggiorna un servizio con l'oggetto servizio specificato.
Il cmdlet restituisce l'oggetto servizio aggiornato.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Set-AzDeploymentManagerService -InputObject $serviceObject
```

Questo comando aggiorna un servizio il cui nome, il cui nome della topologia di servizio e ResourceGroup corrispondono rispettivamente alle proprietà Name, ServiceTopologyName e ResourceGroupName del $serviceObject.
Il servizio verrà aggiornato alle proprietà impostate nel $serviceObject.
