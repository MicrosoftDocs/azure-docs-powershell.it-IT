# Stop-AzDeploymentManagerRollout

## SYNOPSIS
Interrompe l'implementazione.

## SINTASSI

### Interattivo (impostazione predefinita)
```
Stop-AzDeploymentManagerRollout [-ResourceGroupName] <String> [-Name] <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceId
```
Stop-AzDeploymentManagerRollout [-ResourceId] <String> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Stop-AzDeploymentManagerRollout [-InputObject] <PSRollout> [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzDeploymentManagerRollout** interrompe un'implementazione in corso e restituisce un oggetto che rappresenta lo stato corrente dell'implementazione.
Specificare l'implementazione in base al nome e al nome del gruppo di risorse. In alternativa, è possibile specificare l'oggetto Rollout o ResourceId.

Una volta interrotta un'implementazione, non è possibile riprenderla o riavviarla. È possibile creare solo una nuova implementazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Stop-AzDeploymentManagerRollout -ResourceGroupName ContosoResourceGroup -Name ContosoRollout -SkipSucceeded
```

Questo comando interrompe un'implementazione denominata ContosoRollout in ContosoResourceGroup. 

### Esempio 2: Interrompere un'implementazione con l'identificatore di risorsa
```powershell
PS C:\> Restart-AzDeploymentManagerRollout -ResourceId "/subscriptions/subscriptionId/resourcegroups/ContosoResourceGroup/providers/Microsoft.DeploymentManager/rollouts/ContosoRollout"
```

Questo comando interrompe un'implementazione denominata ContosoRollout in ContosoResourceGroup.

### Esempio 3: Interrompere un'implementazione usando l'oggetto di implementazione.
```powershell
PS C:\> Get-AzDeploymentManagerRollout -InputObject $rolloutObject
```

Questo comando interrompe un'implementazione il cui nome e GruppoSocietà Risorse corrispondono rispettivamente alle proprietà Name e ResourceGroupName dell'$rolloutObject.
