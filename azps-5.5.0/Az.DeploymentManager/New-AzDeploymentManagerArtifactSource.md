# New-AzDeploymentManagerArtifactSource

## SYNOPSIS
Crea un'origine dell'elemento.

## SINTASSI

```
New-AzDeploymentManagerArtifactSource -ResourceGroupName <String> -Name <String> -Location <String>
 -SasUri <String> [-Tag <Hashtable>] [-ArtifactRoot <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzDeploymentManagerArtifactSource** crea un'origine dell'elemento.
Specificare le *proprietà Name,* *ResourceGroupName* e required.

È possibile modificare l'oggetto restituito in locale e quindi applicare le modifiche all'origine dell'elemento usando il cmdlet Set-AzDeploymentManagerArtifactSource.

Il cmdlet restituisce un oggetto ArtifactSource con un valore ResourceId a cui è possibile fare riferimento nel cmdlet New-AzDeploymentManagerServiceTopology in modo che da questa posizione sia possibile fare riferimento agli elementi necessari per una risorsa ServiceUnit, i file Template e Parameters.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzDeploymentManagerArtifactSource -ResourceGroupName ContosoResourceGroup -Name ContosoArtifactSource -Location "Central US" -SasUri "https://ContosoStorage.blob.core.windows.net/ContosoArtifacts?sasParameters"
```

Crea un'origine dell'elemento in ContosoResourceGroup con il nome ContosoArtifactSource con Stati Uniti centrali come posizione della risorsa. La proprietà SasUri fornisce un URI della firma di accesso condiviso di Archiviazione di Azure al contenitore di archiviazione in cui sono archiviati gli elementi.
