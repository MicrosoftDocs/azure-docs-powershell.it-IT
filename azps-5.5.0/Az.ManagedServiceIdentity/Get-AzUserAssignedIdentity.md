# Get-AzUserAssignedIdentity

## SYNOPSIS
Recupera le identità/identità assegnate dall'utente.

## SINTASSI

### SuscriptionParameterSet (Impostazione predefinita)
```
Get-AzUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
**Get-AzUserAssignedIdentity ottiene** le identità assegnate dall'utente esistente.
