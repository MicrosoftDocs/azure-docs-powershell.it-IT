# Get-AzDataBoxJob

## SYNOPSIS
Ottiene informazioni sui processi di Databox

## SINTASSI

### ListParameterSet (impostazione predefinita)
```
Get-AzDataBoxJob [-ResourceGroupName <String>] [-Completed] [-CompletedWithError] [-Cancelled] [-Aborted]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByNameParameterSet
```
Get-AzDataBoxJob -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzDataBoxJob -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzDataBoxJobs ottiene** informazioni sui processi di databox in una sottoscrizione di Azure.
Se si specifica il gruppo di risorse, questo cmdlet ottiene tutti i processi della casella dei dati in tale gruppo di risorse. Se si specifica il nome del processo insieme al nome del gruppo di risorse, questo cmdlet ottiene informazioni sul processo specifico della casella dei dati.
Se non si specifica altro che l'ID sottoscrizione, questo cmdlet ottiene informazioni su tutti i processi della casella di dati della sottoscrizione.
