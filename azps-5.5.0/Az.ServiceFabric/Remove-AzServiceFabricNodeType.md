# Remove-AzServiceFabricNodeType

## SYNOPSIS
Rimuovere un tipo di nodo completo da un cluster.

## SINTASSI

```
Remove-AzServiceFabricNodeType [-ResourceGroupName] <String> [-Name] <String> -NodeType <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Usare **Remove-AzServiceFabricNodeType** per rimuovere tutti i nodi da uno specifico tipo di nodo e il tipo di nodo da un cluster. Questo comando non puÃ² essere usato per eliminare il tipo di nodo primario.

## ESEMPI

### Esempio 1
```powershell
PS c:> Remove-AzServiceFabricNodeType -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1'
```

Questo comando rimuove nodeType 'nt1' dal cluster.
