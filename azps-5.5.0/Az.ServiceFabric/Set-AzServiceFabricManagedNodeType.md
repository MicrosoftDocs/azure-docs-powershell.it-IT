# Set-AzServiceFabricManagedNodeType

## SYNOPSIS
Imposta le proprietà delle risorse del tipo di nodo o esegue azioni di reimage su specifici tipi di nodo con il parametro -Reimage.

## SINTASSI

### ByObj (impostazione predefinita)
```
Set-AzServiceFabricManagedNodeType [-InputObject] <PSManagedNodeType> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### WithParamsByName
```
Set-AzServiceFabricManagedNodeType [-ResourceGroupName] <String> [-ClusterName] <String> [-Name] <String>
 [-AsJob] [-InstanceCount <Int32>] [-ApplicationStartPort <Int32>] [-ApplicationEndPort <Int32>]
 [-EphemeralStartPort <Int32>] [-EphemeralEndPort <Int32>] [-Capacity <Hashtable>]

```

```

```

## DESCRIZIONE
Imposta le proprietà delle risorse del tipo di nodo o esegue azioni di reimage su specifici tipi di nodo con il parametro -Reimage. Durante l'operazione di ripristino, i nodi dell'infrastruttura del servizio verranno disabilitati prima di riprovare le macchine virtuali e averli ri abilitati nuovamente una volta tornati. Se questa operazione viene eseguita nei tipi di nodo primario, l'operazione potrebbe richiedere del tempo perché potrebbe non essere possibile visualizzare nuovamente tutti i nodi contemporaneamente. Usare -ForceReimage forzare l'operazione anche se l'infrastruttura del servizio non è in grado di disabilitare i nodi, ma usare con cautela perché ciò potrebbe causare la perdita di dati se nel nodo sono in esecuzione carichi di lavoro con stato.

## ESEMPI

### Esempio 1
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
$NodeTypeName = "nt1"
Set-AzServiceFabricManagedNodeType -ResourceGroupName $rgName -ClusterName $clusterName -name $NodeTypeName -InstanceCount 6 -Verbose
```

Aggiornare il numero di istanze del tipo di nodo.

### Esempio 2
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
$NodeTypeName = "nt1"
Set-AzServiceFabricManagedNodeType -ResourceGroupName $rgName -ClusterName $clusterName -name $NodeTypeName -PlacementProperty @{NodeColor="Red";SomeProperty="6";} -Verbose
```

Aggiornare le properite della posizione del tipo di nodo. In questo modo verranno sovrascritte le precedenti properite del posizionamento, se presenti.

### Esempio 3
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
$NodeTypeName = "nt1"
Set-AzServiceFabricManagedNodeType -ResourceGroupName $rgName -ClusterName $clusterName  -Name $NodeTypeName -Reimage -NodeName nt1_0, nt1_3
```

Reimage node 0 and 3 on the node type.

### Esempio 4
```powershell
$rgName = "testRG"
$clusterName = "testCluster"
$NodeTypeName = "nt1"
$nodeType = Get-AzServiceFabricManagedNodeType -ResourceGroupName $rgName -ClusterName $clusterName -Name $NodeTypeName

$nodeType.VmInstanceCount = 6
$nodeType | Set-AzServiceFabricManagedNodeType
```

Aggiornare il numero di istanze del tipo di nodo con piping.
