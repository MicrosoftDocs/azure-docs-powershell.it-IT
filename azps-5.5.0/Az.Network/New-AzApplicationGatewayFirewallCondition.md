# New-AzApplicationGatewayFirewallCondition

## SYNOPSIS
Crea una condizione di corrispondenza per una regola personalizzata

## SINTASSI

```
New-AzApplicationGatewayFirewallCondition -MatchVariable <PSApplicationGatewayFirewallMatchVariable[]>
 -Operator <String> [-NegationCondition <Boolean>] -MatchValue <String[]> [-Transform <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
**New-AzApplicationGatewayFirewallCondition crea** una condizione di corrispondenza per la regola personalizzata del firewall.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $condition = New-AzApplicationGatewayFirewallCondition -MatchVariable $variable -Operator Contains -NegationCondition false -Transforms Lowercase, Trim -MatchValue abc, cde
```

Il comando crea una nuova condizione di corrispondenza usando la variabile di corrispondenza definita nel $variable, l'operatore è Contains e la condizione di negazione è false, Transfroms including lowercase and trim, the match value is abc and cde. La nuova condizione di corrispondenza viene salvata in $condition.
