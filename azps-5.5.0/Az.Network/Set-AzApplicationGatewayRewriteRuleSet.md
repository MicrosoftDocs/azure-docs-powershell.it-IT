# Set-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Modifica un set di regole di riscrittura per un gateway applicazione.

## SINTASSI

```
Set-AzApplicationGatewayRewriteRuleSet -ApplicationGateway <PSApplicationGateway> -Name <String>
 -RewriteRule <System.Collections.Generic.List`1[Microsoft.Azure.Commands.Network.Models.PSApplicationGatewayRewriteRule]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzApplicationGatewayRewriteRuleSet** modifica una regola di routing delle richieste.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "ruleset1" -RewriteRule $rule
```

Il primo comando recupera il gateway applicazione denominato ApplicationGateway01 e lo archivia nella $AppGw variabile.
Il secondo comando modifica il set di regole di riscrittura in modo che il gateway applicazione usi le regole di riscrittura specificate nella $rule variabile.
