# New-AzLoadBalancerBackendAddressPoolConfig

## SYNOPSIS
Crea una configurazione del pool di indirizzi back-end per una bilanciamento del carico.

## SINTASSI

```
New-AzLoadBalancerBackendAddressPoolConfig -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzLoadBalancerBackendAddressPoolConfig** crea una configurazione del pool di indirizzi back-end per una bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1: Creare una configurazione del pool di indirizzi back-end per un servizio di bilanciamento del carico
```
PS C:\>New-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02"
```

Questo comando crea una configurazione del pool di indirizzi back-end denominata BackendAddressPool02 per una bilanciamento del carico.
