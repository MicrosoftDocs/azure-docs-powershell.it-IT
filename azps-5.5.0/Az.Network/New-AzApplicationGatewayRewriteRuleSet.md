# New-AzApplicationGatewayRewriteRuleSet

## SYNOPSIS
Crea una regola di routing delle richieste per un gateway applicazioni.

## SINTASSI

```
New-AzApplicationGatewayRewriteRuleSet -Name <String>
 -RewriteRule <System.Collections.Generic.List`1[Microsoft.Azure.Commands.Network.Models.PSApplicationGatewayRewriteRule]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzApplicationGatewayRewriteRuleSet crea** un set di regole di riscrittura per un gateway applicazione di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $ruleset = New-AzApplicationGatewayRewriteRuleSet -Name ruleset1 -RewriteRule $rule
```

Questo comando crea un set di regole di riscrittura denominato ruleset1 e archivia il risultato nella variabile $ruleset.
