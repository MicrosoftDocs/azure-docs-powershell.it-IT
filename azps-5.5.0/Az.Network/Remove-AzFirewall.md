# Remove-AzFirewall

## SYNOPSIS
Rimuovere un firewall.

## SINTASSI

```
Remove-AzFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzFirewall** rimuove un firewall di Azure.

## ESEMPI

### Esempio 1: Creare ed eliminare un firewall
```powershell
New-AzFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Questo esempio crea un firewall e quindi lo elimina. Per eliminare la richiesta quando si elimina il firewall, usare il contrassegno -Force.
