# Set-AzApplicationGatewayPrivateLinkConfiguration

## SYNOPSIS
Modifica una configurazione PrivateLink per un gateway applicazioni.

## SINTASSI

```
Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway <PSApplicationGateway> -Name <String>
 -IpConfiguration <PSApplicationGatewayPrivateLinkIpConfiguration[]> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzApplicationGatewayPrivateLinkConfiguration** modifica una configurazione privateLink per un gateway applicazione di Azure.

## ESEMPI

### Esempio 1: Impostare una configurazione PrivateLink
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01" -IpConfiguration $privateLinkIpConfiguration01
```

Il primo comando recupera il gateway applicazione denominato ApplicationGateway01 che appartiene al gruppo di risorse denominato ResourceGroup01 e lo archivia nella $AppGw variabile.
Il secondo comando imposta la configurazione privateLink con il nome privateLinkConfig01 in modo da usare la configurazione IP archiviata in $privateLinkIpConfiguration 01
