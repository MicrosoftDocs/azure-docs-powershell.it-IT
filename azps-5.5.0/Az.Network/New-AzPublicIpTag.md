# New-AzPublicIpTag

## SYNOPSIS
Crea un tag IP.

## SINTASSI

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzPublicIpTag crea** un tag IP.

## ESEMPI

### Esempio 1: Creare un nuovo tag IP
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

Questo comando crea un nuovo tag IP con tagtype come "FirstPartyUsage" e tag like "/Sql". Viene usato nella creazione di publicIpAddress con questi tag specifici per l'allocazione.
