# Get-AzRouteTable

## SYNOPSIS
Ottiene le tabelle delle route.

## SINTASSI

### NoExpand (impostazione predefinita)
```
Get-AzRouteTable [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Espandi
```
Get-AzRouteTable -ResourceGroupName <String> -Name <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzRouteTable** ottiene le tabelle delle route di Azure.
Ãˆ possibile ottenere una singola tabella di route oppure tutte le tabelle di route in un gruppo di risorse o nella sottoscrizione.
