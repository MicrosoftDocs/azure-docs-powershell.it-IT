# Set-AzApplicationGatewaySslProfile

## SYNOPSIS
Aggiorna il profilo SSL per un gateway applicazione.

## SINTASSI

```
Set-AzApplicationGatewaySslProfile -ApplicationGateway <PSApplicationGateway> -Name <String>
 [-SslPolicy <PSApplicationGatewaySslPolicy>]
 [-ClientAuthConfiguration <PSApplicationGatewayClientAuthConfiguration>]
 [-TrustedClientCertificates <PSApplicationGatewayTrustedClientCertificate[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzApplicationGatewaySslProfile** aggiorna il profilo SSL per un gateway applicazione di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $AppGw = Set-AzApplicationGatewaySslProfile -ApplicationGateway $AppGw -Name "Profile01" -ClientAuthConfiguration $newclientconfig
```

Il primo comando recupera il gateway applicazione denominato ApplicationGateway01 che appartiene al gruppo di risorse denominato ResourceGroup01 e lo archivia nella $AppGw variabile. Il secondo comando aggiorna il profilo SSL del gateway applicazione nella variabile $AppGw per aggiornare la configurazione dell'autenticazione client al nuovo valore archiviato in $newclientconfig.
