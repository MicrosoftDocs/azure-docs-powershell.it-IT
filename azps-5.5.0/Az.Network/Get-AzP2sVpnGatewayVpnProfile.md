# Get-AzP2sVpnGatewayVpnProfile

## SYNOPSIS
Genera e restituisce un URL di firma di accesso condiviso che il cliente deve scaricare per il profilo Vpn per la configurazione del client point-to-site in modo che punti alla connettività del sito a P2SVpnGateway.

## SINTASSI

### ByP2SVpnGatewayName (impostazione predefinita)
```
Get-AzP2sVpnGatewayVpnProfile [-Name <String>] -ResourceGroupName <String> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayObject
```
Get-AzP2sVpnGatewayVpnProfile -InputObject <PSP2SVpnGateway> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByP2SVpnGatewayResourceId
```
Get-AzP2sVpnGatewayVpnProfile -ResourceId <String> [-AuthenticationMethod <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzP2sVpnGatewayVpnProfile** consente di generare e ottenere un URL della firma di accesso condiviso per il cliente per scaricare il profilo Vpn per la configurazione del client di puntamento al sito in modo che punti alla connettività del sito a P2SVpnGateway. Puntare alla configurazione del client del sito usando questo profilo Vpn può connettersi solo a questo specifico P2SVpnGateway.
