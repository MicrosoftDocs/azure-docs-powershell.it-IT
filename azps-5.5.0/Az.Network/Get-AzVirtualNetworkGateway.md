# Get-AzVirtualNetworkGateway

## SYNOPSIS
Ottiene un Gateway di rete virtuale

## SINTASSI

```
Get-AzVirtualNetworkGateway [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Gateway di rete virtuale Ã¨ l'oggetto che rappresenta il gateway in Azure.
Il cmdlet **Get-AzVirtualNetworkGateway** restituisce l'oggetto del gateway in Azure in base al nome e al nome del gruppo di risorse.

## ESEMPI

### Esempio 1: Ottenere un Gateway di rete virtuale
```powershell
Get-AzVirtualNetworkGateway -Name myGateway1 -ResourceGroupName myRG
```

Restituisce l'oggetto di Gateway di rete virtuale con il nome "myGateway1" all'interno del gruppo di risorse "myRG"

### Esempio 2: Ottenere un Gateway di rete virtuale
```powershell
Get-AzVirtualNetworkGateway -Name myGateway* -ResourceGroupName myRG
```

Restituisce tutti i gateway di rete virtuale che iniziano con "myGateway" all'interno del gruppo di risorse "myRG"
