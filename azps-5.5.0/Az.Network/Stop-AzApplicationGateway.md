# Stop-AzApplicationGateway

## SYNOPSIS
Arresta un gateway applicazione

## SINTASSI

```
Stop-AzApplicationGateway -ApplicationGateway <PSApplicationGateway> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Stop-AzApplicationGateway** interrompe un gateway applicazione.

## ESEMPI

### Esempio 1: Arrestare un gateway applicazione
```
PS C:\>Stop-AzApplicationGateway -ApplicationGateway $AppGw
```

Questo comando interrompe l'archiviazione del gateway applicazione nella $AppGw variabile.
