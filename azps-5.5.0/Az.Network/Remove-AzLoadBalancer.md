# Remove-AzLoadBalancer

## SYNOPSIS
Rimuove una bilanciamento del carico.

## SINTASSI

```
Remove-AzLoadBalancer -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzLoadBalancer** rimuove una bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1: Rimuovere un servizio di bilanciamento del carico
```
PS C:\>Remove-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
```

Questo comando elimina una bilanciamento del carico denominata MyLoadBalancer nel gruppo di risorse denominato MyResourceGroup.
