# Set-AzNetworkProfile

## SYNOPSIS
Aggiorna un profilo di rete.

## SINTASSI

```
Set-AzNetworkProfile -NetworkProfile <PSNetworkProfile> [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Set-AzPublicIpPrefix aggiorna** un profilo di rete.

## ESEMPI

### Esempio 1
```powershell
$networkProfile = Get-AzNetworkProfile -Name np1 -ResourceGroupName rg1

$networkProfile.Tags = "TestTag"

$networkProfile.ContainerNetworkInterfaceConfigurations = New-AzNetworkProfileContainerNicConfig -Name cnicconfig1

$networkProfile | Set-AzNetworkProfile
```

Il primo comando ottiene un profilo di rete esistente. Il secondo comando aggiorna un tag e il terzo aggiunge una configurazione dell'interfaccia di rete nel profilo di rete. Il quarto comando aggiorna il profilo di rete.
