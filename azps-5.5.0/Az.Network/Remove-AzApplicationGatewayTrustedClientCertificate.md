# Remove-AzApplicationGatewayTrustedClientCertificate

## SYNOPSIS
Rimuove l'oggetto catena di certificati DELLA CA del client attendibile da un gateway applicazioni.

## SINTASSI

```
Remove-AzApplicationGatewayTrustedClientCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGatewayTrustedClientCertificate** rimuove l'oggetto catena di certificati CA del client attendibile da un gateway applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name "TrustedClientCertificate01"
```

Il primo comando ottiene un gateway applicazione e lo archivia nella variabile $gw variabile. Il secondo comando rimuove la catena di certificati CA del client attendibile denominata "TrustedClientCertificate01" dal gateway applicazione archiviato in $gw.
