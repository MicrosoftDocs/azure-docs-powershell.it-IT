# Get-AzNetworkServiceTag

## SYNOPSIS
Ottiene l'elenco di risorse di informazioni sui tag di servizio.

## SINTASSI

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzNetworkServiceTag** ottiene l'elenco delle risorse di informazioni sui tag di servizio.

Tieni presente che le informazioni dell'area geografica di Azure specificate verranno usate come riferimento per la versione (non come filtro in base alla posizione). Ad esempio, anche se si specifica che si otterrà l'elenco dei tag di servizio con prefissi in tutte le aree geografiche ma limitati al cloud a cui appartiene l'abbonamento (ad esempio Pubblico, Governo degli Stati Uniti, Cina o `-Location eastus2` Germania).

```powershell

PS C:\> $sql = $serviceTags.Values | Where-Object { $_.Name -eq "Sql" }
```
