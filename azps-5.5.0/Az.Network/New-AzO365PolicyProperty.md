# New-AzO365PolicyProperty

## SYNOPSIS
Creare un oggetto criterio di interruzione del traffico di Office 365.

## SINTASSI

```
New-AzO365PolicyProperty [-Allow] [-Optimize] [-Default] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Creare un criterio di interruzione di Office 365 da usare con i cmdlet New-AzVpnSite Update-AzVpnSite distribuzione.
## ESEMPI

### Esempio 1
```powershell
PS C:\> $policy = New-AzO365PolicyProperty -Allow -Optimize
```

Creare un criterio di interruzione del traffico di Office 365 con sottoasfondo consentito per consentire e ottimizzare la categoria di traffico.
