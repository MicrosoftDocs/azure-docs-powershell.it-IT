# Get-AzApplicationGatewayUrlPathMapConfig

## SYNOPSIS
Ottiene una matrice di mapping del percorso URL a un pool di server back-end.

## SINTASSI

```
Get-AzApplicationGatewayUrlPathMapConfig [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzApplicationGatewayURLPathMapConfig** ottiene una matrice di mapping del percorso URL a un pool di server back-end.

## ESEMPI

### Esempio 1: Ottenere una configurazione della mappa del percorso URL
```
PS C:\>Get-AzApplicationGatewayUrlPathMapConfig -ApplicationGateway Gateway
```

Questo comando recupera le configurazioni della mappa del percorso URL dal server back-end che si trova nel gateway applicazione denominato Gateway.
