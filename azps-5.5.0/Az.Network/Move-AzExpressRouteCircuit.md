# Move-AzExpressRouteCircuit

## SYNOPSIS
Sposta un circuito ExpressRoute dal modello di distribuzione classica al modello di distribuzione di Gestione risorse.

## SINTASSI

```
Move-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> -Location <String> -ServiceKey <String>
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Move-AzExpressRouteCircuit** sposta un circuito ExpressRoute dal modello di distribuzione classico al modello di distribuzione di Gestione risorse. Dopo lo spostamento, il circuito ExpressRoute si comporta e funziona come qualsiasi altro circuito ExpressRoute creato nel modello di distribuzione di Gestione risorse. I collegamenti di circuito, le reti virtuali e i gateway VPN non vengono spostati attraverso questa operazione. Queste risorse devono essere riconfigurate dopo lo spostamento.

## ESEMPI

### Esempio 1: Spostare un circuito ExpressRoute nel modello di distribuzione di Gestione risorse
```
Move-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $RG -Location $Location -ServiceKey $ServiceKey
```
