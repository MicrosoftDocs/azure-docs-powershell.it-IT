# Start-AzApplicationGateway

## SYNOPSIS
Avvia un gateway applicazione.

## SINTASSI

```
Start-AzApplicationGateway -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Start-AzApplicationGateway** avvia un gateway applicazione di Azure

## ESEMPI

### Esempio1: Avviare un gateway applicazione
```
PS C:\>$AppGw = Start-AzApplicationGateway -ApplicationGateway $AppGw
```

Questo comando avvia il gateway applicazione archiviato nella variabile $AppGw variabile.
