# New-AzFirewallPolicyThreatIntelWhitelist

## SYNOPSIS
Creare un nuovo elenco degli elenchi dei criteri di Threat Intelligence per i criteri del firewall di Azure

## SINTASSI

```
New-AzFirewallPolicyThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzFirewallPolicyThreatIntelWhitelist** crea un oggetto threat intel whitelist, che puÃ² essere usato durante la creazione o l'impostazione di un criterio del firewall di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzFirewallPolicyThreatIntelWhitelist -IpAddress 23.46.72.91,192.79.236.79 -FQDN microsoft.com
```

In questo esempio viene creata una lista bianca di informazioni sulle minacce che contiene un elenco di voci FQDN di una voce e un elenco indirizzi Ip di due voci
