# Get-AzVirtualNetworkGatewayVpnClientConnectionHealth

## SYNOPSIS 
Ottenere l'elenco delle informazioni sull'integrità della connessione del client VPN di un gateway di rete virtuale di Azure per ogni connessione al client VPN

## SINTASSI

```
Get-AzVirtualNetworkGatewayVpnClientConnectionHealth -ResourceName <String> -ResourceGroupName <String> -InputObject <PSVirtualNetworkGateway> -ResourceId <ResourceId>
 [-AsJob] [<CommonParameters>]
```

## DESCRIZIONE
Elencare tutte le informazioni di integrità della connessione del client VPN connesso. Ciò include: vpnConnectionId vpnConnectionDuration vpnConnectionTime publicIpAddress privateIpAddress vpnUserName maxBandwidth egressPacketsTransferred egressBytesTransferred ingressPacketsTransferred ingressBytesTransferred maxPacketsPerSecond
