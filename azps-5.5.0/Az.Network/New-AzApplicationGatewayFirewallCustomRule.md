# New-AzApplicationGatewayFirewallCustomRule

## SYNOPSIS
Crea una nuova regola personalizzata per il criterio firewall del gateway applicazioni.

## SINTASSI

```
New-AzApplicationGatewayFirewallCustomRule -Name <String> -Priority <Int32> -RuleType <String>
 -MatchCondition <PSApplicationGatewayFirewallCondition[]> -Action <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
**New-AzApplicationGatewayFirewallCustomRule** crea una regola personalizzata per i criteri del firewall.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $customRule = New-AzApplicationGatewayFirewallCustomRule -Name example-rule -Priority 1 -RuleType MatchRule -MatchCondition $condtion -Action Allow
```

Il comando crea una nuova regola personalizzata con il nome della regola di esempio, la priorità 1 e il tipo di regola sarà MatchRule con la condizione definita nella variabile della condizione. L'azione sarà consentita. La nuova regola personalizzata di corrispondenza viene salvata in $customRule.
