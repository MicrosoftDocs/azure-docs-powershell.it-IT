# Get-AzCustomIpPrefix

## SYNOPSIS
Ottiene una risorsa CustomIpPrefix

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzCustomIpPrefix [-Name <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzCustomIpPrefix -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzCustomIpPrefix** ottiene uno o piÃ¹ provider CustomIpPrefixes in base al set di parametri di input
