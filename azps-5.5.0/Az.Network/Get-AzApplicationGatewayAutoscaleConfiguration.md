# Get-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Ottiene la configurazione della scala automatica del Gateway applicazioni.

## SINTASSI

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzApplicationGatewayAutoscaleConfiguration** ottiene la configurazione di gradazione automatica del Gateway applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

Il primo comando recupera il gateway applicazione e lo archivia in $gw variabile.
Il secondo comando estrae la configurazione di gradazioni automatica dal gateway applicazione.
