# Add-AzServiceEndpointPolicyDefinition

## SYNOPSIS
Aggiunge una definizione di criterio dell'endpoint del servizio a un criterio specificato.

## SINTASSI

```
Add-AzServiceEndpointPolicyDefinition -Name <String> -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-Description <String>] [-ServiceResource <String[]>] [-Service <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Add-AzServiceEndpointPolicyDefinition** aggiunge una definizione di criterio dell'endpoint del servizio al criterio.

## ESEMPI

### Esempio 1: Aggiorna la definizione di un criterio endpoint del servizio nei criteri dell'endpoint del servizio
```
$policydef= New-AzServiceEndpointPolicyDefinition -Name "ServiceEndpointPolicyDefinition1" -ResourceGroupName "ResourceGroup01" -Service "Microsoft.Storage" -ServiceResources "subscriptions/sub1" -Description "New Definition"
```

Questo comando ha aggiornato la definizione dei criteri dell'endpoint del servizio con il nome ServiceEndpointPolicyDefinition1, il servizio Microsoft.Storage, le sottoscrizioni delle risorse di servizio/sub1 e la descrizione "Nuova definizione" che appartiene al gruppo di risorse denominato ResourceGroup01 e lo archivia nella variabile $policydef.
