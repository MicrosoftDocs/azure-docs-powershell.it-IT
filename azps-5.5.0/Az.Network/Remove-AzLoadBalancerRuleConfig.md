# Remove-AzLoadBalancerRuleConfig

## SYNOPSIS
Rimuove una configurazione di regola per una bilanciamento del carico.

## SINTASSI

```
Remove-AzLoadBalancerRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzLoadBalancerRuleConfig rimuove** una configurazione di regola per una bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1: Rimuovere una configurazione di regole da un servizio di bilanciamento del carico
```
PS C:\>$loadbalancer = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:> Remove-AzLoadBalancerRuleConfig -Name "MyLBruleName" -LoadBalancer $loadbalancer
```

Il primo comando recupera la bilanciamento del carico denominata MyLoadBalancer e quindi la archivia nella $loadbalancer variabile.
Il secondo comando rimuove la configurazione della regola denominata MyLBruleName dal servizio di bilanciamento del carico in $loadbalancer.
