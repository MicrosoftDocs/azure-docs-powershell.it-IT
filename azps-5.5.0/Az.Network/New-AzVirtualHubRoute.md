# New-AzVirtualHubRoute

## SYNOPSIS
Crea un oggetto Percorso hub virtuale di Azure.

## SINTASSI

```
New-AzVirtualHubRoute -AddressPrefix <String[]> -NextHopIpAddress <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Crea un oggetto Percorso hub virtuale di Azure.

## ESEMPI

### Esempio 1

```powershell
PS C:\> $route1 = New-AzVirtualHubRoute -AddressPrefix @("10.0.0.0/16", "11.0.0.0/16") -NextHopIpAddress "12.0.0.5"

AddressPrefixes            NextHopIpAddress
---------------            ----------------
{10.0.0.0/16, 11.0.0.0/16} 12.0.0.5
```

Gli elementi descritti in precedenza creeranno un oggetto percorso dell'hub virtuale che può essere incluso nella tabella delle route dell'hub virtuale.

La route dell'hub virtuale è un oggetto in memoria che può essere usato per creare un oggetto VirtualHubRouteTable.
