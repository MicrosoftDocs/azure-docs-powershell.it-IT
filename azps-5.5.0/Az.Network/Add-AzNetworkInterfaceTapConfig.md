# Add-AzNetworkInterfaceTapConfig

## SYNOPSIS
Crea una risorsa TapConfiguration associata a un'interfaccia di rete. Questo farà riferimento a una risorsa VirtualNetworkTap esistente.

## SINTASSI

### SetByResource (Impostazione predefinita)
```
Add-AzNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTap <PSVirtualNetworkTap>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SetByResourceId
```
Add-AzNetworkInterfaceTapConfig -NetworkInterface <PSNetworkInterface> -Name <String>
 [-VirtualNetworkTapId <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Add-AzNetworkInterfaceTapConfig crea** una risorsa TapConfiguration associata a un'interfaccia di rete. Questo farà riferimento a una risorsa VirtualNetworkTap esistente.

## ESEMPI

### Esempio 1: Aggiungere TapConfiguration a un dato NetworkInterface
```
PS C:\>Add-AzNetworkInterfaceTapConfig -NetworkInterface $sourceNic -VirtualNetworkTap $vVirtualNetworkTap -Name 'myTapConfig'
```

Aggiungere TapConfiguration a un sourceNic. Il traffico da sourceNic VM verrà speculare alla macchina virtuale di destinazione a cui fa riferimento la risorsa vVirtualNetworkTap.
