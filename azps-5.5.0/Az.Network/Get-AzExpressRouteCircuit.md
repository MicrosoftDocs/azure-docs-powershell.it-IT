# Get-AzExpressRouteCircuit

## SYNOPSIS
Ottiene un circuito Azure ExpressRoute da Azure.

## SINTASSI

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzExpressRouteCircuit** viene usato per recuperare un oggetto circuito ExpressRoute dall'abbonamento. L'oggetto circuito restituito pu√≤ essere usato come input di altri cmdlet che operano sui circuiti ExpressRoute.
