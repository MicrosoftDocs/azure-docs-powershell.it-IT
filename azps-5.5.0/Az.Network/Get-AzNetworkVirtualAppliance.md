# Get-AzNetworkVirtualAppliance

## SYNOPSIS
Ottenere o elencare appliance virtuali di rete.

## SINTASSI

### ResourceNameParameterSet (impostazione predefinita)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
I Get-AzNetworkVirtualAppliance di rete recuperano o elencano le risorse di Network Virtual Appliance.
