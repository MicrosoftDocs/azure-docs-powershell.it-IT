# New-AzFirewallHubPublicIpAddress

## SYNOPSIS
Ip pubblico associato al firewall nell'hub virtuale

## SINTASSI

```
New-AzFirewallHubPublicIpAddress [-Count <Int32>] [-Addresses <PSAzureFirewallPublicIpAddress[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Ip pubblico associato al firewall nell'hub virtuale

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzFirewallHubPublicIpAddress -Count 2
```

Verranno creati 2 ip pubblici nel firewall collegato all'hub virtuale. L'indirizzo IP verrà creato nel back-end. Non è possibile fornire gli ipaddress in modo esplicito per un nuovo firewall.

### Esempio 2
```powershell
PS C:\> $publicIp1 = New-AzFirewallPublicIpAddress -Address 10.2.3.4
PS C:\> $publicIp2 = New-AzFirewallPublicIpAddress -Address 20.56.37.46
PS C:\> New-AzFirewallHubPublicIpAddress -Count 3 -Addresses $publicIp1, $publicIp2
```

Verrà creato 1 nuovo ip pubblico nel firewall conservando $publicIp 1, $publicIp 2 già esistenti nel firewall.
