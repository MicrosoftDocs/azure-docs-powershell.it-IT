# Set-AzLoadBalancerProbeConfig

## SYNOPSIS
Aggiorna una configurazione di configurazione per una bilanciamento del carico.

## SINTASSI

```
Set-AzLoadBalancerProbeConfig -LoadBalancer <PSLoadBalancer> -Name <String> [-Protocol <String>] -Port <Int32>
 -IntervalInSeconds <Int32> -ProbeCount <Int32> [-RequestPath <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzLoadBalancerProbeConfig** aggiorna una configurazione di configurazione per una bilanciamento del carico.

## ESEMPI

### Esempio 1: Modificare la configurazione di un servizio di bilanciamento del carico
```powershell
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> $slb | Add-AzLoadBalancerProbeConfig -Name "NewProbe" -Protocol "http" -Port 80 -IntervalInSeconds 15 -ProbeCount 2 -RequestPath "healthcheck.aspx" 
PS C:\> $slb | Set-AzLoadBalancerProbeConfig -Name "NewProbe" -Port 80 -IntervalInSeconds 15 -ProbeCount 2
```

Il primo comando ottiene il bilanciamento del carico denominato MyLoadBalancer e quindi lo archivia nella $slb variabile.
Il secondo comando usa l'operatore della pipeline per passare la bilanciamento del carico in $slb ad Add-AzLoadBalancerProbeConfig, che aggiunge una nuova configurazione di configurazione.
Il terzo comando passa la bilanciamento del carico **a Set-AzLoadBalancerProbeConfig,** che imposta la nuova configurazione.
È necessario specificare diversi degli stessi parametri specificati nel comando precedente perché sono necessari per il cmdlet corrente.
