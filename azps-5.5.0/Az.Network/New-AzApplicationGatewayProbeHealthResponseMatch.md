# New-AzApplicationGatewayProbeHealthResponseMatch

## SYNOPSIS
Crea una corrispondenza di risposta di integrità risposta usata da Health Gateway per un gateway applicazione.

## SINTASSI

```
New-AzApplicationGatewayProbeHealthResponseMatch [-Body <String>] [-StatusCode <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Add-AzApplicationGatewayProbeHealthResponseMatch crea** una corrispondenza health response usata da Health Istanza per un gateway applicazione.

## ESEMPI

### Esempio 1
```
PS C:\>$responsematch = New-AzApplicationGatewayProbeHealthResponseMatch -Body "helloworld" -StatusCode "200-399","503"
```

Questo comando crea una corrispondenza della risposta all'integrità che può essere passata aConfig come parametro.
