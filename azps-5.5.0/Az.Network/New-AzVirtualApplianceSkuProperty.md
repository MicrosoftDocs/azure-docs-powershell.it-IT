# New-AzVirtualApplianceSkuProperty

## SYNOPSIS
Definire una SKU Appliance virtuale di rete per la risorsa.

## SINTASSI

```
New-AzVirtualApplianceSkuProperty -VendorName <String> -BundledScaleUnit <String> -MarketPlaceVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il New-AzVirtualApplianceSkuProperties di ricerca definisce una SKU per la risorsa Appliance virtuale di rete.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $var=New-AzVirtualApplianceSkuProperty -VendorName "barracudasdwanrelease" -BundledScaleUnit 1 -MarketPlaceVersion 'latest'
```

Creare un oggetto Proprietà SKU dell'appliance virtuale da usare con New-AzNetworkVirtualAppliance virtuale.
