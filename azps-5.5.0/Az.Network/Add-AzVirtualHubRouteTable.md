# Add-AzVirtualHubRouteTable

## SYNOPSIS
Crea una risorsa Tabella route hub virtuale figlio di VirtualHub.

## SINTASSI

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
La risorsa tabella delle route dell'hub virtuale contiene l'elenco dei percorsi e un elenco di connessioni a cui può essere collegata e usata per instradare il traffico in un hub virtuale.
