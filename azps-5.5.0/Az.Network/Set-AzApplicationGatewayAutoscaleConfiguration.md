# Set-AzApplicationGatewayAutoscaleConfiguration

## SYNOPSIS
Aggiorna la configurazione della gradazione automatica di un gateway applicazioni.

## SINTASSI

```
Set-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway> -MinCapacity <Int32>
 [-MaxCapacity <Int32>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Set-AzApplicationGatewayAutoscaleConfiguration** modifica la configurazione esistente di gradazioni automatiche di un Gateway applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw -MinCapacity 5
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Il primo comando recupera il gateway applicazione e lo archivia in $gw variabile.
Il secondo comando aggiorna la configurazione di gradazioni automatica dal gateway applicazione.
Il terzo comando aggiorna il gateway applicazione in Azure.
