# New-AzContainerNicConfig

## SYNOPSIS
Crea un nuovo oggetto di configurazione dell'interfaccia di rete del contenitore.

## SINTASSI

```
New-AzContainerNicConfig [-Name <String>] [-IpConfiguration <PSIPConfigurationProfile[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzContainerNicConfig crea** un nuovo oggetto di configurazione dell'interfaccia di rete del contenitore. Questo oggetto determina le caratteristiche dell'interfaccia di rete contenitore creata facendo riferimento al profilo di rete padre.

## ESEMPI

### Esempio 1
```powershell
$containerNicConfig = New-AzContainerNicConfig -Name cnicConfig1

$networkProfile = New-AzNetworkProfile -Name np1 -ResourceGroupName rg1 -Location westus -ContainerNetworkInterfaceConfiguration $containerNicConfig
```

Il primo comando crea una configurazione di interfaccia di rete contenitore vuota. Il secondo crea un nuovo profilo di rete, passando la configurazione dell'interfaccia di rete del contenitore creata in precedenza come argomento al cmdlet New-NetworkProfile.

### Esempio 2

Crea un nuovo oggetto di configurazione dell'interfaccia di rete del contenitore. (autogenerated)
