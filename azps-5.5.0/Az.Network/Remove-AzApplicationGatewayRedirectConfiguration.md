# Remove-AzApplicationGatewayRedirectConfiguration

## SYNOPSIS
Rimuove una configurazione di reindirizzamento da un Gateway applicazioni esistente.

## SINTASSI

```
Remove-AzApplicationGatewayRedirectConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzApplicationGatewayRedirectConfiguration** rimuove una configurazione di reindirizzamento da un Gateway applicazione esistente.

## ESEMPI

### Esempio 1
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\>$AppGw = Remove-AzApplicationGatewayRedirectConfiguration -ApplicationGateway $AppGw -Name "Redirect01"
```

Il primo comando ottiene un gateway applicazione e lo archivia nella $AppGw variabile.
Il secondo comando rimuove la configurazione di reindirizzamento denominata Redirect01 dal gateway applicazione archiviato in $AppGw.
