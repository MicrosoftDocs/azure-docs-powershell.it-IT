# New-AzLoadBalancerOutboundRuleConfig

## SYNOPSIS
Crea una configurazione delle regole in uscita per una bilanciamento del carico.

## SINTASSI

### SetByResource (Impostazione predefinita)
```
New-AzLoadBalancerOutboundRuleConfig -Name <String> [-AllocatedOutboundPort <Int32>] -Protocol <String>
 [-EnableTcpReset] [-IdleTimeoutInMinutes <Int32>] -FrontendIpConfiguration <PSResourceId[]>
 -BackendAddressPool <PSBackendAddressPool> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### SetByResourceId
```
New-AzLoadBalancerOutboundRuleConfig -Name <String> [-AllocatedOutboundPort <Int32>] -Protocol <String>
 [-EnableTcpReset] [-IdleTimeoutInMinutes <Int32>] -FrontendIpConfiguration <PSResourceId[]>
 -BackendAddressPoolId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzLoadBalancerOutboundRuleConfig crea** una configurazione di regola in uscita per una bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1: Creare una configurazione di regole in uscita per una bilanciamento del carico
```powershell
PS C:\>$publicip = New-AzPublicIpAddress -ResourceGroupName "MyResourceGroup" -Name "MyPublicIP" -Location "West US" -AllocationMethod "Dynamic" -Sku "Standard"
PS C:\>$frontend = New-AzLoadBalancerFrontendIpConfig -Name "FrontendIpConfig01" -PublicIpAddress $publicip
PS C:\>$backend = New-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool01"
PS C:\>New-AzLoadBalancerOutboundRuleConfig -Name "MyOutboundRule" -Protocol "Tcp" -FrontendIPConfiguration $frontend -BackendAddressPool $backend
```

Il primo comando crea un indirizzo IP pubblico denominato MyPublicIP nel gruppo di risorse denominato MyResourceGroup e quindi lo archivia nella $publicip variabile.
Il secondo comando crea una configurazione IP front-end denominata FrontendIpConfig01 usando l'indirizzo IP pubblico in $publicip e quindi la archivia nella variabile $frontend>.
Il terzo comando crea una configurazione di pool di indirizzi back-end denominata BackendAddressPool01, che viene quindi archiviata nella variabile $backend back-end.
Il quarto comando crea una configurazione della regola in uscita denominata MyOutboundRule usando gli oggetti front-end e back-end in $frontend e $backend.
I *parametri Protocol,* *FrontendIPConfiguration* e *BackendAddressPool* sono tutti necessari per creare una configurazione della regola in uscita.

### Esempio 2

Crea una configurazione delle regole in uscita per una bilanciamento del carico. (autogenerated)
