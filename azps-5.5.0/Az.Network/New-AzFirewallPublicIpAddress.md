# New-AzFirewallPublicIpAddress

## SYNOPSIS
Segnaposto per l'indirizzo IP che può essere usato per il pip multi-pip in Azure Firewall.

## SINTASSI

```
New-AzFirewallPublicIpAddress [-Address <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Segnaposto per l'indirizzo IP che può essere usato per il pip multi-pip in Azure Firewall.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $publicIp = New-AzFirewallPublicIpAddress -Address 20.2.3.4
```

$publicIp segnaposto per l'indirizzo IP 20.2.3.4
