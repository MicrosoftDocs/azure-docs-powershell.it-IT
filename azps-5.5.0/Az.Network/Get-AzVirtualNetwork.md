# Get-AzVirtualNetwork

## SYNOPSIS
Ottiene una rete virtuale in un gruppo di risorse.

## SINTASSI

### NoExpand
```
Get-AzVirtualNetwork [-Name <String>] [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### Espandi
```
Get-AzVirtualNetwork -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzVirtualNetwork** ottiene una o pi√π reti virtuali n un gruppo di risorse.
