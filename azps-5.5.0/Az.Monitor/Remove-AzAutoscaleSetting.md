# Remove-AzAutoscaleSetting

## SYNOPSIS
Rimuove un'impostazione di scala automatica.

## SINTASSI

```
Remove-AzAutoscaleSetting -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzAutoscaleSetting** rimuove un'impostazione di gradazione automatica.
È necessario specificare il nome dell'impostazione e il nome del gruppo di risorse a cui è assegnata.
Questo cmdlet implementa il criterio ShouldProcess, ad esempio potrebbe richiedere conferma all'utente prima di creare, modificare o rimuovere effettivamente la risorsa.

## ESEMPI

### Esempio 1

Rimuove un'impostazione di scala automatica. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
Remove-AzAutoscaleSetting -Name 'LogAlertRule1' -ResourceGroupName MyResourceGroup
```
