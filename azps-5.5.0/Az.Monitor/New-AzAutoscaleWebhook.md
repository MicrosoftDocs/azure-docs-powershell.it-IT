# New-AzAutoscaleWebhook

## SYNOPSIS
Crea una webhook di gradazioni automatica.

## SINTASSI

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **New-AzAutoscaleWebhook** crea un webhook di gradazioni automatiche.

## ESEMPI

### Esempio 1

Crea una webhook di gradazioni automatica. (autogenerated)

```powershell <!-- Aladdin Generated Example --> 
New-AzAutoscaleWebhook -ServiceUri 'https://contoso.com'
```
