# Remove-AzRelayNamespace

## SYNOPSIS
Rimuove lo spazio dei nomi dal gruppo di risorse specificato. 

## SINTASSI

```
Remove-AzRelayNamespace [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzRelaySburg** rimuove lo spazio dei nomi dal gruppo di risorse specificato.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzRelayNamespace -ResourceGroupName Default-ServiceBus-WestUS -Name TestNameSpace-Relay1
```

Rimuove lo spazio dei nomi `TestNameSpace-Relay1` Relay dal gruppo di risorse `Default-ServiceBus-WestUS` specificato.
