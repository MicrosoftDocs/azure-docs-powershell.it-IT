# Get-AzMigrateProject

## SYNOPSIS
Metodo per eseguire la migrazione di un progetto.

## SINTASSI

```
Get-AzMigrateProject -Name <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## DESCRIZIONE
Metodo per eseguire la migrazione di un progetto.
