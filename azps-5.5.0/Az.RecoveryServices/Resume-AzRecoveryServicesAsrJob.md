# Resume-AzRecoveryServicesAsrJob

## SYNOPSIS
Riprendi un processo sospeso di Azure Site Recovery.

## SINTASSI

### ByObject (impostazione predefinita)
```
Resume-AzRecoveryServicesAsrJob -InputObject <ASRJob> [-Comment <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Resume-AzRecoveryServicesAsrJob -Name <String> [-Comment <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Resume-AzRecoveryServicesAsrJob** riprende un processo sospeso di Azure Site Recovery.

## ESEMPI

### Esempio 1
```
PS C:\> $currentJob = Resume-AzRecoveryServicesAsrJob -Job $Job
```
