# Get-AzRecoveryServicesVaultProperty

## SYNOPSIS
Restituisce le proprietà di un Vault dei servizi di ripristino.

## SINTASSI

```
Get-AzRecoveryServicesVaultProperty [-VaultId <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il **cmdlet Get-AzRecoveryServicesVaultProperty** restituisce le proprietà di un vault dei servizi di recupero.

## ESEMPI

### Esempio 1: Ottenere le proprietà di un vault
```
PS C:\> $vault = Get-AzRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
PS C:\> $props = Get-AzRecoveryServicesVaultProperty -VaultId $vault.Id
PS C:\> $encryption.encryptionProperties
```

Il primo comando ottiene un oggetto Vault e quindi lo archivia nella $vault variabile.
Il secondo comando recupera le proprietà del Vault. A questo punto si accede alle proprietà di crittografia del vault.
