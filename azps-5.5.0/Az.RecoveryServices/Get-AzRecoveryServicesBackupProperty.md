# Get-AzRecoveryServicesBackupProperty

## SYNOPSIS
Recupera le proprietà di backup.

## SINTASSI

```
Get-AzRecoveryServicesBackupProperty -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzRecoveryServicesBackupProperty** ottiene le proprietà di backup per un vault dei servizi di ripristino.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzRecoveryServicesBackupProperty -Vault $vault
```

Ottenere la proprietà del vault di backup per il Vault.
