# Start-AzRecoveryServicesAsrSwitchProcessServerJob

## SYNOPSIS
Passare dalla replica a un altro server di processo per il bilanciamento del carico.

## SINTASSI

```
Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric <ASRFabric> -SourceProcessServer <ASRProcessServer>
 -TargetProcessServer <ASRProcessServer> [-ReplicationProtectedItem <ASRReplicationProtectedItem[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
**Start-AzRecoveryServicesAsrProcessProcessJob** cambia lo spostamento dei dati di replica per le macchine virtuali specificate o per un server di processi specificato nel server di processo di destinazione specificato. Usato per il bilanciamento del carico o il passaggio di replica tra server di processo.

## ESEMPI

### Esempio 1
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer
```

Processo per tenere traccia del passaggio del server di elaborazione per tutto l'elemento protetto da replica dal server di processo di origine a quello di destinazione.

### Esempio 2
```
PS C:\> Start-AzRecoveryServicesAsrSwitchProcessServerJob -Fabric $fabric -SourceProcessServer $sourceProcessServer  -TargetProcessServer $TargetProcessServer -ReplicatedItem $rpList
```

Processo per tenere traccia del passaggio da un server di processo a un altro per un elemento protetto da replica passato da un server di processo di origine a un server di processo di destinazione.
