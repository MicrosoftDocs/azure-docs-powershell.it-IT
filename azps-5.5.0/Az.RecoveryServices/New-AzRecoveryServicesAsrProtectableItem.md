# New-AzRecoveryServicesAsrProtectableItem

## SYNOPSIS
Aggiungere(Individuare) un server fisico all'elenco di elementi proteggibili.

## SINTASSI

```
New-AzRecoveryServicesAsrProtectableItem -ProtectionContainer <ASRProtectionContainer> -FriendlyName <String>
 -IPAddress <String> -OSType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## DESCRIZIONE
**New-AzRecoveryServicesAsrProtectableItem** aggiunge un nuovo elemento proteggibile all'elenco degli elementi individuati come proteggibili in un contenitore di protezione all'interno di un tessuto ASR (applicabile solo per il tipo di tessuto VMware).

## ESEMPI

### Esempio 1

Aggiungere(Individuare) un server fisico all'elenco di elementi proteggibili. (autogenerated)

```powershell <!-- Aladdin Generated Example -->

```
