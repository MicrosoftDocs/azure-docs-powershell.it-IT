# Get-AzRecoveryServicesAsrStorageClassification

## SYNOPSIS
Ottiene le classificazioni di archiviazione ASR disponibili (individuate) nella vault dei servizi di ripristino.

## SINTASSI

### ByFabricObject (impostazione predefinita)
```
Get-AzRecoveryServicesAsrStorageClassification -Fabric <ASRFabric> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByObjectWithName
```
Get-AzRecoveryServicesAsrStorageClassification -Name <String> -Fabric <ASRFabric>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByObjectWithFriendlyName
```
Get-AzRecoveryServicesAsrStorageClassification -FriendlyName <String> -Fabric <ASRFabric>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzRecoveryServicesAsrStorageClassification** ottiene i dettagli delle classificazioni di archiviazione ASR individuate nel vault dei servizi di ripristino.

## ESEMPI

### Esempio 1
```
PS C:\> $StorageClassifications = Get-AzRecoveryServicesAsrStorageClassification -Fabric $Fabric
```

Elencare le classificazioni di archiviazione individuate corrispondenti al tessuto ASR specificato.
