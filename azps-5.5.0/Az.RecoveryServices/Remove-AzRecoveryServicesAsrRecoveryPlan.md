# Remove-AzRecoveryServicesAsrRecoveryPlan

## SYNOPSIS
Elimina il piano di ripristino asr specificato dalla vault dei servizi di ripristino.

## SINTASSI

### ByObject (impostazione predefinita)
```
Remove-AzRecoveryServicesAsrRecoveryPlan -InputObject <ASRRecoveryPlan>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByName
```
Remove-AzRecoveryServicesAsrRecoveryPlan -Name <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Remove-AzRecoveryServicesAsrRecoveryPlan** elimina il piano di ripristino specificato dal vault dei servizi di ripristino.

## ESEMPI

### Esempio 1
```
PS C:\> $currentJob = Remove-AzRecoveryServicesAsrRecoveryPlan -RecoveryPlan $RP
```

Avvia l'eliminazione del piano di ripristino specificato e restituisce il processo a matrice usato per tenere traccia dell'operazione.
