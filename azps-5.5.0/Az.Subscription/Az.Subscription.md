# Modulo Az.Subscription
## Descrizione
Gli argomenti di questa sezione documentano i cmdlet di Azure PowerShell per la sottoscrizione di Azure nel framework di Gestione risorse di Azure (ARM). I cmdlet sono presenti nello spazio dei nomi Microsoft.Azure.Commands.Subscription.

## Cmdlet Az.Subscription
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Recupera i dettagli dell'alias dell'abbonamento

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Crea un nuovo alias e un nuovo abbonamento

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Elimina l'alias dell'abbonamento

### [Update-AzSubscription](Update-AzSubscription.md)
Aggiorna una sottoscrizione di Azure
