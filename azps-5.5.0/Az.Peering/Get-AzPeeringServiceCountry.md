# Get-AzPeeringServiceCountry

## SYNOPSIS
Elenca i paesi disponibili per il servizio di peering.

## SINTASSI

```
Get-AzPeeringServiceCountry [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## DESCRIZIONE
Elenca i paesi disponibili per il servizio di peering.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzPeeringServiceCountry
```

Elenco dei paesi disponibili per il servizio di peering.
