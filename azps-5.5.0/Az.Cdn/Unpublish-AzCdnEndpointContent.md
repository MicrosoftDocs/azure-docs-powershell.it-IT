# Unpublish-AzCdnEndpointContent

## SYNOPSIS
Eliminazione dell'endpoint di una rete CDN.

## SINTASSI

### ByFieldsParameterSet (Impostazione predefinita)
```
Unpublish-AzCdnEndpointContent -EndpointName <String> -ProfileName <String> -ResourceGroupName <String>
 -PurgeContent <String[]> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### ByObjectParameterSet
```
Unpublish-AzCdnEndpointContent -CdnEndpoint <PSEndpoint> -PurgeContent <String[]> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Unpublish-AzCdnEndpointContent** elimina il contenuto da un endpoint della rete per la distribuzione di contenuti (CDN) di Azure.
