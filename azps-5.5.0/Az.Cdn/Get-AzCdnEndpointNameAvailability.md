# Get-AzCdnEndpointNameAvailability

## SYNOPSIS
Ottiene lo stato di disponibilità dell'endpoint della rete CDN.

## SINTASSI

```
Get-AzCdnEndpointNameAvailability -EndpointName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzCdnEndpointNameAvailability** ottiene lo stato di disponibilità dell'endpoint rete per la distribuzione di contenuti (CDN) di Azure.
