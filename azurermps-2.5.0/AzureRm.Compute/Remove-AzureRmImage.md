# Remove-AzureRmImage

## Sinossi
Rimuove un'immagine.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmImage [-ResourceGroupName] <String> [-ImageName] <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmImage** rimuove un'immagine.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzureRmImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' -Force;
```

Questo comando rimuove l'immagine denominata "Image01" nel gruppo di risorse "ResourceGroup01".
