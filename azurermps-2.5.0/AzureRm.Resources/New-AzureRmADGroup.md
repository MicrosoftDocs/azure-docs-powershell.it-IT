# New-AzureRmADGroup

## Sinossi
Crea un nuovo gruppo di Active Directory.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmADGroup -DisplayName <String> -MailNickname <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Crea un nuovo gruppo di Active Directory.

## ESEMPI

### Esempio 1-creare un nuovo gruppo di annunci

```
PS C:\> New-AzureRmADGroup -DisplayName "MyGroupDisplayName" -MailNickname "myemail@domain.com"
```

Crea un nuovo gruppo di annunci con il nome "MyGroupDisplayName" e il Nickname di posta " myemail@domain.com ".
