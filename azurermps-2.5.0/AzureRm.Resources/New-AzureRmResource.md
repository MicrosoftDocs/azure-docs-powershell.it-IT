## Descrizione
Il cmdlet **New-AzureRmResource** crea una risorsa Azure, ad esempio un sito Web, un server di database SQL di Azure o un database SQL di Azure, in un gruppo di risorse.

## ESEMPI

### Esempio 1: creare una risorsa
```
PS> New-AzureRmResource -Location "West US" -Properties @{test="test"} -ResourceName TestSite06 -ResourceType microsoft.web/sites -ResourceGroupName ResourceGroup11 -Force
```

Questo comando crea una risorsa che è un sito Web in ResourceGroup11.

### Esempio 2: creare una risorsa con splatting
```
PS> $prop = @{
    Location          = "West US" 
    Properties        = @{test = "test"} 
    ResourceName      = "TestSite06" 
    ResourceType      = "microsoft.web/sites" 
    ResourceGroupName = "ResourceGroup11" 
    Force             = $true
}

PS> New-AzureRmResource @prop
```

Questo comando crea una risorsa che è un sito Web in ResourceGroup11.
