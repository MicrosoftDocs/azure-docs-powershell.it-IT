# Remove-AzureRmManagedApplication

## Sinossi
Rimuove un'applicazione gestita

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### RemoveByNameAndResourceGroup (impostazione predefinita)
```
Remove-AzureRmManagedApplication -Name <String> -ResourceGroupName <String> [-Force] [-ApiVersion <String>]
 [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveById
```
Remove-AzureRmManagedApplication -Id <String> [-Force] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmManagedApplication** rimuove un'applicazione gestita

## ESEMPI

### Esempio 1: rimuovere l'applicazione gestita dall'ID risorsa
```
PS C:\>$Application = Get-AzureRmManagedApplication -Name "myApp" -ResourceGroupName "myRG"
PS C:\>Remove-AzureRmManagedApplication -Id $Application.ResourceId -Force
```

Il primo comando ottiene un'applicazione gestita denominata myApp tramite il cmdlet Get-AzureRmManagedApplication.
Il comando lo archivia nella variabile $Application.

Il secondo comando rimuove l'applicazione gestita identificata dalla proprietà **resourceId** di $Application.
