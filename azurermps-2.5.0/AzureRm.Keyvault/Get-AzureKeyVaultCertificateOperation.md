# Get-AzureKeyVaultCertificateOperation

## Sinossi
Ottiene lo stato di un'operazione di certificato.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureKeyVaultCertificateOperation** ottiene lo stato di un'operazione di certificato.
