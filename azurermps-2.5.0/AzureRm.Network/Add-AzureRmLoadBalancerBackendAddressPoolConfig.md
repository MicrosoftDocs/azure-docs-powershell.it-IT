# Add-AzureRmLoadBalancerBackendAddressPoolConfig

## Sinossi
Aggiunge una configurazione del pool di indirizzi back-end a un bilanciamento del carico.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Add-AzureRmLoadBalancerBackendAddressPoolConfig -Name <String> -LoadBalancer <PSLoadBalancer>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzureRmLoadBalancerBackend** aggiunge un pool di indirizzi back-end a un bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1 aggiungere una configurazione del pool di indirizzi back-end a un bilanciamento del carico
```
PS C:\>Get-AzureRmLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "myrg" | Add-AzureRmLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" | Set-AzureRmLoadBalancer
```

Questo comando ottiene il bilanciamento del carico denominato MyLoadBalancer, aggiunge il pool di indirizzi back-end denominato BackendAddressPool02 a MyLoadBalancer e quindi usa il cmdlet **set-AzureRmLoadBalancer** per aggiornare MyLoadBalancer.
