# Remove-AzureRmVirtualNetworkGateway

## Sinossi
Elimina un gateway di rete virtuale

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmVirtualNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il gateway di rete virtuale è l'oggetto che rappresenta il gateway in Azure.

Il cmdlet **Get-AzureRmVirtualNetworkGateway** restituisce l'oggetto del gateway in Azure in base al nome e al nome del gruppo di risorse.

## ESEMPI

### 1: eliminare un gateway di rete virtuale
```
Remove-AzureRmVirtualNetworkGateway -Name myGateway -ResourceGroupName myRG
```

Elimina l'oggetto del gateway di rete virtuale con il nome "gateway" nel gruppo di risorse "myRG"

Nota: prima di tutto, è necessario eliminare tutte le connessioni al gateway di rete virtuale usando il cmdlet **Remove-AzureRmVirtualNetworkGatewayConnection** .
