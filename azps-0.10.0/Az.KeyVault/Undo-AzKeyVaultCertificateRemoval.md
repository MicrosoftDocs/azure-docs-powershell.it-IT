# Undo-AzKeyVaultCertificateRemoval

## Sinossi
Recuperare un certificato eliminato in un Vault chiave in uno stato attivo.

## SINTASSI

```
Undo-AzKeyVaultCertificateRemoval [-VaultName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Undo-AzKeyVaultCertificateRemoval** recupererà un certificato eliminato in precedenza.
Il certificato recuperato sarà attivo e potrà essere usato per tutte le operazioni.
Per eseguire questa operazione, il chiamante deve avere l'autorizzazione "Recupera".

## ESEMPI

### Esempio 1
```
PS C:\> Undo-AzKeyVaultCertificateRemoval -VaultName 'MyKeyVault' -Name 'MyCertificate'
```

Questo comando recupererà il certificato "certificato" eliminato in precedenza in uno stato attivo e utilizzabile.
