# Remove-AzKeyVaultCertificateOperation

## Sinossi
Elimina un'operazione di certificato da un Vault chiave.

## SINTASSI

```
Remove-AzKeyVaultCertificateOperation [-VaultName] <String> [-Name] <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzKeyVaultCertificateOperation** Elimina un'operazione di certificato da un Vault chiave.

## ESEMPI

### Esempio 1: rimuovere un'operazione di certificato
```
PS C:\>Remove-AzKeyVaultCertificateOperation -VaultName "ContosoKV01" -Name "TestCert01" -Force
```

Questo comando rimuove l'operazione di certificato denominata TestCert01 dall'archivio della chiave ContosoKV01 senza richiedere conferma.
