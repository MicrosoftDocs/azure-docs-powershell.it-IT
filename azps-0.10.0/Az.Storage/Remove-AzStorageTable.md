# Remove-AzStorageTable

## Sinossi
Rimuove una tabella di archiviazione.

## SINTASSI

```
Remove-AzStorageTable [-Name] <String> [-Force] [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzStorageTable** consente di rimuovere una o più tabelle di archiviazione da un account di archiviazione in Azure.

## ESEMPI

### Esempio 1: rimuovere una tabella
```
PS C:\>Remove-AzStorageTable -Name "TableABC"
```

Questo comando rimuove una tabella.

### Esempio 2: rimuovere più tabelle
```
PS C:\>Get-AzStorageTable table* | Remove-AzStorageTable
```

Questo esempio usa un carattere jolly con il parametro *Name* per ottenere tutte le tabelle che corrispondono alla tabella pattern e quindi passa il risultato sulla pipeline per rimuovere le tabelle.
