# Enable-AzureRmAlias

## Sinossi
Abilita gli alias di prefisso AzureRm per i moduli AZ.

## SINTASSI

```
Enable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Abilita gli alias di prefisso AzureRm per i moduli AZ. Se-Module è specificato, solo i moduli elencati avranno gli alias abilitati. In caso contrario, tutti gli alias di AzureRm sono abilitati.

## ESEMPI

### Esempio 1
```
PS C:\> Enable-AzureRmAlias
```

Abilita tutti i prefissi di AzureRm per la sessione corrente di PowerShell.

### Esempio 1
```
PS C:\> Enable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Abilita gli alias di AzureRm per il modulo AZ. Accounts sia per il processo corrente che per l'utente corrente.
