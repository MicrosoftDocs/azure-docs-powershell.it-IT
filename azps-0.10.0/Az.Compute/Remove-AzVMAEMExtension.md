# Remove-AzVMAEMExtension

## Sinossi
Rimuove l'estensione AEM da una macchina virtuale.

## SINTASSI

```
Remove-AzVMAEMExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>]
 [[-OSType] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzVMAEMExtension** rimuove l'estensione AEM (Azure Enhanced Monitoring) da una macchina virtuale.

## ESEMPI

### Esempio 1: rimuovere l'estensione AEM
```
PS C:\> Remove-AzVMAEMExtension -ResourceGroupName "ResourceGroup11" -VMName "contoso-server"
```

Questo comando rimuove l'estensione AEM per la macchina virtuale denominata Contoso-server.
