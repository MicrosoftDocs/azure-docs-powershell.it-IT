# Update-AzImage

## Sinossi
Aggiorna un'immagine.

## SINTASSI

```
Update-AzImage [-ResourceGroupName] <String> [-ImageName] <String> [-Image] <PSImage> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Update-aziming** aggiorna un'immagine.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzImage -ResourceGroupName 'ResourceGroup01' -ImageName 'Image01' | Remove-AzImageDataDisk -Lun 1 | Update-AzImage;
```

Questo comando rimuove il disco di dati dell'unit√† logica numero 1 dall'immagine esistente "Image01" nel gruppo di risorse "ResourceGroup01".
