# Remove-AzVmssExtension

## Sinossi
Rimuove un'estensione da VMSS.

## SINTASSI

### NameParameterSet
```
Remove-AzVmssExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSet
```
Remove-AzVmssExtension [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [-Id] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzVmssExtension** consente di rimuovere un'estensione dal set di scale della macchina virtuale (VMSS).

## ESEMPI

### Esempio 1: rimuovere l'estensione da VMSS
```
PS C:\> Remove-AzVmssExtension -VirtualMachineScaleSet $VMSS -Name $extName
```
