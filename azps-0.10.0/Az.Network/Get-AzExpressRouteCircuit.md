# Get-AzExpressRouteCircuit

## Sinossi
Ottiene un circuito di Azure ExpressRoute da Azure.

## SINTASSI

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzExpressRouteCircuit** viene usato per recuperare un oggetto Circuit ExpressRoute dall'abbonamento. L'oggetto Circuit restituito pu√≤ essere usato come input per altri cmdlet che operano su circuiti ExpressRoute.

## ESEMPI

### Esempio 1: ottenere il circuito ExpressRoute da eliminare
```
Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzExpressRouteCircuit
```
