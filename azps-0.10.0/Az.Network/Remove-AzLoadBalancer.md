# Remove-AzLoadBalancer

## Sinossi
Rimuove un bilanciamento del carico.

## SINTASSI

```
Remove-AzLoadBalancer -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzLoadBalancer** rimuove un bilanciamento del carico di Azure.

## ESEMPI

### Esempio 1: rimuovere un bilanciamento del carico
```
PS C:\>Remove-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
```

Questo comando Elimina un bilanciamento del carico denominato MyLoadBalancer nel gruppo di risorse denominato MyResourceGroup.
