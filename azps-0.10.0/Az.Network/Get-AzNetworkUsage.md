# Get-AzNetworkUsage

## Sinossi
Elenca gli usi di rete per un abbonamento

## SINTASSI

```
Get-AzNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzNetworkUsage ottiene i limiti e l'utilizzo corrente per le risorse di rete.

## ESEMPI

