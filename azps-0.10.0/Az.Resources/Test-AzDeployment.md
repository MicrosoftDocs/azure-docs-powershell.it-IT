# Test-AzDeployment

## Sinossi
Convalida una distribuzione.

## Descrizione
Il cmdlet **test-AzDeployment** determina se un modello di distribuzione e i relativi valori di parametro sono validi.

## ESEMPI

### Esempio 1: testare la distribuzione con un modello e un file di parametro personalizzati
```
PS C:\>Test-AzDeployment -Location "West US" -TemplateFile "D:\Azure\Templates\EngineeringSite.json" -TemplateParameterFile "D:\Azure\Templates\EngSiteParms.json"
```

Questo comando verifica una distribuzione nell'ambito dell'abbonamento corrente usando il file di modello e il file di parametri specificati.
