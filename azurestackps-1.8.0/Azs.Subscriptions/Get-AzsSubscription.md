# Get-AzsSubscription

## Sinossi
Ottenere l'elenco degli abbonamenti.

## SINTASSI

### Elenco (impostazione predefinita)
```
Get-AzsSubscription [<CommonParameters>]
```

### Ottieni
```
Get-AzsSubscription [-SubscriptionId] <String> [<CommonParameters>]
```

## Descrizione
Ottenere l'elenco degli abbonamenti.

## ESEMPI

### ESEMPIO 1
```
Get-AzsSubscription
```

Ottenere l'elenco degli abbonamenti.
