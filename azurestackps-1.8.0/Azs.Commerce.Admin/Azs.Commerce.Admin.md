# Modulo di AZS. Commerce. admin
## Descrizione
Versione di anteprima del modulo di amministrazione di AzureStack commerce che consente di visualizzare l'utilizzo complessivo dei dati in un timbro AzureStack. 

## Cmdlet AZS. Commerce. admin
### [Get-AzsSubscriberUsage](Get-AzsSubscriberUsage.md)
Recuperare i dati sull'utilizzo durante l'intervallo specificato.
