# Get-AzsLoadBalancer

## Sinossi
Ottenere un elenco di tutti i bilanciatori di carico.

## SINTASSI

```
Get-AzsLoadBalancer [[-Filter] <String>] [[-OrderBy] <String>] [[-Skip] <Int32>] [[-Top] <Int32>]
 [[-InlineCount] <String>] [<CommonParameters>]
```

## Descrizione
Ottenere un elenco di tutti i bilanciatori di carico.

## ESEMPI

### ESEMPIO 1
```
Get-AzsLoadBalancer
```

Ottenere un elenco di tutti i bilanciatori di carico.
