# Get-AzsStorageDestinationShare

## Sinossi
Restituisce un elenco di condivisioni di destinazione che il sistema considera come migliori candidati per la migrazione.

## SINTASSI

```
Get-AzsStorageDestinationShare [-SourceShareName] <String> [-FarmName] <String> [[-ResourceGroupName] <String>]
 [<CommonParameters>]
```

## Descrizione
Restituisce un elenco di condivisioni di destinazione che il sistema considera come migliori candidati per la migrazione.

## ESEMPI

### --------------------------ESEMPIO 1--------------------------
```
Get-AzsStorageDestinationShare -FarmName f9b8e2e2-e4b4-44e0-9d92-6a848b1a5376 -SourceShareName "||SU1FileServer.azurestack.local|SU1_ObjStore"
```

Ottenere un elenco delle condivisioni di destinazione che il sistema ritiene migliori candidati per la migrazione.
