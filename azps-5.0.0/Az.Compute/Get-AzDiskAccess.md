# Get-AzDiskAccess

## Sinossi
Ottiene le proprietà degli accessi su disco

## SINTASSI

### DefaultParameterSet (impostazione predefinita)
```
Get-AzDiskAccess [[-ResourceGroupName] <String>] [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ResourceIDParameterSet
```
Get-AzDiskAccess [-ResourceId] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzDiskAccess** ottiene le proprietà degli accessi su disco

## ESEMPI

### Esempio 1: uso del set di parametri predefinito 
```
PS C:\> Get-AzDiskAccess -ResourceGroupName 'ResourceGroup01' -Name 'DiskAccess01'

ProvisioningState          : Succeeded
TimeCreated                : 8/13/2020 7:02:50 PM

Name                       : DiskAccess01
Type                       : Microsoft.Compute/diskAccesses
Tags                       : {}
```

Questo comando consente di ottenere le proprietà di una risorsa di Access su disco denominata "DiskAccess01" nel gruppo di risorse "ResourceGroup01".
