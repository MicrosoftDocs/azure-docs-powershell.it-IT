# Modulo AZ. CustomProviders
## Descrizione
PowerShell di Microsoft Azure: cmdlet di provider personalizzati

## Cmdlet AZ. CustomProviders
### [Get-AzCustomProvider](Get-AzCustomProvider.md)
Ottiene il manifesto del provider di risorse personalizzato.

### [Get-AzCustomProviderAssociation](Get-AzCustomProviderAssociation.md)
Ottenere un'associazione.

### [New-AzCustomProvider](New-AzCustomProvider.md)
Crea o aggiorna il provider di risorse personalizzato.

### [New-AzCustomProviderAssociation](New-AzCustomProviderAssociation.md)
Creare o aggiornare un'associazione.

### [Remove-AzCustomProvider](Remove-AzCustomProvider.md)
Elimina il provider di risorse personalizzato.

### [Remove-AzCustomProviderAssociation](Remove-AzCustomProviderAssociation.md)
Eliminare un'associazione.

### [Update-AzCustomProvider](Update-AzCustomProvider.md)
Aggiorna un provider di risorse personalizzato esistente.
L'unico valore che può essere aggiornato tramite PATCH è attualmente i tag.
