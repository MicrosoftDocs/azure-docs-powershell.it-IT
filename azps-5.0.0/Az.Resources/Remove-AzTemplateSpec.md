# Remove-AzTemplateSpec

## Sinossi
Rimuove una specifica modello

## SINTASSI

### RemoveByNameParameterSet (impostazione predefinita)
```
Remove-AzTemplateSpec [-Force] [-ResourceGroupName] <String> [-Name] <String> [[-Version] <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByIdParameterSet
```
Remove-AzTemplateSpec [-Force] [[-Version] <String>] [-ResourceId] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Rimuove una specifica del modello specificata. Se il parametro version **-Version** viene specificato, solo la versione specificata verrà rimossa. Se non è disponibile una versione specifica, la specifica del modello e tutte le relative versioni verranno rimosse. Se il contrassegno di forza è presente **,** non verrà visualizzata alcuna richiesta di conferma per la rimozione.

## ESEMPI

### Esempio 1: rimozione di una versione specifica per nome
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'MyTemplateSpec' -Version 'v1.0'
```

Rimuove la versione "v 1.0" della specifica del modello denominata "MyTemplateSpec" nel gruppo di risorse "myRG".

### Esempio 2: rimozione di una versione specifica tramite ID risorsa
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceId '/subscriptions/{subId}/resourceGroups/myRG/providers/Microsoft.Resources/templateSpecs/MyTemplateSpec' -Version 'v1.0'
```

Rimuove la versione "v 1.0" della specifica del modello denominata "MyTemplateSpec" nel gruppo di risorse "myRG" dell'abbonamento \{ subId \} .

### Esempio 3: rimozione di una specifica modello e di tutte le versioni per nome
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceGroupName 'myRG' -Name 'MyTemplateSpec'
```

Rimuove la specifica del modello denominata "MyTemplateSpec" e tutte le relative versioni nel gruppo di risorse "myRG".

### Esempio 3: rimozione di una specifica modello e di tutte le versioni per ID risorsa
```powershell
PS C:\> Remove-AzTemplateSpec -ResourceId '/subscriptions/{subId}/resourceGroups/myRG/providers/Microsoft.Resources/templateSpecs/MyTemplateSpec' -ResourceGroupName 'myRG'
```

Rimuove la specifica del modello denominata "MyTemplateSpec" e tutte le relative versioni nel gruppo di risorse "myRG" dell'abbonamento \{ subId \} .
