# New-AzADGroup

## Sinossi
Crea un nuovo gruppo di Active Directory.

## SINTASSI

```
New-AzADGroup -DisplayName <String> -MailNickname <String> [-Description <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Crea un nuovo gruppo di Active Directory. Di seguito sono elencate le autorizzazioni necessarie:

- Grafico di Azure Active Directory
  - Directory. ReadWrite. All
- Microsoft Graph
  - Directory. ReadWrite. All
  - PrivilegedAccess. ReadWrite. AzureADGroup

## ESEMPI

### Esempio 1: creare un nuovo gruppo di annunci

```powershell
PS C:\> New-AzADGroup -DisplayName "MyGroupDisplayName" -MailNickname "MyGroupNick"
```

Crea un nuovo gruppo di annunci con il nome "MyGroupDisplayName" e il Nickname di posta "MyGroupNick".
