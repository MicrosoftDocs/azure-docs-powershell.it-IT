# Invoke-AzDataBoxEdgeDevice

## Sinossi
Richiama azioni specifiche nel dispositivo.

## SINTASSI

## Descrizione
Il cmdlet **Invoke-AzDataBoxEdgeDevice** richiama le azioni per analizzare, scaricare e installare gli aggiornamenti nel dispositivo Edge della casella dati. Un'analisi automatica viene eseguita giornalmente sul dispositivo, Ã¨ possibile attivare l'analisi in modo esplicito eseguendo questo cmdlet.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -ScanForUpdate
```

### Esempio 2
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -FetchUpdate
```

### Esempio 3
```powershell
PS C:\> Invoke-AzDataBoxEdgeDevice -ResourceGroupName resourceGroupName -Name deviceName -InstallUpdate
```
