# Get-AzRedisCache

## Sinossi
Ottiene una cache di Redis.

## SINTASSI

```
Get-AzRedisCache [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzRedisCache** ottiene la cache di Azure Redis specificata.
Se non si specificano parametri, questa operazione ottiene ogni cache di redis per l'abbonamento corrente.

## ESEMPI

### Esempio 1: ottenere una cache Redis per nome

```
        Type               : Microsoft.Cache/Redis
        HostName           : myexists.redis.cache.windows.net
        Port               : 6379

        RedisConfiguration : {}
        EnableNonSslPort   : False
        RedisVersion       : 2.8
        Size               : 1GB
        Sku                : Basic
        Tag                : {}
        Zone               : []
```

Questo comando consente di ottenere la cache Redis denominata esisto.
