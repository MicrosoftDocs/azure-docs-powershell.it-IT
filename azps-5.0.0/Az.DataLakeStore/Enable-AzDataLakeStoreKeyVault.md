# Enable-AzDataLakeStoreKeyVault

## Sinossi
Consente di consentire a un Vault Key gestito dall'utente di crittografare l'account di data Lake Store specificato.

## SINTASSI

```
Enable-AzDataLakeStoreKeyVault [-Account] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Enable-AzDataLakeStoreKeyVault** tenta di abilitare un Vault Key gestito dall'utente per la crittografia dell'account di data Lake Store specificato.

## ESEMPI

### Esempio 1: abilitare il Vault chiave per l'account di ContosoADLS
```
PS C:\>Enable-AzDataLakeStoreKeyVault -Name "ContosoADLS"
```

Questo comando tenta di abilitare l'archiviazione delle chiavi gestite dall'utente per l'account di data Lake Store denominato ContosoADLS.
