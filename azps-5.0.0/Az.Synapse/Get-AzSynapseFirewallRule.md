# Get-AzSynapseFirewallRule

## Sinossi
Ottiene una regola del firewall di analisi sinapsi.

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzSynapseFirewallRule [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseFirewallRule -WorkSpaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSynapseFirewallRule** ottiene una regola del firewall di analisi di Azure sinapsi.
Se non specifichi un nome di regola, questo cmdlet ottiene tutte le regole.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSynapseFirewallRule -ResourceGroupName ContosoResourceGroup -WorkspaceName ContosoWorkspace
```

Questo comando consente di ottenere tutte le regole del firewall in un'area di lavoro.

### Esempio 2
```powershell
PS C:\> Get-AzSynapseFirewallRule -ResourceGroupName ContosoResourceGroup -WorkspaceName ContosoWorkspace -Name ContosoFirewallRule
```

Questo comando consente di ottenere la regola del firewall in area di lavoro ContosoWorkspace con nome ContosoFirewallRule.

### Esempio 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseFirewallRule
```

Questo comando consente di ottenere tutte le regole del firewall in un'area di lavoro tramite pipeline.
