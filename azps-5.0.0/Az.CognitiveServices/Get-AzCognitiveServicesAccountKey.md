# Get-AzCognitiveServicesAccountKey

## Sinossi
Ottiene le chiavi dell'API per un account.

## SINTASSI

```
Get-AzCognitiveServicesAccountKey [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzCognitiveServicesAccountKey** ottiene le chiavi API per un account di servizi cognitivi con provisioning.
Un account di servizi cognitivi dispone di due chiavi API: key1 e key2.
I tasti consentono l'interazione con l'endpoint dell'account dei servizi cognitivi.
Usare New-AzCognitiveServicesAccountKey per rigenerare una chiave.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzCognitiveServicesAccountKey -ResourceGroupName cognitive-services-resource-group -name myluis

Key1                             Key2
----                             ----
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
