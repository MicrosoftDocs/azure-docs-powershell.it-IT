# Update-AzApplicationInsights

## Sinossi
aggiornare una risorsa approfondimenti applicazione esistente

## SINTASSI

```
Update-AzApplicationInsights [-ResourceGroupName] <String> [-Name] <String> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [[-RetentionInDays] <Int32>]
 [[-PublicNetworkAccessForIngestion] <String>] [[-PublicNetworkAccessForQuery] <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
aggiornare una risorsa approfondimenti applicazione esistente

## ESEMPI

### Esempio 1 componente approfondimenti dell'applicazione di aggiornamento
```powershell
Update-AzApplicationInsights -ResourceGroupName "rgName" -Name "aiName" -PublicNetworkAccessForIngestion "Disabled" -PublicNetworkAccessForQuery "Disabled"
```

aggiornare il componente Insights applicazione "aiName" PublicNetworkAccessForIngestion/PublicNetworkAccessForQuery sia per "disabled"
