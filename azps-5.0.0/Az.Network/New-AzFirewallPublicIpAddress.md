# New-AzFirewallPublicIpAddress

## Sinossi
Questo è il segnaposto per l'indirizzo IP che può essere usato per multi PIP in Azure firewall.

## SINTASSI

```
New-AzFirewallPublicIpAddress [-Address <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Questo è il segnaposto per l'indirizzo IP che può essere usato per multi PIP in Azure firewall.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $publicIp = New-AzFirewallPublicIpAddress -Address 20.2.3.4
```

$publicIp sarà il segnaposto per l'indirizzo IP 20.2.3.4
