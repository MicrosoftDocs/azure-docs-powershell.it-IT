# Add-AzVirtualHubRouteTable

## Sinossi
Crea una risorsa della tabella route hub virtuale che è un elemento figlio di VirtualHub.

## SINTASSI

```
Add-AzVirtualHubRouteTable -Name <String> -Route <PSVirtualHubRoute[]> -Connection <String[]>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
La risorsa tabella route hub virtuale contiene l'elenco di route e un elenco di connessioni a cui può essere associato e viene usato per instradare il traffico in un hub virtuale.
