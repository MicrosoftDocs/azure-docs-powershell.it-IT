# Update-AzVpnServerConfiguration

## Sinossi

## Descrizione
Il cmdlet **Update-AzVpnServerConfiguration** consente di aggiornare il VpnServerConfiguration esistente con diversi VpnProtocols, VpnAuthenticationTypes, IpsecPolicies e di impostare i parametri correlati per il tipo di autenticazione VPN selezionati in base al requisito del cliente per la connettività punto a sito.
