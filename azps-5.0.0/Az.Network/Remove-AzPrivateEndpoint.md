# Remove-AzPrivateEndpoint

## Sinossi
Rimuove un endpoint privato.

## SINTASSI

```
Remove-AzPrivateEndpoint -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzPrivateEndpoint** rimuove un endpoint privato. 

## ESEMPI

### Esempio 1
```
Remove-AzPrivateEndpoint -Name MyPrivateEndpoint1 -ResourceGroupName TestResourceGroup
```

In questo esempio viene rimosso un endpoint privato denominato MyPrivateEndpoint1.
