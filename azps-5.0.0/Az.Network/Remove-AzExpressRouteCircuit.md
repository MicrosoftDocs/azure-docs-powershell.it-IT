# Remove-AzExpressRouteCircuit

## Sinossi
Rimuove un circuito ExpressRoute.

## SINTASSI

```
Remove-AzExpressRouteCircuit -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzExpressRouteCircuit** rimuove un circuito ExpressRoute.

## ESEMPI

### Esempio 1: eliminare un circuito ExpressRoute
```
Remove-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg
```

### Esempio 2: eliminare un circuito ExpressRoute con la pipeline
```
Get-AzExpressRouteCircuit -Name $CircuitName -ResourceGroupName $rg | Remove-AzExpressRouteCircuit
```
