# Remove-AzNetworkInterfaceIpConfig

## Sinossi
Rimuove una configurazione IP dell'interfaccia di rete da un'interfaccia di rete.

## SINTASSI

```
Remove-AzNetworkInterfaceIpConfig -Name <String> -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzNetworkInterfaceIpConfig** consente di rimuovere una configurazione IP di interfaccia di rete da un'interfaccia di rete Azure.

## ESEMPI

### Esempio 1: eliminare una configurazione IP da un'interfaccia di rete
```powershell
$nic = Get-AzNetworkInterface -Name mynic -ResourceGroupName myrg

Remove-AzNetworkInterfaceIpConfig -Name IPConfig-1 -NetworkInterface $nic

Set-AzNetworkInterface -NetworkInterface $nic
```

Il primo comando ottiene un'interfaccia di rete denominata MYNIC e la archivia nella variabile $nic. Il secondo comando rimuove la configurazione IP chiamata IPConfig-1 associata a questa interfaccia di rete. Il terzo comando imposta le modifiche apportate all'interfaccia di rete.
