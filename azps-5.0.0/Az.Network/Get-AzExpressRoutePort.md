# Get-AzExpressRoutePort

## Sinossi
Ottiene una risorsa di Azure ExpressRoutePort.

## SINTASSI

### ResourceNameParameterSet (impostazione predefinita)
```
Get-AzExpressRoutePort [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePort -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzExpressRoutePort** viene usato per recuperare un oggetto ExpressRoutePort dall'abbonamento. L'oggetto expressrouteport restituito puÃ² essere usato come input per altri cmdlet che operano in ExpressRoutePort.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Ottiene l'oggetto ExpressRoutePort con il nome $PortName nel gruppo di risorse $rg nell'abbonamento.

### Esempio 2
```powershell
PS C:\> Get-AzExpressRoutePort -Name test*
```

Ottiene tutti gli oggetti ExpressRoutePort il cui nome inizia con "test".

### Esempio 3
```powershell
PS C:\> Get-AzExpressRoutePort -ResourceId $id
```

Ottiene l'oggetto ExpressRoutePort con ResourceId $id.
