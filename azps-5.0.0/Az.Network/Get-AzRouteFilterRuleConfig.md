# Get-AzRouteFilterRuleConfig

## Sinossi
Ottiene una regola di filtro route in un filtro di route.

## SINTASSI

```
Get-AzRouteFilterRuleConfig [-Name <String>] -RouteFilter <PSRouteFilter>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzRouteFilterRuleConfig** ottiene una regola di filtro route o un elenco di regole di filtro delle route in un filtro di route.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $rf = Get-AzRouteFilter -Name "MyRouteFilter" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf -Name "Rule01"
PS C:\> Get-AzRouteFilterRuleConfig -RouteFilter $rf
```

Il primo comando ottiene il filtro della route denominato MyRouteFilter e lo archivia nella variabile $rf.
Il secondo comando consente di ottenere la regola di filtro route denominata Rule01 associata al filtro della route.
Il terzo comando ottiene un elenco delle regole di filtro delle route associate al filtro della route.
