# Test-AzAnalysisServicesServer

## Sinossi
Verifica l'esistenza di un'istanza di Analysis Services server

## SINTASSI

```
Test-AzAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Test-AzAnalysisServicesServer verifica l'esistenza di un'istanza di Analysis Services server

## ESEMPI

### Esempio 1
```
PS C:\> Test-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Questo comando tester√† se esiste un server denominato TestServer nel ResourceGroup TestGroup
