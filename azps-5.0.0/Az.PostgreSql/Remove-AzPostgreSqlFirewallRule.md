# Remove-AzPostgreSqlFirewallRule

## Sinossi
Elimina una regola del firewall del server.

## SINTASSI

### Elimina (impostazione predefinita)
```
Remove-AzPostgreSqlFirewallRule -Name <String> -ResourceGroupName <String> -ServerName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzPostgreSqlFirewallRule -InputObject <IPostgreSqlIdentity> [-DefaultProfile <PSObject>] [-AsJob]
 [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrizione
Elimina una regola del firewall del server.

## ESEMPI

### Esempio 1: rimuovere la regola del firewall PostgreSql per nome
```powershell
PS C:\> Remove-AzPostgreSqlFirewallRule -Name rule -ResourceGroupName PostgreSqlTestRG -ServerName PostgreSqlTestServer

```

Questo cmdlet rimuove la regola del firewall PostgreSql per nome.

### Esempio 2: rimuovere la regola del firewall PostgreSql per identità
```powershell
PS C:\> $ID = "/subscriptions/<SubscriptionId>/resourceGroups/PostgreSqlTestRG/providers/Microsoft.DBforPostgreSQL/servers/PostgreSqlTestServer/firewallRules/rule"
PS C:\> Remove-AzPostgreSqlFirewallRule -InputObject $ID
 
```

Questi cmdlet eliminano la regola del firewall PostgreSql per identità.
