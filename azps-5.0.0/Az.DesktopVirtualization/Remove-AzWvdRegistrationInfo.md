# Remove-AzWvdRegistrationInfo

## Sinossi
Rimuovere le informazioni di registrazione di Windows Virtual Desktop.

## SINTASSI

```
Remove-AzWvdRegistrationInfo -HostPoolName <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrizione
Rimuovere le informazioni di registrazione di Windows Virtual Desktop.

## ESEMPI

### Esempio 1: eliminare un token di registrazione desktop virtuale di Windows
```powershell
PS C:\> Remove-AzWvdRegistrationInfo -ResourceGroupName ResourceGroupName -HostPoolName HostPoolName
```

Questo comando Elimina un token di registrazione desktop virtuale di Windows in un pool host.
