# Remove-AzTimeSeriesInsightsEnvironment

## Sinossi
Elimina l'ambiente con il nome specificato nel gruppo di sottoscrizioni e risorse specificato.

## SINTASSI

### Elimina (impostazione predefinita)
```
Remove-AzTimeSeriesInsightsEnvironment -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzTimeSeriesInsightsEnvironment -InputObject <ITimeSeriesInsightsIdentity> [-DefaultProfile <PSObject>]
 [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrizione
Elimina l'ambiente con il nome specificato nel gruppo di sottoscrizioni e risorse specificato.

## ESEMPI

### Esempio 1: rimuovere un ambiente Insights serie temporali per nome
```powershell
PS C:\> Remove-AzTimeSeriesInsightsEnvironment -ResourceGroupName testgroup -Name tsill

```

Questo comando rimuove un ambiente Insights serie temporali.

### Esempio 2: rimuovere un ambiente Insights serie temporali per oggetto
```powershell
PS C:\> $env = Get-AzTimeSeriesInsightsEnvironment -ResourceGroupName testgroup -Name tsill
PS C:\> Remove-AzTimeSeriesInsightsEnvironment -InputObject $env

```

Questo comando rimuove un ambiente Insights serie temporali.
