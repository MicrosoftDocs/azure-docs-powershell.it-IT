# Get-AzDataFactoryGatewayAuthKey

## Sinossi
Ottiene la chiave di autenticazione del gateway per una factory di dati di Azure.

## SINTASSI

### ByFactoryName (impostazione predefinita)
```
Get-AzDataFactoryGatewayAuthKey [-DataFactoryName] <String> [-GatewayName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByFactoryObject
```
Get-AzDataFactoryGatewayAuthKey [-InputObject] <PSDataFactory> [-GatewayName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzDataFactoryGatewayAuthKey** ottiene la chiave di autenticazione del gateway per un gateway di Azure Data Factory specificato.
Il gateway viene registrato con un servizio cloud usando questo Key1 o Key2 di questa chiave auth.

## ESEMPI

### Esempio 1: ottiene la chiave AUTH di un gateway
```
PS C:\> Get-AzDataFactoryGatewayAuthKey -ResourceGroup ADFResource -GatewayName 'MyGateway' -DataFactoryName MyADF

       gGHqUg1THLqA=
Key2 : DMG@632e739e-1053-4070-9102-8591f067526e@41fcbc45-c594-4152-a8f1-fcbcd6452aea@wu@kFXxBdFCEBeL7LPB3hA3LqLd1uNFbyv
       YmWxtV4WD3JQ=
```

Questo comando ottiene la chiave di autenticazione del gateway per il gateway di data factory denominato gateway.
