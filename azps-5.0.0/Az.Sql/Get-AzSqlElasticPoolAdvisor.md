# Get-AzSqlElasticPoolAdvisor

## Sinossi
Ottiene uno o più consulenti per un pool elastico di Azure SQL.

## SINTASSI

```
Get-AzSqlElasticPoolAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -ElasticPoolName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlElasticPoolAdvisor** ottiene uno o più consulenti di Azure SQL Elastic pool Advisor per un pool di Azure SQL Elastic.
