# Remove-AzSqlServer

## Sinossi
Rimuove un server di database SQL di Azure.

## SINTASSI

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzSqlServer** rimuove un server di database SQL di Azure.
L'operazione di eliminazione è asincrona e può richiedere del tempo, quindi verificare che l'operazione sia terminata prima di eseguire altre operazioni che dipendono dal server che viene completamente eliminato.
Ad esempio, devi creare un nuovo server che usa lo stesso nome.

## ESEMPI

### Esempio 1: rimuovere un server
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

Questo comando rimuove il server di database SQL di Azure denominato Server01.
