# Remove-AzApiManagementCertificate

## Sinossi
Rimuove un certificato di gestione delle API.

## SINTASSI

```
Remove-AzApiManagementCertificate -Context <PsApiManagementContext> -CertificateId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApiManagementCertificate** rimuove un certificato di gestione delle API di Azure.

## ESEMPI

### Esempio 1: rimuovere un certificato
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementCertificate -Context $ApiMgmtContext -CertificateId "0123456789" -Force
```

Questo comando rimuove il certificato di gestione API specificato.
Dato che il parametro *Force* è specificato, non è richiesta alcuna conferma.
