# Remove-AzBatchJobSchedule

## Sinossi
Rimuove la pianificazione di un processo batch.

## SINTASSI

```
Remove-AzBatchJobSchedule [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzBatchJobSchedule** rimuove una pianificazione dei processi batch di Azure.

## ESEMPI

### Esempio 1: eliminare una pianificazione processo batch
```
PS C:\>Remove-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context
```

Questo comando Elimina la pianificazione del processo con ID MyJobSchedule.
Il comando richiede conferma prima di eliminare il processo.
Usa il cmdlet Get-AzBatchAccountKey per assegnare un contesto alla variabile $Context.

### Esempio 2: eliminare una procedura batch senza conferma tramite la pipeline
```
PS C:\>Get-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context | Remove-AzBatchJobSchedule -Force -BatchContext $Context
```

Questo comando consente di ottenere la pianificazione del processo che contiene l'ID MyJobSchedule usando il cmdlet Get-AzBatchJobSchedule.
Il comando passa la pianificazione del processo al cmdlet corrente usando l'operatore pipeline.
Il comando Elimina la pianificazione del processo.
Poiché il comando include il parametro *Force* , non chiede conferma.

## PARAMETRI

### -BatchContext
Specifica l'istanza di **BatchAccountContext** utilizzata dal cmdlet per interagire con il servizio batch.
Se si usa il cmdlet Get-AzBatchAccount per ottenere il BatchAccountContext, l'autenticazione di Azure Active Directory verrà usata durante l'interazione con il servizio batch. Per usare invece l'autenticazione con chiave condivisa, usa il cmdlet Get-AzBatchAccountKey per ottenere un oggetto BatchAccountContext con i relativi tasti di scelta popolati. Quando si usa l'autenticazione con chiave condivisa, il tasto di scelta principale viene usato per impostazione predefinita. Per cambiare la chiave da usare, imposta la proprietà BatchAccountContext.
