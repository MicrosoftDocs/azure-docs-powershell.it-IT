# Remove-AzAutomationSchedule

## Sinossi
Elimina una programmazione di automazione.

## SINTASSI

```
Remove-AzAutomationSchedule [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzAutomationSchedule** Elimina una programmazione da Azure Automation.

## ESEMPI

### Esempio 1: rimuovere una pianificazione
```
PS C:\>Remove-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -ResourceGroupName "ResourceGroup01"
```

Questo comando Elimina la pianificazione denominata Schedule01 in account di automazione Contoso17 nel gruppo risorse ResourceGroup01.
