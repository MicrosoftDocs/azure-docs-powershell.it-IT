# Remove-AzAutomationWebhook

## Sinossi
Rimuove un webhook da un Runbook di automazione.

## SINTASSI

```
Remove-AzAutomationWebhook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzAutomationWebhook** rimuove un webhook da un Runbook di automazione di Azure.
Il webhook viene eliminato.

## ESEMPI

### Esempio 1: rimuovere un hook
```
PS C:\>Remove-AzAutomationWebhook -Name "Webhook11" -ResourceGroup "ResourceGroup01" -AutomationAccountName "AutomationAccount01" -Force
```

Questo comando rimuove un webhook denominato Webhook11 nell'account di automazione denominato AutomationAccount01.
Il comando specifica il parametro *Force* .
Pertanto, non chiede conferma.
