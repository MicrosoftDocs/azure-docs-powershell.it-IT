# Add-AzsVMExtension

## Sinossi
Creare una nuova immagine dell'estensione della macchina virtuale.

## SINTASSI

```
Add-AzsVMExtension [-Publisher] <String> [-Type] <String> [-Version] <String> [-SourceBlob] <Object>
 [-VmOsType] <Object> [-ComputeRole] <String> [-VmScaleSetEnabled] [-SupportMultipleExtensions]
 [-IsSystemExtension] [[-Location] <String>] [-AsJob] [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Creare un'immagine dell'estensione della macchina virtuale.

## ESEMPI

### --------------------------ESEMPIO 1--------------------------
```
Add-AzsVMExtension -Publisher "Microsoft" -Type "MicroExtension" -Version "0.1.0" -ComputeRole "IaaS" -SourceBlob "https://github.com/Microsoft/PowerShell-DSC-for-Linux/archive/v1.1.1-294.zip" -SupportMultipleExtensions -VmOsType "Linux"
```

Aggiungere una nuova estensione di immagine della piattaforma.
