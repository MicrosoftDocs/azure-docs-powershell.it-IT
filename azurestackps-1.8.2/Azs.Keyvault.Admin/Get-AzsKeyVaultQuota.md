# Get-AzsKeyVaultQuota

## Sinossi
Ottenere un elenco di tutti gli oggetti quota per il Vault in una posizione.

## SINTASSI

```
Get-AzsKeyVaultQuota [[-Location] <String>] [<CommonParameters>]
```

## Descrizione
Ottenere un elenco di tutti gli oggetti quota per il Vault in una posizione.

## ESEMPI

### ESEMPIO 1
