# Modulo AZ. DataLakeStore
## Descrizione
Gli argomenti di questa sezione documentano i cmdlet di PowerShell di Azure per Azure Data Lake Store in Azure Resource Manager (ARM) Framework. I cmdlet sono presenti nello spazio dei nomi Microsoft. Azure. Commands. DataLakeStore. Questi cmdlet funzionano solo con gli account di archiviazione di Gen1 di Azure Data Lake.

## Cmdlet AZ. DataLakeStore
### [Add-AzDataLakeStoreFirewallRule](Add-AzDataLakeStoreFirewallRule.md)
Aggiunge una regola del firewall all'account di data Lake Store specificato.

### [Add-AzDataLakeStoreItemContent](Add-AzDataLakeStoreItemContent.md)
Aggiunge contenuto a un elemento in uno Store di data Lake.

### [Add-AzDataLakeStoreTrustedIdProvider](Add-AzDataLakeStoreTrustedIdProvider.md)
Aggiunge un provider di identità attendibile all'account di data Lake Store specificato.

### [Add-AzDataLakeStoreVirtualNetworkRule](Add-AzDataLakeStoreVirtualNetworkRule.md)
Aggiunge una regola di rete virtuale all'account di data Lake Store specificato.

### [Enable-AzDataLakeStoreKeyVault](Enable-AzDataLakeStoreKeyVault.md)
Consente di consentire a un Vault Key gestito dall'utente di crittografare l'account di data Lake Store specificato.

### [Export-AzDataLakeStoreChildItemProperty](Export-AzDataLakeStoreChildItemProperty.md)
Esporta le proprietà (utilizzo del disco e ACL) per l'intero albero dal percorso specificato a un percorso di output

### [Export-AzDataLakeStoreItem](Export-AzDataLakeStoreItem.md)
Scarica un file da data Lake Store.

### [Get-AzDataLakeStoreAccount](Get-AzDataLakeStoreAccount.md)
Ottiene i dettagli di un account di data Lake Store.

### [Get-AzDataLakeStoreChildItem](Get-AzDataLakeStoreChildItem.md)
Ottiene l'elenco di elementi in una cartella in data Lake Store.

### [Get-AzDataLakeStoreChildItemSummary](Get-AzDataLakeStoreChildItemSummary.md)
Ottiene il riepilogo delle dimensioni totali, dei file e delle directory contenuti nel percorso specificato

### [Get-AzDataLakeStoreFirewallRule](Get-AzDataLakeStoreFirewallRule.md)
Ottiene le regole del firewall specificate nello Store Data Lake specificato.
Se non è specificata alcuna regola del firewall, elenca tutte le regole del firewall per l'account.

### [Get-AzDataLakeStoreItem](Get-AzDataLakeStoreItem.md)

### [Get-AzDataLakeStoreItemAclEntry](Get-AzDataLakeStoreItemAclEntry.md)

### [Get-AzDataLakeStoreItemContent](Get-AzDataLakeStoreItemContent.md)
Ottiene il contenuto di un file in data Lake Store.

### [Get-AzDataLakeStoreItemOwner](Get-AzDataLakeStoreItemOwner.md)
Ottiene il proprietario di un file o di una cartella in data Lake Store.

### [Get-AzDataLakeStoreItemPermission](Get-AzDataLakeStoreItemPermission.md)
Ottiene l'ottale delle autorizzazioni di un file o di una cartella in data Lake Store.

### [Get-AzDataLakeStoreTrustedIdProvider](Get-AzDataLakeStoreTrustedIdProvider.md)
Ottiene il provider di identità attendibile specificato nell'archivio dati specificato del lago.
Se non viene specificato alcun provider, elenca tutti i provider per l'account.

### [Get-AzDataLakeStoreVirtualNetworkRule](Get-AzDataLakeStoreVirtualNetworkRule.md)
Ottiene le regole di rete virtuale specificate nello Store Data Lake.
Se non è specificata alcuna regola di rete virtuale, elenca tutte le regole di rete virtuale per l'account.
