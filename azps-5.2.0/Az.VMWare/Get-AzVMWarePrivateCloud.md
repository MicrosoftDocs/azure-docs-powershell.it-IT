# Get-AzVMWarePrivateCloud

## Sinossi
Ottenere un cloud privato

## SINTASSI

### List1 (impostazione predefinita)
```
Get-AzVMWarePrivateCloud [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Ottieni
```
Get-AzVMWarePrivateCloud -Name <String> -ResourceGroupName <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### GetViaIdentity
```
Get-AzVMWarePrivateCloud -InputObject <IVMWareIdentity> [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### Elenco
```
Get-AzVMWarePrivateCloud -ResourceGroupName <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [<CommonParameters>]
```

## Descrizione
Ottenere un cloud privato

## ESEMPI

### Esempio 1: elencare il cloud privato in abbonamento
```powershell
PS C:\> Get-AzVMWarePrivateCloud

Location      Name            Type
--------      ----            ----
australiaeast azps-test-cloud Microsoft.AVS/privateClouds
```

Elenco cloud privato in abbonamento

### Esempio 2: elenco cloud privato in gruppo risorse
```powershell
PS C:\> Get-AzVMWarePrivateCloud -ResourceGroupName azps-test-group

Location      Name            Type
--------      ----            ----
australiaeast azps-test-cloud Microsoft.AVS/privateClouds
```

Elenco cloud privato in gruppo risorse

### Esempio 3: ottenere cloud privato
```powershell
PS C:\> Get-AzVMWarePrivateCloud -ResourceGroupName azps-test-group -Name azps-test-cloud

Location      Name            Type
--------      ----            ----
australiaeast azps-test-cloud Microsoft.AVS/privateClouds
```

Ottenere cloud privato

PROPRIETÀ COMPLESSE DI PARAMETRI

Per creare i parametri descritti di seguito, Costruisci una tabella hash contenente le proprietà appropriate. Per informazioni sulle tabelle hash, eseguire Get-Help about_Hash_Tables.


INPUTOBJECT <IVMWareIdentity> : parametro Identity
  - `[AuthorizationName <String>]`: Nome dell'autorizzazione del circuito ExpressRoute nel cloud privato
  - `[ClusterName <String>]`: Nome del cluster nel cloud privato
  - `[HcxEnterpriseSiteName <String>]`: Nome del sito aziendale di HCX nel cloud privato
  - `[Id <String>]`: Percorso identità risorse

  - `[SubscriptionId <String>]`: ID dell'abbonamento di destinazione.
