# Get-AzSmartGroup

## Sinossi
Ottiene le informazioni sui gruppi smart

## SINTASSI

### SmartGroupsListByFilter (impostazione predefinita)
```
Get-AzSmartGroup [-SortBy <String>] [-SortOrder <String>] [-TimeRange <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SmartGroupById
```
Get-AzSmartGroup -SmartGroupId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSmartGroup** ottiene le informazioni sui gruppi smart.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSmartGroup -TimeRange "1h"
```

Elenca tutti i gruppi di Smart formati nell'ultima ora. Usare Format-List per ottenere i dettagli completi di ogni gruppo di Smart in elenco.

### Esempio 2
```powershell
PS C:\> Get-AzSmartGroup -SmartGroupId "afbf1b3a-0a6c-4f19-9c9b-644ccd7b1529" | Format-List
```

Ottenere i dettagli del gruppo smart per ID (GUID) o ID risorsa (ID ARM completo)
