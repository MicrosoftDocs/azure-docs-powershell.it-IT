# Modulo AZ. Subscription
## Descrizione
Gli argomenti di questa sezione documentano i cmdlet di PowerShell di Azure per l'abbonamento a Azure in Azure Resource Manager (ARM) Framework. I cmdlet sono presenti nello spazio dei nomi Microsoft. Azure. Commands. Subscription.

## Cmdlet AZ. Subscription
### [Get-AzSubscriptionAlias](Get-AzSubscriptionAlias.md)
Ottiene i dettagli degli alias di sottoscrizione

### [New-AzSubscriptionAlias](New-AzSubscriptionAlias.md)
Crea un nuovo alias e un abbonamento

### [Remove-AzSubscriptionAlias](Remove-AzSubscriptionAlias.md)
Elimina l'alias della sottoscrizione

### [Update-AzSubscription](Update-AzSubscription.md)
Aggiorna un abbonamento a Azure
