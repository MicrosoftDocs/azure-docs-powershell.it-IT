# Get-AzSubscriptionAlias

## Sinossi
Ottiene i dettagli degli alias di sottoscrizione

## SINTASSI

```
Get-AzSubscriptionAlias [-AliasName <String>] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSubscriptionAlias** ottiene i dettagli degli alias di sottoscrizione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSubscriptionAlias -AliasName "ExistingAliasName"
```

Ottiene i dettagli degli alias di sottoscrizione
