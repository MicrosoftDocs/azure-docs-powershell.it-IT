# Get-AzStorageSyncServerEndpoint

## Sinossi
Questo comando elenca tutti gli endpoint del server all'interno di un gruppo di sincronizzazione specifico.

## SINTASSI

### StringParameterSet (impostazione predefinita)
```
Get-AzStorageSyncServerEndpoint [-ResourceGroupName] <String> [-StorageSyncServiceName] <String>
 [-SyncGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ObjectParameterSet
```
Get-AzStorageSyncServerEndpoint [-ParentObject] <PSSyncGroup> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ParentStringParameterSet
```
Get-AzStorageSyncServerEndpoint [-ParentResourceId] <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Questo comando elenca tutti gli endpoint del server all'interno di un gruppo di sincronizzazione specifico. PuÃ² essere usato anche per elencare gli attributi di ogni endpoint server.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzStorageSyncServerEndpoint -ResourceGroupName "myResourceGroup" -StorageSyncServiceName "myStorageSyncServiceName" -SyncGroupName "mySyncGroupName"
```

Questo comando consente di ottenere tutti gli endpoint del server contenuti nel gruppo di sincronizzazione specificato. Specifica-ServerEndpointName per restituirne uno specifico.
