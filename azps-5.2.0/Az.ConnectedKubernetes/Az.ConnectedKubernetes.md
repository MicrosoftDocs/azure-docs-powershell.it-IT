# Modulo AZ. ConnectedKubernetes
## Descrizione
PowerShell di Microsoft Azure: cmdlet ConnectedKubernetes

## Cmdlet AZ. ConnectedKubernetes
### [Get-AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
Restituisce le proprietà del cluster connesso specificato, inclusi nome, identità, proprietà e dettagli del cluster aggiuntivi.

### [New-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
API per registrare un nuovo cluster di K8s e quindi creare una risorsa rilevata in ARM

### [Remove-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
Eliminare un cluster connesso, rimuovendo la risorsa rilevata in Azure Resource Manager (ARM).

### [Update-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
API per aggiornare determinate proprietà della risorsa cluster connessa
