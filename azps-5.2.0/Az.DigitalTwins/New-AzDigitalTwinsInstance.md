# New-AzDigitalTwinsInstance

## Sinossi
Creare o aggiornare i metadati di un DigitalTwinsInstance.
Il modello usuale per modificare una proprietà consiste nel recuperare i metadati di DigitalTwinsInstance e di sicurezza e quindi combinarli con i valori modificati in un nuovo corpo per aggiornare DigitalTwinsInstance.

## SINTASSI

### CreateExpanded (impostazione predefinita)
```
New-AzDigitalTwinsInstance -ResourceGroupName <String> -ResourceName <String> -Location <String>
 [-SubscriptionId <String>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm]
 [-WhatIf] [<CommonParameters>]
```

### Creare
```
New-AzDigitalTwinsInstance -ResourceGroupName <String> -ResourceName <String>
 -DigitalTwinsCreate <IDigitalTwinsDescription> [-SubscriptionId <String>] [-DefaultProfile <PSObject>]
 [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrizione
Creare o aggiornare i metadati di un DigitalTwinsInstance.
Il modello usuale per modificare una proprietà consiste nel recuperare i metadati di DigitalTwinsInstance e di sicurezza e quindi combinarli con i valori modificati in un nuovo corpo per aggiornare DigitalTwinsInstance.

## ESEMPI

### Esempio 1: creare un AzDigitalTwinsInstance per impostazione predefinita.
```powershell
PS C:\> New-AzDigitalTwinsInstance -ResourceGroupName youritest -ResourceName youriDigitalTwin -Location eastus

-------- ----             ------- ----

```

Creare un AzDigitalTwinsInstance per impostazione predefinita

### Esempio 2: creare un oggetto AzDigitalTwinsInstance per AzDigitalTwins.
```powershell
PS C:\> $GetAzDigTwin = Get-AzDigitalTwinsInstance -ResourceGroupName youritemp -ResourceName youriDigitalTwinsTest
```
