# Invoke-AzVMReimage

## Sinossi
Riimmagine di una macchina virtuale di Azure.

## SINTASSI

```
Invoke-AzVMReimage [-ResourceGroupName] <String> [-VMName] <String> [-TempDisk] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Invoke-AzVMReimage** riimmagini una macchina virtuale di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Invoke-AzVMReimage -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

Questo comando riimmaginer√† la macchina virtuale denominata VirtualMachine07 in ResourceGroup11.
