# New-AzDiskAccess

## Sinossi
Crea una risorsa di Access su disco

## SINTASSI

```
New-AzDiskAccess [-ResourceGroupName] <String> [-Name] <String> [-Location] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzDiskAccess** crea una risorsa di Access su disco

## ESEMPI

### Esempio 1
```
PS C:\> New-AzDiskAccess -ResourceGroupName "ResourceGroup01" -Name "DiskAccess01" -Location "NorthCentralUS"
```

Questo comando creerà un accesso su disco con proprietà specificate.
