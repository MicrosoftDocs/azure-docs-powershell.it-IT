# Get-AzSynapseDataFlow

## Sinossi
Ottiene informazioni sui flussi di dati nell'area di lavoro.

## SINTASSI

### GetByName (impostazione predefinita)
```
Get-AzSynapseDataFlow -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseDataFlow -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSynapseDataFlow** ottiene le informazioni sui flussi di dati nell'area di lavoro.
Se specifichi il nome di un flusso di dati, questo cmdlet ottiene le informazioni sul flusso di dati.
Se non specifichi un nome, questo cmdlet recupera le informazioni su tutti i flussi di dati nell'area di lavoro.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace
```

Questo comando consente di ottenere informazioni su tutti i flussi di dati nell'area di lavoro denominata ContosoWorkspace.

### Esempio 2
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace -Name ContosoDataFlow
```

Questo comando consente di ottenere informazioni sul flusso di dati denominato ContosoDataFlow nell'area di lavoro denominata ContosoWorkspace.
