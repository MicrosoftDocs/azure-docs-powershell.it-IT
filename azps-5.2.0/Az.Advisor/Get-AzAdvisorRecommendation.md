# Get-AzAdvisorRecommendation

## Sinossi
Ottiene un elenco di suggerimenti di Azure Advisor.

## SINTASSI

### NameParameterSet (impostazione predefinita)
```
Get-AzAdvisorRecommendation [-Category <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdParameterSet
```
Get-AzAdvisorRecommendation [-ResourceId] <String> [-Category <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Ottiene l'elenco di suggerimenti di Azure Advisor. Può essere filtrato per categoria, ID risorsa, nome e così via.
