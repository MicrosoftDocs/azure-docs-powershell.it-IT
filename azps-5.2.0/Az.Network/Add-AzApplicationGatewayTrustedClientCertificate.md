# Add-AzApplicationGatewayTrustedClientCertificate

## Sinossi
Aggiunge una catena di certificati CA client attendibile a un gateway dell'applicazione.

## SINTASSI

```
Add-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway <PSApplicationGateway> -Name <String>
 -CertificateFile <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzApplicationGatewayTrustedClientCertificate** aggiunge una catena di certificati CA client attendibile a un gateway dell'applicazione Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClient = New-AzApplicationGatewayTrustedClientCertificate -Name "ClientCert" -CertificateFile "C:\clientCAChain.cer"
PS C:\> $AppGw = Add-AzApplicationGatewaySslProfile -Name $sslProfileName -ApplicationGateway $gw -TrustedClientCertificates $trustedClient
PS C:\> $gw = Set-AzApplicationGateway -ApplicationGateway $gw
```

Il primo comando ottiene il gateway dell'applicazione e lo archivia in $gw variabile. Il secondo comando crea una nuova catena di certificati CA del client attendibile che prende il percorso del certificato della CA client come input. Il terzo comando crea un profilo SSL usando un certificato client attendibile. Il quarto comando Aggiorna il gateway dell'applicazione.
