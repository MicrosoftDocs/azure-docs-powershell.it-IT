# New-AzFirewallHubIpAddress

## Sinossi
Indirizzi IP assoicated al firewall nell'hub virtuale

## SINTASSI

```
New-AzFirewallHubIpAddress [-PrivateIPAddress <String>] [-PublicIPs <PSAzureFirewallHubPublicIpAddresses>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Indirizzi IP assoicated al firewall nell'hub virtuale. Possono essere indirizzi pubblici e privati

## ESEMPI

### Esempio 1
```powershell
PS C:\> $fwpips = New-AzFirewallHubPublicIpAddress -Count 2
PS C:\> New-AzFirewallHubIpAddress -PublicIPs $fwpips
```

Questo esempio crea un oggetto indirizzo IP Hub con un conteggio di 2 IP pubblici. L'oggetto HubIPAddress viene ssociated al firewall nell'hub virtuale.
