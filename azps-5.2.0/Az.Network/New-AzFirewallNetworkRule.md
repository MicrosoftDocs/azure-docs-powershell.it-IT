# New-AzFirewallNetworkRule

## Sinossi
Crea una regola di rete firewall.

## SINTASSI

```
New-AzFirewallNetworkRule -Name <String> [-Description <String>] -SourceAddress <String[]>
 [-SourceIpGroup <String[]>] [-DestinationAddress <String[]>] [-DestinationIpGroup <String[]>]
 [-DestinationFqdn <String[]>] -DestinationPort <String[]> -Protocol <String[]>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzFirewallNetworkRule** crea una regola di rete per Azure firewall.

## ESEMPI

### Esempio 1: creare una regola per tutto il traffico TCP
```powershell
$rule = New-AzFirewallNetworkRule -Name "all-tcp-traffic" -Description "Rule for all TCP traffic" -Protocol TCP -SourceAddress "*" -DestinationAddress "*" -DestinationPort "*"
```

Questo esempio crea una regola per tutto il traffico TCP. L'utente impone se il traffico sarà consentito o negato per una regola in base alla raccolta di regole a cui è associato.

### Esempio 2: creare una regola per tutto il traffico TCP da 10.0.0.0 a 60.1.5.0:4040
```powershell
$rule = New-AzFirewallNetworkRule -Name "partial-tcp-rule" -Description "Rule for all TCP traffic from 10.0.0.0 to 60.1.5.0:4040" -Protocol TCP -SourceAddress "10.0.0.0" -DestinationAddress "60.1.5.0" -DestinationPort "4040"
```

Questo esempio crea una regola per tutto il traffico TCP da 10.0.0.0 a 60.1.5.0:4040. L'utente impone se il traffico sarà consentito o negato per una regola in base alla raccolta di regole a cui è associato.

### Esempio 3: creare una regola per tutto il traffico TCP e ICMP da qualsiasi origine a 10.0.0.0/16
```powershell
$rule = New-AzFirewallNetworkRule -Name "tcp-and-icmp-rule" -Description "Rule for all TCP and ICMP traffic from any source to 10.0.0.0/16" -Protocol TCP,ICMP -SourceAddress * -DestinationAddress "10.0.0.0/16" -DestinationPort *
```

Questo esempio crea una regola per tutto il traffico TCP da qualsiasi origine a 10.0.0.0/16. L'utente impone se il traffico sarà consentito o negato per una regola in base alla raccolta di regole a cui è associato.
