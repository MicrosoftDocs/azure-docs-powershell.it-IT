# Get-AzApplicationGatewayTrustedClientCertificate

## Sinossi
Ottiene la catena di certificati CA del client attendibile con un nome specifico dal gateway dell'applicazione.

## SINTASSI

```
Get-AzApplicationGatewayTrustedClientCertificate [-Name <String>] -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzApplicationGatewayTrustedClientCertificate ottiene la catena di certificati CA del client attendibile con un nome specifico dal gateway dell'applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $trustedClientCert = Get-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name $certName
```

Il primo comando ottiene il gateway dell'applicazione e lo archivia in $gw variabile. Il secondo comando consente di ottenere la catena di certificati CA del client attendibile con un nome specificato dal gateway dell'applicazione.
