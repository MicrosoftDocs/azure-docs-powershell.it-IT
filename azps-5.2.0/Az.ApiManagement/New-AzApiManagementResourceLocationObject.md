# New-AzApiManagementResourceLocationObject

## Sinossi
Creare un nuovo contratto per la posizione delle risorse (usato nei gateway).

## SINTASSI

```
New-AzApiManagementResourceLocationObject -Name <String> [-City <String>] [-District <String>]
 [-CountryOrRegion <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzApiManagementResourceLocationObject** crea un nuovo contratto per la posizione delle risorse (usato nei gateway).

## ESEMPI

### Esempio 1: creare un contratto per la posizione delle risorse
```powershell
PS C:\>$location = New-AzApiManagementResourceLocationObject -Name "n1" -City "c1" -District "d1" -CountryOrRegion "r1"
```

Questo comando crea un percorso di risorsa.
