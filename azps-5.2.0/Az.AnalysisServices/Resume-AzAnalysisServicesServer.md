# Resume-AzAnalysisServicesServer

## Sinossi
Riprende un'istanza di Analysis Services server

## SINTASSI

```
Resume-AzAnalysisServicesServer [[-ResourceGroupName] <String>] [-Name] <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Resume-AzAnalysisServicesServer riprende un'istanza di Analysis Services server

## ESEMPI

### Esempio 1
```
PS C:\> Resume-AzAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Questo comando riprenderà un server in pausa denominato TestServer nel ResourceGroup TestGroup
