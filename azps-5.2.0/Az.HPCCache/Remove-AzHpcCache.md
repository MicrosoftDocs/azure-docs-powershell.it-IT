# Remove-AzHpcCache

## Sinossi
Rimuove una cache HPC.

## SINTASSI

```
Remove-AzHpcCache -ResourceGroupName <String> -Name <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzHpcCache** rimuove una cache HPC di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzHpcCache -ResourceGroupName testRG -CacheName testCache
```
