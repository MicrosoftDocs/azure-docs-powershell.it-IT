# Get-AzKustoClusterLanguageExtension

## Sinossi
Restituisce un elenco di estensioni di lingua che possono essere eseguite in query KQL.

## SINTASSI

```
Get-AzKustoClusterLanguageExtension -ClusterName <String> -ResourceGroupName <String>
 [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrizione
Restituisce un elenco di estensioni di lingua che possono essere eseguite in query KQL.

## ESEMPI

### Esempio 1: elencare tutte le estensioni di lingua impostate per un cluster
```powershell
PS C:\> Get-AzKustoClusterLanguageExtension -ResourceGroupName testrg -ClusterName testnewkustocluster

Name
----
R
PYTHON
```

Il comando precedente restituisce un elenco di estensioni di lingua che possono essere eseguite in query KQL.
