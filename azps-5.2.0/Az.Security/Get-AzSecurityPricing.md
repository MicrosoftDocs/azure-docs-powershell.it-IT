# Get-AzSecurityPricing

## Sinossi

Ottiene i piani di Azure Defender per un abbonamento in Azure Security Center.

## SINTASSI

### SubscriptionScope (impostazione predefinita)

```powershell
Get-AzSecurityPricing [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SubscriptionLevelResource

```powershell
Get-AzSecurityPricing -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId

```powershell
Get-AzSecurityPricing -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione

Puoi visualizzare ogni piano di Azure Defender, per abbonamento, usando questo cmdlet.

Per informazioni dettagliate su Azure Defender e sui piani disponibili, vedere [Introduzione a Azure Defender](https://docs.microsoft.com/azure/security-center/azure-defender).

### Esempio 2

```powershell
PS C:\> Get-AzSecurityPricing -ResourceId
```

Ottiene i dettagli sui prezzi dell'ID risorsa specifico. Dove ResourceId è uno degli ID restituiti da `Get-AzSecurityPricing` .

### Esempio 3

```powershell
PS C:\> Get-AzSecurityPricing -Name
```

Ottiene i dettagli sui prezzi del piano denominato Azure Defender. Dove `name` è uno dei nomi restituiti da `Get-AzSecurityPricing` .
