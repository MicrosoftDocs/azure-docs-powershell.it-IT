# Get-AzIotSecurityAnalyticsAggregatedAlert

## Sinossi
Ottenere avviso aggregato di sicurezza

## SINTASSI

### SolutionScope (impostazione predefinita)
```
Get-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName <String> -SolutionName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### SolutionLevelResource
```
Get-AzIotSecurityAnalyticsAggregatedAlert -ResourceGroupName <String> -SolutionName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzIotSecurityAnalyticsAggregatedAlert restituisce uno o più avvisi aggregati nei dispositivi di hub.
Il nome degli avvisi aggregati è una combinazione del tipo di avviso e della data di aggragted di avviso, separati da "/".
