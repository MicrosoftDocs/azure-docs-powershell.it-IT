# Get-AzConsumptionPriceSheet

## Sinossi
Ottenere i fogli di prezzo dell'abbonamento.

## SINTASSI

```
Get-AzConsumptionPriceSheet [-BillingPeriodName <String>] [-ExpandMeterDetail] [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzConsumptionPriceSheet** ottiene i fogli di prezzo dell'abbonamento.
