# Remove-AzStorageShareStoredAccessPolicy

## Sinossi
Rimuove un criterio di accesso archiviato da una condivisione di archiviazione.

## SINTASSI

```
Remove-AzStorageShareStoredAccessPolicy [-ShareName] <String> [-Policy] <String> [-PassThru]
 [-Context <IStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-ConcurrentTaskCount <Int32>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzStorageShareStoredAccessPolicy** rimuove un criterio di accesso archiviato da una condivisione di archiviazione di Azure.

## ESEMPI

### Esempio 1: rimuovere un criterio di accesso archiviato da una condivisione di archiviazione di Azure
```
PS C:\>Remove-AzStorageShareStoredAccessPolicy -ShareName "ContosoShare" -Policy "GeneralPolicy"
```

Questo comando rimuove un criterio di accesso archiviato denominato GeneralPolicy da ContosoShare.
