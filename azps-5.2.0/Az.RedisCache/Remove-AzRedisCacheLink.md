# Remove-AzRedisCacheLink

## Sinossi
Rimuovere un collegamento di replica geo tra due cache Redis.

## SINTASSI

```
Remove-AzRedisCacheLink -PrimaryServerName <String> -SecondaryServerName <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Rimuovere un collegamento di replica geo tra due cache Redis.

## ESEMPI

### Esempio 1: rimuovere un collegamento di replica geo
```
PS C:\>Remove-AzRedisCacheLink -PrimaryServerName "mycache1" -SecondaryServerName "mycache2"
```

Questo comando rimuove i collegamenti di Geo-replica in cui la cache di redis denominata myCache1 è primaria e la cache di redis denominata mycache2 è secondaria.
