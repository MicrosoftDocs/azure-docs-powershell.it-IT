# Set-AzEventHubNamespace

## Sinossi
Aggiorna lo spazio dei nomi degli hub eventi specificati.

## SINTASSI

### NamespaceParameterSet (impostazione predefinita)

 [[-SkuName] <String>] [[-SkuCapacity] <Int32>] [[-State] <NamespaceState>] [[-Tag] <Hashtable>] [-EnableKafka]
 [-Identity] [-IdentityUserDefined <String>] [-KeySource <String>]

## Descrizione
Il cmdlet Set-AzEventHubNamespace aggiorna le proprietà dello spazio dei nomi degli hub eventi specificati.
