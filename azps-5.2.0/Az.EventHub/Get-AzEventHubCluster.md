# Get-AzEventHubCluster

## Sinossi
Ottiene i dettagli di un singolo cluster di hub eventi o restituisce un elenco di cluster di hub eventi.

## SINTASSI

```
Get-AzEventHubCluster [-ResourceGroupName] <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzEventHubCluster restituisce i dettagli di un cluster di hub eventi o un elenco di tutti i cluster di hub eventi in un gruppo di risorse specifico.
Se viene fornito il nome del cluster, vengono restituiti i dettagli di un singolo cluster.
Se non viene specificato un nome di cluster, viene restituito un elenco di tutti i cluster nel gruppo di risorse specificato.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzEventHubCluster -ResourceGroupName RSG-Cluster27651 -Name Eventhub-Cluster-5557

            ntHub/clusters/Eventhub-Cluster-5557
Name      : Eventhub-Cluster-5557
Location  : southcentralus
CreatedAt : 09/10/2020 22:09:57
UpdatedAt : 09/10/2020 23:02:48
MetricId  :
Status    :
Sku       : Microsoft.Azure.Commands.EventHub.Models.PSEventHubsClusterSkuAttributes
Tags      : {[ClusterTag2, Tag4], [ClusterTag1, Tag3]}

```

Restituisce il detials del cluster "Eventhub-cluster-5557" dal gruppo di risorse "RSG-Cluster27651"
