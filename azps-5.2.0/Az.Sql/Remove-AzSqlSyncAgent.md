# Remove-AzSqlSyncAgent

## Sinossi
Rimuove un agente di sincronizzazione SQL di Azure.

## SINTASSI

```
Remove-AzSqlSyncAgent [-Name] <String> [-Force] [-PassThru] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzSqlSyncAgent** rimuove un agente di sincronizzazione SQL di Azure.

## ESEMPI

### Esempio 1: rimuovere un agente di sincronizzazione
```
PS C:\>Remove-AzSqlSyncAgent -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -Name "syncAgent01"
```

Questo comando rimuove l'agente di sincronizzazione di Azure SQL denominato syncAgent01.
