# Modulo AZ. KubernetesConfiguration
## Descrizione
PowerShell di Microsoft Azure: cmdlet KubernetesConfiguration

## Cmdlet AZ. KubernetesConfiguration
### [Get-AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
Ottiene i dettagli della configurazione del controllo del codice sorgente.

### [New-AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
Creare una nuova configurazione del controllo del codice sorgente Kubernetes.

### [Remove-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
Questo eliminerà il file YAML usato per configurare la configurazione del controllo del codice sorgente, impedendo così la sincronizzazione futura dal repo di origine.
