# Remove-AzDataLakeAnalyticsCatalogCredential

## Sinossi
Elimina una credenziale di analisi di Azure Data Lake.

## SINTASSI

```
Remove-AzDataLakeAnalyticsCatalogCredential [-Account] <String> [-DatabaseName] <String> [-Name] <String>
 [[-Password] <PSCredential>] [-Recurse] [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Remove-AzDataLakeAnalyticsCatalogCredential Elimina le credenziali di un catalogo di analisi del Lago di dati di Azure.

## ESEMPI

### Esempio 1: rimuovere una credenziale
```
PS C:\> Remove-AzDataLakeAnalyticsCatalogCredential -AccountName "ContosoAdla" `
                      -DatabaseName "DatabaseName" `
                      -Name "CredName"
```

Questo comando consente di rimuovere le credenziali del catalogo dei dati del lago analitico specificato.
