# Get-AzMarketplacePrivateStoreOffer

## Sinossi
Ottenere una o più offerte di uno store privato.

## SINTASSI

```
Get-AzMarketplacePrivateStoreOffer -PrivateStoreId <String> [-OfferId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Ottenere una o più offerte di un archivio privato con i piani public + private aggiunti in ambito tenant. Se l'ID abbonamento è presente, ottenere una o più offerte private Store con piani privati solo in ambito abbonamento
