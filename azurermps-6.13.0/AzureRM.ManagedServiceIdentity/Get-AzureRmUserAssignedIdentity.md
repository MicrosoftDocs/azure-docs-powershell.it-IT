# Get-AzureRmUserAssignedIdentity

## Sinossi
Ottiene l'identità/identità assegnate dall'utente.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### SuscriptionParameterSet (impostazione predefinita)
```
Get-AzureRmUserAssignedIdentity [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceGroupParameterSet
```
Get-AzureRmUserAssignedIdentity -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
**Get-AzureRmUserAssignedIdentity** ottiene le identità assegnate agli utenti esistenti.
