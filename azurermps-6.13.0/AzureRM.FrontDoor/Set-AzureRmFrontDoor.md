# Set-AzureRmFrontDoor

## Sinossi
Aggiornare un bilanciamento del carico della porta anteriore

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### ByFieldsParameterSet (impostazione predefinita)
```
Set-AzureRmFrontDoor -ResourceGroupName <String> -Name <String> [-RoutingRule <PSRoutingRule[]>]
 [-BackendPool <PSBackendPool[]>] [-FrontendEndpoint <PSFrontendEndpoint[]>]
 [-LoadBalancingSetting <PSLoadBalancingSetting[]>] [-HealthProbeSetting <PSHealthProbeSetting[]>]

 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureRmFrontDoor** aggiorna un bilanciamento del carico di porta anteriore. Se non vengono forniti parametri di input, verranno usati i vecchi parametri della porta anteriore esistente.
