# Resume-AzureRmAutomationJob

## Sinossi
Riprende un processo di automazione sospesa.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Resume-AzureRmAutomationJob [-Id] <Guid> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Resume-AzureRmAutomationJob** riprende un processo di automazione di Azure sospesa.
Specificare il processo sospeso.
Per sospendere un processo, usare il cmdlet Suspend-AzureRmAutomationJob.

## ESEMPI

### Esempio 1: riprendere un processo sospeso
```
PS C:\>Resume-AzureRmAutomationJob -AutomationAccountName "Contoso17" -Id 2989b069-24fe-40b9-b3bd-cb7e5eac4b64 -ResourceGroupName "ResourceGroup01"
```

Questo comando riprende il processo con l'ID specificato.
