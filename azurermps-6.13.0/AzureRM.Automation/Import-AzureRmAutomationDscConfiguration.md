# Import-AzureRmAutomationDscConfiguration

## Sinossi
Importa una configurazione DSC in automazione.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Import-AzureRmAutomationDscConfiguration -SourcePath <String> [-Tags <IDictionary>] [-Description <String>]
 [-Published] [-Force] [-LogVerbose <Boolean>] [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Import-AzureRmAutomationDscConfiguration** importa una configurazione DSC (APS desired state Configuration) in Azure Automation.
Specifica il percorso di uno script APS che contiene una singola configurazione DSC.

## ESEMPI

### Esempio 1: importare una configurazione DSC in automazione
```
PS C:\>Import-AzureRmAutomationDscConfiguration -AutomationAccountName "Contoso17"-ResourceGroupName "ResourceGroup01" -SourcePath "C:\DSC\client.ps1" -Force
```

Questo comando importa la configurazione DSC nel file denominato client.ps1 nell'account di automazione denominato Contoso17. Il comando specifica il parametro *Force* . Se esiste una configurazione DSC esistente, questo comando lo sostituisce.
