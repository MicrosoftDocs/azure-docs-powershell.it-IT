# Remove-AzureRmAutomationHybridWorkerGroup

## Sinossi
Rimuove il gruppo di lavoratori ibridi dall'automazione.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmAutomationHybridWorkerGroup [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet Remove-AzureRmAutomationHybridWorkerGroup rimuove un gruppo di lavoro ibrido dall'automazione.

## ESEMPI

### Esempio 1
Questo comando rimuove un lavoratore ibrido per nome.

```powershell
PS C:\> Remove-AzureRmAutomationHybridWorkerGroup -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "GroupName" `
                                                  -Force
```
