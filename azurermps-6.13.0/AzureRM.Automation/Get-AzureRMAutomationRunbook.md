# Get-AzureRmAutomationRunbook

## Sinossi
Ottiene un Runbook.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### ByAll (impostazione predefinita)
```
Get-AzureRmAutomationRunbook [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByRunbookName
```
Get-AzureRmAutomationRunbook [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmAutomationRunbook** ottiene manuali operativi di automazione di Azure.
Per ottenere uno specifico Runbook, specificarne il nome.

## ESEMPI

### Esempio 1: ottenere tutti manuali operativi
```
PS C:\>Get-AzureRmAutomationRunbook -AutomationAccountName "Contoso17" -ResourceGroupName "ResourceGroup01"
```

Questo comando ottiene tutti i manuali operativi nell'account di automazione di Azure denominato Contoso17.
