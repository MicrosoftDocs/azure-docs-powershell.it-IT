# Remove-AzureRmAutomationRunbook

## Sinossi
Rimuove un Runbook.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmAutomationRunbook [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmAutomationRunbook** rimuove un Runbook da Azure Automation.

## ESEMPI

### Esempio 1: rimuovere un Runbook
```
PS C:\>Remove-AzureRmAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbook03" -ResourceGroupName "ResourceGroup01"
```

Questo comando rimuove il Runbook denominato Runbook03 nell'account di automazione di Azure denominato Contoso17.
