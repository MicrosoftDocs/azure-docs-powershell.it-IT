# Publish-AzureRmAutomationRunbook

## Sinossi
Pubblica un Runbook.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Publish-AzureRmAutomationRunbook [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Publish-AzureRmAutomationRunbook** pubblica un Runbook per l'uso nell'ambiente di produzione di Azure Automation.

## ESEMPI

### Esempio 1: pubblicare un Runbook
```
PS C:\>Publish-AzureRmAutomationRunbook -AutomationAccountName "Contoso17" -Name "Runbk01" -ResourceGroupName "ResourceGroup01"
```

Questo comando pubblica il Runbook denominato Runbk01 nell'account di automazione di Azure denominato Contoso17.
