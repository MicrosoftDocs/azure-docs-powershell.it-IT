# Remove-AzureRmAutomationConnection

## Sinossi
Rimuove una connessione di automazione.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmAutomationConnection [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmAutomationConnection** rimuove una connessione da Azure Automation.

## ESEMPI

### Esempio 1: rimuovere una connessione
```
PS C:\>Remove-AzureRmAutomationConnection -AutomationAccountName "Contoso17" -Name "ContosoConnection" -ResourceGroupName "ResourceGroup01"
```

Questo comando rimuove un certificato denominato ContosoConnection nell'account di automazione denominato Contoso17.
