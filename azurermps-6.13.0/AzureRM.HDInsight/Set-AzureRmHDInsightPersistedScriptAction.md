# Set-AzureRmHDInsightPersistedScriptAction

## Sinossi
Imposta un'azione di script eseguita in precedenza come azione di script persistente.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Set-AzureRmHDInsightPersistedScriptAction [-ClusterName] <String> [-ScriptExecutionId] <Int64>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureRmHDInsightPersistedScriptAction** imposta un'azione di script eseguita in precedenza come azione di script persistente.
L'azione di script specificata deve essere stata completata in precedenza.
L'azione script verrÃ  eseguita ogni volta che viene ridimensionato il cluster di Azure HDInsight.

## ESEMPI

### Esempio 1: impostare un'azione di script in precedenza riuscita per essere mantenuta oppure eseguirla in scala cluster
```
PS C:\>Set-AzureRmHDInsightPersistedScriptAction `
            -ClusterName "your-hadoop-001" `
            -ScriptExecutionId "<id>"
```

Questo comando imposta un'azione di script in precedenza riuscita come azione di script persistente.
