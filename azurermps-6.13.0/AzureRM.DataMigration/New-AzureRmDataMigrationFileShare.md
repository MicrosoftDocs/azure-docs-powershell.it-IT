# New-AzureRmDataMigrationFileShare

## Sinossi
Crea l'oggetto FileShare per il servizio di migrazione del database di Azure, che specifica la condivisione di rete locale a cui eseguire il backup del database di origine.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmDataMigrationFileShare -Path <String> -Credential <PSCredential>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet New-AzureRmDataMigrationFileShare crea l'oggetto FileShare che specifica la condivisione di rete locale in cui il servizio di migrazione del database di Azure puÃ² eseguire i backup del database di origine. L'account del servizio che usa l'istanza di SQL Server di origine deve avere privilegi di scrittura per questa condivisione di rete.
