# Remove-AzureRmPolicyRemediation

## Sinossi
Elimina una correzione dei criteri.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### ByName (impostazione predefinita)
```
Remove-AzureRmPolicyRemediation -Name <String> [-Scope <String>] [-ManagementGroupName <String>]
 [-ResourceGroupName <String>] [-AllowStop] [-AsJob] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Remove-AzureRmPolicyRemediation -ResourceId <String> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Remove-AzureRmPolicyRemediation -InputObject <PSRemediation> [-AllowStop] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmPolicyRemediation** Elimina una correzione dei criteri.

## ESEMPI

### Esempio 1: eliminare un criterio di correzione nell'ambito di un gruppo di risorse
```
PS C:\> Remove-AzureRmPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1"
```

Questo comando Elimina il risanamento denominato "remediation1" nel gruppo di risorse "myRG".

### Esempio 2: eliminare una correzione del gruppo di gestione tramite piping
```
PS C:\> $remediation = Get-AzureRmPolicyRemediation -ManagementGroupName "mg1" -Name "remediation1"
PS C:\> $remediation | Remove-AzureRmPolicyRemediation -Confirm
```

Questo comando Elimina il risanamento denominato "remediation1" dal gruppo di gestione "MG1". Verrà visualizzata una richiesta di conferma prima di eliminare la risorsa.

### Esempio 3: annullare ed eliminare una correzione dei criteri
```
PS C:\> Remove-AzureRmPolicyRemediation -ResourceGroupName "myRG" -Name "remediation1" -AllowStop
```

Questo comando Elimina il risanamento denominato "remediation1" nel gruppo di risorse "myRG". Se la correzione è in corso, verrà annullata prima di essere eliminata.
