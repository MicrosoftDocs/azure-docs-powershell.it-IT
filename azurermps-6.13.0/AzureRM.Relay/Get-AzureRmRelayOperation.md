# Get-AzureRmRelayOperation

## Sinossi
Elenco delle operazioni di inoltro supportate

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmRelayOperation** elenca le operazioni supportate da relay.
