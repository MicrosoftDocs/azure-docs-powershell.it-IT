# Get-AzureRmSqlDatabaseAdvisor

## Sinossi
Ottiene uno o più consulenti per un database SQL di Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSqlDatabaseAdvisor [-AdvisorName <String>] [-ExpandRecommendedActions] -ServerName <String>
 -DatabaseName <String> [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlDatabaseAdvisor** ottiene uno o più consulenti di database SQL di Azure per un database SQL di Azure.
