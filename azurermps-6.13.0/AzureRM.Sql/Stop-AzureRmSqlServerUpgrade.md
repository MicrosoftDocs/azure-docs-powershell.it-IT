# Stop-AzureRmSqlServerUpgrade

## Sinossi
Interrompe l'aggiornamento di un server di database SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureRmSqlServerUpgrade [-Force] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureRmSqlServerUpgrade** interrompe l'aggiornamento di un server di database SQL di Azure.

## ESEMPI

### Esempio 1: arrestare un aggiornamento del server
```
PS C:\>Stop-AzureRmSqlServerUpgrade -ResourceGroupName "ResourceGroup01" -ServerName "Server02"
```

Questo comando interrompe la richiesta di aggiornamento del server denominato Server02 assegnato al gruppo di risorse denominato ResourceGroup01.
