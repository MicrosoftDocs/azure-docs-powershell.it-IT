# Set-AzureRmSqlServerRecommendedActionState

## Sinossi
Aggiorna lo stato di un'azione consigliata di Azure SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Set-AzureRmSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureRmSqlServerRecommendedActionState** aggiorna lo stato di un'azione consigliata di Azure SQL Server.
Questo cmdlet applica, ripristina o Elimina l'azione consigliata in base al nuovo stato.
