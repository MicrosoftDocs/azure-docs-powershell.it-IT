# Stop-AzureRmSqlElasticPoolActivity

## Sinossi
Annulla l'operazione di aggiornamento asincrono in un pool elastico.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureRmSqlElasticPoolActivity [-PassThru] [-ServerName] <String> [-ElasticPoolName] <String>
 [-OperationId <Guid>] [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureRmSqlElasticPoolActivity** Annulla l'operazione di aggiornamento asincrono in un pool elastico.

## ESEMPI

### Esempio 1: annullare l'operazione di aggiornamento asincrono in un pool elastico
```
PS C:\> Stop-AzureRmSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -OperationId af97005d-9243-4f8a-844e-402d1cc855f5
```

Questo comando Annulla l'operazione di aggiornamento asincrono sul pool elastico.
