# Get-AzureRmSqlInstance

## Sinossi
Restituisce informazioni sull'istanza di database gestito di Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### GetInstanceByResourceGroup (impostazione predefinita)
```
Get-AzureRmSqlInstance [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetInstanceByNameAndResourceGroup
```
Get-AzureRmSqlInstance [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlInstance** restituisce informazioni su una o piÃ¹ istanze gestite da SQL Azure.
Specificare il nome di un'istanza per visualizzare le informazioni solo per quell'istanza.
