# Remove-AzureRmSqlServerDnsAlias

## Sinossi
Rimuove l'alias DNS di Azure SQL Server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### Rimuovere un alias DNS del server dai parametri di input dei cmdlet
```
Remove-AzureRmSqlServerDnsAlias -Name <String> -ServerName <String> [-ResourceGroupName] <String> [-Force]
 [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Rimuovere un alias DNS del server dalla definizione dell'istanza di AzureSqlServerDnsAliasModel
```
Remove-AzureRmSqlServerDnsAlias -InputObject <AzureSqlServerDnsAliasModel> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Rimuovere un alias DNS del server da un ID di risorsa Azure
```
Remove-AzureRmSqlServerDnsAlias -ResourceId <String> [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Questo comando rimuove l'alias DNS di Azure SQL Server dal server lasciando intatto il server.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzureRmSqlServerDnsAlias -DnsAliasName aliasName -ServerName serverName -ResourceGroupName rg
```

Rimuove l'alias DNS di Azure SQL Server con il nome aliasname dal server con il nome nomeserver
