# Revoke-AzureRmDiskAccess

## Sinossi
Revoca un accesso a un disco.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Revoke-AzureRmDiskAccess [-ResourceGroupName] <String> [-DiskName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Revoke-AzureRmDiskAccess** revoca un accesso a un disco.

## ESEMPI

### Esempio 1
```
PS C:\> Revoke-AzureRmDiskAccess -ResourceGroupName 'ResourceGroup01' -DiskName 'Disk01'
```

Revocare l'accesso al disco denominato "Disk01" nel gruppo di risorse denominato "ResourceGroup01"
