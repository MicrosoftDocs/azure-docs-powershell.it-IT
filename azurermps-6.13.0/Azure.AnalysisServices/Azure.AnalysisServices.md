# Modulo di Azure. AnalysisServices
## Descrizione
Questo modulo contiene comandi per l'esecuzione di operazioni in un ambiente di Analysis Services specifico di Azure

## Cmdlet di Azure. AnalysisServices
### [Add-AzureAnalysisServicesAccount](Add-AzureAnalysisServicesAccount.md)
Aggiunge un account autenticato da usare per le richieste di cmdlet del server di Analysis Services Azure.

### [Export-AzureAnalysisServicesInstanceLog](Export-AzureAnalysisServicesInstanceLog.md)
Esporta un log da un'istanza del server di Analysis Services nell'ambiente attualmente connesso, come specificato nel comando Add-AzureAnalysisServicesAccount

### [Riavviare-AzureAnalysisServicesInstance](Restart-AzureAnalysisServicesInstance.md)
Riavvia un'istanza del server di Analysis Services nell'ambiente attualmente connesso, come specificato in comando Add-AzureAnalysisServicesAccount

### [Sincronizzazione-AzureAnalysisServicesInstance](Sync-AzureAnalysisServicesInstance.md)
Sincronizza un database specificato nell'istanza specificata di Analysis Services server in tutte le istanze di scalabilit√† delle query nell'ambiente attualmente connesso, come specificato in comando Add-AzureAnalysisServicesAccount
