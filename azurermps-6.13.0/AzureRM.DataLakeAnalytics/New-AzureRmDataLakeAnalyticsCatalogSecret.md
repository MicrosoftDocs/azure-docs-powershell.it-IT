# New-AzureRmDataLakeAnalyticsCatalogSecret

## Sinossi
Crea un segreto del catalogo dei dati di Lake Analytics.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### CreateByFullURI
```
New-AzureRmDataLakeAnalyticsCatalogSecret [-Account] <String> [-DatabaseName] <String> [-Secret] <PSCredential>
 [-DatabaseHost] <String> [-Port] <Int32> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### CreateByHostNameAndPort
```
New-AzureRmDataLakeAnalyticsCatalogSecret [-Account] <String> [-DatabaseName] <String> [-Secret] <PSCredential>
 [-Uri] <Uri> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRmDataLakeAnalyticsCatalogSecret** crea un segreto da usare in un catalogo di dati di analisi di Azure Data Lake.

## ESEMPI

### Esempio 1: ottenere il segreto per un catalogo
```
PS C:\>New-AzureRmDataLakeAnalyticsCatalogSecret -Account "ContosoAdlAccount" -DatabaseName "databaseName" -Secret (Get-Credential) -Host "https://example.contoso.com" -Port 8080
```

Questo comando ottiene il segreto corrispondente all'account, al database, alle credenziali e all'host specificati.
