# Remove-AzureRmApplicationGatewayUrlPathMapConfig

## Sinossi
Rimuove i mapping del percorso URL in un pool di server back-end.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmApplicationGatewayUrlPathMapConfig -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmApplicationGatewayUrlPathMapConfig** rimuove i mapping del percorso URL in un pool di server back-end.
