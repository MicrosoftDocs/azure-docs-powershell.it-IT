# Remove-AzureRmFirewall

## Sinossi
Rimuovere un firewall.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmFirewall -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmFirewall** rimuove un firewall di Azure.

## ESEMPI

### 1: creare ed eliminare un firewall
```
New-AzureRmFirewall -Name "azFw" -ResourceGroupName "rgName" -Location centralus 

Remove-AzureRmFirewall -Name "azFw" -ResourceGroupName "rgName"
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Questo esempio crea un firewall e lo elimina. Per eliminare il prompt quando si elimina il firewall, usare il flag-Force.

### 2: deallocare il firewall, quindi eliminare il firewall
```
$firewall=Get-AzureRmFirewall -ResourceGroupName rgName -Name azFw
$firewall.Deallocate()
Remove-AzureRmFirewall -ResourceGroupName rgName -Name azFw
Confirm
Are you sure you want to remove resource 'azFw'
[Y] Yes  [N] No  [S] Suspend  [?] Help (default is "Y"): y
```

Questo esempio recupera un firewall, rilascia il firewall e quindi Elimina il firewall. Il comando deallocate rimuove il servizio in corso mantenendo la configurazione del firewall. Se l'utente vuole riavviare il servizio, il metodo allocate deve essere chiamato sul firewall.
Per eliminare il prompt quando si elimina il firewall, usare il flag-Force.
