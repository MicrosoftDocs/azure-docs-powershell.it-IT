# New-AzureRmApplicationGatewayTrustedRootCertificate

## Sinossi
Crea un certificato radice attendibile per un gateway dell'applicazione.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmApplicationGatewayTrustedRootCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRmApplicationGatewayTrustedRootCertificate** crea un certificato radice attendibile per un gateway dell'applicazione Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $certFilePath = ".\rootCA.cer"
PS C:\> $trc = New-AzureRmApplicationGatewayTrustedRootCertificate -Name "trc1" --CertificateFile $certFilePath
```

Questo comando crea un certificato radice attendibile denominato List "TRC1" e archivia il risultato nella variabile denominata $trc.
