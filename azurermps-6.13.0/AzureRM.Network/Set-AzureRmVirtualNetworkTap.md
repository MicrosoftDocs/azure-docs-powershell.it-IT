# Set-AzureRmVirtualNetworkTap

## Sinossi
Imposta lo stato obiettivo per un tocco di rete virtuale.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Set-AzureRmVirtualNetworkTap -VirtualNetworkTap <PSVirtualNetworkTap> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
**Set-AzureRmVirtualNetworkTap** imposta lo stato obiettivo per un tocco di rete virtuale di Azure.

## ESEMPI

### Esempio 1: configurare un tocco di rete virtuale
```
PS C:\>$vTap = Get-AzureRmVirtualNetworkTap -ResourceGroupName "ResourceGroup1" -Name "VirtualTap1"
PS C:\>$vTap.DestinationNetworkInterfaceIPConfiguration = $newDestinationNic.IpConfigurations[0]
PS C:\>Set-AzureRmVirtualNetworkTap -VirtualNetworkTap $vTap
```

Il comando Aggiorna il IpConfiguration di destinazione e aggiorna il tocco di rete virtuale.
Se sono presenti configurazioni tap che la fanno riferimento, tutto il traffico di origine non inizier√† a essere speculare alla nuova configurazione IP di destinazione post-aggiornamento.
