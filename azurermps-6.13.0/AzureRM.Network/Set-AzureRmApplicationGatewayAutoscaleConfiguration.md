# Set-AzureRmApplicationGatewayAutoscaleConfiguration

## Sinossi
Aggiorna la configurazione di un gateway applicazione in scala automatica.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Set-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 -MinCapacity <Int32> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureRmApplicationGatewayAutoscaleConfiguration** modifica la configurazione di scala automatica esistente di un gateway dell'applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzureRmApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Set-AzureRmApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw -MinCapacity 5
PS C:\> $gw = Set-AzureRmApplicationGateway -ApplicationGateway $gw
```

Il primo comando ottiene il gateway dell'applicazione e lo archivia in $gw variabile.
Il secondo comando Aggiorna la configurazione di scala automatica dal gateway applicationg.
Il terzo comando Aggiorna il gateway dell'applicazione in Azure.
