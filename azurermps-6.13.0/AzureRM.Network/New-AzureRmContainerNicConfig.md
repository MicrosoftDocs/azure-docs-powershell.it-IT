# New-AzureRmNetworkProfileContainerNicConfig

## Sinossi
Crea un nuovo oggetto di configurazione dell'interfaccia di rete contenitore.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmNetworkProfileContainerNicConfig [-Name <String>] [-IpConfiguration <PSIPConfigurationProfile[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRmNetworkProfileContainerNicConfig** crea un nuovo oggetto di configurazione dell'interfaccia di rete contenitore. Questo oggetto determina le caratteristiche della rete di contenitori interfacs creata facendo riferimento al profilo di rete padre.

## ESEMPI

### Esempio 1
```powershell
$containerNicConfig = New-AzureRmNetworkProfileContainerNicConfig -Name cnicConfig1

$networkProfile = New-AzureRmNetworkProfile -Name np1 -ResourceGroupName rg1 -Location westus -ContainerNetworkInterfaceConfiguration $containerNicConfig
```

Il primo comando crea una configurazione di interfaccia di rete contenitore vuota. Il secondo crea un nuovo profilo di rete, passando la configurazione dell'interfaccia di rete del contenitore creato in precedenza come argomento al cmdlet New-NetworkProfile.
