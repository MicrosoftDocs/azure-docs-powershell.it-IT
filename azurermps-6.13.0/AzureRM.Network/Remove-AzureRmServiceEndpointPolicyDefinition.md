## SINTASSI

```
Remove-AzureRmServiceEndpointPolicyDefinition [-Name <String>] -ServiceEndpointPolicy <PSServiceEndpointPolicy>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmServiceEndpointPolicy** rimuove un criterio endpoint del servizio.

## ESEMPI

### Esempio 1: rimuove un criterio endpoint di servizio tramite Name
```
Remove-AzureRmServiceEndpointPolicyDefinition -Name "PolicyDef1"
```

Questo comando rimuove un criterio endpoint servizio con nome PolicyDef1
