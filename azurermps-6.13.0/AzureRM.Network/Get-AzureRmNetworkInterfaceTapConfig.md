# Get-AzureRmNetworkInterfaceTapConfig

## Sinossi
Ottiene una risorsa di configurazione tap.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzureRmNetworkInterfaceTapConfig -ResourceGroupName <String> -NetworkInterfaceName <String>
 [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzureRmNetworkInterfaceTapConfig -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmNetworkInterfaceTapConfig** ottiene una configurazione di Azure tap per un gruppo di risorse specifico, un'interfaccia di rete e un nome di configurazione di tocco o un elenco di configurazioni di tocco in un gruppo di risorse e in un'interfaccia di rete.

## ESEMPI

### Esempio 1: ottenere tutte le configurazioni di tocco per una determinata interfaccia di rete
```
PS C:\>Get-AzureRmNetworkInterfaceTapConfig -ResourceGroupName "ResourceGroup1" -NetworkInterface "sourceNicName"
```

Questo comando ottiene il tocco delle configurazioni aggiunte per l'interfaccia di rete specificata.

### Esempio 2: ottenere una configurazione di tocco specificata
```
PS C:\>Get-AzureRmNetworkInterface -ResourceGroupName "ResourceGroup1" -NetworkInterface "sourceNicName" -Name "tapconfigName"
```

Con questo comando viene aggiunta una configurazione di tocco specifica per l'interfaccia di rete specificata.
