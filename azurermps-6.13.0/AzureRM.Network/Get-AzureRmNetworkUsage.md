# Get-AzureRmNetworkUsage

## Sinossi
Elenca gli usi di rete per un abbonamento

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmNetworkUsage -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzureRmNetworkUsage ottiene i limiti e l'utilizzo corrente per le risorse di rete.
