# Remove-AzureRmApplicationGatewayTrustedRootCertificate

## Sinossi
Rimuove un certificato radice attendibile da un gateway dell'applicazione.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmApplicationGatewayTrustedRootCertificate -Name <String>
 -ApplicationGateway <PSApplicationGateway> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmApplicationGatewayTrustedRootCertificate** rimuove un certificato radice attendibile da un gateway applicazione esistente.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzureRmApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzureRmApplicationGatewayTrustedRootCertificate -ApplicationGateway $gw -Name "myRootCA"
PS C:\> $gw = Set-AzureRmApplicationGateway -ApplicationGateway $gw
```

Il primo comando ottiene un gateway dell'applicazione e lo archivia nella variabile $gw.
Il secondo comando rimuove il certificato radice attendibile denominato myRootCA dal gateway dell'applicazione archiviato in $gw.
Il terzo comando Aggiorna il gateway dell'applicazione in Azure.
