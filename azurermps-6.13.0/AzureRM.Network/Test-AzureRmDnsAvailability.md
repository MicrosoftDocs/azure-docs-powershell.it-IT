# Test-AzureRmDnsAvailability

## Sinossi
Controlla se un nome di dominio nella zona cloudapp.azure.com è disponibile per l'uso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Test-AzureRmDnsAvailability -DomainNameLabel <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Controlla se un nome di dominio nella zona cloudapp.azure.com è disponibile per l'uso.

## ESEMPI

### Esempio 1: verificare se contoso.cloudapp.azure.com è disponibile per l'uso.
```
Test-AzureRmDnsAvailability -DomainNameLabel contoso -Location westus
```
