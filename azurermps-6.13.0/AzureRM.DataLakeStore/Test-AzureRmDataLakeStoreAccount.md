# Test-AzureRmDataLakeStoreAccount

## Sinossi
Verifica l'esistenza di un account di data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Test-AzureRmDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **test-AzureRmDataLakeStoreAccount** verifica l'esistenza di un account di data Lake Store.

## ESEMPI

### Esempio 1: testare un account
```
PS C:\>Test-AzureRmDataLakeStoreAccount -Name "ContosoADL"
```

Questo comando verifica se l'account denominato ContosoADL esiste.
