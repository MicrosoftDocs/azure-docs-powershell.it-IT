# Join-AzureRmDataLakeStoreItem

## Sinossi
Unisce uno o più file per creare un file in data Lake Store.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Join-AzureRmDataLakeStoreItem [-Account] <String> [-Paths] <DataLakeStorePathInstance[]>
 [-Destination] <DataLakeStorePathInstance> [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **join-AzureRmDataLakeStoreItem** unisce uno o più file per creare un file in data Lake Store.

## ESEMPI

### Esempio 1: unire due elementi
```
PS C:\>Join-AzureRmDataLakeStoreItem -AccountName "ContosoADL" -Paths "/MyFiles/File01.txt","/MyFiles/File02.txt" -Destination "/MyFiles/CombinedFile.txt"
```

Questo comando unisce File01.txt e File02.txt per creare il file CombinedFile.txt.
