# Get-AzureRmApiManagementCertificate

## Sinossi
Ottiene i certificati di gestione delle API configurati per l'autenticazione reciproca con backend nel servizio.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### GetAllCertificates (impostazione predefinita)
```
Get-AzureRmApiManagementCertificate -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### GetByCertificateId
```
Get-AzureRmApiManagementCertificate -Context <PsApiManagementContext> -CertificateId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmApiManagementCertificate** ottiene tutti i certificati o i certificati di gestione delle API di Azure specificati.

## ESEMPI

### Esempio 1: ottenere tutti i certificati
```
PS C:\>$ApiMgmtContext = New-AzureRmApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzureRmApiManagementCertificate -Context $ApiMgmtContext
```

Questo comando ottiene tutti i certificati di gestione delle API.

### Esempio 2: ottenere un certificato in base all'ID
```
PS C:\>$ApiMgmtContext = New-AzureRmApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzureRmApiManagementCertificate -Context $ApiMgmtContext -CertificateId "0123456789"
```

Questo comando ottiene il certificato di gestione delle API con l'ID specificato.
