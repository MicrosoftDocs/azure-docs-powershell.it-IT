# Remove-AzureRmApiManagement

## Sinossi
Rimuove un servizio di gestione API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmApiManagement -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmApiManagement** rimuove un servizio di gestione delle API di Azure.

## ESEMPI

### Esempio 1: rimuovere un servizio di gestione API
```
PS C:\>Remove-AzureRmApiManagement -ResourceGroupName "ContosoGroup02" -Name "ContosoApi"
```

Questo comando rimuove il servizio di gestione delle API denominato ContosoApi.
