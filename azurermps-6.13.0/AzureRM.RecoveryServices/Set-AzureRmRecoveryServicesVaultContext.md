# Set-AzureRmRecoveryServicesVaultContext

## Sinossi
Imposta il contesto del Vault.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Set-AzureRmRecoveryServicesVaultContext -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureRmRecoveryServicesVaultContext** imposta il contesto del Vault per i servizi di ripristino del sito Azure.

## ESEMPI

### Esempio 1
```
PS C:\> Set-AzureRmRecoveryServicesVaultContext -Vault $vault
```

Imposta il contesto del Vault.
