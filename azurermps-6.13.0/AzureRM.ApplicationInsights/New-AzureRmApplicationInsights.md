# New-AzureRmApplicationInsights

## Sinossi
Creare una nuova risorsa approfondimenti applicazione

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmApplicationInsights [-ResourceGroupName] <String> [-Name] <String> [-Location] <String>
 [-Kind <String>] [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Creare una nuova risorsa approfondimenti applicazione

## ESEMPI

### Esempio 1 creare una nuova risorsa approfondimenti applicazione
