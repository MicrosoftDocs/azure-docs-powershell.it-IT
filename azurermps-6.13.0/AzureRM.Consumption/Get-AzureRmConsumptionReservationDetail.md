# Get-AzureRmConsumptionReservationDetail

## Sinossi
Ottenere i dettagli delle prenotazioni per l'intervallo di date specificato.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmConsumptionReservationDetail -StartDate <DateTime> -EndDate <DateTime> -ReservationOrderId <String>
 [-ReservationId <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmConsumptionReservationDetail** ottiene i dettagli delle prenotazioni per l'intervallo di date specificato.
