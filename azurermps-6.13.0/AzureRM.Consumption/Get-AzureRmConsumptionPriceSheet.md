# Get-AzureRmConsumptionPriceSheet

## Sinossi
Ottenere i fogli di prezzo dell'abbonamento.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmConsumptionPriceSheet [-BillingPeriodName <String>] [-ExpandMeterDetail] [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmConsumptionPriceSheet** ottiene i fogli di prezzo dell'abbonamento.
