# Remove-AzureRmStreamAnalyticsOutput

## Sinossi
Elimina un output da un processo di analisi del flusso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmStreamAnalyticsOutput** Elimina in modo asincrono un output da un processo di analisi del flusso in Azure.

## ESEMPI

### ESEMPIO 1: rimuovere un output da un processo
```
PS C:\>Remove-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Questo comando rimuove l'output di output in Job StreamingJob.
