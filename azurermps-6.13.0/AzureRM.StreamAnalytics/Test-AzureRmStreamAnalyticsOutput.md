# Test-AzureRmStreamAnalyticsOutput

## Sinossi
Verifica lo stato di connessione di un output.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Test-AzureRmStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **test-AzureRmStreamAnalyticsOutput** verifica la possibilitÃ  di una connessione di analisi dello stream a un output.

## ESEMPI

### ESEMPIO 1: verificare lo stato di connessione di un output
```
PS C:\>Test-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Questo verifica lo stato di connessione dell'output di output in StreamingJob.
