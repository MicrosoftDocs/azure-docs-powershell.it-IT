# Add-AzureRmKeyVaultNetworkRule

## Sinossi
Aggiunge una regola che consente di limitare l'accesso a un Vault chiave in base all'indirizzo Internet del client.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### ByVaultName (impostazione predefinita)
```
Add-AzureRmKeyVaultNetworkRule [-VaultName] <String> [[-ResourceGroupName] <String>]
 [-IpAddressRange <String[]>] [-VirtualNetworkResourceId <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByInputObject
```
Add-AzureRmKeyVaultNetworkRule [-InputObject] <PSKeyVault> [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### ByResourceId
```
Add-AzureRmKeyVaultNetworkRule [-ResourceId] <String> [-IpAddressRange <String[]>]
 [-VirtualNetworkResourceId <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzureRmKeyVaultNetworkRule** concede o limita l'accesso a un Vault chiave a un set di chiamante designato dagli indirizzi IP o dalla rete virtuale a cui appartengono. La regola offre la possibilità di limitare l'accesso ad altri utenti, applicazioni o gruppi di sicurezza a cui sono state concesse le autorizzazioni tramite i criteri di accesso.
