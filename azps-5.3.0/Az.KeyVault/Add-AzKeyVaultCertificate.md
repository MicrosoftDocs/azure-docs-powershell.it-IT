# Add-AzKeyVaultCertificate

## Sinossi
Aggiunge un certificato a un Vault chiave.

## SINTASSI

```
Add-AzKeyVaultCertificate [-VaultName] <String> [-Name] <String>
 [-CertificatePolicy] <PSKeyVaultCertificatePolicy> [-Tag <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzKeyVaultCertificate** avvia il processo di registrazione di un certificato in un Vault chiave in Azure Key Vault.

Il primo comando usa il cmdlet New-AzKeyVaultCertificatePolicy per creare un criterio di certificato e lo archivia nella variabile $Policy.
Il secondo comando USA **Add-AzKeyVaultCertificate** per avviare il processo per creare un certificato.
Il terzo comando usa il cmdlet Get-AzKeyVaultCertificateOperation per eseguire il polling dell'operazione per verificare che sia completa.
Il comando finale usa il cmdlet Get-AzKeyVaultCertificate per ottenere il certificato.
