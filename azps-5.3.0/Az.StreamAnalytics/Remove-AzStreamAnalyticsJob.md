# Remove-AzStreamAnalyticsJob

## Sinossi
Rimuove un processo di analisi del flusso.

## SINTASSI

```
Remove-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzStreamAnalyticsJob** Elimina in modo asincrono un processo specifico di analisi del flusso in Azure.

## ESEMPI

### ESEMPIO 1: rimuovere un processo
```
PS C:\>Remove-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

Questo comando rimuove il job StreamingJob.
