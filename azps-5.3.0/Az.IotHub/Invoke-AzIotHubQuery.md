# Invoke-AzIotHubQuery

## Sinossi
Eseguire una query su un hub molto usando un potente linguaggio simile a SQL.

## SINTASSI

### ResourceSet (impostazione predefinita)
```
Invoke-AzIotHubQuery [-ResourceGroupName] <String> [-IotHubName] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectSet
```
Invoke-AzIotHubQuery [-InputObject] <PSIotHub> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdSet
```
Invoke-AzIotHubQuery [-ResourceId] <String> [-Query] <String> [-Top <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Eseguire una query su un hub molto usato con una potente lingua simile a SQL per recuperare le informazioni riguardanti i gemelli di dispositivi e moduli, i processi e il routing dei messaggi.
https://docs.microsoft.com/azure/iot-hub/iot-hub-devguide-query-languagePer altre informazioni, vedere.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices"
```

Eseguire una query su tutti i dati dei dispositivi Twin in un hub di Azure.

### Esempio 2
```powershell
PS C:\> Invoke-AzIotHubQuery -ResourceGroupName "myresourcegroup" -IotHubName "myiothub" -Query "select * from devices.modules where devices.deviceId = 'myDevice1'" -Top 2
```

Query Top 2 Module dati Twin su un dispositivo di destinazione.
