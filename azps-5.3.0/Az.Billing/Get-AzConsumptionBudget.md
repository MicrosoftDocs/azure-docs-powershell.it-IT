# Get-AzConsumptionBudget

## Sinossi
Ottenere un elenco di budget in un abbonamento o in un gruppo di risorse.

## SINTASSI

```
Get-AzConsumptionBudget [-DefaultProfile <IAzureContextContainer>] [-ResourceGroupName <String>]
 [-Name <String>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzConsumptionBudget ottiene un elenco di budget in un abbonamento o in un gruppo di risorse.

## ESEMPI

### Esempio 1: ottenere un elenco di budget a livello di abbonamento
```powershell
PS C:\> Get-AzConsumptionBudget

CurrentSpend:  null

TimePeriod:  EndDate:  11/1/2018 12:00:00 AM
             StartDate:  6/1/2018 12:00:00 AM
Type:  Microsoft.Consumption/budgets
```
