# Get-AzSynapseSparkPool

## Sinossi
Ottiene un pool di scintille di analisi sinapsi.

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzSynapseSparkPool [-ResourceGroupName <String>] -WorkspaceName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByParentObjectParameterSet
```
Get-AzSynapseSparkPool [-Name <String>] -WorkspaceObject <PSSynapseWorkspace>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzSynapseSparkPool -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSynapseSparkPool** ottiene le informazioni su un pool di analisi di Azure sinapsi.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzSynapseSparkPool -WorkspaceName ContosoWorkspace
```

Questo comando consente di ottenere tutti i pool di scintille in un'area di lavoro.

### Esempio 2
```powershell
PS C:\> Get-AzSynapseSparkPool -WorkspaceName ContosoWorkspace -Name ContosoSparkPool
```

Questo comando consente di ottenere il pool di scintille in area di lavoro ContosoWorkspace con nome ContosoSparkPool.

### Esempio 3
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Get-AzSynapseSparkPool
```

Questo comando consente di ottenere tutti i pool di scintille in un'area di lavoro tramite pipeline.

### Esempio 4
```powershell
PS C:\> Get-AzSynapseSparkPool -ResourceId "/subscriptions/21686af7-58ec-4f4d-9c68-f431f4db4edd/resourceGroups/ContosoResourceGroup/providers/Microsoft.Synapse/workspaces/ContosoWorkspace/bigDataPools/ContosoSparkPool"
```

Questo comando consente di ottenere il pool di scintille con l'ID risorsa specificato.
