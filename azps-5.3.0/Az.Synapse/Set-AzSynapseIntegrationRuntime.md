## Descrizione
Il cmdlet **set-AzSynapseIntegrationRuntime** aggiorna un runtime di integrazione con parametri specifici.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' -Description 'New description'
```

Il cmdlet aggiorna la descrizione di Integration runtime denominata "test-SelfHost-IR".

### Esempio 2
```powershell
PS C:\> Set-AzSynapseIntegrationRuntime -WorkspaceName ContosoWorkspace -Name 'test-selfhost-ir' `
                                        -SharedIntegrationRuntimeResourceId '/subscriptions/b3ee3a7f-7614-4644-ad07-afa832620b4b/resourceGroups/rg-test-dfv2/providers/Microsoft.DataFactory/factories/test-df-eu2/integrationruntimes/test-selfhost-ir' -Type "SelfHosted"
```

Il cmdlet aggiunge l'area di lavoro per usare il runtime di integrazione condivisa. Quando `-SharedIntegrationRuntimeResourceId` si usa il parametro `-Type` deve anche essere incluso. Tieni presente che nell'area di lavoro deve essere concessa l'autorizzazione per usare Integration runtime prima di eseguire cmdlet.
