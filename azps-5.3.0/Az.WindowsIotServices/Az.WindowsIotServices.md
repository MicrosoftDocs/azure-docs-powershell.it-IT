# Modulo AZ. WindowsIotServices
## Descrizione
PowerShell di Microsoft Azure: cmdlet WindowsIotServices

## Cmdlet AZ. WindowsIotServices
### [Get-AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
Ottenere i metadati correlati non relativi alla sicurezza di un servizio di dispositivo Windows.

### [New-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
Creare o aggiornare i metadati di un servizio per dispositivi Windows.
Il modello usuale per la modifica di una proprietà consiste nel recuperare i metadati del servizio di dispositivo Windows e i metadati di sicurezza e quindi combinarli con i valori modificati in un nuovo corpo per aggiornare il servizio Windows sacco.

### [Remove-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
Eliminare un servizio Windows sacco.

### [Update-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
Aggiorna i metadati di un servizio per dispositivi Windows.
Il modello usuale per la modifica di una proprietà consiste nel recuperare i metadati del servizio di dispositivo Windows e i metadati di sicurezza e quindi combinarli con i valori modificati in un nuovo corpo per aggiornare il servizio Windows sacco.
