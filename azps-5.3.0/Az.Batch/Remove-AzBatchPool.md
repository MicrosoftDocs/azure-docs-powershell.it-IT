# Remove-AzBatchPool

## Sinossi
Elimina il pool di batch specificato.

## SINTASSI

```
Remove-AzBatchPool [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzBatchPool** Elimina il pool di batch di Azure specificato.
Viene richiesto di confermare a meno che non si usi il parametro *Force* .

## ESEMPI

### Esempio 1: eliminare un pool di batch per ID pool
```
PS C:\>Remove-AzBatchPool -Id "MyPool" -BatchContext $Context
```

Questo comando Elimina il pool con ID pool.
All'utente viene richiesta la conferma prima che venga applicata l'operazione di eliminazione.

### Esempio 2: eliminare tutti i pool di batch per forza
```
PS C:\>Get-AzBatchPool -BatchContext $Context | Remove-AzBatchPool -Force -BatchContext $Context
```

Questo comando Elimina tutti i pool di batch.
Dato che il parametro *Force* è presente, il messaggio di conferma viene soppresso.

Per usare invece l'autenticazione con chiave condivisa, usa il cmdlet Get-AzBatchAccountKey per ottenere un oggetto BatchAccountContext con i relativi tasti di scelta popolati. Quando si usa l'autenticazione con chiave condivisa, il tasto di scelta principale viene usato per impostazione predefinita. Per cambiare la chiave da usare, imposta la proprietà BatchAccountContext.
