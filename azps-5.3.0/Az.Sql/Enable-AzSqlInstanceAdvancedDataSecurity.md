# Enable-AzSqlInstanceAdvancedDataSecurity

## Sinossi
Abilita la sicurezza dei dati avanzata in un'istanza gestita.

## SINTASSI

```
Enable-AzSqlInstanceAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlManagedInstanceModel>] -InstanceName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Enable-AzSqlInstanceAdvancedDataSecurity** Abilita la sicurezza dei dati avanzata in un'istanza gestita. Advanced Data Security è un pacchetto di sicurezza unificato che include la classificazione dei dati, la valutazione delle vulnerabilità e la protezione avanzata delle minacce per l'istanza gestita. Verrà creato automaticamente un nuovo account di archiviazione per il salvataggio delle valutazioni della vulnerabilità. Se un account di archiviazione è stato creato in precedenza per questo scopo, verrà usato al suo posto)

## ESEMPI

### Esempio 1: abilitare la sicurezza dei dati avanzata dell'istanza gestita
```powershell
PS C:\>  Enable-AzSqlInstanceAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -InstanceName "ManagedInstance01" 

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```

### Esempio 2: abilitare la sicurezza dei dati avanzata dell'istanza gestita da una risorsa del server
```powershell
PS C:\>  Get-AzSqlInstance `
           -ResourceGroupName "ResourceGroup01" `
           -Name "ManagedInstance01" `
           | Enable-AzSqlInstanceAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ManagedInstanceName          : ManagedInstance01
IsEnabled                    : True
```
