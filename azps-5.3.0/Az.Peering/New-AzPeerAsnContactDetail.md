# New-AzPeerAsnContactDetail

## Sinossi
Crea un dettaglio di contatto in memoria per PeerAsn. 

## SINTASSI

```
New-AzPeerAsnContactDetail -Role <String> -Email <String> [-Phone <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Creare un dettaglio di contatto PeerAsn in memoria.

## ESEMPI

### Creare dettagli di contatto e aggiungere a PeerAsn
```powershell
PS C:\> $nocContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> $customerContact = New-AzPeerAsnContactDetail -Role Noc -Email "noc@contoso.com" -Phone "+1 (887) 888-8088"
PS C:\> New-AzPeerAsn -Name $name -PeerName "Contoso Networks Limited" -PeerAsn 65000 -ContactDetail $nocContact,$customerContact
```

Ruolo e posta elettronica sono obbligatori. Il telefono è facoltativo. Il telefono supporta +-() o gli spazi. Un PeerAsn deve includere almeno un dettaglio di contatto di tipo "NOC"
