# Set-AzDataLakeAnalyticsDataSource

## Sinossi
Modifica i dettagli di un'origine dati di un account di analisi dei dati sul lago.

## SINTASSI

```
Set-AzDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-AccessKey] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzDataLakeAnalyticsDataSource** modifica i dettagli di un'origine dati di un account di analisi di Azure Data Lake.

## ESEMPI

### Esempio 1: cambiare il tasto di scelta per un'origine dati
```
PS C:\>Set-AzDataLakeAnalyticsDataSource -Account "ContosoAdlAccount" -Blob "contosowasb" -AccessKey "...newaccesskey..."
```

Questo comando modifica il tasto di scelta archiviato per un'origine dati di archiviazione BLOB di Azure.
