# Add-AzRecoveryServicesAsrReplicationProtectedItemDisk

## Sinossi
Aggiungere il disco per la protezione per la macchina virtuale di Azure già protetta.

## SINTASSI

```
Add-AzRecoveryServicesAsrReplicationProtectedItemDisk -InputObject <ASRReplicationProtectedItem>
 -AzureToAzureDiskReplicationConfiguration <ASRAzuretoAzureDiskReplicationConfig[]> [-WaitForCompletion]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzRecoveryServicesAsrReplicationProtectedItemDisk** aggiunge il disco per la protezione per la macchina virtuale Azure già protetta.

## ESEMPI

### Esempio 1
```powershell
PS C:> Add-AzRecoveryServicesAsrReplicationProtectedItemDisk -ReplicationProtectedItem $rpi -AzureToAzureDiskReplicationConfiguration $disk1,$disk2
```

Avviare l'operazione per aggiungere la configurazione del disco specificata per la protezione.

### Esempio 2
```powershell
PS C:> $ReplicationProtectedItem |Add-AzRecoveryServicesAsrReplicationProtectedItemDisk -AzureToAzureDiskReplicationConfiguration $disk1,$disk2
```

Avviare l'operazione per aggiungere la configurazione del disco specificata per la protezione. Elemento protetto per la replica di input piping.
