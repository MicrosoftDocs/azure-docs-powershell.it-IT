# Set-AzApiManagementNamedValue

## Sinossi
Modifica un valore denominato per la gestione delle API.

## SINTASSI

```
Set-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-Name <String>]
 [-Value <String>] [-Secret <Boolean>] [-Tag <String[]>] [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzApiManagementNamedValue** modifica una gestione delle API di Azure denominata Value.

## ESEMPI

### Esempio 1: modificare i tag nel valore denominato
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>$Tags = 'sdk', 'powershell'
PS C:\> Set-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -Tags $Tags -PassThru
```

Il primo comando assegna due valori alla variabile $Tags.
Il secondo comando modifica il valore denominato che contiene l'ID Property11.
Il comando assegna le stringhe in $Tags come tag nel valore denominato.

### Esempio 2: modificare il valore denominato per avere un valore segreto
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Set-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property12" -Secret $True -PassThru
```

Questo comando modifica il valore denominato da crittografare.
