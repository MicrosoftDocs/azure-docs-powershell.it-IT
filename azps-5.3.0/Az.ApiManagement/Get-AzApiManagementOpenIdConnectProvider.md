# Get-AzApiManagementOpenIdConnectProvider

## Sinossi
Ottiene i provider di connessione OpenID.

## SINTASSI

### GetAllOpenIdConnectProviders (impostazione predefinita)
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByOpenIdConnectProviderId
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-OpenIdConnectProviderId <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### GetByName
```
Get-AzApiManagementOpenIdConnectProvider -Context <PsApiManagementContext> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzApiManagementOpenIdConnectProvider** ottiene i provider di connessione OpenID in gestione API di Azure.
ClientSecret non verrÃ  incluso nei dettagli dei risultati. Per ottenere il segreto del client, USA **Get-AzApiManagementOpenIdConnectProviderClientSecret**.

## ESEMPI

### Esempio 1: ottenere tutti i provider
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext
```

Questo comando ottiene tutti i provider di connessione OpenID per il contesto specificato.

### Esempio 2: ottenere un provider tramite un ID
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -OpenIdConnectProviderId "OICProvider01"
```

Questo comando ottiene il provider con ID OICProvider01.

### Esempio 3: ottenere un provider usando un nome
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementOpenIdConnectProvider -Context $apimContext -Name "Contoso OpenID Connect Provider"
```

Questo comando ottiene il provider denominato contoso OpenID Connect provider.
