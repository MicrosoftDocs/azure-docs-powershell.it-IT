# New-AzApiManagementSamplingSetting

## Sinossi
Creare una nuova impostazione di campionamento per la diagnostica

## SINTASSI

```
New-AzApiManagementSamplingSetting [-SamplingType <String>] [-SamplingPercentage <Double>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzApiManagementSamplingSetting** crea una nuova impostazione di campionamento per la diagnostica

## ESEMPI

### Esempio 1: creare un'impostazione di campionamento di base
```powershell
PS C:\> New-AzApiManagementSamplingSetting -SamplingType fixed -Percentage 100

SamplingType Percentage
------------ ----------
fixed               100
```

Crea un'impostazione di campionamento di `Fixed` tipo con registrazione per 100% delle richieste/risposte

### Esempio 2

Creare una nuova impostazione di campionamento per la diagnostica. AutoGenerated

```powershell
<!-- Aladdin Generated Example --> 
New-AzApiManagementSamplingSetting -SamplingPercentage 100 -SamplingType fixed
```
