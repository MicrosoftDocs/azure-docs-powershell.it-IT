# Remove-AzApiManagementNamedValue

## Sinossi
Rimuove un valore denominato per la gestione delle API.

## SINTASSI

```
Remove-AzApiManagementNamedValue -Context <PsApiManagementContext> -NamedValueId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApiManagementNamedValue** consente di rimuovere una gestione API di Azure **denominata value**.

## ESEMPI

### Esempio 1: rimuovere il valore denominato
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementNamedValue -Context $apimContext -NamedValueId "Property11" -PassThru
```

Questo comando rimuove il valore denominato che contiene l'ID Property11.
