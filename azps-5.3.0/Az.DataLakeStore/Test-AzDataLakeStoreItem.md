# Test-AzDataLakeStoreItem

## Sinossi
Verifica l'esistenza di un file o una cartella in data Lake Store.

## SINTASSI

```
Test-AzDataLakeStoreItem [-Account] <String> [-Path] <DataLakeStorePathInstance> [[-PathType] <PathType>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **test-AzDataLakeStoreItem** verifica l'esistenza di un file o di una cartella in data Lake Store.

## ESEMPI

### Esempio 1: testare un file
```powershell
PS C:\>Test-AzDataLakeStoreItem -AccountName "ContosoADL" -Path "/MyFiles/Test.csv"
```

Questo comando verifica se il file Test.csv esiste nell'account ContosoADL.

### Esempio 2

Verifica l'esistenza di un file o una cartella in data Lake Store. AutoGenerated
