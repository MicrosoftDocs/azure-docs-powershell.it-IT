# New-AzWebAppDatabaseBackupSetting

## Sinossi

## SINTASSI

```
New-AzWebAppDatabaseBackupSetting [-Name] <String> [-DatabaseType] <String> [-ConnectionString] <String>
 [[-ConnectionStringName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzWebAppDatabaseBackupSetting** crea una nuova impostazione di backup di Azure Web App.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzWebAppDatabaseBackupSetting -ResourceGroupName "Default-Web-WestUS" -Name "ContosoWebApp" -ConnectionString "MyConnectionString" -DatabaseType "SqlAzure"
```

Crea un'impostazione di backup del database (stringa di connessione) di tipo sqlazure per l'app specificata ContosoWebApp che si trova all'interno del gruppo di risorse predefinito-Web-Westus.

### Esempio 2

Il cmdlet New-AzWebAppDatabaseBackupSetting crea una nuova impostazione di backup di Azure Web App. AutoGenerated

```powershell <!-- Aladdin Generated Example --> 
New-AzWebAppDatabaseBackupSetting -ConnectionString 'MyConnectionString' -ConnectionStringName <String> -DatabaseType 'SqlAzure' -Name 'ContosoWebApp'
```
