# New-AzCosmosDBGremlinConflictResolutionPolicy

## Sinossi
Crea un nuovo oggetto di tipo PSConflictResolutionPolicy. Può essere passato come valore di parametro per set-AzCosmosDBGremlinGraph.

## SINTASSI

```
New-AzCosmosDBGremlinConflictResolutionPolicy -Type <String> [-Path <String>]
 [-ConflictResolutionProcedure <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Oggetto corrispondente alla ConflictResolutionPolicy dell'API Gremlin.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzCosmosDBGremlinConflictResolutionPolicy -Type LastWriterWins -Path "/myPath"

Mode           ConflictResolutionPath ConflictResolutionProcedure
----           ---------------------- ---------------------------
LastWriterWins /myPath
```
