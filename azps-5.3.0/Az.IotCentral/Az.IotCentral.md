# Modulo AZ. IotCentral
## Descrizione
Fornisce la gestione CRUD per le applicazioni centrali di un sacco.

## Cmdlet AZ. IotCentral
### [Get-AzIotCentralApp](Get-AzIotCentralApp.md)
Ottiene le proprietà per una o più applicazioni centrali di un sacco.

### [New-AzIotCentralApp](New-AzIotCentralApp.md)
Crea una nuova applicazione centrale di un sacco.

### [Remove-AzIotCentralApp](Remove-AzIotCentralApp.md)
Elimina un'applicazione centrale di un sacco.

### [Set-AzIotCentralApp](Set-AzIotCentralApp.md)
Aggiorna i metadati per un'applicazione centrale di un sacco.
