# Get-AzAutomationSchedule

## Sinossi
Ottiene una programmazione di automazione.

## SINTASSI

### ByAll (impostazione predefinita)
```
Get-AzAutomationSchedule [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByName
```
Get-AzAutomationSchedule [-Name] <String> [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzAutomationSchedule** ottiene una pianificazione di Azure Automation.

## ESEMPI

### Esempio 1: ottenere una pianificazione
```
PS C:\>Get-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "DailySchedule08" -ResourceGroupName "ResourceGroup01"
```

Questo comando consente di ottenere la pianificazione denominata DailySchedule08.
