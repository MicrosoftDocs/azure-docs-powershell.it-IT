# Set-AzAutomationSchedule

## Sinossi
Modifica una programmazione di automazione.

## SINTASSI

```
Set-AzAutomationSchedule [-Name] <String> [-IsEnabled <Boolean>] [-Description <String>]
 [-ResourceGroupName] <String> [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzAutomationSchedule** modifica una programmazione in Azure Automation.

## ESEMPI

### Esempio 1: modificare una programmazione
```
PS C:\>Set-AzAutomationSchedule -AutomationAccountName "Contoso17" -Name "Schedule01" -Description "Automation Schedule" -ResourceGroupName "ResourceGroup01"
```

Questo comando modifica la descrizione della programmazione denominata Schedule01.
