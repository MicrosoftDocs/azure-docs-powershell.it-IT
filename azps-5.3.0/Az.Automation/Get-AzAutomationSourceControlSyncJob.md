# Get-AzAutomationSourceControlSyncJob

## Sinossi
Ottiene i processi di sincronizzazione del controllo del codice sorgente di Azure Automation.

## SINTASSI

```
Get-AzAutomationSourceControlSyncJob -SourceControlName <String> [-JobId <Guid>] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzAutomationSourceControlSyncJob ottiene i processi di sincronizzazione del controllo del codice sorgente di Azure Automation. Per ottenere un processo di sincronizzazione del controllo del codice sorgente specifico, specificare il relativo ID.

## ESEMPI

### Esempio 1
Questo comando consente di ottenere tutti i processi di sincronizzazione del controllo del codice sorgente di automazione per il controllo VSTSNative.


```powershell
PS C:\> Get-AzAutomationSourceControlSyncJob -ResourceGroupName "rg1" `
                                                  -AutomationAccountName "devAccount" `
                                                  -Name "VSTSNative"

```
