# Connect-AzContainerRegistry

## Sinossi
Accedere a un registro di sistema di Azure container.

## SINTASSI

### WithoutNameAndPasswordParameterSet (impostazione predefinita)
```
Connect-AzContainerRegistry -Name <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### WithNameAndPasswordParameterSet
```
Connect-AzContainerRegistry -Name <String> -UserName <String> -Password <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Accedere a un registro di sistema di Azure container.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName
```

Accedere a ACR senza credenziali quando si accede giÃ  all'account Azure.

### Esempio 2
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $RegistryName -Password $AdminPassWord
```

Accedere a ACR con il nome utente e la password di amministratore quando l'utente ha abilitato l'amministratore.

### Esempio 3
```powershell
PS C:\> Connect-AzContainerRegistry -Name $RegistryName -UserName $ServicePrincipal -Password $ServicePrincipalPassword
```

Accedere a ACR con l'ID applicazione e la password di Service Principal.
