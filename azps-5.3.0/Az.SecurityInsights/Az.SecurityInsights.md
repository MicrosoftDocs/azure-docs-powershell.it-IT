# Modulo AZ. SecurityInsights
## Descrizione
Microsoft Azure Sentinel è una soluzione scalabile, cloud-native, Security Information Event Management (SIEM) e Security Orchestration Automatic Response (SOAR). Azure Sentinel offre un'analisi intelligente della sicurezza e un'intelligenza delle minacce nell'azienda, offrendo una soluzione unica per il rilevamento degli avvisi, la visibilità delle minacce, la ricerca proattiva e la risposta alle minacce.
