# Get-AzSentinelAlertRuleTemplate

## Sinossi
Ottenere il modello di regola analitica.

## SINTASSI

### WorkspaceScope (impostazione predefinita)
```
Get-AzSentinelAlertRuleTemplate -ResourceGroupName <String> -WorkspaceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### AlertRuleTemplateId
```
Get-AzSentinelAlertRuleTemplate -ResourceGroupName <String> -WorkspaceName <String>
 -AlertRuleTemplateId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSentinelAlertRuleTemplate -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSentinelAlertRuleTemplate** ottiene un modello di regola di avviso nell'area di lavoro specificata.
Se specifichi il parametro *AlertRuleTemplateId* , viene restituito un singolo oggetto **AlertRuleTemplate** .
Se non specifichi il parametro *AlertRuleTemplateId* , viene restituita una matrice contenente tutti i modelli di regola di avviso nell'area di lavoro specificata.
Puoi usare l'oggetto **AlertRuleTemplate** per creare una nuova regola di avviso.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $AlertRuleTemplates = Get-AzSentinelAlertRuleTemplate -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName"
```

Questo esempio recupera tutti i **AlertRuleTemplates** nell'area di lavoro specificata e lo archivia nella variabile $AlertRuleTemplates.

### Esempio 2
```powershell
PS C:\> $AlertRuleTemplate = Get-AzSentinelAlertRuleTemplate -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -AlertRuleTemplateId "MyAlertRuleTemplateId"
```

Questo esempio ottiene un **AlertRuleTemplate** nell'area di lavoro specificata e lo archivia nella variabile $AlertRuleTemplate.
