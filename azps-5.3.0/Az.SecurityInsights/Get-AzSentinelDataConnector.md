# Get-AzSentinelDataConnector

## Sinossi
Ottenere un connettore dati.

## SINTASSI

### WorkspaceScope (impostazione predefinita)
```
Get-AzSentinelDataConnector -ResourceGroupName <String> -WorkspaceName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### DataConnectorId
```
Get-AzSentinelDataConnector -ResourceGroupName <String> -WorkspaceName <String> -DataConnectorId <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzSentinelDataConnector -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSentinelDataConnector** ottiene un connettore di dati dall'area di lavoro specificata.
Se specifichi il parametro *DataConnectorId* , viene restituito un singolo oggetto **DataConnector** .
Se non specifichi il parametro *DataConnectorId* , viene restituita una matrice contenente tutti i connettori dati nell'area di lavoro specificata.
Puoi usare l'oggetto **DataConnector** per **aggiornare il connettore** di dati, ad esempio puoi disabilitare il DataConnector.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $DataConnectors = Get-AzSentinelDataConnector -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName"
```

Questo esempio recupera tutti i **DataConnector** nell'area di lavoro specificata e lo archivia nella variabile $DataConnectors.

### Esempio 2
```powershell
PS C:\> $DataConnector = Get-AzSentinelDataConnector -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -DataConnectorId "MyDataConnectorId"
```

Questo esempio ottiene un **DataConnector** nell'area di lavoro specificata e lo archivia nella variabile $DataConnector.
