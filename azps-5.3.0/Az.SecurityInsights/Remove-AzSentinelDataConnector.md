# Remove-AzSentinelDataConnector

## Sinossi
Rimuovere un connettore dati.

## SINTASSI

### DataConnectorId (impostazione predefinita)
```
Remove-AzSentinelDataConnector -ResourceGroupName <String> -WorkspaceName <String> -DataConnectorId <String>
 [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzSentinelDataConnector -InputObject <PSSentinelDataConnector> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzSentinelDataConnector** Elimina definitivamente un connettore di dati da un'area di lavoro specificata.
Puoi passare un oggetto **DataConnector** usando l'operatore pipeline o in alternativa puoi specificare i parametri obbligatori.
Puoi usare il parametro Confirm e $ConfirmPreference variabile di Windows PowerShell per controllare se il cmdlet richiede conferma.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzSentinelDataConnector -ResourceGroupName "MyResourceGroup" -WorkspaceName "MyWorkspaceName" -DataConnectorId "MyDataConnectorId"
```

Questo comando rimuove il DataConnector dall'area di lavoro.
