# Start-AzMigrateTestMigrationCleanup

## Sinossi
Pulisce la migrazione dei test per il server di replica.

## SINTASSI

### ByIDVMwareCbt (impostazione predefinita)
```
Start-AzMigrateTestMigrationCleanup -TargetObjectID <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### ByInputObjectVMwareCbt
```
Start-AzMigrateTestMigrationCleanup -InputObject <IMigrationItem> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Start-AzMigrateTestMigrationCleanup avvia la pulizia della migrazione di test per il server di replica.
