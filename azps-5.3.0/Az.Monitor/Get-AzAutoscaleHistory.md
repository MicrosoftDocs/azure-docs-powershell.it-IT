# Get-AzAutoscaleHistory

## Sinossi
Ottiene la cronologia di scalabilità verticale.

## SINTASSI

```
Get-AzAutoscaleHistory [-ResourceId <String>] [-StartTime <DateTime>] [-EndTime <DateTime>] [-Status <String>]
 [-Caller <String>] [-DetailedOutput] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzAutoscaleHistory** ottiene la cronologia degli eventi correlati a un'impostazione di scala.

## ESEMPI

### Esempio 1: ottenere tutti gli eventi associati a un abbonamento
```
PS C:\>Get-AzAutoscaleHistory -StartTime 2015-02-09T18:35:00 -EndTime 2015-02-09T18:40:00 -DetailedOutput
```

Questo comando ottiene tutti gli eventi correlati alla scalabilità associata alla sottoscrizione corrente.
