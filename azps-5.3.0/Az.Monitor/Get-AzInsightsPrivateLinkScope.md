# Get-AzInsightsPrivateLinkScope

## Sinossi
Ottenere l'ambito del collegamento privato

## SINTASSI

### ByResourceGroupParameterSet (impostazione predefinita)
```
Get-AzInsightsPrivateLinkScope [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByResourceNameParameterSet
```
Get-AzInsightsPrivateLinkScope -ResourceGroupName <String> -Name <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzInsightsPrivateLinkScope -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Elenco o ottenere un ambito di collegamento privato 

## ESEMPI

### Esempio 1
```powershell
Get-AzInsightsPrivateLinkScope -ResourceGroupName "rg_name"
```

Ambiti di collegamento elenco privato in gruppo risorse "rg_name"

### Esempio 2
```powershell
Get-AzInsightsPrivateLinkScope -ResourceGroupName "rg_name" -Name "scope_name"
```

Ottenere l'ambito dei collegamenti privati con il nome "scope_name" in gruppo risorse "rg_name"
