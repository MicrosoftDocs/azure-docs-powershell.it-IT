# Get-AzLogicAppRunHistory

## Sinossi
Ottiene la cronologia di esecuzione di un'app logica.

## SINTASSI

```
Get-AzLogicAppRunHistory -ResourceGroupName <String> -Name <String> [-RunName <String>] [-FollowNextPageLink]
 [-MaximumFollowNextPageLink <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzLogicAppRunHistory** ottiene la cronologia di esecuzione di un'app logica.
Questo cmdlet restituisce una raccolta di oggetti **WorkflowRun** .
Specifica il gruppo delle app e delle risorse logiche.
Questo modulo supporta i parametri dinamici.
Per usare un parametro dinamico, digitarlo nel comando.
Per individuare i nomi dei parametri dinamici, digitare un segno meno (-) dopo il nome del cmdlet e quindi premere ripetutamente TAB per spostarsi tra i parametri disponibili.
Se si omette un parametro di modello obbligatorio, il cmdlet richiede il valore.

### Esempio 3

```powershell
Get-AzLogicAppRunHistory -Name 'LogicApp03' -ResourceGroupName MyResourceGroup -FollowNextPageLink
```

Questo comando consente di ottenere l'intera cronologia di esecuzione di un'app logica denominata LogicApp03 seguendo NextPageLink.

### Esempio 4

```powershell
Get-AzLogicAppRunHistory -Name 'LogicApp03' -ResourceGroupName MyResourceGroup -FollowNextPageLink -MaximumFollowNextPageLink 1
```

Questo comando consente di ottenere le prime due pagine della cronologia di esecuzione di un'app logica denominata LogicApp03 seguendo NextPageLink e limitando le dimensioni dei risultati a due pagine.
Ogni pagina contiene trenta risultati.
