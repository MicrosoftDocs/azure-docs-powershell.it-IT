# Remove-AzDdosProtectionPlan

## Sinossi
Rimuove un piano di protezione DDoS.

## SINTASSI

```
Remove-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Remove-AzDdosProtectionPlan rimuove un piano di protezione DDoS.

## ESEMPI

### Esempio 1: rimuovere un piano di protezione DDoS vuoto
```
D:\> Remove-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

In questo caso, rimuoviamo un piano di protezione DDoS come specificato.

### Esempio 2: rimuovere un piano di protezione DDoS associato a una rete virtuale

I piani di protezione DDoS non possono essere eliminati se associati a una rete virtuale. Il primo passaggio consiste nel disassociare entrambi gli oggetti. Qui otteniamo la versione più aggiornata della rete virtuale associata al piano e impostiamo la proprietà **DdosProtectionPlan** su un valore vuoto e il contrassegno **EnableDdosProtection** (questo contrassegno non può essere vero senza un piano).
Quindi, persistiamo il nuovo stato eseguendo il piping della variabile locale in **set-AzVirtualNetwork**. A questo punto, il piano non è più associato alla rete virtuale.
Se questa è l'ultima associata al piano, è possibile rimuovere il piano di protezione DDoS usando il comando Rimuovi-AzDdosProtectionPlan.
