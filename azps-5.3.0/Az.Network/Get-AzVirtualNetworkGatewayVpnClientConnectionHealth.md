# Get-AzVirtualNetworkGatewayVpnClientConnectionHealth

## Sinossi 
Ottenere l'elenco di integrità della connessione client VPN di un gateway di rete virtuale di Azure per ogni connessione client VPN

## SINTASSI

```
Get-AzVirtualNetworkGatewayVpnClientConnectionHealth -ResourceName <String> -ResourceGroupName <String> -InputObject <PSVirtualNetworkGateway> -ResourceId <ResourceId>
 [-AsJob] [<CommonParameters>]
```

## Descrizione
Elenca tutti i servizi di integrità della connessione client VPN connessi. Questo include: vpnConnectionId vpnConnectionDuration vpnConnectionTime publicIpAddress privateIpAddress vpnUserName maxBandwidth egressPacketsTransferred egressBytesTransferred ingressPacketsTransferred ingressBytesTransferred maxPacketsPerSecond
