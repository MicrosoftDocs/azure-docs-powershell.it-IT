# Remove-AzApplicationGatewayTrustedClientCertificate

## Sinossi
Rimuove l'oggetto della catena di certificati CA client attendibile da un gateway dell'applicazione.

## SINTASSI

```
Remove-AzApplicationGatewayTrustedClientCertificate -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApplicationGatewayTrustedClientCertificate** rimuove l'oggetto della catena di certificati CA del client attendibile da un gateway dell'applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $gw = Remove-AzApplicationGatewayTrustedClientCertificate -ApplicationGateway $gw -Name "TrustedClientCertificate01"
```

Il primo comando ottiene un gateway dell'applicazione e lo archivia nella variabile $gw. Il secondo comando rimuove la catena di certificati CA client attendibile denominata "TrustedClientCertificate01" dal gateway dell'applicazione archiviato in $gw.
