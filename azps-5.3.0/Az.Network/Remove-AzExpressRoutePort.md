# Remove-AzExpressRoutePort

## Sinossi
Rimuove un ExpressRoutePort.

## SINTASSI

### ResourceNameParameterSet (impostazione predefinita)
```
Remove-AzExpressRoutePort -ResourceGroupName <String> -Name <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzExpressRoutePort -InputObject <PSExpressRoutePort> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Remove-AzExpressRoutePort -ResourceId <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzExpressRoutePort** rimuove un ExpressRoutePort.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $PortName -ResourceGroupName $rg
```

Rimuove $PortName risorsa ExpressRoutePort in $rg gruppo di risorse nell'abbonamento.

### Esempio 2
```powershell
PS C:\> Remove-AzExpressRoutePort -InputObject $erPort
```

Rimuove la risorsa ExpressRoutePort in InputObject.

### Esempio 3
```powershell
PS C:\> Remove-AzExpressRoutePort -Name $ResourceId $id
```

Rimuove la risorsa ExpressRoutePort con ResourceId $id.
