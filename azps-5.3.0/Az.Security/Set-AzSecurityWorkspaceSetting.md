# Set-AzSecurityWorkspaceSetting

## Sinossi
Aggiorna le impostazioni dell'area di lavoro per l'abbonamento.

## SINTASSI

```
Set-AzSecurityWorkspaceSetting -Name <String> -Scope <String> -WorkspaceId <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Aggiorna le impostazioni dell'area di lavoro per l'abbonamento.
L'area di lavoro configurata conterrà i dati di sicurezza raccolti dall'agente di analisi del log di Azure che è installato in VM all'interno di questo abbonamento.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Set-AzSecurityWorkspaceSetting -Name "default" -Scope "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869" -WorkspaceId  "/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/resourcegroups/mainws/providers/microsoft.operationalinsights/workspaces/securityuserws"

Id                                                                                                         Name    WorkspaceId 
--                                                                                                         ----    ----
/subscriptions/487bb485-b5b0-471e-9c0d-10717612f869/providers/Microsoft.Security/workspaceSettings/default default /...
```

Imposta l'area di lavoro "area di lavoro" per contenere tutti i dati di sicurezza raccolti dall'agente di analisi di Azure log.
