# Get-AzRegulatoryComplianceControl

## Sinossi
Ottiene i controlli di conformità normativi

## SINTASSI

### SubscriptionLevelResource (impostazione predefinita)
```
Get-AzRegulatoryComplianceControl [-Name <String>] -StandardName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceId
```
Get-AzRegulatoryComplianceControl -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Ottenere i dettagli del controllo spcific o elencare tutti i controlli in uno specifico standard di conformità normativo.
