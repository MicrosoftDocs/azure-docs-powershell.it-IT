# New-AzCdnProfile

## Sinossi
Crea un profilo CDN.

## SINTASSI

```
New-AzCdnProfile -ProfileName <String> -Location <String> -Sku <PSSkuName> -ResourceGroupName <String>
 [-Tag <Hashtable>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzCdnProfile** crea un profilo CDN (Azure Content delivering Network).

## ESEMPI

### Esempio 1

Crea un profilo CDN. AutoGenerated
