# Get-AzureRmServiceBusRule

## Sinossi
Ottiene una descrizione della regola specificata per una determinata sottoscrizione di argomenti. 

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmServiceBusRule [-ResourceGroupName] <String> [-Namespace] <String> [-Topic] <String>
 [-Subscription] <String> [[-Name] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmServiceBusRule** ottiene la descrizione della regola specificata nell'abbonamento specificato di topic.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzureRmServiceBusRule -ResourceGroup Default-ServiceBus-WestUS -Namespace SBExample1 -Topic SBTopic -Subscription SBSubscription -NAme SBRule
```

Restituisce la descrizione della regola specificata per un abbonamento specificato.
