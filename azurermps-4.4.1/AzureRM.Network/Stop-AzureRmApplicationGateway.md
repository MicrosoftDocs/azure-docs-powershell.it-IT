# Stop-AzureRmApplicationGateway

## Sinossi
Interrompe un gateway applicazione

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureRmApplicationGateway -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureRmApplicationGateway** arresta un gateway dell'applicazione.

## ESEMPI

### Esempio 1: arrestare un gateway applicazione
```
PS C:\>Stop-AzureRmApplicationGateway -ApplicationGateway $AppGw
```

Questo comando Arresta il gateway dell'applicazione archiviato nella variabile $AppGw.
