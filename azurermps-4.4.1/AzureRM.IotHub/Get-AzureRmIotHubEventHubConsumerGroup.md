# Get-AzureRmIotHubEventHubConsumerGroup

## Sinossi
Ottiene tutte le consumergroups eventhub.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmIotHubEventHubConsumerGroup [-ResourceGroupName] <String> [-Name] <String>
 [-EventHubEndpointName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Ottiene tutte le consumergroups eventhub per i diversi EventHubs usati da IotHub.

## ESEMPI

### L'esempio 1 restituisce tutte le consumergroups eventhub per la eventhub di telemetria
```
PS C:\> Get-AzureRmIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubEndpointName "events"
```

Ottiene tutte le consumergroups eventhub per il eventhub di telemetria per il iothub denominato myiothub

### L'esempio 2 recupera tutte le consumergroups eventhub per operationsmonitoring eventhub
```
PS C:\> Get-AzureRmIotHubEventHubConsumerGroup -ResourceGroupName "myresourcegroup" -Name "myiothub" -EventHubEndpointName "operationsMonitoringEvents"
```

Ottiene tutte le consumergroups eventhub per il eventhub operationsMonitoringEvents per il iothub denominato myiothub
