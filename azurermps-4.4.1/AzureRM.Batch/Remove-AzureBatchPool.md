# Remove-AzureBatchPool

## Sinossi
Elimina il pool di batch specificato.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureBatchPool [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureBatchPool** Elimina il pool di batch di Azure specificato.
Viene richiesto di confermare a meno che non si usi il parametro *Force* .

## ESEMPI

### Esempio 1: eliminare un pool di batch per ID pool
```
PS C:\>Remove-AzureBatchPool -Id "MyPool" -BatchContext $Context
```

Questo comando Elimina il pool con ID pool.
All'utente viene richiesta la conferma prima che venga applicata l'operazione di eliminazione.

### Esempio 2: eliminare tutti i pool di batch per forza
```
PS C:\>Get-AzureBatchPool -BatchContext $Context | Remove-AzureBatchPool -Force -BatchContext $Context
```

Questo comando Elimina tutti i pool di batch.
Dato che il parametro *Force* è presente, il messaggio di conferma viene soppresso.
