# Stop-AzureBatchJob

## Sinossi
Interrompe un processo batch.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureBatchJob [-Id] <String> [[-TerminateReason] <String>] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureBatchJob** interrompe un processo batch di Azure.
Questo comando contrassegna il processo come completato.

## ESEMPI

### Esempio 1: interrompere un processo batch
```
PS C:\>Stop-AzureBatchJob -Id "Job-000001" -TerminateReason "No more tasks to run" -BatchContext $Context
```

Questo comando Arresta il processo con ID Job-000001.
Il comando specifica un motivo per cui si è scelto di interrompere il processo.
Usa il cmdlet Get-AzureRmBatchAccountKeys per assegnare un contesto alla variabile $Context.
