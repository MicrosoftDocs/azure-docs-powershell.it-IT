# Disable-AzureBatchJobSchedule

## Sinossi
Disabilita la pianificazione di un processo batch.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Disable-AzureBatchJobSchedule [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Disable-AzureBatchJobSchedule Disabilita** una pianificazione dei processi batch di Azure.
Se si disattiva una programmazione, i processi non vengono creati in base a tale programmazione.
Ãˆ possibile abilitare una pianificazione disabilitata in un secondo momento.

## ESEMPI

### Esempio 1: disabilitare una pianificazione del processo
```
PS C:\>Disable-AzureBatchJobSchedule -Id "JobSchedule17" -BatchContext $Context
```

Questo comando Disabilita la pianificazione del processo con ID JobSchedule17.
Usa il cmdlet **Get-AzureRmBatchAccountKeys** per assegnare un contesto alla variabile $context.
