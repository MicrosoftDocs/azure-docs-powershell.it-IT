# Remove-AzureBatchComputeNodeUser

## Sinossi
Elimina un account utente da un nodo di calcolo batch.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureBatchComputeNodeUser [-PoolId] <String> [-ComputeNodeId] <String> [-Name] <String>
 -BatchContext <BatchAccountContext> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureBatchComputeNodeUser** Elimina un account utente da un nodo di calcolo batch di Azure.

## ESEMPI

### Esempio 1: eliminare un utente da un nodo Compute senza conferma
```
PS C:\>Remove-AzureBatchComputeNodeUser -PoolId "Pool01" -ComputeNodeId "ComputeNode01" -Name "User14" -Force -BatchContext $Context
```

Questo comando Elimina l'utente denominato User14 dal nodo Compute denominato ComputeNode01.
Il nodo Compute si trova nel pool denominato pool01.
Questo comando specifica il parametro *Force* .
Di conseguenza, il comando non chiede conferma.
