# Stop-AzureBatchJobSchedule

## Sinossi
Interrompe la pianificazione di un processo batch.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureBatchJobSchedule [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureBatchJobSchedule** interrompe la pianificazione di un processo batch di Azure.

## ESEMPI

### Esempio 1: interrompere la pianificazione di un processo
```
PS C:\>Stop-AzureBatchJobSchedule -Id "JobSchedule17" -BatchContext $Context
```

Questo comando interrompe la pianificazione del processo che contiene l'ID JobSchedule17.
Usa il cmdlet Get-AzureRmBatchAccountKeys per assegnare un contesto alla variabile $Context.
