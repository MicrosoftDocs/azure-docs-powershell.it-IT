# Get-AzureRmBatchAccountKeys

## Sinossi
Ottiene le chiavi di un account batch.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmBatchAccountKeys [-AccountName] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmBatchAccountKeys** ottiene le chiavi di un account batch di Azure nell'abbonamento corrente.
