# Add-AzureRmDataLakeAnalyticsDataSource

## Sinossi
Aggiunge un'origine dati a un account di analisi dei dati sul lago.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

### Aggiungere un account di archiviazione dati per i laghi
```
Add-AzureRmDataLakeAnalyticsDataSource [-Account] <String> [-DataLakeStore] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### Aggiungere un account di archiviazione BLOB
```
Add-AzureRmDataLakeAnalyticsDataSource [-Account] <String> [-Blob] <String> [-AccessKey] <String>
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzureRmDataLakeAnalyticsDataSource** aggiunge un'origine dati a un account di analisi di Azure Data Lake.

## ESEMPI

### Esempio 1: aggiungere un'origine dati a un account
```
PS C:\>Add-AzureRmDataLakeAnalyticsDataSource -Account "ContosoAdlA" -DataLakeStore "ContosoAdlS"
```

Questo comando aggiunge un'origine dati di data Lake Store a un account di analisi dei dati sul lago.
