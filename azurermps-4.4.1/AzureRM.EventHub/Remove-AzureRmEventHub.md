# Remove-AzureRmEventHub

## Sinossi
Rimuove l'hub eventi specificato.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmEventHub [-ResourceGroupName] <String> -Namespace <String> -Name <String> [-WhatIf] [-Confirm]
```

## Descrizione
Il cmdlet Remove-AzureRmEventHub rimuove ed Elimina il hub dell'evento specificato dallo spazio dei nomi specifico.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzureRmEventHub -ResourceGroupName MyResourceGroupName -Namespace MyNamespaceName -Name MyEventHubName
```

Rimuove l'hub dell'evento \` MyEventHubName \` .
