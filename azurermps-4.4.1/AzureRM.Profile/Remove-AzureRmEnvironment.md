# Remove-AzureRmEnvironment

## Sinossi
Rimuove gli endpoint e i metadati per la connessione a una determinata istanza di Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmEnvironment [-Name] <String> [-Scope <ContextModificationScope>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Remove-AzureRmEnvironment rimuove gli endpoint e le informazioni sui metadati per la connessione a una determinata istanza di Azure.
