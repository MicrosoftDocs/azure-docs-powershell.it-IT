# Disable-AzureRmBackupProtection

## Sinossi
Disabilita la protezione per un elemento protetto di backup.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Disable-AzureRmBackupProtection [-RemoveRecoveryPoints] [-Force] [-Item] <AzureRMBackupItem>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Disable-AzureRmBackupProtection Disabilita** la protezione per un elemento protetto di Azure Backup.
Questo cmdlet interrompe il backup pianificato regolare di un elemento.
Questo cmdlet pu√≤ eliminare i punti di ripristino esistenti per l'elemento di backup.
