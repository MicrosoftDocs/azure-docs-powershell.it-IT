# New-AzureRmSiteRecoveryVault

## Sinossi
Crea un Vault di servizi di ripristino del sito.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmSiteRecoveryVault -Name <String> -ResourceGroupName <String> -Location <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzureRmSiteRecoveryVault** crea un Vault di servizi di ripristino di Azure site.
