# Get-AzureRmSiteRecoveryVaultSettings

## Sinossi
Ottiene le informazioni sulle impostazioni per il Vault di ripristino del sito corrente.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSiteRecoveryVaultSettings [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSiteRecoveryVaultSettings** ottiene le informazioni sulle impostazioni correlate all'attuale archivio di ripristino del sito Azure.
