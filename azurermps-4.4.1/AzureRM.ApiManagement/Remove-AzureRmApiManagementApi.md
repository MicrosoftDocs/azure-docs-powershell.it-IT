# Remove-AzureRmApiManagementApi

## Sinossi
Rimuove un'API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmApiManagementApi -Context <PsApiManagementContext> -ApiId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmApiManagementApi** rimuove un'API esistente.

## ESEMPI

### Esempio 1: rimuovere un'API
```
PS C:\>Remove-AzureRmApiManagementApi -Context $ApiMgmtContext -ApiId "0123456789"
```

Questo comando rimuove l'API con l'ID specificato.
