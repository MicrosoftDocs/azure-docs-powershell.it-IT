# Remove-AzureRmApiManagementOpenIdConnectProvider

## Sinossi
Rimuove un provider OpenID Connect.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmApiManagementOpenIdConnectProvider -Context <PsApiManagementContext>
 -OpenIdConnectProviderId <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmApiManagementOpenIdConnectProvider** rimuove un provider OpenID Connect per la gestione delle API di Azure.

## ESEMPI

### Esempio 1: rimuovere un provider
```
PS C:\>Remove-AzureRmApiManagementOpenIdConnectProvider -Context $ApimContext -OpenIdConnectProviderId "OICProvicer01" -PassThru
```

Questo comando rimuove un provider con ID OICProvicer01.
