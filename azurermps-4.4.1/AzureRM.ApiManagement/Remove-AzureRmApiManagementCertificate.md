# Remove-AzureRmApiManagementCertificate

## Sinossi
Rimuove un certificato di gestione delle API.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmApiManagementCertificate -Context <PsApiManagementContext> -CertificateId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmApiManagementCertificate** rimuove un certificato di gestione delle API di Azure.

## ESEMPI

### Esempio 1: rimuovere un certificato
```
PS C:\>Remove-AzureRmApiManagementCertificate -Context $ApiMgmtContext -CertificateId "0123456789" -Force
```

Questo comando rimuove il certificato di gestione API specificato.
Dato che il parametro *Force* è specificato, non è richiesta alcuna conferma.
