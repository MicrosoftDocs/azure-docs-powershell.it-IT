# Stop-AzureRmHDInsightJob

## Sinossi
Interrompe un processo in uso specificato in un cluster.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Stop-AzureRmHDInsightJob [-ClusterName] <String> [-JobId] <String> [-HttpCredential] <PSCredential>
 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Stop-AzureRmHDInsightJob** interrompe un processo in uso specificato in un cluster di Azure HDInsight.

## ESEMPI

### Esempio 1: arrestare un processo nel cluster specificato
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential
PS C:\> Stop-AzureRmHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds `
            -JobId $jobId
```

Questo comando Arresta un processo nel cluster denominato your-Hadoop-001.
