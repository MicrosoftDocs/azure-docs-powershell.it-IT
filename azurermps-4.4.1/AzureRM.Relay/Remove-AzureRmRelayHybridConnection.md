# Remove-AzureRmRelayHybridConnection

## Sinossi
Rimuove HybridConnection dallo spazio dei nomi HybridConnection specificato.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmRelayHybridConnection [-ResourceGroupName] <String> [-Namespace] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmRelayHybridConnection** rimuove HybridConnection dallo spazio dei nomi Relay specificato.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzureRmRelayHybridConnection -ResourceGroupName Default-ServiceBus-WestUS -Namespace TestNameSpace-Relay1 -Name TestHybridConnection
```

Rimuove HybridConnection `TestHybridConnection` dallo spazio dei nomi `TestNameSpace-Relay1` .
