# Test-AzureRmAnalysisServicesServer

## Sinossi
Verifica l'esistenza di un'istanza di Analysis Services server

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Test-AzureRmAnalysisServicesServer [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Test-AzureRmAnalysisServicesServer verifica l'esistenza di un'istanza di Analysis Services server

## ESEMPI

### Esempio 1
```
PS C:\> Test-AzureRmAnalysisServicesServer -Name "testserver" -ResourceGroupName "testgroup"
```

Questo comando tester√† se esiste un server denominato TestServer nel ResourceGroup TestGroup
