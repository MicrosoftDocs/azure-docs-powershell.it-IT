# New-AzureRmAnalysisServicesServer

## Sinossi
Crea un nuovo server di Analysis Services

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
New-AzureRmAnalysisServicesServer [-ResourceGroupName] <String> [-Name] <String> [-Location] <String>
 [-Sku] <String> [[-Tag] <Hashtable>] [[-Administrator] <String>] [[-BackupBlobContainerUri] <String>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet New-AzureRmAnalysisServicesServer crea un nuovo server di Analysis Services

## ESEMPI

### Esempio 1
```
PS C:\> New-AzureRmAnalysisServicesServer -ResourceGroupName "testresourcegroup" -Name "testserver" -Location "West-US" -Sku "S1"
```

Crea un server denominato TestServer in Azure Region West-US e nel gruppo di risorse testresrourcegroup. Il livello SKU per il server sarà S1.
