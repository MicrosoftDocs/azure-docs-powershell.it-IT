# Remove-AzureRmStreamAnalyticsInput

## Sinossi
Elimina un input da un processo di analisi del flusso.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmStreamAnalyticsInput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmStreamAnalyticsInput** Elimina in modo asincrono un input da un processo di analisi del flusso in Azure.

## ESEMPI

### ESEMPIO 1: rimuovere un flusso di input da un processo
```
PS C:\>Remove-AzureRmStreamAnalyticsInput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "EventStream"
```

Questo comando rimuove l'input EventStream da StreamingJob.
