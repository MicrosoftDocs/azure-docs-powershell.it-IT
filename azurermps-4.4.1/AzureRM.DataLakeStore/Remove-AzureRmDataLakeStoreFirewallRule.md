# Remove-AzureRmDataLakeStoreFirewallRule

## Sinossi
Rimuove la regola del firewall specificata nello Store Data Lake specificato.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmDataLakeStoreFirewallRule [-Account] <String> [[-Name] <String>] [-PassThru]
 [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzureRmDataLakeStoreFirewallRule** rimuove la regola del firewall specificata nello Store Data Lake.

## ESEMPI

### Esempio 1: rimuovere una regola del firewall da un account
```
PS C:\> Remove-AzureRmDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyFirewallRule
```

Rimuove la regola del firewall "MyFirewallRule" dall'account "ContosoADL"
