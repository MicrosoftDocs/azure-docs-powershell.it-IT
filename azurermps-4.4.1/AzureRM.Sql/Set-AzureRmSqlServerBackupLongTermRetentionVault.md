# Set-AzureRmSqlServerBackupLongTermRetentionVault

## Sinossi
Imposta un Vault di conservazione a lungo termine del server.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Set-AzureRmSqlServerBackupLongTermRetentionVault -ResourceId <String> [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzureRmSqlServerBackupLongTermRetentionVault** imposta il Vault di conservazione a lungo termine registrato nel server.
Il Vault Ã¨ una risorsa di backup di Azure usata per archiviare i dati di backup.
