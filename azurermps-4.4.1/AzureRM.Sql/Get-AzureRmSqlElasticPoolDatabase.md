# Get-AzureRmSqlElasticPoolDatabase

## Sinossi
Ottiene i database elastici in un pool elastico e i relativi valori di proprietà.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlElasticPoolDatabase** ottiene i database elastici in un pool elastico e i relativi valori di proprietà.
Puoi specificare il nome di un database elastico in Azure SQL database per visualizzare i valori delle proprietà solo per il database.

## ESEMPI

### Esempio 1: ottenere tutti i database in un pool elastico
```
PS C:\>Get-AzureRmSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Questo comando consente di ottenere tutti i database in un pool elastico denominato ElasticPool01.
