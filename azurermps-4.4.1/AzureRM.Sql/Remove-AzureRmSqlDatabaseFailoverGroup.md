# Remove-AzureRmSqlDatabaseFailoverGroup

## Sinossi
Rimuove un gruppo di failover di database SQL di Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Remove-AzureRmSqlDatabaseFailoverGroup [-ServerName] <String> [-FailoverGroupName] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Questo comando rimuove il gruppo di failover con il nome specificato, lasciando intatti tutti i database e le relazioni di replica. L'endpoint del listener verrÃ  annullata la registrazione dal DNS.

Il server principale del gruppo di failover deve essere usato per eseguire il comando.

## ESEMPI

### Esempio 1
```
PS C:\> Remove-AzureRmSqlDatabaseFailoverGroup -ResourceGroupName rg -ServerName primaryserver -FailoverGroupName fg
```

Rimuovere un gruppo di failover.
