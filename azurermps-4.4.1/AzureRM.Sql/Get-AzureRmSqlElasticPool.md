# Get-AzureRmSqlElasticPool

## Sinossi
Ottiene i pool elastici e i relativi valori di proprietà in un database SQL di Azure.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSqlElasticPool [[-ElasticPoolName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlElasticPool** ottiene i pool elastici e i relativi valori di proprietà.
Specificare il nome di un pool elastico esistente per visualizzare i valori delle proprietà solo per il pool.
