# Get-AzureRmSqlSyncAgent

## Sinossi
Restituisce informazioni sugli agenti di sincronizzazione di Azure SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlSyncAgent** restituisce informazioni su uno o piÃ¹ agenti di sincronizzazione SQL di Azure.
Specificare il nome di un agente di sincronizzazione per visualizzare le informazioni solo per l'agente di sincronizzazione.
