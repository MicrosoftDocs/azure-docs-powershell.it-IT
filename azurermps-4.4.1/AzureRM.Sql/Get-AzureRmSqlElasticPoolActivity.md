# Get-AzureRmSqlElasticPoolActivity

## Sinossi
Ottiene lo stato delle operazioni in un pool elastico.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Get-AzureRmSqlElasticPoolActivity [-ServerName] <String> [-ElasticPoolName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzureRmSqlElasticPoolActivity** ottiene lo stato delle operazioni in un pool elastico per un database SQL di Azure.
Puoi vedere lo stato della creazione del pool e degli aggiornamenti della configurazione.

## ESEMPI

### Esempio 1: ottenere lo stato delle operazioni per un pool elastico
```
PS C:\>Get-AzureRmSqlElasticPoolActivity -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

Questo comando consente di ottenere lo stato delle operazioni per il pool elastico denominato ElasticPool01.
