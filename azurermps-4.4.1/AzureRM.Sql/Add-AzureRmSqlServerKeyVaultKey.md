# Add-AzureRmSqlServerKeyVaultKey

## Sinossi
Aggiunge una chiave di volta chiave a un server SQL.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Add-AzureRmSqlServerKeyVaultKey [-KeyId] <String> [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Add-AzureRmSqlServerKeyVaultKey aggiunge una chiave di volta chiave al server SQL fornito.
Il server deve avere le autorizzazioni ' Get, wrapKey, unwrapKey ' per il Vault.

## ESEMPI

### --------------------------Esempio 1: aggiungere chiave di volta chiave--------------------------
```
PS C:\> Add-AzureRmSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

Questo comando aggiunge la chiave di volta chiave con ID ' https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 ' a SQL Server denominato "ContosoServer" nel gruppo di risorse "ContosoResourceGroup".
