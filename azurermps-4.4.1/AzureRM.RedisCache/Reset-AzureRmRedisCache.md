# Reset-AzureRmRedisCache

## Sinossi
Riavvia i nodi di una cache.

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## SINTASSI

```
Reset-AzureRmRedisCache -ResourceGroupName <String> -Name <String> -RebootType <String> [-ShardId <Int32>]
 [-Force] [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Reset-AzureRmRedisCache** riavvia i nodi di un'istanza della cache di Azure Redis.

## ESEMPI

### Esempio 1: riavviare entrambi i nodi
```
PS C:\>Reset-AzureRmRedisCache -ResourceGroupName "ResourceGroup13" -Name "RedisCache06" -RebootType "AllNodes" -Force
```

Questo comando Riavvia entrambi i nodi per la cache denominata RedisCache06.
