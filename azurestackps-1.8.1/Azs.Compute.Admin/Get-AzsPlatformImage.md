# Get-AzsPlatformImage

## Sinossi
Restituisce le immagini della macchina virtuale caricate nel repository di immagini della piattaforma.

## SINTASSI

### Elenco (impostazione predefinita)
```
Get-AzsPlatformImage [-Publisher <String>] [-Offer <String>] [-Sku <String>] [-Location <String>]
 [<CommonParameters>]
```

### Ottieni
```
Get-AzsPlatformImage -Publisher <String> -Offer <String> -Sku <String> -Version <String> [-Location <String>]
 [<CommonParameters>]
```

### ResourceId
```
Get-AzsPlatformImage -ResourceId <String> [<CommonParameters>]
```

## Descrizione
Restituisce immagini della piattaforma.

## ESEMPI

### --------------------------ESEMPIO 1--------------------------
```
Get-AzsPlatformImage
```

Restituisce le immagini della macchina virtuale caricate nel repository di immagini della piattaforma nella posizione locale.

### --------------------------ESEMPIO 2--------------------------
```
Get-AzsPlatformImage -Publisher Canonical -Offer UbuntuServer -Sku 16.04-LTS -Version 0.1.0
```

Ottenere una specifica immagine della piattaforma.
