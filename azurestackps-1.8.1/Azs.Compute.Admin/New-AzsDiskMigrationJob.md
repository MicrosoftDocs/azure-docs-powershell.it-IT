# New-AzsDiskMigrationJob

## Sinossi
Avvia un processo di migrazione del disco gestito per eseguire la migrazione dei dischi gestiti alla condivisione di destinazione specificata.

## SINTASSI

```
New-AzsDiskMigrationJob [-Disks] <Disk[]> [-TargetShare] <String> [[-Location] <String>] [-Name] <String>
 [<CommonParameters>]
```

## Descrizione
Creare un processo di migrazione disco.

## ESEMPI

### --------------------------ESEMPIO 1--------------------------
```
New-AzsDiskMigrationJob -Name "MyMigrationJob" -Disks $disks -location local -TargetShare "\\SU1FileServer.azurestack.local\SU1_ObjStore"
```

Avviare un processo di migrazione su disco gestito per i primi 20 dischi
