# Remove-AzApiManagementProperty

## Sinossi
Rimuove una proprietà di gestione API.

## SINTASSI

```
Remove-AzApiManagementProperty -Context <PsApiManagementContext> -PropertyId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApiManagementProperty** rimuove una **Proprietà** di gestione delle API di Azure.

## ESEMPI

### Esempio 1: rimuovere una proprietà
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementProperty -Context $apimContext -PropertyId "Property11" -PassThru
```

Questo comando rimuove la proprietà con ID Property11.
