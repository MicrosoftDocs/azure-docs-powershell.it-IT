# Remove-AzApiManagementLogger

## Sinossi
Rimuove un logger di gestione API.

## SINTASSI

```
Remove-AzApiManagementLogger -Context <PsApiManagementContext> -LoggerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApiManagementLogger** rimuove un **logger** di gestione delle API di Azure.

## ESEMPI

### Esempio 1: rimuovere un logger
```
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementLogger -Context $apimContext -LoggerId "Logger123" -Force
```

Questo comando rimuove un logger con ID Logger123.
