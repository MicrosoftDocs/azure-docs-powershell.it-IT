# Remove-AzStreamAnalyticsFunction

## Sinossi
Elimina una funzione da un processo di analisi del flusso.

## SINTASSI

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzStreamAnalyticsFunction** Elimina una funzione in modo asincrono da un processo di analisi di Azure Stream.

## ESEMPI

### Esempio 1: rimuovere una funzione di analisi del flusso
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

Questo comando rimuove la funzione denominata ScoreTweet dal processo denominato StreamJob22.
