# Get-AzSqlDatabaseAudit

## Sinossi
Ottiene le impostazioni di controllo di un database SQL di Azure.

## SINTASSI

### DatabaseParameterSet (impostazione predefinita)
```
Get-AzSqlDatabaseAudit [-ResourceGroupName] <String> [-ServerName] <String> [-DatabaseName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### DatabaseObjectParameterSet
```
Get-AzSqlDatabaseAudit -DatabaseObject <AzureSqlDatabaseModel> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlDatabaseAudit** ottiene le impostazioni di controllo di un database SQL di Azure.
Per usare il cmdlet, usare i parametri *ResourceGroupName* , *ServerName* e *DatabaseName* per identificare il database.
