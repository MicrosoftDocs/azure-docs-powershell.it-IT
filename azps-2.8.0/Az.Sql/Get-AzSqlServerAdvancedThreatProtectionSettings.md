# Get-AzSqlServerAdvancedThreatProtectionSettings

## Sinossi
Ottiene le impostazioni avanzate per la protezione delle minacce per un server.

## SINTASSI

```
Get-AzSqlServerAdvancedThreatProtectionSettings -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlServerAdvancedThreatProtectionSettings** ottiene le impostazioni avanzate per la protezione delle minacce di un server SQL di Azure.
Per usare questo cmdlet, specificare i parametri *ResourceGroupName* e *ServerName* per identificare il server per cui questo cmdlet ottiene le impostazioni.

## ESEMPI

### Esempio 1: ottenere le impostazioni avanzate di protezione dalle minacce per un server
```
PS C:\>Get-AzSqlServerAdvancedThreatProtectionSettings -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Questo comando consente di ottenere le impostazioni avanzate di protezione dalle minacce per un server denominato Server01.
Il server viene assegnato al gruppo di risorse ResourceGroup11.
