# Get-AzSqlElasticJobTargetGroup

## Sinossi
Ottiene uno o pi√π gruppi di destinazione del processo

## SINTASSI

### DefaultSet (impostazione predefinita)
```

 [-Name <String>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ObjectSet
```

```

### ResourceIdSet

## Descrizione
Il cmdlet Get-AzSqlElasticJobTargetGroup ottiene un gruppo di destinazione e le destinazioni

## ESEMPI

### Esempio 1
```
PS C:\> $agent = Get-AzSqlElasticJobAgent -ResourceGroupName rg -ServerName elasticjobserver -Name agent
$agent | Get-AzSqlElasticJobTargetGroup -Name tg1

TargetGroupName Targets
--------------- -------
tg1             (s1,db1)
```

Ottiene un gruppo di destinazione e le destinazioni
