# Remove-AzApplicationGatewayIPConfiguration

## Sinossi
Rimuove una configurazione IP da un gateway dell'applicazione.

## SINTASSI

```
Remove-AzApplicationGatewayIPConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApplicationGatewayIPConfiguration** consente di rimuovere una configurazione IP da un gateway dell'applicazione Azure.

## ESEMPI

### Esempio 1: rimuovere una configurazione IP da un gateway dell'applicazione Azure
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayIPConfiguration -ApplicationGateway $AppGw -Name "Subnet02"
```

Il primo comando ottiene un gateway dell'applicazione e lo archivia nella variabile $AppGw.
Il secondo comando rimuove la configurazione IP denominata Subnet02 dal gateway dell'applicazione archiviato in $AppGw.
