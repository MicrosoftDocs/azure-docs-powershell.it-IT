# Remove-AzApplicationGatewayRewriteRuleSet

## Sinossi
Rimuove un set di regole di riscrittura da un gateway dell'applicazione.

## SINTASSI

```
Remove-AzApplicationGatewayRewriteRuleSet -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApplicationGatewayRewriteRuleSet** rimuove un set di regole di riscrittura da un gateway dell'applicazione Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayRewriteRuleSet -ApplicationGateway $AppGw -Name "RuleSet02"
```

Il primo comando ottiene un gateway dell'applicazione e lo archivia nella variabile $AppGw.
Il secondo comando rimuove il set di regole di riscrittura denominato RuleSet02 dal gateway dell'applicazione archiviato in $AppGw.
