# New-AzApplicationGatewayAuthenticationCertificate

## Sinossi
Crea un certificato di autenticazione per un gateway dell'applicazione.

## SINTASSI

```
New-AzApplicationGatewayAuthenticationCertificate -Name <String> -CertificateFile <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzApplicationGatewayAuthenticationCertificate** crea un certificato di autenticazione per un gateway dell'applicazione Azure.

## ESEMPI

### Esempio 1: creare un certificato di autenticazione
```
PS C:\> $cert = New-AzApplicationGatewayAuthenticationCertificate -Name "cert01" -CertificateFile "C:\cert.cer"
```

Il primo comando crea il certificato di autenticazione denominato cert01.
