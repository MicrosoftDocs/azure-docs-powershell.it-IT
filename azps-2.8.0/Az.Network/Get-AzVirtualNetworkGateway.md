# Get-AzVirtualNetworkGateway

## Sinossi
Ottiene un gateway di rete virtuale

## SINTASSI

```
Get-AzVirtualNetworkGateway [-Name <String>] -ResourceGroupName <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il gateway di rete virtuale Ã¨ l'oggetto che rappresenta il gateway in Azure.
Il cmdlet **Get-AzVirtualNetworkGateway** restituisce l'oggetto del gateway in Azure in base al nome e al nome del gruppo di risorse.

## ESEMPI

### 1: ottenere un gateway di rete virtuale
```
Get-AzVirtualNetworkGateway -Name myGateway1 -ResourceGroupName myRG
```

Restituisce l'oggetto del gateway di rete virtuale con il nome "myGateway1" nel gruppo di risorse "myRG"

### 2: ottenere un gateway di rete virtuale
```
Get-AzVirtualNetworkGateway -Name myGateway* -ResourceGroupName myRG
```

Restituisce tutti i gateway di rete virtuali che iniziano con "gateway" nel gruppo di risorse "myRG"
