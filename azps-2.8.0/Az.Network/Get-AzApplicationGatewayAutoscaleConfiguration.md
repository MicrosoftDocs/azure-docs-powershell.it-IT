# Get-AzApplicationGatewayAutoscaleConfiguration

## Sinossi
Ottiene la configurazione di scala automatica del gateway dell'applicazione.

## SINTASSI

```
Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzApplicationGatewayAutoscaleConfiguration** ottiene la configurazione di scala automatica del gateway dell'applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $gw = Get-AzApplicationGateway -Name $appgwName -ResourceGroupName $resgpName
PS C:\> $autoscaleConfiguration = Get-AzApplicationGatewayAutoscaleConfiguration -ApplicationGateway $gw
PS C:\> $autoscaleConfiguration.MinCapacity
```

Il primo comando ottiene il gateway dell'applicazione e lo archivia in $gw variabile.
Il secondo comando estrae la configurazione di scala automatica dal gateway dell'applicazione.
