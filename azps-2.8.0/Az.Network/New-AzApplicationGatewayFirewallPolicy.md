# New-AzApplicationGatewayFirewallPolicy

## Sinossi
Crea un criterio firewall di Application Gateway.

## SINTASSI

```
New-AzApplicationGatewayFirewallPolicy -Name <String> -ResourceGroupName <String> -Location <String>
 [-CustomRule <PSApplicationGatewayFirewallCustomRule[]>] [-Tag <Hashtable>] [-Force] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzApplicationGatewayFirewallPolicy** crea un criterio del firewall di gateway applicazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> $firewallPolicy = New-AzApplicationGatewayFirewallPolicy -Name wafResource1 -ResourceGroupName "rg1"  -Location  "westus" -CustomRules $customRule
```

Questo comando crea un nuovo criterio firewall di gateway di applicazioni di Azure denominato "wafResource1" nel gruppo di risorse "RG1" in location "westus" con le regole personalizzate definite nella variabile $customRule
