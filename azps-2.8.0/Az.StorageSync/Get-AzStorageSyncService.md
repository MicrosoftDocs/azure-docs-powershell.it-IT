# Get-AzStorageSyncService

## Sinossi
Questo comando elenca tutti i servizi di sincronizzazione dello storage all'interno di un ambito specifico del gruppo sottoscrizione/risorsa.

## SINTASSI

### ParentStringParameterSet (impostazione predefinita)
```
Get-AzStorageSyncService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### StringParameterSet
```
Get-AzStorageSyncService [-ResourceGroupName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Questo comando elenca tutti i servizi di sincronizzazione dello storage all'interno di un ambito specifico del gruppo sottoscrizione/risorsa. Può essere usato anche per elencare gli attributi di ogni servizio di sincronizzazione dello spazio di archiviazione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzStorageSyncService -ResourceGroupName "myResourceGroup"
```

Questo comando elenca tutte le risorse del servizio di sincronizzazione dello storage in un gruppo di risorse specifico. Può essere usato anche per elencare gli attributi di ogni servizio di sincronizzazione dello spazio di archiviazione. Specifica-StorageSyncServiceName per restituirne uno specifico.
