# Remove-AzAttestation

## Sinossi
Elimina un'attestazione.

## SINTASSI

### ByAvailableAttestation (impostazione predefinita)
```
Remove-AzAttestation [-Name] <String> [-ResourceGroupName] <String> [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ResourceIdByAvailableAttestation
```
Remove-AzAttestation [-ResourceId] <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectByAvailableAttestation
```
Remove-AzAttestation [-InputObject] <PSAttestation> [-PassThru] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet Remove-AzAttestation Elimina l'attestazione specificata.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzAttestation -Name example -ResourceGroupName rg1 
```

Eliminare l'attestazione "esempio" dall'abbonamento corrente e dal gruppo di risorse "RG1".
