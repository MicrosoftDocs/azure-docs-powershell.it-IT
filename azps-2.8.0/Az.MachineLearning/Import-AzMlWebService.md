# Import-AzMlWebService

## Sinossi
Importa un oggetto JSON in una definizione di servizio Web.

## SINTASSI

### ImportFromJSONFile
```
Import-AzMlWebService -InputFile <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ImportFromJSONString.
```
Import-AzMlWebService -JsonString <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Import-AzMlWebService importa, specificato direttamente o in un file a cui si fa riferimento, e crea un oggetto definizione di servizio Web che pu√≤ essere passato al cmdlet New-AzMlWebService.

## ESEMPI

### Esempio 1: importare da una stringa
```
Import-AzMlWebService -JsonString $jsonDefinition
```

### Esempio 2: importare da un percorso di file
```
Import-AzMlWebService -InputFile "C:\mlservice.json"
```
