# Uninstall-AzureRm

## Sinossi
Rimuove tutti i moduli AzureRm da un computer.

## SINTASSI

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Rimuove tutti i moduli AzureRm da un computer.

## ESEMPI

### Esempio 1
```
PS C:\> Uninstall-AzureRm
```

L'esecuzione di questo comando consente di rimuovere tutti i moduli AzureRm dal computer per la versione di PowerShell in cui viene eseguito il cmdlet.
