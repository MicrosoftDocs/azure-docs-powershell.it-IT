# Disable-AzureRmAlias

## Sinossi
Disabilita gli alias di prefisso AzureRm per i moduli AZ.

## SINTASSI

```
Disable-AzureRmAlias [-Scope <String>] [-Module <String[]>] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Disabilita gli alias di prefisso AzureRm per i moduli AZ. Se-Module è specificato, solo i moduli elencati avranno disabilitato alias. In caso contrario, tutti gli alias di AzureRm sono disabilitati.

## ESEMPI

### Esempio 1
```
PS C:\> Disable-AzureRmAlias
```

Disabilita tutti i prefissi di AzureRm per la sessione corrente di PowerShell.

### Esempio 1
```
PS C:\> Disable-AzureRmAlias -Module Az.Accounts -Scope CurrentUser
```

Disabilita gli alias di AzureRm per il modulo AZ. Accounts sia per il processo corrente che per l'utente corrente.
