# Set-AzDtlAutoShutdownPolicy

## Sinossi
Imposta il criterio di arresto automatico di un Lab DevTest Labs.

## SINTASSI

### Enable (impostazione predefinita)
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Enable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### Disabilitare
```
Set-AzDtlAutoShutdownPolicy [[-Time] <DateTime>] [-Disable] [-LabName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **set-AzDtlAutoShutdownPolicy** imposta i criteri di arresto automatico di un Lab, che arresta automaticamente tutte le macchine virtuali nel Lab in un determinato momento della giornata.
Il cmdlet usa il gruppo di risorse specificato e il nome del Lab per impostare i criteri.
