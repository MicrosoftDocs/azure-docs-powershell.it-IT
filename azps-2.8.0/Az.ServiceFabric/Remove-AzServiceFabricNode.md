# Remove-AzServiceFabricNode

## Sinossi
Rimuovere i nodi dal tipo di nodo specifico da un cluster.

## SINTASSI

```
Remove-AzServiceFabricNode -NumberOfNodesToRemove <Int32> [-ResourceGroupName] <String> [-Name] <String>
 -NodeType <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
USA **Remove-AzServiceFabricNode** per rimuovere i nodi da un tipo di nodo specifico da un cluster. La rimozione procede solo se soddisfa le metriche di integritÃ  del cluster.

## ESEMPI

### Esempio 1
```powershell
PS c:> Remove-AzServiceFabricNode -ResourceGroupName 'Group1' -Name 'Contoso01SFCluster' -NodeTypeName 'nt1' -NumberOfNodesToRemove 2
```

Questo comando consente di rimuovere 2 nodi da NodeType ' NT1'.
