# Get-AzMetricAlertRuleV2

## Sinossi
Ottiene le regole di avviso metriche V2 (non classiche)

## SINTASSI

### ByResourceGroupName (impostazione predefinita)
```
Get-AzMetricAlertRuleV2 [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByRuleName
```
Get-AzMetricAlertRuleV2 -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### ByRuleId
```
Get-AzMetricAlertRuleV2 -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzMetricAlertRuleV2** ottiene una regola di avviso metrica in base al nome o all'URI o a tutte le regole di avviso metriche da un gruppo di risorse o da un abbonamento specificato.
