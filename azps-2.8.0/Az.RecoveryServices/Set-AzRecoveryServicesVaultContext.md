# Set-AzRecoveryServicesVaultContext

## Sinossi

Imposta il contesto del Vault.

## SINTASSI

```
Set-AzRecoveryServicesVaultContext -Vault <ARSVault> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione

Il cmdlet **set-AzRecoveryServicesVaultContext** imposta il contesto del Vault per i servizi di ripristino del sito Azure.

Avviso: questo cmdlet Ã¨ deprecato in una versione futura per le modifiche alle interruzioni. Non saranno disponibili sostituzioni. Usare il parametro-VaultId in tutti i comandi per i servizi di recupero in corso.

## ESEMPI

### Esempio 1

```powershell
PS C:\> $vault = Get-AzRecoveryServicesVault -ResourceGroupName "resourceGroup" -Name "vaultName"
PS C:\> Set-AzRecoveryServicesVaultContext -Vault $vault
```

Imposta il contesto del Vault.
