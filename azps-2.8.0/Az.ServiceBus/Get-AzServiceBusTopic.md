# Get-AzServiceBusTopic

## Sinossi
Restituisce una descrizione per l'argomento Service Bus specificato.

## SINTASSI

```
Get-AzServiceBusTopic [-ResourceGroupName] <String> [-Namespace] <String> [[-Name] <String>]
 [-MaxCount <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzServiceBusTopic** restituisce una descrizione dell'argomento per lo spazio dei nomi Service Bus specificato.

## ESEMPI

### Esempio 1
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -TopicName SB-Topic_exampl1
```

Restituisce la descrizione dell'argomento specificato per lo spazio dei nomi di Service Bus specificato.

### Esempio 2
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1
```

Restituisce un elenco di argomenti per lo spazio dei nomi di Service Bus specificato. Per impostazione predefinita verranno restituiti gli argomenti di 100, se sono piÃ¹ di 100 argomenti da restituire, usare il parametro-MaxCount.

### Esempio 3
```
PS C:\> Get-AzServiceBusTopic -ResourceGroup Default-ServiceBus-WestUS -NamespaceName SB-Example1 -MaxCount 150
```

Restituisce l'elenco dei primi argomenti di 150 per lo spazio dei nomi di Service Bus specificato.
