# Modulo di AZS. Commerce. admin
## Descrizione
Versione di anteprima del modulo operatore di AzureStack commerce che consente di visualizzare l'utilizzo di dati aggregati attraverso il proprio timbro AzureStack.

## Cmdlet AZS. Commerce. admin
### [Get-AzsSubscriberUsage](Get-AzsSubscriberUsage.md)
Recuperare i dati sull'utilizzo durante l'intervallo specificato.
