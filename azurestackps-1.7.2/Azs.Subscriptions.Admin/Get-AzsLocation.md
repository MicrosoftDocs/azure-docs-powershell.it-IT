# Get-AzsLocation

## Sinossi
Ottenere un elenco di tutte le posizioni di AzureStack.

## SINTASSI

### Elenco (impostazione predefinita)
```
Get-AzsLocation [<CommonParameters>]
```

### Ottieni
```
Get-AzsLocation [-Name] <String> [<CommonParameters>]
```

## Descrizione
Ottenere un elenco di tutte le posizioni di AzureStack.

## ESEMPI

### --------------------------ESEMPIO 1--------------------------
```
Get-AzsLocation
```
