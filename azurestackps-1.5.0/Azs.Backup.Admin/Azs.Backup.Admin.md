# Modulo di AZS. backup. admin
## Descrizione
Versione di anteprima del modulo operatore di backup di AzureStack.  Questo modulo consente agli operatori di monitorare e gestire i backup.

## Cmdlet AZS. backup. admin
### [Get-AzsBackup](Get-AzsBackup.md)
Restituisce un backup da una posizione in base al nome.

### [Get-AzsBackupConfiguration](Get-AzsBackupConfiguration.md)
Restituisce l'elenco delle configurazioni di backup.

### [New-AzsEncryptionKeyBase64](New-AzsEncryptionKeyBase64.md)
Genera chiave di crittografia per i backup dell'infrastruttura.

### [Ripristinare-AzsBackup](Restore-AzsBackup.md)
Ripristinare un backup.

### [Set-AzsBackupConfiguration](Set-AzsBackupConfiguration.md)
Impostare la configurazione di backup nella posizione specificata.

### [Start-AzsBackup](Start-AzsBackup.md)
Eseguire il backup di una posizione specifica.
