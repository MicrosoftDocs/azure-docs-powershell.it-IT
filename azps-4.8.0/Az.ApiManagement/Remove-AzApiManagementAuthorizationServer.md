# Remove-AzApiManagementAuthorizationServer

## Sinossi
Rimuove un server di autorizzazione.

## SINTASSI

```
Remove-AzApiManagementAuthorizationServer -Context <PsApiManagementContext> -ServerId <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApiManagementAuthorizationServer** rimuove un server di autorizzazione Gestione API di Azure.

## ESEMPI

### Esempio 1: rimuovere un server di autorizzazione
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Remove-AzApiManagementAuthorizationServer -Context $ApiMgmtContext -ServerId "authserverid" -Force
```

Questo comando rimuove il server di autorizzazione Gestione API specificato.
Dato che il parametro *Force* è specificato, non è richiesta alcuna conferma.
