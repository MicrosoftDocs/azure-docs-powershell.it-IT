# New-AzApiManagementHttpMessageDiagnostic

## Sinossi
Crea un'istanza di **PsApiManagementHttpMessageDiagnostic** che è un'impostazione di diagnostica del messaggio http della diagnostica

## SINTASSI

```
New-AzApiManagementHttpMessageDiagnostic [-HeadersToLog <String[]>] [-BodyBytesToLog <Int32>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzApiManagementHttpMessageDiagnostic** crea l'impostazione di diagnostica del messaggio http.

## ESEMPI

### Esempio 1: creare un'impostazione di diagnostica messaggio http di base
```powershell
PS C:\>  New-AzApiManagementHttpMessageDiagnostic -Headers 'Content-Type', 'UserAgent' -BodyBytes 100

Headers                   Body
-------                   ----
{Content-Type, UserAgent} Microsoft.Azure.Commands.ApiManagement.ServiceManagement.Models.PsApiManagementBodyDiagnosticSetting
```

Creare un'impostazione di diagnostica per i messaggi HTTP per registrare `Content-Type` e `User-Agent` intestazioni insieme a 100 byte di `body`
