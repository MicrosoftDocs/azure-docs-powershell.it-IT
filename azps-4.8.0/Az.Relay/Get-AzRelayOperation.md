# Get-AzRelayOperation

## Sinossi
Elenco delle operazioni di inoltro supportate

## SINTASSI

```
Get-AzRelayOperation [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzRelayOperation** elenca le operazioni supportate da relay.
