# Start-AzHDInsightJob

## Sinossi
Avvia un processo di Azure HDInsight definito in un cluster specificato.

## SINTASSI

```
Start-AzHDInsightJob [-ClusterName] <String> [-JobDefinition] <AzureHDInsightJobDefinition>
 [-HttpCredential] <PSCredential> [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Start-AzHDInsightJob** avvia un processo di Azure HDInsight definito in un cluster specificato.
PuÃ² trattarsi di un processo MapReduce, di un processo di MapReduce in streaming, di un processo hive o di un processo di maiale.

## ESEMPI

### Esempio 1: avviare un processo nel cluster specificato
```
PS C:\># Cluster info
PS C:\> $clusterName = "your-hadoop-001"
PS C:\> $clusterCreds = Get-Credential

# Hive job details
PS C:\> $statusFolder = "tempStatusFolder/"
PS C:\> $query = "SHOW TABLES"

PS C:\> New-AzHDInsightHiveJobDefinition -StatusFolder $statusFolder `
            -Query $query `
        | Start-AzHDInsightJob `
            -ClusterName $clusterName `
            -ClusterCredential $clusterCreds
```

Questo comando avvia un processo nel cluster denominato your-Hadoop-001.
