# Get-AzSqlSyncAgent

## Sinossi
Restituisce informazioni sugli agenti di sincronizzazione di Azure SQL.

## SINTASSI

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzSqlSyncAgent** restituisce informazioni su uno o piÃ¹ agenti di sincronizzazione SQL di Azure.
Specificare il nome di un agente di sincronizzazione per visualizzare le informazioni solo per l'agente di sincronizzazione.
