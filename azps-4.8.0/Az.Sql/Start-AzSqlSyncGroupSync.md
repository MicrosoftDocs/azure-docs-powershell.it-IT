# Start-AzSqlSyncGroupSync

## Sinossi
Avvia la sincronizzazione di un gruppo di sincronizzazione.

## SINTASSI

```
Start-AzSqlSyncGroupSync [-SyncGroupName] <String> [-PassThru] [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Start-AzSqlSyncGroupSync** avvia la sincronizzazione di un gruppo di sincronizzazione.

## ESEMPI

### Esempio 1
```
PS C:\> Start-AzSqlSyncGroupSync -SyncGroupName mysg [-ServerName] mysrv [-DatabaseName] mydb [-ResourceGroupName] myrg
```

Questo comando avvia un ciclo di sincronizzazione per il gruppo di sincronizzazione mysg.
