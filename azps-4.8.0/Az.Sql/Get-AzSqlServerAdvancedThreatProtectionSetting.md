# Get-AzSqlServerAdvancedThreatProtectionSetting

## SYNOPSIS
Ottiene le impostazioni di Protezione dalle minacce avanzate per un server.

## SINTASSI

```
Get-AzSqlServerAdvancedThreatProtectionSetting -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIZIONE
Il cmdlet **Get-AzSqlServerAdvancedThreatProtectionSetting** ottiene le impostazioni di Protezione avanzata dalle minacce di un server SQL Azure.
Per usare questo cmdlet, specificare i *parametri ResourceGroupName* e *ServerName* per identificare il server per cui il cmdlet ottiene le impostazioni.

## ESEMPI

### Esempio 1: Ottenere le impostazioni di Protezione avanzata dalle minacce per un server
```
PS C:\>Get-AzSqlServerAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
ResourceGroupName            : ResourceGroup11
ServerName                   : Server01
ThreatDetectionState         : Enabled
NotificationRecipientsEmails : admin@myCompany.com
StorageAccountName           : mystorage
EmailAdmins                  : True
ExcludedDetectionTypes       : {}
RetentionInDays              : 0
```

Questo comando recupera le impostazioni di Protezione avanzata dalle minacce per un server denominato Server01.
Il server Ã¨ assegnato al gruppo di risorse ResourceGroup11.
