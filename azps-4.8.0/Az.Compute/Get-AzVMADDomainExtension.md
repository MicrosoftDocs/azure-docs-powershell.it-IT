# Get-AzVMADDomainExtension

## Sinossi
Ottiene informazioni su un'estensione del dominio Active Directory.

## SINTASSI

```
Get-AzVMADDomainExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzVMADDomainExtension** ottiene le informazioni sull'estensione di dominio di Azure Active Directory specificata (ad).

## ESEMPI

### Esempio 1

Ottiene informazioni su un'estensione del dominio Active Directory. AutoGenerated
