# Get-AzDiskEncryptionSet

## Sinossi
Ottenere o elencare i set di crittografia del disco.

## SINTASSI

```
Get-AzDiskEncryptionSet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Ottenere o elencare i set di crittografia del disco.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzDiskEncryptionSet -ResourceGroupName rg1 -Name enc1;

Name              : enc1

```

Ottenere il set di crittografia del disco ' ENC1'
