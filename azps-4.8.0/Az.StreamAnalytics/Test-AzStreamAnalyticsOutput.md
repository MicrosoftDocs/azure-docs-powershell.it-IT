# Test-AzStreamAnalyticsOutput

## Sinossi
Verifica lo stato di connessione di un output.

## SINTASSI

```
Test-AzStreamAnalyticsOutput [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **test-AzStreamAnalyticsOutput** verifica la possibilitÃ  di una connessione di analisi dello stream a un output.

## ESEMPI

### Esempio 1: verificare lo stato di connessione di un output
```powershell
PS C:\>Test-AzStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamingJob" -Name "Output"
```

Questo verifica lo stato di connessione dell'output di output in StreamingJob.
