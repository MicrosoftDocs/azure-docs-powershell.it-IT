# Get-AzDataShareDataSet

## Sinossi
Ottiene informazioni sui set di dati in Azure Data Share.

## SINTASSI

### ByFieldsParameterSet (impostazione predefinita)
```
Get-AzDataShareDataSet -ResourceGroupName <String> -AccountName <String> -ShareName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByResourceIdParameterSet
```
Get-AzDataShareDataSet -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzDataShareDataSet** ottiene le informazioni sui set di dati aggiunti in una condivisione in un account di condivisione dati di Azure. Se specifichi il nome del set di dati, questo cmdlet recupera le informazioni sul set di dati. Se non specifichi un nome, questo cmdlet riceverÃ  informazioni su tutti i set di dati in una condivisione. Ho
