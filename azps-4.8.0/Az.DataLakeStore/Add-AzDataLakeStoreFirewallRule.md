# Add-AzDataLakeStoreFirewallRule

## Sinossi
Aggiunge una regola del firewall all'account di data Lake Store specificato.

## SINTASSI

```
Add-AzDataLakeStoreFirewallRule [-Account] <String> [-Name] <String> [-StartIpAddress] <String>
 [-EndIpAddress] <String> [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzDataLakeStoreFirewallRule** aggiunge una regola del firewall all'account di data Lake Store specificato.

## ESEMPI

### Esempio 1: aggiungere una nuova regola del firewall a un account di data Lake Store
```
PS C:\> Add-AzDataLakeStoreFirewallRule -AccountName "ContosoADL" -Name MyRule -StartIpAddress "127.0.0.1" -EndIpAddress "127.0.0.2"
```

In questo modo viene creata una nuova regola del firewall denominata "regola" in account "ContosoADL" con un intervallo IP di 127.0.0.1-127.0.0.2
