# Test-AzDataLakeStoreAccount

## Sinossi
Verifica l'esistenza di un account di data Lake Store.

## SINTASSI

```
Test-AzDataLakeStoreAccount [-Name] <String> [[-ResourceGroupName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **test-AzDataLakeStoreAccount** verifica l'esistenza di un account di data Lake Store.

## ESEMPI

### Esempio 1: testare un account
```
PS C:\>Test-AzDataLakeStoreAccount -Name "ContosoADL"
```

Questo comando verifica se l'account denominato ContosoADL esiste.
