# New-AzFirewallThreatIntelWhitelist

## Sinossi
Creare una nuova whitelist di Threat Intelligence per Azure firewall

## SINTASSI

```
New-AzFirewallThreatIntelWhitelist [-FQDN <String[]>] [-IpAddress <String[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzFirewallThreatIntelWhitelist** crea un oggetto di tipo whitelist di Intel, che puÃ² essere usato per la creazione o l'impostazione di un firewall di Azure.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzFirewallThreatIntelWhitelist -IpAddress @("2.2.2.2", "3.3.3.3") -FQDN @("bing.com", "yammer.com")
```

In questo esempio viene creato un elenco di informazioni di base su un elenco di indirizzi di una whitelist contenente un elenco FQDN di due voci e un indirizzo IP whitelist
