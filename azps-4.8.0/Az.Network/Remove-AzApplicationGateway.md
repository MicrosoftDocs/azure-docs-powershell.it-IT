# Remove-AzApplicationGateway

## Sinossi
Rimuove un gateway dell'applicazione.

## SINTASSI

```
Remove-AzApplicationGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzApplicationGateway** rimuove un gateway dell'applicazione.

## ESEMPI

### Esempio 1: rimuovere un gateway di applicazione specificato
```
PS C:\>Remove-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
```

Questo comando rimuove il gateway dell'applicazione denominato ApplicationGateway01 nel gruppo di risorse denominato ResourceGroup01.
