# Get-AzLoadBalancerFrontendIpConfig

## Sinossi
Ottiene una configurazione IP front-end in un bilanciamento del carico.

## SINTASSI

```
Get-AzLoadBalancerFrontendIpConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzLoadBalancerFrontendIpConfig** ottiene una configurazione IP front-end o un elenco di configurazioni IP front-end in un bilanciamento del carico.

## ESEMPI

### Esempio 1: ottenere una configurazione IP front-end in un bilanciamento del carico
```
PS C:\>$slb = Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
PS C:\> Get-AzLoadBalancerFrontendIpConfig -Name "MyFrontEnd" -LoadBalancer $slb
```

Il primo comando ottiene il bilanciamento del carico denominato MyLoadBalancer e lo archivia nella variabile $slb.
Il secondo comando consente di ottenere la configurazione IP front-end associata a tale bilanciamento del carico.
