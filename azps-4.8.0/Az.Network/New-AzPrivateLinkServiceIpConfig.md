# New-AzPrivateLinkServiceIpConfig

## Sinossi
Creare una configurazione IP di un servizio di collegamento privato.

## SINTASSI

```
New-AzPrivateLinkServiceIpConfig -Name <String> [-PrivateIpAddressVersion <String>]
 [-PrivateIpAddress <String>] [-PublicIpAddress <PSPublicIpAddress>]
 [-Subnet <PSSubnet>] [-Primary]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **New-AzPrivateLinkServiceIpConfig** crea una configurazione IP del servizio di collegamento privato. Questa configurazione viene usata per l'origine NAT-ing il traffico in arrivo da endpoint privato. 

## ESEMPI

### Esempio 1
```powershell
New-AzPrivateLinkServiceIpConfig -Name $IpConfigurationName -PrivateIpAddress "10.0.0.5" -Primary
```

Questo esempio crea una configurazione IP del servizio di collegamento privato in memoria.
