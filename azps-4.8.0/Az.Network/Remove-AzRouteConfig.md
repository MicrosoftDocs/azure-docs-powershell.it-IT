# Remove-AzRouteConfig

## Sinossi
Consente di rimuovere una route da una tabella di route.

## SINTASSI

```
Remove-AzRouteConfig -RouteTable <PSRouteTable> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzRouteConfig** consente di rimuovere una route da una tabella di route di Azure.

## ESEMPI

### Esempio 1: rimuovere una route
