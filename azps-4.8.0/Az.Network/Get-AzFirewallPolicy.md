# Get-AzFirewallPolicy

## Sinossi
Ottiene un criterio di Azure firewall

## SINTASSI

### GetByNameParameterSet (impostazione predefinita)
```
Get-AzFirewallPolicy -Name <String> -ResourceGroupName <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByResourceIdParameterSet
```
Get-AzFirewallPolicy -ResourceId <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzFirewallPolicy** ottiene uno o piÃ¹ firewall in un gruppo di risorse

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzFirewallPolicy -Name firwallPolicy -ResourceGroupName TestRg
```

Questo esempio ottiene un criterio firewall denominato "firewallPolicy" nel gruppo risorse "TestRg"
