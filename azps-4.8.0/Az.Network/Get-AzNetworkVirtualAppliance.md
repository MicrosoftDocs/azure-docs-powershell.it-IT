# Get-AzNetworkVirtualAppliance

## Sinossi
Ottenere o elencare gli elettrodomestici virtuali di rete.

## SINTASSI

### ResourceNameParameterSet (impostazione predefinita)
```
Get-AzNetworkVirtualAppliance [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzNetworkVirtualAppliance -ResourceId <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
I comandi Get-AzNetworkVirtualAppliance ottengono o elencano le risorse di Virtual Appliance di rete.
