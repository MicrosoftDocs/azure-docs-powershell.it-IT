# New-AzNetworkWatcherConnectionMonitor

## Sinossi
Crea una risorsa di monitoraggio connessione.

## SINTASSI

### SetByName (impostazione predefinita)
```
New-AzNetworkWatcherConnectionMonitor -NetworkWatcherName <String> -ResourceGroupName <String> -Name <String>
 -SourceResourceId <String> -MonitoringIntervalInSeconds <Int32> [-SourcePort <Int32>]
 [-DestinationResourceId <String>] -DestinationPort <Int32> [-DestinationAddress <String>] [-ConfigureOnly]
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet New-AzNetworkWatcherConnectionMonitor crea una risorsa Monitor di connessione per l'origine e la destinazione specificati (SingleSourcedestination Connection Monitor) o set di gruppi di test (MultiEndpointConnectionMonitor).
