# Add-AzSynapseTriggerSubscription

## Sinossi
Sottoscrivere il trigger dell'evento per gli eventi del servizio esterno.

## SINTASSI

### AddByName (impostazione predefinita)
```
Add-AzSynapseTriggerSubscription -WorkspaceName <String> -Name <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddByObject
```
Add-AzSynapseTriggerSubscription -WorkspaceObject <PSSynapseWorkspace> -Name <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### AddByInputObject
```
Add-AzSynapseTriggerSubscription -InputObject <PSTriggerResource> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Add-AzSynapseTriggerSubscription** sottoscrive il trigger di evento agli eventi del servizio esterno specificati dal trigger definizione.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Add-AzSynapseTriggerSubscription -WorkspaceName ContosoWorkspace -Name ContosoTrigger
```

Questo comando eseguirà la sottoscrizione del trigger denominato ContosoTrigger agli eventi specificati dal trigger definizione.

### Esempio 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Add-AzSynapseTriggerSubscription -Name ContosoTrigger
```

Questo comando eseguirà la sottoscrizione del trigger denominato ContosoTrigger agli eventi specificati dal trigger definizione tramite pipeline.

### Esempio 3
```powershell
PS C:\> $trigger = Get-AzSynapseTrigger -WorkspaceName ContosoWorkspace -Name ContosoTrigger
PS C:\> $trigger | Add-AzSynapseTriggerSubscription
```

Questo comando eseguirà la sottoscrizione del trigger denominato ContosoTrigger agli eventi specificati dal trigger definizione tramite pipeline.
