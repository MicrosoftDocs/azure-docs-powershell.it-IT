# Remove-AzSynapseWorkspace

## Sinossi
Elimina un'area di lavoro di analisi sinapsi.

## SINTASSI

### DeleteByNameParameterSet (impostazione predefinita)
```
Remove-AzSynapseWorkspace [-ResourceGroupName <String>] -Name <String> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByInputObjectParameterSet
```
Remove-AzSynapseWorkspace -InputObject <PSSynapseWorkspace> [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### DeleteByResourceIdParameterSet
```
Remove-AzSynapseWorkspace -ResourceId <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzSynapseWorkspace** Elimina definitivamente un'area di lavoro di Azure sinapsi Analytics.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Remove-AzSynapseWorkspace -Name ContosoWorkspace
```

Questo comando Elimina un'area di lavoro di Azure sinapsi Analytics.

### Esempio 2
```powershell
PS C:\> $ws = Get-AzSynapseWorkspace -Name ContosoWorkspace
PS C:\> $ws | Remove-AzSynapseWorkspace
```

Questo comando Elimina un'area di lavoro di analisi di Azure sinapsi tramite pipeline.

### Esempio 3
```powershell
PS C:\> Remove-AzSynapseWorkspace -ResourceId /subscriptions/21686af7-58ec-4f4d-9c68-f431f4db4edd/resourceGroups/ContosoResourceGroup/providers/Microsoft.Synapse/workspaces/ContosoWorkspace
```

Questo comando Elimina un'area di lavoro di analisi di Azure sinapsi tramite pipeline con l'ID risorsa specificato.
