# Get-AzHpcCacheSku

## Sinossi
Ottiene tutte le SKU disponibili in abbonamento.

## SINTASSI

```
Get-AzHpcCacheSku [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzHpcCacheSku** restituisce un elenco di SKU disponibili in abbonamento.

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzHpcCacheSku
```
