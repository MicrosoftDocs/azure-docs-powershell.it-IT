# Remove-AzMySqlServer

## Sinossi
Elimina un server.

## SINTASSI

### Elimina (impostazione predefinita)
```
Remove-AzMySqlServer -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzMySqlServer -InputObject <IMySqlIdentity> [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrizione
Elimina un server.

## ESEMPI

### Esempio 1: rimuovere il server MySql per resourceGroup e il nome del server
```powershell
PS C:\> Remove-AzMySqlServer -ResourceGroupName PowershellMySqlTest -Name mysql-test

```

Questo cmdlet rimuove il server MySql per resourceGroup e il nome del server.

### Esempio 2: rimuovere il server MySql per identità
```powershell
PS C:\> $ID = "/subscriptions/<SubscriptionId>/resourceGroups/PowershellMySqlTest/providers/Microsoft.DBforMySQL/servers/mysql-test"
PS C:\> Remove-AzMySqlServer -InputObject $ID
 
```

Questi cmdlet rimuovono MySql Server per identità.
