# Modulo AZ. search
## Descrizione
Questo argomento Mostra gli argomenti della Guida per i cmdlet di ricerca di Azure.

## Cmdlet AZ. search
### [Get-AzSearchAdminKeyPair](Get-AzSearchAdminKeyPair.md)
Ottiene la coppia di chiavi di amministrazione del servizio di ricerca di Azure.

### [Get-AzSearchQueryKey](Get-AzSearchQueryKey.md)
Ottiene i tasti di query del servizio di ricerca di Azure.

### [Get-AzSearchService](Get-AzSearchService.md)
Ottiene un servizio di ricerca Azure.

### [New-AzSearchAdminKey](New-AzSearchAdminKey.md)
Rigenera una chiave di amministratore del servizio di ricerca di Azure.

### [New-AzSearchQueryKey](New-AzSearchQueryKey.md)
Creare una nuova chiave di query per il servizio di ricerca di Azure.

### [New-AzSearchService](New-AzSearchService.md)
Crea un servizio di ricerca Azure.

### [Remove-AzSearchQueryKey](Remove-AzSearchQueryKey.md)
Rimuovere la chiave di query dal servizio di ricerca di Azure.

### [Remove-AzSearchService](Remove-AzSearchService.md)
Rimuovere un servizio di ricerca di Azure.

### [Set-AzSearchService](Set-AzSearchService.md)
Aggiornare un servizio di ricerca di Azure.
