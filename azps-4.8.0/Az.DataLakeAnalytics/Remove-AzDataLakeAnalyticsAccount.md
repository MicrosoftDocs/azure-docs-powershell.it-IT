# Remove-AzDataLakeAnalyticsAccount

## Sinossi
Elimina un account di analisi dei dati sul lago.

## SINTASSI

```
Remove-AzDataLakeAnalyticsAccount [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzDataLakeAnalyticsAccount** Elimina definitivamente un account di analisi di Azure Data Lake.

## ESEMPI

### Esempio 1: rimuovere un account
```
PS C:\>Remove-AzDataLakeAnalyticsAccount -Name "ContosoAdlAccount"
```

Questo comando rimuove l'account di analisi dei dati del lago specificato.
