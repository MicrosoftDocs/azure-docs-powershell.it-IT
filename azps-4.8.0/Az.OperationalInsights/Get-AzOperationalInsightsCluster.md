# Get-AzOperationalInsightsCluster

## Sinossi
Ottenere o elencare i cluster

## SINTASSI

```
Get-AzOperationalInsightsCluster [[-ResourceGroupName] <String>] [[-ClusterName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Ottenere o elencare i cluster, i cluster di elenco in gruppo risorse quando non Ã¨ stato fornito "-clustername", elencare i cluster in abbonamento quando non sono stati forniti "-clustername" e "ResourceGroupName".
