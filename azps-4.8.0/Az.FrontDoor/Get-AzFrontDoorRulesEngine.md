# Get-AzFrontDoorRulesEngine

## Sinossi
Ottenere le configurazioni del motore regole.

## SINTASSI

```
Get-AzFrontDoorRulesEngine -ResourceGroupName <String> -FrontDoorName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet **Get-AzFrontDoorRulesEngine** ottiene una configurazione specifica del motore di regole o ottiene tutte le configurazioni del motore di regole associate a uno sportello anteriore. 

## ESEMPI

### Esempio 1
```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name rulesengine3

Name         RulesEngineRules
----         ----------------
rulesEngine3 {rules1}
```

Ottenere la configurazione del motore di regole specifiche.

### Esempio 2

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName

Name         RulesEngineRules
----         ----------------
rulesEngine1 {Rule1}
rulesEngine2 {Rule1}
rulesEngine3 {rules1}
```

Ottenere tutte le configurazioni del motore di regole in una porta principale.

### Esempio 3

```powershell
PS C:\> Get-AzFrontDoorRulesEngine -ResourceGroupName $resourceGroupName -FrontDoorName $frontDoorName -Name nonexistent
```
