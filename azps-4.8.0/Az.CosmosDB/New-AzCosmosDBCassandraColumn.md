# New-AzCosmosDBCassandraColumn

## Sinossi
Crea una nuova colonna di CosmosDB Cassandra.

## SINTASSI

```
New-AzCosmosDBCassandraColumn -Name <String> -Type <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## Descrizione
Il **nuovo AzCosmosDBCassandraColumn** crea una nuova colonna di CosmosDB Cassandra.

## ESEMPI

### Esempio 1
```powershell
PS C:\> New-AzCosmosDBCassandraColumn -Name "name" -Type int

Name Type
---- ----
name int
```
