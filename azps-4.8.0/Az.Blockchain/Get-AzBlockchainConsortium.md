# Get-AzBlockchainConsortium

## Sinossi
Elenca i consorzi disponibili per un abbonamento.

## SINTASSI

```
Get-AzBlockchainConsortium -Location <String> [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## Descrizione
Elenca i consorzi disponibili per un abbonamento.

## ESEMPI

### Esempio 1: ottenere consorzi blockchain.
```powershell
PS C:\> Get-AzBlockchainConsortium -Location eastus

```

Questo comando elenca i consorzi in un abbonamento per una posizione specifica.
