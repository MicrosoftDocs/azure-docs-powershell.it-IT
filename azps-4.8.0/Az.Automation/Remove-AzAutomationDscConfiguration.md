# Remove-AzAutomationDscConfiguration

## Sinossi
Rimuove le configurazioni DSC dall'automazione.

## SINTASSI

```
Remove-AzAutomationDscConfiguration [-Name] <String> [-Force] [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## Descrizione
Il cmdlet **Remove-AzAutomationDscConfiguration** rimuove le configurazioni DSC (APS desired state Configuration) di Azure Automation.
