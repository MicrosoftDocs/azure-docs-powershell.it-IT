# Get-AzIotSecurityAnalytics

## Sinossi
Ottenere analisi della sicurezza degli altri

## SINTASSI

```
Get-AzIotSecurityAnalytics -ResourceGroupName <String> -SolutionName <String> [-Default]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Descrizione
Il cmdlet Get-AzIotSecurityAnalytics restituisce un set di analisi della sicurezza di una soluzione di sicurezza molto specifica
